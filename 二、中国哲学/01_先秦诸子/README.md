# 01 道与仁义：先秦诸子的百家争鸣

## 核心张力

**道(自然) vs 仁义(人文)**

- 道(dào)：自然无为、宇宙本源、超越人为
- 仁义(rén yì)：人文教化、道德规范、社会建构

## 思维流向

中国哲学的开端，围绕"如何安顿人生与社会"展开两条路线：
- 道家路线：回归自然之道，批判人为造作——老子、庄子
- 儒家路线：建立仁义礼乐，成就人文秩序——孔子、孟子、荀子
- 墨家挑战：兼爱非攻，质疑儒家差等之爱——墨子
- 法家综合：以法治国，超越道德理想主义——商鞅、韩非
- 名家追问：名实关系，语言如何指称世界——惠施、公孙龙

## 人物与文献

### 老子/《道德经》(约前6-前4世纪)
- **原文**：《道德经》(Dàodé Jīng，八十一章)
- **核心学说**：道法自然、无为而治、反者道之动
- **关键概念**：
  - 道(dào)：万物本源，先天地生，独立不改
  - 无为(wú wéi)：不妄为，顺应自然
  - 反(fǎn)：相反相成，物极必反

### 孔子(前551-前479)
- **原文**：《论语》(Lúnyǔ)
- **核心学说**：仁、礼、义、君子、中庸
- **关键概念**：
  - 仁(rén)：爱人，己欲立而立人
  - 礼(lǐ)：社会规范，克己复礼
  - 君子(jūn zǐ)：道德人格的理想

### 墨子(约前468-前376)
- **原文**：《墨子》(Mòzǐ)
- **核心学说**：兼爱、非攻、尚贤、天志
- **关键概念**：
  - 兼爱(jiān ài)：无差等的爱，视人之国若视其国
  - 三表法(sān biǎo fǎ)：经验验证标准——本、原、用

### 孟子(前372-前289)
- **原文**：《孟子》(Mèngzǐ，七篇)
- **核心学说**：性善论、仁政、四端、养气
- **关键概念**：
  - 性善(xìng shàn)：恻隐之心人皆有之
  - 四端(sì duān)：恻隐、羞恶、辞让、是非
  - 浩然之气(hào rán zhī qì)：至大至刚，配义与道

### 庄子(约前369-前286)
- **原文**：《庄子》(Zhuāngzǐ，三十三篇)
- **核心学说**：逍遥游、齐物论、心斋坐忘
- **关键概念**：
  - 逍遥(xiāo yáo)：无待而游，精神自由
  - 齐物(qí wù)：万物齐一，是非相对
  - 心斋(xīn zhāi)：虚静其心，唯道集虚

### 荀子(前313-前238)
- **原文**：《荀子》(Xúnzǐ，三十二篇)
- **核心学说**：性恶论、化性起伪、礼义之统
- **关键概念**：
  - 性恶(xìng è)：人之性恶，其善者伪也
  - 伪(wěi)：人为，化性起伪——通过礼义改造本性
  - 天行有常(tiān xíng yǒu cháng)：天道自然，不为尧存不为桀亡

### 韩非(约前280-前233)
- **原文**：《韩非子》(Hánfēizǐ，五十五篇)
- **核心学说**：法、术、势合一、人性自利
- **关键概念**：
  - 法(fǎ)：成文法，布之于众
  - 术(shù)：君主驾驭臣下的权术
  - 势(shì)：君主的威势与权力

### 公孙龙(约前320-前250)
- **原文**：《公孙龙子》(Gōngsūn Lóngzǐ)
- **核心学说**：白马非马、离坚白、指物论
- **关键概念**：
  - 白马非马(bái mǎ fēi mǎ)：个别与一般的分离
  - 离坚白(lí jiān bái)：属性可分离，坚与白离

## 核心问题

1. **人性本善还是本恶？** 孟子：性善，四端固有；荀子：性恶，化性起伪；告子：性无善恶
2. **爱是否有差等？** 儒家：仁爱有差等（亲亲→仁民→爱物）；墨家：兼爱无差等
3. **道与仁义能否调和？** 道家：大道废有仁义；儒家：人伦日用即道——两条路线的根本分歧
