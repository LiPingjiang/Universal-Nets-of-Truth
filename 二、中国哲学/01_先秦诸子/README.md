# 二、中国哲学 · 先秦诸子

## 时间跨度

约公元前6世纪 — 公元前221年秦统一

## 核心问题

如何治国？如何做人？天人关系如何？

---

## 1.1 儒家（Confucianism / Ruism）

> **核心问题**：仁是什么？如何成为君子？

| 人物 | 生卒年 | 代表著作/文本 | 核心学说 | 文件 |
|------|--------|--------------|----------|------|
| **孔子**（Confucius / Kongzi / 孔丘） | 前551-前479 | 《论语》（Lunyu，弟子及再传弟子记录） | 仁（ren）、礼（li）、义（yi）、君子（junzi）、中庸（zhongyong）；为政以德；有教无类；正名 | 待建 |
| **曾子**（Zengzi / 曾参） | 前505-前435 | 《大学》（传为曾子所作，收入《礼记》） | 三纲领（明明德、亲民、止于至善）；八条目（格物、致知、诚意、正心、修身、齐家、治国、平天下） | 待建 |
| **子思**（Zisi / 孔伋，孔子之孙） | 前483-前402 | 《中庸》（传为子思所作，收入《礼记》） | 天命之谓性；诚者天之道；中和 | 待建 |
| **孟子**（Mencius / Mengzi / 孟轲） | 前372-前289 | 《孟子》（Mengzi，七篇） | 性善论；四端（恻隐、羞恶、辞让、是非）；仁政；民贵君轻；养气（浩然之气）；王道 vs 霸道 | 待建 |
| **荀子**（Xunzi / 荀况） | 前313-前238 | 《荀子》（Xunzi，三十二篇） | 性恶论；礼义是人为（伪，wei）；化性起伪；天行有常；制天命而用之；正名论 | 待建 |

## 1.2 道家（Daoism）

> **核心问题**：道的本质是什么？如何达到无为？

| 人物/文本 | 年代 | 代表著作/文本 | 核心学说 | 文件 |
|-----------|------|--------------|----------|------|
| **老子/《道德经》**（Laozi / Daodejing） | 约前6-前4世纪（作者与年代有争议） | 《道德经》（Daodejing，八十一章） | 道可道非常道；无为（wuwei）；反者道之动；柔弱胜刚强；小国寡民；道法自然 | 待建 |
| **庄子**（Zhuangzi / 庄周） | 约前369-前286 | 《庄子》（Zhuangzi，三十三篇：内篇七、外篇十五、杂篇十一） | 逍遥游；齐物论（万物齐一）；心斋坐忘；庄周梦蝶；无用之用；庖丁解牛；生死观 | 待建 |
| **列子**（Liezi / 列御寇） | 约前5-前3世纪（真伪有争议） | 《列子》（Liezi，八篇） | 贵虚；生死齐一；力命论 | 待建 |
| **杨朱**（Yang Zhu） | 约前4世纪 | 无直接传世著作，思想见于《孟子》《列子·杨朱篇》 | 为我；贵生；轻物重生；不拔一毛以利天下 | 待建 |
