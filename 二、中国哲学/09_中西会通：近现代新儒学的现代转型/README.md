# 09 中西会通：近现代新儒学的现代转型

## 核心张力

**中(传统) vs 西(现代)**

- 中(zhōng)：中国传统、儒家道统、内圣外王
- 西(xī)：西方现代、民主科学、理性启蒙

## 思维流向

从第八章"知行合一"的传统实践哲学，到第九章"中西会通"的现代转型：
- 清末民初：中学为体西学为用（张之洞）→ 全盘西化 vs 文化保守主义
- 新儒家第一代：梁漱溟、熊十力——会通中西印三大传统
- 新儒家第二代：牟宗三、唐君毅——儒家内圣开出民主科学
- 新儒家第三代：杜维明、刘述先——儒家与全球伦理
- 当代：安乐哲、余英时——比较哲学与思想史路径

## 人物与文献

### 张之洞(1837-1909)
- **原文**：《劝学篇》
- **核心观点**：中学为体，西学为用
- **关键概念**：体(tǐ)：纲常名教；用(yòng)：工商技艺

### 梁漱溟(1893-1988)
- **原文**：《东西文化及其哲学》《中国文化要义》
- **核心观点**：世界文化三路向——西方向前、中国持中、印度向后
- **关键概念**：意欲(will)：文化根本，西方意欲向前，中国意欲持中

### 熊十力(1885-1968)
- **原文**：《新唯识论》《体用论》
- **核心观点**：体用不二、翕辟成变
- **关键概念**：
  - 体(tǐ)：本体，本心
  - 用(yòng)：现象，翕辟成变
  - 翕辟(xī pì)：本体流行的两种势用

### 牟宗三(1909-1995)
- **原文**：《心体与性体》《智的直觉与中国哲学》《现象与物自身》
- **核心观点**：良知坎陷说、内圣开出新外王
- **关键概念**：
  - 智的直觉(zhì de zhí jué)：良知本体之知，对应康德intellectuelle Anschauung
  - 良知坎陷(liáng zhī kǎn xiàn)：良知自我否定，转出知性主体
  - 两层存有论：执的存有论（现象界）vs 无执的存有论（本体界）

### 唐君毅(1909-1978)
- **原文**：《生命存在与心灵境界》
- **核心观点**：心灵九境——从客观境到主观境到超主客境
- **关键概念**：心通九境，层层超升

### 杜维明(1940-)
- **核心观点**：儒家与全球伦理、文明对话
- **关键概念**：
  - 文化中国(Cultural China)：超越地域的儒家文化圈
  - 启蒙反思：反思西方启蒙心态，发掘儒家资源

## 核心问题

1. **儒家能否开出民主科学？** 牟宗三：良知坎陷可开出；批评者：坎陷说过于曲折
2. **中西哲学能否通约？** 比较哲学的方法论问题——概念对应、语境差异、翻译困境
3. **儒家在现代的角色？** 从道统到学统到政统：心性儒学、政治儒学、生活儒学
