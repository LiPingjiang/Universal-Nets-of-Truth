# 08 知行合一：从孔孟到船山的实践哲学

## 核心张力

**知(认识) vs 行(实践)**

- 知(zhī)：道德认识、理性自觉、学问思辨
- 行(xíng)：道德实践、身体力行、习行工夫

## 思维流向

从第七章"心学与理学的分途"，到第八章"知行合一的实践哲学"：
- 先秦：孔子"行有余力则以学文"、孟子"强恕而行"
- 宋明：朱熹"知行常相须"、阳明"知行合一"
- 明清：船山"行先知后"、颜元"习行哲学"
- 近代：孙中山"知难行易"、毛泽东"实践论"

## 人物与文献

### 孔子(前551-前479)
- **原文**：《论语》
- **核心观点**：先行其言而后从之、听其言而观其行
- **关键概念**：行(xíng)：身体力行，道德实践

### 孟子(前372-前289)
- **原文**：《孟子》
- **核心观点**：强恕而行、求仁莫近焉
- **关键概念**：行有不得者皆反求诸己

### 朱熹
- **核心观点**：知行常相须，论先后知为先，论轻重行为重
- **关键概念**：知(zhī)：对理的认识；行(xíng)：依理而行

### 王阳明
- **核心观点**：知行合一、真知即所以为行
- **关键论证**：
  - "知而不行，只是未知"
  - "如好好色，如恶恶臭"——真知必有行

### 王船山
- **核心观点**：行先知后、知行相资以为用
- **关键论证**：批评阳明"以知为行"，主张行可兼知而知不可兼行

### 颜元(1635-1704)
- **原文**：《四存编》《习斋记余》
- **核心学说**：习行哲学、实学实用
- **关键概念**：习行(xí xíng)：亲身习践，实用实学

### 孙中山(1866-1925)
- **核心观点**：知难行易
- **论证**：饮食、用钱、建筑等十事证明行易知难

## 核心问题

1. **知行先后：知先行后？行先知后？知行合一？** 朱：知先行后；船山：行先知后；阳明：知行合一
2. **知行轻重：知重行轻？行重知轻？知行相资？** 朱：行为重；阳明：知行为一；船山：知行相资
3. **知的性质：德性之知？见闻之知？** 张载：德性之知不萌于见闻；阳明：真知即行
