# 04 名教与自然：魏晋玄学的核心张力

## 核心张力

**名教(礼法) vs 自然(无为)**

- 名教(míng jiào)：儒家礼法、社会规范、人伦秩序
- 自然(zì rán)：道家无为、本性自发、超越人为

## 思维流向

从第三章"天人感应"的汉代宇宙论，到第四章"名教与自然"的魏晋玄学：
- 正始玄学：何晏、王弼——贵无论，以无为本，圣人体无
- 竹林玄学：阮籍、嵇康——越名教而任自然，政治反抗与个体觉醒
- 元康玄学：裴頠、郭象——崇有论与独化论，名教即自然
- 东晋玄佛合流：僧肇——不真空，物不迁，玄学化的般若学

## 人物与文献

### 何晏（约190-249）
- **原文**：《论语集解》《无名论》《道论》（残篇）
- **核心学说**：贵无论先驱，圣人无情说
- **关键概念**：
  - 以无为本(yǐ wú wéi běn)：无名为道之本
  - 圣人无情(shèng rén wú qíng)：圣人纯理，无喜怒哀乐

### 王弼(226-249)
- **原文**：《周易注》《老子注》《论语释疑》（残篇）
- **核心学说**：贵无论集大成，以一统众，得意忘言
- **关键概念**：
  - 以无为本(yǐ wú wéi běn)：天下万物皆以有为生，有之所始，以无为本
  - 得意忘言(dé yì wàng yán)：言者所以明象，得象而忘言；象者所以存意，得意而忘象
  - 圣人体无(shèng rén tǐ wú)：圣人能体无，但不可言说

### 阮籍(210-263)
- **原文**：《阮籍集》（含《咏怀诗》《大人先生传》《达庄论》）
- **核心学说**：越名教而任自然，隐晦的政治反抗
- **关键概念**：
  - 越名教而任自然(yuè míng jiào ér rèn zì rán)：超越礼法束缚，顺从自然本性
  - 大人先生(dà rén xiān sheng)：理想人格，与天地并生

### 嵇康(223-262)
- **原文**：《嵇康集》（含《声无哀乐论》《养生论》《释私论》《与山巨源绝交书》）
- **核心学说**：非汤武而薄周孔，声无哀乐，养生
- **关键概念**：
  - 声无哀乐(shēng wú āi yuè)：声音本身无哀乐，哀乐在听者之心
  - 越名教而任自然：刚肠疾恶，轻肆直言，与司马氏政权决裂

### 裴頠(267-300)
- **原文**：《崇有论》
- **核心学说**：崇有论——无不能生有，有是自生的
- **关键概念**：
  - 崇有论(chóng yǒu lùn)：反对贵无，主张有自生
  - 无不能生有(wú bù néng shēng yǒu)：无即虚无，不能产生实有

### 郭象(约252-312)
- **原文**：《庄子注》
- **核心学说**：独化论——物各自造，名教即自然
- **关键概念**：
  - 独化(dú huà)：物各自造而无所待，无造物主
  - 性分(xìng fèn)：万物各有其性分，各任其性即为逍遥
  - 名教即自然(míng jiào jí zì rán)：礼法名教即是自然之性的体现

### 僧肇(384-414)
- **原文**：《肇论》（《物不迁论》《不真空论》《般若无知论》《涅槃无名论》）
- **核心学说**：不真空、物不迁、般若无知——玄学化的般若学
- **关键概念**：
  - 不真空(bù zhēn kōng)：万物非有非无，缘起故空
  - 物不迁(wù bù qiān)：动静未始异，虽动而常静

## 核心问题

1. **名教与自然能否调和？** 王弼：名教出于自然；郭象：名教即自然；嵇康：越名教而任自然——三种立场
2. **贵无与崇有：本体论之争** 王弼：以无为本；裴頠：无不能生有；郭象：独化，无亦无
3. **言意之辩：语言能否表达真理？** 王弼：得意忘言；欧阳建：言尽意——两种认识论立场
