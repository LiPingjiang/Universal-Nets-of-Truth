# 张君劢：新儒家与科玄论战（白话解读）

## 一、张君劢是谁？

张君劢（1887-1969），字嘉森，江苏宝山（今上海宝山）人。他是现代新儒家的开创者之一，"科学与人生观论战"的发起者，中国现代思想史上最重要的思想家之一。

张君劢年轻时留学日本和德国，师从德国哲学家倭铿和法国哲学家柏格森。他深受康德哲学影响，认为科学有其边界，边界之外是哲学和宗教的领域。1923年，他在清华大学发表《人生观》演讲，提出"科学不能解决人生观"，引发了现代中国思想史上最大规模的论战。

论战中，科学派（丁文江、胡适等）批评他是"玄学鬼"，但他不为所动。他说：我不是反对科学，我是反对科学主义——认为科学可以解决一切问题的迷信。科学能解释自然现象，但不能解决人生意义问题。

张君劢一生著述极丰，涉及哲学、政治、文化等多个领域。他还积极参与政治，起草了《中华民国宪法》，被誉为"中华民国宪法之父"。1949年后他移居海外，继续传播新儒家思想。1969年在美国去世。

## 二、"科学不能解决人生观"：划定科学的边界

张君劢最著名的命题是：科学不能解决人生观问题。

他说："科学无论如何发达，而人生观问题之解决，决非科学所能为力，惟赖诸人类之自身。"

这是什么意思？张君劢认为，科学和人生观是两个不同的领域：
- **科学** = 客观的、因果的、分析的、普遍的
  - 科学问："为什么天会下雨？"
  - 科学回答：因为水蒸气凝结成云，云中的水滴增大后落下
- **人生观** = 主观的、直觉的、综合的、单一性的
  - 人生观问："人生的意义是什么？"
  - 人生观回答：这需要每个人自己去体会和选择

科学能解释自然现象，但不能告诉你人生的意义。为什么？因为人生的意义不是因果律能决定的，而是每个人自由选择的结果。

张君劢打了个比方：
- **科学** = 地图，告诉你世界的地理情况
- **人生观** = 方向，告诉你该往哪里走
- **关系** = 地图不能告诉你方向，方向需要你根据价值观自己决定

这对现代人的启发：现代社会过于迷信科学，认为科学可以解决一切问题。张君劢提醒我们：科学有其边界。医学能延长寿命，但不能告诉你生命是否有意义；经济学能解释市场规律，但不能告诉你是否应该赚钱。人生的意义问题，需要哲学、宗教、艺术来回答。

## 三、"科玄论战"：现代中国思想的大争论

1923年，张君劢的演讲引发了"科学与人生观论战"，这是现代中国思想史上最重要的争论。

**论战四方**：
- **张君劢（玄学派）**：科学不能解决人生观，需要哲学和直觉
- **丁文江（科学派）**：科学可以解决一切，包括人生观
- **陈独秀（马克思主义派）**：科学与人生观都受经济基础决定
- **胡适（自由主义派）**：多研究具体问题，少谈抽象主义

论战的结果：科学派在舆论上占了上风，但张君劢提出的问题没有被真正解决。直到今天，科学与人文的关系仍然是现代社会的核心问题。

张君劢的观点在今天特别有启发。比如：
- **基因编辑技术** = 科学能告诉我们怎么做，但不能告诉我们应不应该做
- **人工智能** = 科学能制造AI，但不能告诉我们AI是否有道德地位
- **生命延长** = 科学能延长寿命，但不能告诉我们长寿是否有意义

这些问题都是"科学不能解决的人生观问题"，需要哲学和伦理学来回答。

## 四、"新儒家"：让儒学活起来

张君劢是现代新儒家的开创者之一。他提出：新儒家不是旧儒家的翻版，而是用现代哲学方法重新阐释儒家。

他说："新儒家者，非旧儒家之翻版，乃以现代哲学方法，重新阐释儒家之义理者也。其目的，在使儒学适应现代世界，而非使现代世界适应儒学。"

什么意思？新儒家不是要回到古代，而是要让儒学适应现代社会。用现代哲学（康德、黑格尔、柏格森）重新解释儒家思想，使儒学在现代社会中仍然有价值。

张君劢对儒学性质的重新界定：
- 儒学不是物理学，不是生物学，而是心灵学、道德学
- 儒学的核心不是认识世界，而是修养人格
- 儒学的认识方法是直觉，不是科学分析
- 儒学的目的是道德，不是知识

他打了个比方：
- **科学** = 教你怎么做一把椅子（知识）
- **儒学** = 教你为什么要做椅子，为谁做椅子（道德）
- **关系** = 两者都需要，但不能互相替代

这对现代人的启发：现代社会过于重视知识，忽视了道德。张君劢提醒我们：知识不能代替道德，科学不能解决人生的意义问题。儒学的价值不在于提供知识，而在于提供道德方向。

## 五、"文化认同"：国家的根基

张君劢特别重视文化认同。他说："一国之立，必有其特殊之文化。"

什么意思？一个国家的根基不是军事力量或经济实力，而是文化认同。文化认同是几千年来形成的，不是一朝一夕可以建立的。如果忘记了文化，国家就失去灵魂，即使富裕强大，也只是木偶。

张君劢提出了文化复兴的路径：
- **不是全盘西化**：不能放弃自己的传统文化
- **不是复古守旧**：不能固守传统，拒绝现代化
- **而是创造性的转化**：用现代方法整理传统文化，使之为现代所用

他打了个比方：
- **全盘西化** = 把自己的房子拆了，住别人的房子
- **复古守旧** = 住在自己的老房子里，拒绝修葺
- **创造性的转化** = 修缮自己的老房子，配上现代设施，既保留传统，又适应现代

这对现代人的启发：在全球化时代，文化认同越来越重要。张君劢提醒我们：既要开放，又要坚守；既要学习西方，又要保留传统。不是非此即彼，而是创造性地融合。

## 六、"国家社会主义"：第三条道路

张君劢提出了"国家社会主义"的方案，试图调和资本主义和社会主义。

他说："资本主义之病，在贫富悬殊；社会主义之病，在剥夺自由。吾人所求者，国家社会主义也——以计划经济求公平，以民主政治保自由。"

什么意思？资本主义的问题是贫富差距太大，社会主义的问题是剥夺个人自由。张君劢想走"第三条道路"：经济上计划经济（保证公平），政治上民主制度（保证自由）。

他还提出了"经济民主"的概念：民主不仅是选举权（政治民主），也是生产关系的公平（经济民主）。如果贫富差距太大，即使有了选举权，也不是真正的民主。

这一方案非常超前。现代西方的"福利国家"理论也强调经济民主：以市场经济为基础，以社会保障为补充。但批评者认为，计划经济与民主政治可能有冲突：追求经济公平可能损害政治自由。

这对现代人的启发：现代社会的核心问题是如何平衡效率与公平、自由与平等。张君劢提醒我们：这不是非此即彼的选择，而是需要智慧地平衡。不是放弃自由追求平等，也不是放弃平等追求自由，而是找到两者之间的平衡点。

## 七、一句话总结

张君劢告诉我们：科学有其边界，不能解决人生的意义问题；人生的意义需要哲学、直觉和自由意志来回答；儒学要现代化，不是复古，而是创造性地转化；国家的根基是文化认同，不能忘记传统；要走第三条道路，既求经济公平，又保政治自由。这就是张君劢的新儒家智慧。

## 八、对今天的意义

张君劢的思想在今天仍有巨大启发：

**1. 关于科学边界**：现代社会过于迷信科学，认为科技可以解决一切问题。张君劢提醒我们：科学有其边界。基因编辑、人工智能、生命延长——这些技术带来的伦理问题，科学本身不能回答。需要哲学和伦理学来思考。

**2. 关于人文精神**：在科技至上的时代，人文精神被边缘化。张君劢提醒我们：人生的意义不是科学能决定的，需要哲学、宗教、艺术来回答。不要只做技术专家，也要做人文学家。

**3. 关于文化认同**：全球化时代，文化认同越来越重要。张君劢提醒我们：既要开放学习，又要坚守传统。不是全盘西化，也不是复古守旧，而是创造性地转化。

**4. 关于社会公平**：贫富差距是现代社会的大问题。张君劢的"经济民主"提醒我们：民主不仅是政治权利，也是经济权利。没有经济公平，政治民主就是空洞的。

**5. 关于自由与平等**：效率和公平、自由和平等之间的平衡是永恒难题。张君劢提醒我们：这不是非此即彼的选择，而是需要智慧地平衡。第三条道路可能比极端立场更合理。
