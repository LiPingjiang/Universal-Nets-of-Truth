## 核心论点

1. **才性合**：才能与本性不是分离的，也不是完全同一的，而是相互结合、融合为一的。
2. **四本论**：才性关系有四种理论，李丰持"合"说，试图调和"同"与"异"之间的对立。
3. **正始清谈**：作为正始玄学核心人物，参与并推动了才性论议题的讨论。
4. **才性论的政治意义**：才性合说试图为门阀制度下的选官标准提供折中方案。
5. **人物品藻**：以才性合说为基础，参与人物品藻活动，与傅嘏、夏侯玄等形成辩论。

## 章节关联

| 关联人物 | 关系类型 | 说明 |
|----------|----------|------|
| 傅嘏 | 同道（论敌） | 傅嘏主张"才性同"，李丰"才性合"与之相近但有别，两人在四本论中各持一说 |
| 夏侯玄 | 论敌 | 夏侯玄主张"才性异"，与李丰"才性合"说对立，四本论核心辩论双方 |
| 钟会 | 同道（论敌） | 钟会主张"才性离"，与李丰"才性合"形成对照，四本论四派之一 |
| 何晏 | 正始同道 | 同为正始玄学代表人物，共同参与清谈，李丰曾参与何晏集团 |
| 王弼 | 正始同道 | 王弼注《老子》《周易》，与李丰共同开创正始玄学风气 |
| 刘劭 | 思想先驱 | 刘劭《人物志》系统讨论才性问题，为四本论提供理论基础 |
| 王广 | 同道 | 王广主张"才性离"（一说为钟会），四本论之四派之一 |
| 司马师 | 政治对立 | 李丰因密谋诛司马师事泄被处死，与司马氏集团有政治冲突 |
| 嵇康 | 思想间接关联 | 嵇康竹林玄学主张自然主义，与正始才性论属不同路径 |
| 裴頠 | 后世回应 | 裴頠《崇有论》批判玄学贵无，可视为对正始玄学的反拨 |
