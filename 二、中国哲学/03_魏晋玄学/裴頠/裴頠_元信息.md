# 裴頠 · 元信息

## 基本信息
- **原名**：裴頠，字逸民
- **生卒年**：公元267年—300年（西晋）
- **学派**：玄学（崇有派）
- **核心领域**：本体论、名实之辩、政治哲学

## 关键术语（多语言对照）
| 中文 | 英文/梵文 | 备注 |
|------|----------|------|
| 崇有 | Exalting Being | 以"有"为本体核心主张，反对贵无 |
| 贵无 | Exalting Non-being | 以"无"为本体，王弼、何晏等主张 |
| 无 | Wu / Non-being | 裴頠批判的对象，认为"无"不能生"有" |
| 有 | You / Being | 裴頠认为"有"是万物之本，万有可自生 |
| 自生 | Self-generation | 万物自己生成，不需要"无"作为根据 |
| 济有者皆有 | What Completes Being Is All Being | 万物的生成和完成只能依靠"有" |
| 总混群本 | The Totality of All Roots | 万物之总体，即"有"之全体 |
| 名教 | Mingjiao / Moral Codes | 社会伦理规范，裴頠主张维护名教 |
| 理既有之众 | The Multitude That Already Has Principle | 万物已存在，自有其理 |

## 核心论点
- 崇有反无：批判王弼"以无为本"，主张"有"才是万物的真实本体
- 无不能生有："无"是"有"的缺失状态，不能作为万物生成的根据
- 济有者皆有：万物的生成和完成只能依靠其他"有"，而非"无"
- 自生论：万物自生，不需要"无"作为本体，也不需要造物主
- 维护名教：反对玄学放达之风，主张以礼法维护社会秩序

## 与本书其他章节的关联
- 直接回应王弼"以无为本"，是魏晋玄学内部最重要的理论争论（03_魏晋玄学·王弼）
- 郭象"独化论"在裴頠"自生"论基础上发展，但扬弃了"崇有"的实体论（03_魏晋玄学·郭象）
- 其维护名教的立场与嵇康"非汤武而薄周孔"形成对立（03_魏晋玄学·嵇康）
- 其"自生"思想对宋明理学"理一分殊"有间接影响（05_宋明理学·朱熹）

## 学术定位
- 西晋玄学崇有派的代表人物，与王弼贵无派形成魏晋玄学的两大阵营
- 以《崇有论》系统批判"贵无"思想，是中国哲学史上第一次系统的本体论批判
- 政治上以维护名教著称，反对当时玄学名士的放达之风
- 在"八王之乱"中被赵王司马伦杀害，是西晋政治悲剧的牺牲品
