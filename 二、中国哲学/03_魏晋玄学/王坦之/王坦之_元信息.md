## 核心论点

1. **废庄论**：庄子哲学破坏社会秩序，应当废除。
2. **礼法不可废**：社会必须依靠礼法来维持秩序，玄学主张的自然放任会导致混乱。
3. **批判虚无之谈**：玄学清谈以"无"为本，是空洞的虚谈，无补于世。
4. **儒术为治国之本**：主张回归儒家经典和礼法制度，以儒术治国。
5. **老庄之弊**：老庄哲学强调无为、放达，实际上是为放纵行为提供理论借口。

## 章节关联

| 关联人物 | 关系类型 | 说明 |
|----------|----------|------|
| 庄子 | 批判对象 | 王坦之《废庄论》直接批判庄子哲学，认为其破坏社会秩序 |
| 郭象 | 思想对立 | 郭象注《庄子》，主张"名教即自然"，王坦之则主张废除庄子 |
| 王弼 | 思想对立 | 王弼以"无"为本的玄学本体论，王坦之视为"虚无之谈" |
| 何晏 | 思想对立 | 何晏开启玄学清谈，王坦之批判其浮华风气 |
| 裴頠 | 思想同道 | 裴頠《崇有论》批判"贵无"，与王坦之反玄学立场相近 |
| 范宁 | 思想同道 | 范宁同样批判玄学，排拒老庄，两人立场一致 |
| 孙盛 | 思想同道 | 孙盛亦批判玄学，老子非大贤论与王坦之废庄论相呼应 |
| 王羲之 | 家族关系 | 王坦之为王羲之族侄，同属太原王氏，但王羲之亦受玄学影响 |
| 桓温 | 政治同僚 | 王坦之与桓温在东晋政坛共事，反对桓温篡权 |
| 谢安 | 政治同僚 | 谢安代表玄学化政治，王坦之与之有政见分歧 |
