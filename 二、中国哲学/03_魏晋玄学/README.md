# 03 天人感应：两汉经学与经典解释

## 核心张力

**天(自然) vs 人(人文)**

- 天(tiān)：自然秩序、宇宙法则、神圣权威
- 人(rén)：人文建构、道德实践、政治治理

## 思维流向

从第二章"人性之辩"的道德内省，到第三章"天人感应"的宇宙论建构：
- 汉初黄老：道法结合，无为而治——《黄帝四经》、陆贾、贾谊
- 董仲舒神学：天人感应，儒家神学化——为中央集权提供宇宙论基础
- 今古文之争：微言大义 vs 训诂考证——两种不同的经典解释方法
- 谶纬神学：预言与经典的神秘结合——政治预言的神学化
- 王充批判：疾虚妄，元气自然论——对天人感应的理性反思

## 人物与文献

### 陆贾（约前240-前170）
- **原文**：《新语》(Xīnyǔ，十二篇)
- **核心观点**：逆取顺守，文武并用——马上得天下不能马上治之
- **关键概念**：
  - 逆取顺守(nì qǔ shùn shǒu)：以武力夺取，以文德守成

### 董仲舒(前179-前104)
- **原文**：《春秋繁露》(Chūnqiū Fánlù)
- **核心学说**：天人感应、罢黜百家、三纲五常
- **关键概念**：
  - 天人感应(tiān rén gǎn yìng)：天以灾异谴告人君，人君以德配天
  - 三纲五常(sān gāng wǔ cháng)：君为臣纲、父为子纲、夫为妻纲；仁义礼智信
  - 春秋大一统(chūn qiū dà yī tǒng)：政治统一与思想统一的结合

### 刘歆（约前50-23）
- **原文**：《七略》（已佚，班固《汉书·艺文志》据此）
- **核心贡献**：古文经学倡导者——发现古文《春秋左氏传》《毛诗》《古文尚书》
- **关键概念**：
  - 古文经学(gǔ wén jīng xué)：以先秦古文字书写的经典，重训诂考证

### 许慎（约58-约147）
- **原文**：《说文解字》(Shuōwén Jiězì，十五卷)
- **核心贡献**：中国第一部系统字典——540部首，形声义结合
- **关键概念**：
  - 六书(liù shū)：象形、指事、会意、形声、转注、假借

### 郑玄(127-200)
- **原文**：《毛诗传笺》《三礼注》《论语注》
- **核心贡献**：遍注群经，今古文融合——礼学集大成
- **关键概念**：
  - 通学(tōng xué)：打破今古文壁垒，综合两派

### 王充(27-约97)
- **原文**：《论衡》(Lùnhéng，八十五篇)
- **核心学说**：疾虚妄、元气自然论、无神论
- **关键概念**：
  - 元气(yuán qì)：天地万物之本，自然无为
  - 疾虚妄(jí xū wàng)：批判天人感应、谶纬神学的虚妄
  - 天无意志(tiān wú yì zhì)：天即自然，无目的无意志

## 核心问题

1. **天是否有意志？** 董仲舒：天有意志，以灾异谴告；王充：天无意志，元气自然
2. **今古文之争：微言大义还是训诂考证？** 今文：通经致用，重义理发挥；古文：实事求是，重文字考证
3. **儒家神学化是发展还是异化？** 董仲舒将儒家与阴阳五行结合，是创造性转化还是背离孔子人文精神？
