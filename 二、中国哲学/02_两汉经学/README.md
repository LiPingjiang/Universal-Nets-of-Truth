# 02 人性之辩：从孟子性善到荀子性恶

## 核心张力

**性善 vs 性恶**

- 性善(xìng shàn)：道德本心固有，仁义内在
- 性恶(xìng è)：自然本性趋利，善由人为

## 思维流向

从第一章"道与仁义"的百家争鸣，到第二章"人性之辩"的深化：
- 先秦奠基：孟子性善 vs 荀子性恶——两条人性论路线的对立
- 两汉综合：董仲舒性三品——圣人之性、中民之性、斗筲之性
- 扬雄调和：性善恶混——修善则为善人，修恶则为恶人
- 王充批判：用气为性——禀气厚薄决定善恶，命定论色彩
- 韩愈复兴：性三品说的回归——为宋明理学人性论铺路

## 人物与文献

### 孟子(前372-前289)
- **原文**：《孟子·告子上》《孟子·尽心上》
- **核心论证**：性善四端——恻隐之心仁之端，羞恶之心义之端
- **关键概念**：
  - 四端(sì duān)：恻隐、羞恶、辞让、是非——仁义礼智之端
  - 扩充(kuò chōng)：扩而充之，若火之始然
  - 养气(yǎng qì)：浩然之气，配义与道

### 荀子(前313-前238)
- **原文**：《荀子·性恶篇》
- **核心论证**：人之性恶，其善者伪也——生而有好利、疾恶、耳目之欲
- **关键概念**：
  - 伪(wěi)：人为，通过礼义法度改造本性
  - 化性起伪(huà xìng qǐ wěi)：变化本性，兴起人为
  - 师法(shī fǎ)：贵师重法，化性必由学

### 告子（约前4世纪）
- **原文**：《孟子·告子上》（告子思想见于孟子引述）
- **核心观点**：性无善恶——性犹湍水，决诸东方则东流
- **关键概念**：生之谓性(shēng zhī wèi xìng)：性即生之自然，无道德含义

### 董仲舒(前179-前104)
- **原文**：《春秋繁露·深察名号》《春秋繁露·实性》
- **核心学说**：性三品说——圣人之性、中民之性、斗筲之性
- **关键概念**：
  - 性三品(xìng sān pǐn)：上品善、下品恶、中品可善可恶
  - 待教而善(dài jiào ér shàn)：中民之性待王教而善
  - 天人感应(tiān rén gǎn yìng)：天以阴阳五行生人，性即天之所赋

### 扬雄(前53-18)
- **原文**：《法言·修身》
- **核心观点**：人之性也善恶混——修其善则为善人，修其恶则为恶人
- **关键概念**：善恶混(shàn è hùn)：人性兼具善恶两端，如材木之善恶

### 王充(27-约97)
- **原文**：《论衡·率性篇》《论衡·本性篇》
- **核心观点**：用气为性——禀气厚薄决定善恶，命定论
- **关键概念**：
  - 用气为性(yòng qì wéi xìng)：性即气之禀受，气有清浊
  - 命定论(mìng dìng lùn)：禀气不可变，善恶有定

### 韩愈(768-824)
- **原文**：《原性》
- **核心观点**：性三品说的复兴——上者可教而下者可制
- **关键概念**：
  - 性三品(xìng sān pǐn)：上品善、中品可导、下品恶
  - 性情论(xìng qíng lùn)：性为先天，情为后天发动

## 核心问题

1. **性善论如何解释恶？** 孟子：放其良心，陷溺其心；荀子：性恶，善由礼义
2. **性恶论如何解释善？** 荀子：化性起伪，师法礼义；孟子：善端固有，非由外铄
3. **性三品是调和还是逃避？** 董仲舒、韩愈将人性分品，是否回避了性善性恶的根本争论？
