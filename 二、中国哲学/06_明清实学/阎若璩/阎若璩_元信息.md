# 阎若璩：古文尚书辨伪（元信息）

## 基本信息

- **生卒年**：1636年—1704年（清代）
- **字**：百诗
- **号**：潜丘
- **世称**：潜丘先生
- **籍贯**：山西太原（后迁江苏淮安）
- **学派**：经学考据（清代汉学代表人物）
- **主要著作**：《尚书古文疏证》《四书释地》《潜丘札记》
- **师承**：无明确师承，承毛奇龄等考据学传统
- **影响**：戴震（清代汉学集大成）、惠栋（吴派汉学）、崔述（古史辨先驱）

## 核心论点

1. **古文尚书为伪书**：系统论证《古文尚书》是东晋梅赜伪托，非先秦真本
2. **内证外证结合**：从文本内部和外部同时找证据，方法严谨
3. **辨伪方法论**：建立系统的辨伪方法，成为清代考据学的典范
4. **尊崇今文**：肯定汉代伏生所传《今文尚书》的真实性
5. **疑古精神**：对古代文献保持怀疑，不轻易信古
6. **实事求是**：一切以证据为准，不凭主观臆断

## 学术定位

- **思想史地位**：清代考据学的典范人物，以《尚书古文疏证》证明《古文尚书》为伪书，是清代学术最重要的成就之一，对后世疑古思潮产生深远影响
- **现代诠释**：钱穆以"清代学术之典范"释阎若璩；梁启超以"科学方法"评价其考据方法；顾颉刚以"古史辨先驱"定位之
- **争议焦点**：辨伪是否过于疑古？对古文尚书的全盘否定是否公允？考据方法是否过于繁琐？
- **当代价值**：批判性思维、证据意识、文献考证方法、疑古精神
