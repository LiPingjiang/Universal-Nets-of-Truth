# 07 心学与理学：朱熹与王阳明的分途

## 核心张力

**心(主体) vs 理(客体)**

- 心(xīn)：主体意识、道德本心、良知
- 理(lǐ)：客观法则、宇宙秩序、性理

## 思维流向

从第六章"理事之辩"的格物穷理，到第七章"心学与理学的分途"：
- 朱熹走"性即理"路线：理在心外，须格物穷理
- 王阳明走"心即理"路线：理在心中，须致良知
- 两条路线在明清之际合流与反思（王船山、黄宗羲）

## 人物与文献

### 朱熹(1130-1200)
- **原文**：《四书章句集注》《近思录》《朱子语类》
- **核心学说**：性即理、格物穷理、理一分殊
- **关键概念**：
  - 理(lǐ)：形而上者，生物之本
  - 气(qì)：形而下者，生物之具
  - 性(xìng)：理在人身，天命之性
  - 格物(gé wù)：穷至事物之理

### 陆九渊(1139-1193)
- **原文**：《象山语录》《陆九渊集》
- **核心学说**：心即理、宇宙便是吾心
- **与朱熹之争**：鹅湖之会（1175年）——心学与理学的首次正面交锋

### 王阳明(1472-1529)
- **原文**：《传习录》《大学问》
- **核心学说**：心即理、致良知、知行合一
- **关键概念**：
  - 良知(liáng zhī)：心之本体，不虑而知
  - 致良知(zhì liáng zhī)：推扩良知于事事物物
  - 知行合一(zhī xíng hé yī)：真知必行，不行非知

### 王船山(1619-1692)
- **原文**：《周易外传》《尚书引义》《读通鉴论》
- **核心学说**：理在气中、性日生日成、行先知后
- **对朱王的综合与超越**：批评朱子理先气后，也批评阳明心即理，主张理气合一、性在习中

## 核心问题

1. **理在心内还是心外？** 朱熹：理在心外，须即物穷理；阳明：理在心中，心即理也
2. **良知是现成还是修养？** 阳明后学分化：现成派（王畿）vs 工夫派（钱德洪）
3. **知行如何合一？** 阳明：真知即所以为行，不行不足谓之知；船山：行先知后，知行相资
