# 06 理事之辩：从华严法界到宋明格物

## 核心张力

**理(本体) vs 事(现象)**

- 理(lǐ)：本体、法则、形而上者
- 事(shì)：现象、事物、形而下者

## 思维流向

从第五章"佛性如何"的隋唐佛学，到第六章"理事之辩"的宋明理学：
- 华严宗：法界缘起，理事无碍——理遍于事，事遍于理
- 禅宗：即事即真——平常心是道，穿衣吃饭即是佛法
- 周敦颐：太极图说——无极而太极，理学的开端
- 张载：太虚即气——气本论，民胞物与
- 程颢程颐：天理——性即理，格物致知
- 朱熹：理一分殊——理在事先，格物穷理
- 陆九渊：心即理——发明本心，六经注我

## 人物与文献

### 杜顺(557-640)
- **原文**：《华严法界观门》
- **核心学说**：华严法界观——真空观、理事无碍观、周遍含容观
- **关键概念**：
  - 法界(fǎ jiè)：一切法的总相，理与事的统一体

### 法藏(643-712)
- **原文**：《华严金狮子章》
- **核心学说**：六相圆融、十玄缘起
- **关键概念**：
  - 十玄门(shí xuán mén)：同时具足相应门、广狭自在无碍门等——事事无碍的十种表现

### 周敦颐(1017-1073)
- **原文**：《太极图说》《通书》
- **核心学说**：无极而太极，主静立人极
- **关键概念**：
  - 无极而太极(wú jí ér tài jí)：太极本无极，有无相生
  - 诚(chéng)：圣人之本，诚者天之道

### 张载(1020-1077)
- **原文**：《正蒙》《横渠易说》
- **核心学说**：太虚即气、民胞物与、性二元
- **关键概念**：
  - 太虚即气(tài xū jí qì)：太虚不能无气，气不能不聚为万物
  - 民胞物与(mín bāo wù yǔ)：民吾同胞，物吾与也——横渠四句
  - 天地之性(tiān dì zhī xìng)：太虚本性，善；气质之性：气禀之偏，可善可恶

### 程颢(1032-1085)
- **原文**：《定性书》《识仁篇》
- **核心学说**：仁者与天地万物为一体，心学倾向
- **关键概念**：
  - 识仁(shí rén)：学者须先识仁，仁浑然与物同体
  - 定性(dìng xìng)：动亦定，静亦定，无将迎，无内外

### 程颐(1033-1107)
- **原文**：《伊川易传》《程氏遗书》
- **核心学说**：性即理、格物致知、涵养须用敬
- **关键概念**：
  - 性即理(xìng jí lǐ)：性即理也，所谓理性是也
  - 格物(gé wù)：穷至事物之理
  - 涵养(hán yǎng)：涵养须用敬，进学则在致知

### 朱熹(1130-1200)
- **原文**：《四书章句集注》《近思录》《朱子语类》
- **核心学说**：理一分殊、理气论、心统性情
- **关键概念**：
  - 理一分殊(lǐ yī fēn shū)：太极只是一个理，万物各具一理
  - 理气论(lǐ qì lùn)：理也者，形而上之道也；气也者，形而下之器也
  - 心统性情(xīn tǒng xìng qíng)：性是体，情是用，心则统之

### 陆九渊(1139-1193)
- **原文**：《象山语录》《陆九渊集》
- **核心学说**：心即理、宇宙便是吾心
- **关键概念**：
  - 心即理(xīn jí lǐ)：宇宙便是吾心，吾心即是宇宙
  - 发明本心(fā míng běn xīn)：学问之道，求其放心而已

## 核心问题

1. **理在事先还是理在事中？** 朱熹：理在事先，理先气后；陆九渊：心即理，理在心中；张载：太虚即气，理气合一
2. **格物穷理还是发明本心？** 朱熹：即物穷理，今日格一物，明日格一物；陆九渊：六经注我，先立乎其大者
3. **理事关系：华严的理事无碍如何影响宋明理学？** 华严：理遍于事，事遍于理；朱熹：理一分殊，事事有理——概念传承与转化
