# 程颢：识仁与定性（元信息）

## 基本信息

- **生卒年**：1032年—1085年（北宋）
- **字**：伯淳
- **世称**：明道先生
- **籍贯**：河南洛阳
- **学派**：洛学（宋明理学开创者）
- **主要著作**：《识仁篇》《定性书》（收入《二程遗书》）
- **师承**：周敦颐
- **影响**：陆王心学先驱，与程颐并称"二程"

## 核心论点

1. **天理本体论**："天理"是宇宙本体兼道德根源，客观实在，不因人的活动而增减
2. **仁的存在论化**：仁不是道德规范，而是"浑然与物同体"的存在状态
3. **超越动静的定**："动亦定，静亦定"，消解主客对立，"廓然大公，物来顺应"
4. **直觉认识论**："体贴""体认"，反对格物穷理，主张"存久自明"
5. **自然辩证法**："无独必有对"，对立是存在的本体论结构

## 学术定位

- **思想史地位**：宋明理学开创者之一，洛学奠基人，心学先驱
- **现代诠释**：牟宗三以"智的直觉"释程颢；余英时以"内在超越"定位
- **争议焦点**：泛道德主义（混淆事实与价值）、直觉主义（缺乏公共标准）
- **当代价值**：生态伦理（万物一体）、心灵修养（定性）、辩证法（无独必有对）
