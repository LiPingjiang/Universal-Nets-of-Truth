# 05 佛性如何：隋唐佛学的宗派繁荣

## 核心张力

**佛性(本有) vs 修行(渐修)**

- 佛性(fó xìng)：众生本具成佛之性，内在本有
- 修行(xiū xíng)：渐修积累，外铄工夫，待缘而显

## 思维流向

从第四章"名教与自然"的玄佛合流，到第五章"佛性如何"的隋唐宗派繁荣：
- 三论宗：吉藏——中道二谛，破邪显正，继承般若学
- 天台宗：智顗——一念三千，三谛圆融，中国第一个系统佛教宗派
- 华严宗：法藏——法界缘起，事事无碍，中国化本体论巅峰
- 法相宗：玄奘——唯识无境，转识成智，印度瑜伽行的中国化
- 禅宗：慧能——顿悟成佛，明心见性，彻底中国化的佛教革命
- 净土宗：善导——他力往生，念佛为本，平民化的佛教实践

## 人物与文献

### 吉藏(549-623)
- **原文**：《三论玄义》《中论疏》《大乘玄论》
- **核心学说**：中道、二谛、八不中道
- **关键概念**：
  - 二谛(èr dì)：世俗谛（有）与胜义谛（空）
  - 四重二谛(sì chóng èr dì)：从低到高四层二谛，逐步破执

### 智顗(538-597)
- **原文**：《法华玄义》《摩诃止观》
- **核心学说**：一念三千、三谛圆融、止观双修
- **关键概念**：
  - 一念三千(yī niàn sān qiān)：一念心中具足三千世间（十法界×十如是×三种世间）
  - 三谛圆融(sān dì yuán róng)：空谛、假谛、中谛，三谛在一心中
  - 止观(zhǐ guān)：止（定）与观（慧）双修

### 法藏(643-712)
- **原文**：《华严金狮子章》《华严探玄记》
- **核心学说**：法界缘起、事事无碍、六相圆融
- **关键概念**：
  - 法界缘起(fǎ jiè yuán qǐ)：一切法相互缘起，一即一切，一切即一
  - 理事无碍(lǐ shì wú ài)：理遍于事，事遍于理，理事交融
  - 六相圆融(liù xiàng yuán róng)：总别、同异、成坏六相，相即相入

### 玄奘(602-664)
- **原文**：《成唯识论》（编译印度十大论师释论）《大唐西域记》
- **核心学说**：唯识无境、八识、转识成智
- **关键概念**：
  - 八识(bā shì)：眼耳鼻舌身意末那阿赖耶
  - 转识成智(zhuǎn shí chéng zhì)：转前五识为成所作智，转第六识为妙观察智，转第七识为平等性智，转第八识为大圆镜智

### 慧能(638-713)
- **原文**：《六祖坛经》
- **核心学说**：顿悟成佛、明心见性、无念为宗
- **关键概念**：
  - 顿悟(dùn wù)：直指人心，见性成佛，不立文字
  - 无念(wú niàn)：于念而不念，不执着于念
  - 自性(zì xìng)：本心佛性，本来清净，不假外求

### 善导(613-681)
- **原文**：《观无量寿经疏》《往生礼赞》
- **核心学说**：他力往生、念佛为本、正定业
- **关键概念**：
  - 他力(tā lì)：依靠阿弥陀佛愿力往生净土
  - 正定业(zhèng dìng yè)：念佛为正定业，决定往生

## 核心问题

1. **佛性是本有还是始有？** 地论师南道：本有；北道：始有；天台：性具善恶；华严：性起
2. **顿悟还是渐修？** 禅宗南宗：顿悟；北宗神秀：渐修；天台：圆顿止观——中国佛教的核心分歧
3. **自力还是他力？** 禅宗：自性自度，自力解脱；净土：他力往生，依靠弥陀——两种修行路线
