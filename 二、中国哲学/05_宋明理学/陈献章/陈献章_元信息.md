# 陈献章：白沙心学与自得之学（元信息）

## 基本信息

- **生卒年**：1428年—1500年（明代）
- **字**：公甫
- **号**：石斋
- **世称**：白沙先生
- **籍贯**：广东新会
- **学派**：白沙心学（明代心学先驱）
- **主要著作**：《白沙子集》《白沙语录》
- **师承**：吴与弼（康斋先生，程朱理学）
- **影响**：湛若水（弟子）、王阳明（同时代，受白沙影响）

## 核心论点

1. **自得之学**：真理不是从外学来的，而是心中自有的，需要通过修养自得
2. **以静为本**：以静坐为修养的根本方法，通过静养恢复心的本体
3. **心具万理**：心包含万理，不需要逐物穷理
4. **自然之乐**：修养的极致是自然之乐，不假安排，随处自在
5. **鸢飞鱼跃**：心体流行的自然境界，活泼泼的，非枯寂
6. **学宗自然**：以自然为宗，反对人为造作和繁琐支离

## 学术定位

- **思想史地位**：明代心学先驱，上承陆九渊，下启王阳明，构成明代思想转折的关键人物
- **现代诠释**：陈来以"心学转型"释白沙，视其为程朱到陆王的过渡；牟宗三以"自然之教"释之
- **争议焦点**："静坐"是否近禅？"自然"是否取消道德努力？"自得"是否主观独断？
- **当代价值**：静坐修养的实践方法、自然主义生活态度、自得之学的自信确立
