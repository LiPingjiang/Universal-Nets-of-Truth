# 王廷相：气学与性日生日成（元信息）

## 基本信息

- **生卒年**：1474年—1544年（明代）
- **字**：子衡
- **号**：浚川
- **世称**：浚川先生
- **籍贯**：河南仪封
- **学派**：气学（明代气学集大成者）
- **主要著作**：《慎言》《雅述》《王氏家藏集》
- **师承**：无明确师承，承张载气学传统
- **影响**：王夫之（继承发展气学）、黄宗羲（受其影响）

## 核心论点

1. **气学本体论**：气是宇宙本体，万物皆气的聚散，上承张载"太虚即气"
2. **性日生日成**：性不是先天的、固定的，而是后天在实践中日生日成的
3. **离气无性**：性不能脱离气而存在，性即气之理
4. **理在气中**：理不是独立实体，而是气运行的条理和规律
5. **实践认识论**：知识来自实践和感官经验，反对先验的"德性之知"
6. **批判程朱**：批评朱熹"理在气先"的形而上学，反对"性即理"的先验人性论

## 学术定位

- **思想史地位**：明代气学集大成者，上承张载，下启王夫之、戴震，构成宋明理学气学传统的重要环节
- **现代诠释**：张岱年以唯物主义释王廷相，视其为明清之际思想解放的先声；侯外庐以"早期启蒙思想"定位之
- **争议焦点**："性日生日成"是否消解道德普遍性？经验论是否忽视道德直觉？批判程朱是否过于极端？
- **当代价值**：实践认识论、后天养成论、经验主义方法论
