# 胡宏：性本论与知言（学术解读）

## 一、核心论题：性本论的建构

胡宏（1105-1161），字仁仲，号五峰，福建崇安人。他是南宋湖湘学派的开创者，其父胡安国为著名经学家。胡宏上承程门弟子杨时、侯仲良，下启张栻，与朱熹、陆九渊同时代而思想独立。其代表作《知言》以"性"为核心概念，建构了与程朱理学、陆王心学并列的"性本论"哲学体系。

### 1.1 "性为天地之所以立"：性本体的确立

胡宏哲学的核心命题："性也者，天地之所以立也。"

**问题背景**：程颐提出"性即理"，以"理"为性之内容；朱熹综合程颐，以"性即理"为理学人性论基石。胡宏则另辟蹊径：性不是理，而是理之所以可能的根据——性统摄理、气、心、物，是"统会"。

**胡宏的回应**：
- **性非体非用**：性超越体用二分，既不是本体，也不是作用
- **性统会一切**：本体、作用、器物、道都是性的"统会"
- **性为终极根据**：天地之所以存在，是因为有性

这一概念非常独特：它既非西方形而上学的"实体"（substance），也非佛教的"空"，而是中国哲学特有的"本体-发用"统一体。性作为"统会"，先于一切分化，是一切分化之所以可能的条件。

### 1.2 与程朱理学的对话

| 程颐/朱熹 | 胡宏 |
|-----------|------|
| 性即理 | 性为统会，非理 |
| 心统性情 | 心为已发，性为未发 |
| 性有善恶（天地之性善，气质之性可善可恶） | 性无善恶 |
| 天理人欲对立 | 天理人欲同体异用 |
| 先涵养后察识 | 先察识后涵养 |

胡宏与程朱的分歧不是细节之争，而是本体论立场之争：程朱以"理"为本，胡宏以"性"为本。这一分歧决定了他们在心性论、工夫论、伦理学上的系统性差异。

## 二、心性论：心为已发

### 2.1 "心以成性"的哲学结构

胡宏提出"心也者，知天地，宰万物，以成性者也"——心的功能是"成性"，而非性本身。

**哲学分析**：
- **心为已发**：心是性的发用，不是性的本体
- **心以成性**：心通过认知、主宰来成就性
- **性先心后**：性为本体，心为发用

这与朱熹"心统性情"（心包含性与情）形成对立：朱熹认为心是一个更大的容器，包含未发之性和已发之情；胡宏则认为心只是已发，性在心上。

### 2.2 与陆王心学的对话

陆九渊提出"心即理"，王阳明发展为"心即性"。胡宏的"心为已发"与陆王形成鲜明对立：

| 胡宏 | 陆王 |
|------|------|
| 心为已发 | 心即理/性 |
| 性为本体 | 心为本体 |
| 性先心后 | 心先性后 |
| 先察识后涵养 | 致良知 |

胡宏的立场更保守：心不是本体，而是本体（性）的发用。这意味着人不能直接相信自己的心，而必须通过性的观照来审查心。这一立场对现代"自我肯定"文化有批判意义。

## 三、性无善恶：本体论的超越

### 3.1 "性无善恶"的哲学内涵

胡宏最著名的命题："性无善恶。"

**这不是道德相对主义**，而是本体论判断：
- **善恶是价值判断**：属于情（已发）的层面
- **性超越价值判断**：作为本体，不可作善恶判断
- **性为善恶之根据**：善恶是性的不同发用

这类似于康德的区分：现象界有善恶，物自体不可作善恶判断。但胡宏的意图不是不可知论，而是强调：性作为"统会"，包含善恶的可能性，但不等于善恶本身。

### 3.2 与朱熹的辩论

朱熹批评胡宏"性无善恶"混淆了性之善恶。朱熹的论证：
- 如果性无善恶，则道德标准从何而来？
- 如果性无善恶，则"天命之谓性"如何解释？
- 如果性无善恶，则修养工夫失去方向

胡宏的回应（隐含）：
- 道德标准来自心对性的"察识"，不是来自性本身
- "天命之谓性"说的是性作为本体，不是性作为道德标准
- 修养工夫的方向是"尽其性"，而非"变善性"

### 3.3 "好恶以己/以道"的伦理分析

胡宏区分"好恶"（性之发用）与"以己/以道"（价值选择）：
- **好恶本身不是恶**：好恶是性的自然发用
- **以己之私好恶是恶**：心不以道导之
- **以道之正好恶是善**：心以道导之

这一分析非常精细：性提供好恶的能力（中性），心决定好恶的方向（善恶）。道德责任在心，不在性。这与康德的"自由意志"有相似之处：自然因果不提供道德判断，道德判断来自自由意志的选择。

## 四、天理人欲同体异用

### 4.1 "同体异用"的哲学结构

胡宏最具争议的命题："天理人欲，同体而异用，同行而异情。"

**哲学分析**：
- **同体**：天理与人欲不是两个实体，而是同一实体的不同表现
- **异用**：同一实体可以表现为天理，也可以表现为人欲
- **同行**：天理与人欲走的同一条路
- **异情**：但实情不同

朱熹批评此说"将人欲混入天理"，但胡宏的哲学意图是：消灭欲望不是消灭一个实体，而是转化一种发用。如同水与冰：同一本质，不同形态。我们不是要消灭冰，而是要让冰融化成水。

### 4.2 "欲望辩证法"

胡宏的"欲望辩证法"：
- **从性的角度**：欲望不是不善的——欲望是性的自然发用
- **从心的角度**：欲望不是善的——心如果不以道导之，欲望就是恶的

这意味着道德责任在心，不在欲望本身。欲望是中性的，心的选择决定善恶。这一思想对现代欲望哲学有启发：不是压制欲望，而是引导欲望。

## 五、工夫论：先察识后涵养

### 5.1 察识与涵养的次序

胡宏的工夫论："先察识后涵养"。

- **察识**：察识心之所发，了解性的表现
- **涵养**：在察识的基础上涵养扩充

这与朱熹"先涵养后察识"形成对立。朱熹认为：如果不先通过静坐涵养使心清明，察识就是妄察识；胡宏则认为：如果不先察识心之所发，涵养就是空涵养。

### 5.2 湖湘学统的工夫论

胡宏的工夫论影响湖湘学统：
- **张栻**：继承"先察识后涵养"，但更强调"居敬"
- **朱熹**：批评湖湘学"重察识轻涵养"，主张"居敬穷理"
- **王阳明**：发展"致良知"，与察识有相似之处

## 六、历史定位与影响

### 6.1 在理学谱系中的位置

胡宏上承程门，下启湖湘学统。在理学内部：
- **与程朱**：被朱熹批评，但影响深远
- **与陆王**：心性论对立，但工夫论有相似
- **与张栻**：湖湘学统的传承
- **与王夫之**：王夫之继承"天理人欲同体异用"

### 6.2 现代诠释

- **牟宗三**：以"性体"释胡宏，视其为宋明理学重要一支
- **钱穆**：重视湖湘学统，认为胡宏被朱熹遮蔽
- **陈来**：分析胡宏与朱熹的辩论，指出胡宏的哲学价值

### 6.3 批评与争议

- **道德相对主义**："性无善恶"是否消解道德标准？
- **欲望辩护**："天理人欲同体"是否为欲望辩护？
- **心外有性**："心为已发"是否矮化心的地位？
