# 程颐：涵养与格物（学术解读）

## 一、核心论题：理性主义的理学建构

程颐（1033-1107），字正叔，世称伊川先生，程颢之弟。与兄共创洛学，但思想取向不同：程颢重直觉体悟，程颐重理性修养。程颐的学说经朱熹发扬光大，成为理学正统，影响中国思想七百年。

### 1.1 "涵养须用敬，进学则在致知"

此语是程颐工夫论的总纲，包含两个维度：

**"敬"的工夫**：
- "敬"即"主一""无适"——精神专注，不旁骛
- 敬是道德修养的根基，"入道莫如敬"
- 敬则自然和，敬是工夫，和是效验
- 与程颢"识仁"不同：程颢从本体入手，程颐从工夫入手

**"致知"的学问**：
- "致知在格物"——知识来源于穷究事物之理
- 格物不是接触事物，而是穷理
- 积累既久，自然贯通（渐进主义）

二者关系：敬是致知的心理条件，致知是敬的理性支撑。没有敬，知成玩物；没有致知，敬成空寂。

### 1.2 "性即理也"：人性论的本体论转向

程颐提出"性即理"，标志理学人性论的确立：

- **孟子**：性善是就"四端"（恻隐等心理倾向）而言
- **程颐**：性善是就"理"（本体）而言——性不是气质，而是义理
- **恶的来源**：恶不是性，而是"气禀"（气质禀赋）

这一区分被朱熹发展为"天命之性"（纯善）与"气质之性"（可善可恶）的二分。问题：如果性即理，理是纯善的，恶从何来？程颐以"气禀"解释，但气禀为何有清浊？这是否将恶的根源推给宇宙论，从而减轻道德责任？

## 二、格物穷理：理学认识论

### 2.1 对《大学》"格物"的重新诠释

《大学》言"格物致知"，但未解释"格物"何义。程颐重新诠释：

- **"格"**：穷究、至极
- **"物"**：理（非具体事物，而是事物之理）
- **"格物"**：穷究事物之理，至极其理

这一诠释将认识论形上学化：格物不是经验研究，而是对形上之理的把握。

### 2.2 积累与贯通

程颐的格物论是渐进主义：

> "今日格一物，明日格一物，积习既久，脱然自有贯通处。"

- **渐进**：一件一件穷理，不期待顿悟
- **贯通**：积累既久，自然豁然贯通
- **目的**：最终达到"众物之表里精粗无不到，吾心之全体大用无不明"

这与禅宗"顿悟"对立，与科学归纳法有相似。但程颐的"理"是形上实体，非经验规律；"贯通"是本体论的顿悟，非逻辑的归纳。

### 2.3 与程颢认识论的比较

| 程颢（明道） | 程颐（伊川） |
|-----------|-----------|
| "体贴""体认" | "格物""穷理" |
| 直觉主义 | 理性主义 |
| "存久自明" | "积习贯通" |
| 重本体 | 重工夫 |
| 心学先驱 | 理学正统 |

## 三、理一分殊：形上学架构

### 3.1 "理一"与"分殊"

程颐提出"理一分殊"，解决"一"与"多"的形上学问题：

- **理一**：统一的太极之理，超越具体事物
- **分殊**：万物各有其理，理在事中
- **关系**：万物之理即太极之理，非另有一理

朱熹以"月印万川"喻之：一月在天，万川映月，月只是一月，影有万影。程颐的"理一分殊"既保持统一性（理一），又承认差异性（分殊），避免了一元论与多元论的对立。

### 3.2 与华严宗"一即一切"的比较

华严宗言"一即一切，一切即一"，与"理一分殊"有表面相似：

- **相似**：都强调统一性与差异性的圆融
- **差异**：华严宗以"法界缘起"为基，理学以"天理实体"为基；华严宗重"性起"，理学重"格物"

程颐可能受华严宗影响，但将佛教的本体论转化为儒家的道德形上学。

## 四、"道学"的自我意识

### 4.1 区别于文士与讲师

程颐区分三种学者：

- **文士**：能文者，追求辞章之美
- **讲师**：谈经者，追求训诂之详
- **儒者**：知道者，追求义理之真

这是"道学"（理学）的自我定位：儒学不是文学，不是经学，而是"道学"——对道的体认与践行。

### 4.2 与王安石"新学"的对立

程颐反对王安石的"新学"：
- 王安石以《三经新义》为科举标准，程颐认为是"曲学"
- 王安石重功利，程颐重义理
- 这一对立延续为南宋"朱陆之争""庆元党禁"

## 五、历史定位与影响

### 5.1 在理学谱系中的位置

程颐上承周敦颐、程颢，下启朱熹。朱熹以程颐为正统，建构了严密的理学体系：
- **理气论**：理是形上实体，气是形质材料
- **心性论**：天命之性（理）与气质之性（气）
- **工夫论**：格物穷理与涵养用敬

### 5.2 现代诠释

- **冯友兰**：以新实在论释程颐，"理"为客观共相
- **牟宗三**：批评程颐"性即理"是"只存有不活动"，非"即存有即活动"
- **余英时**：以"士的自觉"释程颐，道学是士大夫的政治文化运动

### 5.3 批评与争议

- **形式主义**："敬"是否沦为外在仪态？
- **理性主义**：格物穷理是否过于强调知识，忽视实践？
- **保守主义**："理"的客观性是否阻碍社会变革？

## 六、关键文本细读

### 6.1 "涵养须用敬，进学则在致知"

此语的深层结构：
- **敬**：道德修养的工夫，心理层面的专注
- **致知**：知识增长的途径，理性层面的穷理
- **关系**：敬是致知的前提（无敬则知成玩物），致知是敬的深化（无知则敬成空寂）

程颐的"敬"不同于宗教的"虔敬"——不是对人格神的敬畏，而是对"理"的专注。这既是道德修养，也是认识论态度。

### 6.2 "性即理也"

此语的革命性：
- **孟子**：性善是心理描述（人皆有恻隐之心）
- **程颐**：性善是本体论命题（性即理，理是纯善的）
- **问题**：恶从何来？程颐以"气禀"解释，但气禀为何有清浊？

牟宗三批评程颐"性即理"是"只存有不活动"——理是静态的实体，非动态的创造。这与程颢"天理"的"活动性"不同。
