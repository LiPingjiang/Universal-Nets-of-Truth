# 湛若水：甘泉心学与随处体认天理（元信息）

## 基本信息

- **生卒年**：1466年—1560年（明代）
- **字**：元明
- **号**：甘泉
- **世称**：甘泉先生
- **籍贯**：广东增城
- **学派**：甘泉心学（明代心学重要一支）
- **主要著作**：《甘泉集》《心性图说》《圣学格物通》
- **师承**：陈献章（白沙先生）
- **影响**：王阳明（同时代，思想对话）、吕坤（后学）

## 核心论点

1. **随处体认天理**：修养工夫不是在静坐中，而是在日常生活中随时随地体认天理
2. **心体与万物同体**：心的本体与宇宙万物同体，心无处不在
3. **天理即心**：天理就是心，心就是天理，体认天理就是体认心
4. **勿忘勿助**：修养要不失本心，不刻意助长，在日用常行中自然体认
5. **格物即正念头**：重新解释"格物"为端正心中念头，非逐物穷理
6. **调和朱陆**：试图综合朱熹的"道问学"与陆九渊的"尊德性"

## 学术定位

- **思想史地位**：明代心学重要一支，甘泉学派的创始人，与阳明心学并称明代两大心学，试图调和朱陆
- **现代诠释**：陈来以"心学之调和"释甘泉，视其为朱陆融合的关键尝试；钱明重视其与阳明的思想对话
- **争议焦点**："随处体认"是否过于宽泛？调和朱陆是否两边不讨好？"心体同物"是否泛化心的概念？
- **当代价值**：生活化修养的实践智慧、调和思维的方法论启示、身心统一的哲学
