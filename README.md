# 窅纂 · Compendium of Abstruse Truth

遍历天下各方文明的思想疆域，汇总所有用以探求本源真理的思辨工具与各家学说。

---

## 书名释义

**《窅纂》**

- **窅**（yǎo）：深远幽杳，代指人类哲学不断追问的幽深终极问题；隐微难察，各类流派底层暗藏的分歧、精微思辨；虚空玄冥，对应东西方形而上学里不可直观的本体领域。
- **纂**（zuǎn）：汇集、编录、汇编群书。古代《四库纂要》《诸子纂录》皆用此字，专指将分散、各成体系的文献、学说整合为一部完整典籍。

全名意境：一部汇编古今所有幽深终极疑问，以及各家给出的思辨解答的典籍。二字极简，古卷感极强，含蓄厚重。

**英文书名**：Compendium of Abstruse Truth

- Compendium 直译"汇编、集要"，精准对应"纂"
- Abstruse 意为幽深玄奥、艰深难解，完美对应"窅"

---

## 核心定位

本书不是一部传统意义上的"哲学史"，而是一部**以问题为纲、以解法为目**的全球思想汇编。

在纵向时间线的主干上，以当代人的视角重新挖掘和阐释古今各家学说最本质的思想碰撞与精髓，用熟语和白话打破术语壁垒，让今天的读者能像古人一样自然听懂那些曾改变世界的思考。

---

## 三大文明传统

本书遵循国际学界通用框架，以三大文明哲学传统为主线：

1. **西方哲学（Western Philosophy）**：古希腊源头，经中世纪、近代、当代，内含分析哲学与欧陆哲学两大路线
2. **中国哲学（Chinese / East Asian Philosophy）**：儒道源头，经魏晋玄学、宋明理学、近现代新儒学
3. **印度哲学（Indian / South Asian Philosophy）**：吠陀传统，经婆罗门教、佛教、六派哲学

伊斯兰哲学、非洲哲学、拉美哲学等归入各传统交汇与影响脉络中叙述，不作为独立顶层传统。

## 西方哲学：思维流变的主线

西方哲学十三章按人类思维关注点的历史流变展开，每一章标题本身就是一个核心张力或问题：

| 章节 | 核心张力 | 思维流变 |
|------|----------|----------|
| 01 本原与存在 | 万物从何而来？ | 从泰勒斯"水是本原"到柏拉图"理念是真实存在"——人类第一次用自然本身解释自然，随后发现感官世界不可靠，转向追问背后的永恒实体 |
| 02 变化与永恒 | 流变与静止哪个更真实？ | 赫拉克利特"万物流变"与巴门尼德"存在不动"的正面碰撞→柏拉图调和（理念永恒、现象流变）→亚里士多德"潜能与实现"的动态存在论 |
| 03 德性与知识 | 什么是善的生活？ | 苏格拉底"德性即知识"把哲学从自然转向人→希腊化时期追问"如何在不确定世界中获得内心宁静"——从公共政治伦理退缩到个人心灵安顿 |
| 04 信仰与理性 | 雅典与耶路撒冷如何调和？ | 奥古斯丁用柏拉图框架理解基督教→阿奎那用亚里士多德综合信仰与理性→奥卡姆剃刀最终分离二者，为近代铺路 |
| 05 共相与个别 | 概念是否对应实在？ | 安瑟尔谟"共相是实在"→罗瑟林"共相只是声音"→阿奎那温和调和→奥卡姆"只有个别事物存在"——摧毁经院哲学基础，引出经验主义 |
| 06 经验与理性 | 知识来源于经验还是理性？ | 培根归纳法→洛克白板说→笛卡尔我思→斯宾诺莎神即自然→莱布尼茨单子论→休谟怀疑论摧毁因果——近代认识论的大震荡 |
| 07 心灵与物质 | 心灵与身体是什么关系？ | 笛卡尔二元论→斯宾诺莎一元两面→莱布尼茨单子论→贝克莱观念论→当代功能主义、取消论、泛心论——四百年的心物难题 |
| 08 主体与绝对 | 主体如何构造客体？ | 康德"哥白尼式革命"→费希特绝对自我→谢林自然哲学→黑格尔辩证法→费尔巴哈唯物主义——德国古典哲学的内在张力与崩溃 |
| 09 自由与必然 | 人有自由意志吗？ | 康德先验自由→黑格尔"自由是对必然的认识"→马克思历史必然性中的阶级自由→萨特"人被判定为自由"→法兰克福学派"被操纵的自由" |
| 10 语言与逻辑 | 语言与世界的关系？ | 弗雷格现代逻辑→罗素摹状词→维特根斯坦早期"语言的界限"→维也纳学派证实原则→日常语言分析→后分析哲学整体论 |
| 11 存在与权力 | 存在如何被理解？ | 尼采"权力意志"→海德格尔"存在之遗忘"→萨特"存在先于本质"→福柯"权力/知识"→德里达"解构"——从追问存在到追问权力与话语 |
| 12 真理与解构 | 什么是真理？ | 符合论→融贯论→实用论→海德格尔"解蔽"→德里达"延异"——真理概念从对应事实到被彻底解构的历程 |
| 13 意识、正义与生态 | 21世纪人类面临什么？ | 心灵哲学意识难题→政治哲学正义分配→生态哲学人类世危机——当代哲学从抽象理论转向具体生存困境 |

## 书写体例

- **甲译（白话意译）**：现代口语，通顺自然，正文加粗，括注解释与脚注标记常规字重
- **乙译（学术直译）**：术语精确，逐句对应原文，不加粗
- **熟语速读**：字面与本义有隔阂的条目，开头用现代大白话一句点破"到底是什么"
- **括注用原文词**：甲译正文用熟语，括注必须填真正的原文词，不得再做意译替换
