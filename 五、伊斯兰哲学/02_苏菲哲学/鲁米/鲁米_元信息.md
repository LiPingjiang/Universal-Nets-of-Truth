# 鲁米：元信息

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 贾拉勒丁·鲁米 |
| **外文名** | Jalal al-Din Rumi / Mevlana / Mawlana |
| **生卒年** | 1207年–1273年 |
| **国籍/地域** | 生于中亚巴尔赫（Balkh，今阿富汗），卒于土耳其孔亚（Konya） |
| **身份/职业** | 苏菲诗人、神秘主义哲学家、法学教授、梅夫拉维教团创始人 |
| **核心主题** | 苏菲神秘主义、爱的哲学、灵魂回归、旋转舞（萨玛）、宗教多元主义、自我消融 |

| **所属传统** | 波斯苏菲主义、梅夫拉维教团（Mevlevi Order）、伊斯兰神秘主义 |
| **历史地位** | 伊斯兰文明史上最伟大的诗人之一；苏菲神秘主义哲学的集大成者；跨宗教对话的先驱；当代西方最畅销的"灵性诗人"之一 |

## 三、核心论点列表

1. **爱的本体论**：爱是唯一的真实；爱是宇宙的本源、认识神的媒介、伦理学的原则。
2. **爱高于知识**：知识把人带到神的门口，爱把人带进神的房间；爱比理性更深刻。
3. **灵魂流亡论**：灵魂来自光的世界，在物质世界中处于流亡状态；死亡是回归故乡。
4. **自我消融论**："我"是最大的谎言；溶解"我"即找到神；个体在爱中转化为更高存在。
5. **旋转舞哲学**：旋转是宇宙隐喻、自我消融的身体体验、通往静止的运动之路。
6. **宗教多元主义**：真理是唯一的，门是众多的；不同宗教在形式上不同，在本质上相同。
7. **形式与本质**：宗教的"形式"（仪式、教条）是外在的，"本质"（爱、真理）是内在的。
8. **批判教条主义**：执着于"我的宗教是唯一正确的"是自我幻觉；应超越形式触及本质。
9. **镜子隐喻**：心灵是镜子，被"自我"的灰尘覆盖时无法反映神；清除自我即见神。
10. **酒与醉的隐喻**：神圣的爱如酒，使灵魂醉而忘自我；醉是超越理性的途径。
11. **蜡烛隐喻**：灵魂是蜡烛，在黑暗世界中燃烧；燃烧即痛苦，也是希望。
12. **友谊与分离**：鲁米与沙姆斯的友谊是"爱的原型"；分离的痛苦转化为创作的动力。

## 四、章节关联

### 同章节关联（五、伊斯兰哲学 / 02_苏菲哲学）

- **伊本·阿拉比**：伊本·阿拉比的"存在单一论"（wahdat al-wujud）与鲁米的"爱的本体论"在苏菲形而上学上形成核心对话；两者在"合一""完美人"等主题上有深层亲和性。

### 跨章节关联

- **五、伊斯兰哲学 / 01_古典伊斯兰哲学 / 伊本·巴哲**：伊本·巴哲的"理智主义"与鲁米的"爱的哲学"在"理性vs.爱"的维度上形成伊斯兰哲学内部的重要张力。
- **五、伊斯兰哲学 / 01_古典伊斯兰哲学 / 伊本·哈兹姆**：伊本·哈兹姆的"爱的宿命论"与鲁米的"爱的本体论"在"爱的哲学"主题上有对话空间。
- **五、伊斯兰哲学 / 01_古典伊斯兰哲学 / 加扎利**：加扎利的"圣学复苏"中的苏菲转向与鲁米的"爱的转向"在"从理智到爱"的维度上有继承关系。
- **五、伊斯兰哲学 / 01_古典伊斯兰哲学 / 苏赫拉瓦尔迪**：苏赫拉瓦尔迪的"光照哲学"（Ishraq）与鲁米的"光的世界"在"光之形而上学"上有深层对话。
- **一、西方哲学 / 03_德性与知识 / 柏拉图**：柏拉图的《会饮》中的"爱欲"（Eros）与鲁米的"爱"（'ishq）在"爱的形而上学"上有跨文化比较空间。
- **一、西方哲学 / 03_德性与知识 / 普罗提诺**：普罗提诺的"流溢论"和"灵魂回归"与鲁米的"灵魂回归"和"合一"在新柏拉图主义传统上有直接继承关系。
- **一、西方哲学 / 11_存在与权力 / 叔本华**：叔本华的"意志与表象"和"艺术作为解脱"与鲁米的"爱作为合一"和"旋转舞作为解脱"有比较空间。
- **一、西方哲学 / 11_存在与权力 / 尼采**：尼采的"酒神精神"（Dionysian）和"权力意志"与鲁米的"狂喜"（wajd）和"爱之力"有深层对话。
- **一、西方哲学 / 11_存在与权力 / 海德格尔**：海德格尔的"存在之思"和"诗意栖居"与鲁米的"存在即爱"和"诗歌作为哲学"有对话空间。
- **三、印度哲学 / 01_吠陀与奥义书 / 奥义书**：奥义书中的"梵我合一"（Atman-Brahman unity）与鲁米的"灵魂与神合一"在"合一"形而上学上有跨文化比较。
- **三、印度哲学 / 03_佛教哲学 / 佛陀**：佛教的"无我"（anatta）与鲁米的"消融自我"在"自我消解"的维度上有深层对话。
- **二、中国哲学 / 01_先秦诸子 / 老子**：老子的"道法自然"和"无为"与鲁米的"顺从爱"和"消融自我"在"顺应"哲学上有比较空间。
- **二、中国哲学 / 01_先秦诸子 / 庄子**：庄子的"逍遥游"和"齐物"与鲁米的"狂喜"和"宗教多元"在"超越分别"的维度上有对话空间。
- **四、交汇编 / 03_西学东渐 / 伊本·阿拉比**：鲁米与伊本·阿拉比共同构成苏菲哲学传统最重要的两个节点。

### 主题关联

- **"爱的哲学"主题**：可与柏拉图《会饮》、奥古斯丁《忏悔录》、斯宾诺莎《伦理学》中的"爱"概念形成跨文化比较研究。
- **"神秘主义"主题**：可与基督教神秘主义（埃克哈特大师、伪狄奥尼修斯）、印度神秘主义（商羯罗、拉玛纳·马哈希）、中国神秘主义（禅宗）形成比较研究。
- **"宗教多元主义"主题**：可与约翰·希克（宗教多元主义）、保罗·尼特（宗教对话）、弗朗西斯·克洛尼（比较神学）形成对话。
- **"身体与灵性"主题**：可与梅洛-庞蒂（身体现象学）、福柯（身体技术）、当代舞蹈哲学形成比较研究。

## 五、延伸阅读

- **代表作**：Jalal al-Din Rumi, *Mathnawi*（《玛斯纳维》）——鲁米最重要的系统著作，六卷
- **代表作**：Jalal al-Din Rumi, *Fihi Ma Fihi*（《沉默之书》）——鲁米的日常谈话和讲道
- **英译本**：Coleman Barks, *The Essential Rumi*（1995）——使鲁米在西方广为人知的英译本
- **研究著作**：Annemarie Schimmel, *The Triumphal Sun: A Study of the Works of Jalaloddin Rumi*（1993）——鲁米研究的经典之作
- **研究著作**：William Chittick, *The Sufi Path of Love: The Spiritual Teachings of Rumi*（1983）——从苏菲传统内部解读鲁米
- **比较研究**：Leonard Lewisohn, *The Heritage of Sufism*（1999）——将鲁米置于波斯苏菲传统的整体框架中
- **当代应用**：Kabir Helminski, *The Rumi Collection*（2000）——鲁米诗歌的当代精神应用
- **舞蹈研究**：Irene Markoff, "Mevlevi Sufi Music and Semah" in *The Garland Encyclopedia of World Music*（2002）——鲁米旋转舞的音乐和仪式研究
- **跨文化研究**：Elif Shafak, *The Forty Rules of Love*（2010）——以鲁米和沙姆斯的友谊为背景的当代小说
