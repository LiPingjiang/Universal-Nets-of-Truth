| **生卒年** | 1849年–1905年 |
| **国籍/地域** | 埃及，生于盖勒尤卜省马哈拉特纳斯尔（Maḥallat Naṣr），卒于亚历山大 |
| **身份/职业** | 伊斯兰学者、法学家、神学家、社会改革家、埃及大穆夫提（1899-1905） |
| **核心主题** | 理性与启示的调和、伊智提哈德复兴、反对盲从、爱资哈尔改革、法目的论、跨宗教对话 |

| **历史地位** | 伊斯兰现代主义的理论奠基者；20世纪伊斯兰改革思想的源头性人物；爱资哈尔大学改革的先驱 |
| **学术背景** | 爱资哈尔大学（传统宗教法学、神学教育）；1882年参与乌拉比起义被流放；流亡法国期间与阿富汗尼合作 |
| **主要师承** | 贾迈勒丁·阿富汗尼（政治改革思想）；爱资哈尔传统经学（法学与神学基础） |

## 核心论点列表

1. **理性与启示的本质相容性**：真正的理性和真正的伊斯兰启示不可能互相矛盾。两者冲突的表象，要么源于对宗教的错误理解，要么源于理性运用的缺陷。理性思考最终会引向对独一真主的认识。
2. **伊智提哈德的开放性**：独立判断之门从未真正关闭，也不应该关闭。每一时代的伊斯兰学者都有责任对新出现的问题进行独立的理性推理，而不是机械地套用中世纪法学结论。
3. **法目的高于字面条文**：伊斯兰法的目的是保护人类的根本利益，法律条文是实现这些目的的手段。当字面条文与时代需要发生张力时，应以更高的法律目的为准绳，进行创造性的重新诠释。
4. **穆斯林落后的内因论**：伊斯兰世界的落后根源不在伊斯兰教本身，而在于穆斯林放弃了伊斯兰鼓励的理性探究精神，沉溺于盲目模仿和迷信。改革方向是向内的——回归伊斯兰的真正精神。
5. **教育改革的迫切性**：爱资哈尔等传统宗教教育机构必须引入现代科学和批判性思维，伊斯兰知识分子应当具备理解和回应现代世界的能力。
6. **公民宗教的相容性**：在一个多元社会中，虔诚的穆斯林信仰与尊重不同信仰者的公民资格完全相容。宗教属于真主，祖国属于全体公民。
7. **跨宗教对话的可能性**：伊斯兰与基督教、犹太教同属亚伯拉罕宗教传统，共享理性基础，可以进行实质性的知识对话，而非对抗。
8. **《古兰经》的目的性诠释**：《古兰经》是指引书，而非自然科学教科书。它关于宇宙的论述目的是引导理性沉思，而非提供科学理论。对其字面科学意义的过度解读是一种误读。
