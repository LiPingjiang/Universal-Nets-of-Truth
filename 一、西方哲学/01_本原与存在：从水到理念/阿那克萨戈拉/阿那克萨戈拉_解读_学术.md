# 阿那克萨戈拉 · 解读（学术版）

## 一、思想史定位

### 1.1 西方哲学史上第一位将心灵作为宇宙秩序原则的人

阿那克萨戈拉（约公元前500-428年）是西方哲学史上**第一位明确提出"心灵"（νοῦς）作为宇宙秩序原则的人**。他的"心灵"概念影响了柏拉图、亚里士多德和整个西方哲学传统。

### 1.2 西方哲学史上第一位将哲学引入雅典的人

阿那克萨戈拉是西方哲学史上**第一位将哲学引入雅典的人**。他在雅典教授哲学，成为伯里克利的朋友和老师。他的思想对雅典的文化和政治产生了深远影响。

### 1.3 对后世哲学的影响

阿那克萨戈拉的影响贯穿整个西方哲学史：
- **柏拉图**："灵魂"（ψυχή）概念直接受阿那克萨戈拉影响——灵魂是认识理念的能力
- **亚里士多德**："不动的推动者"直接继承阿那克萨戈拉的"心灵"——神是纯粹的思想（νοῦς）
- **斯多葛学派**："宇宙灵魂"（πνεῦμα）概念来自阿那克萨戈拉——心灵渗透万物
- **基督教**："上帝是心灵"的概念来自阿那克萨戈拉——上帝是宇宙的秩序原则

## 二、核心命题分析

### 2.1 "万物包含一切"

**核心命题**：在一切事物中都包含一切事物的种子（ἐν παντὶ πᾶν）。

**哲学问题**：

**（1）"种子"（σπέρματα）的含义**
- 阿那克萨戈拉认为：宇宙最初是一个**混沌的混合物**——一切事物的"种子"混合在一起
- 每个事物都包含**一切事物的种子**——如面包中含有肉、骨、血等的种子
- 事物的变化不是从非存在到存在，而是**种子的分离和组合**

**（2）与巴门尼德的关系**
- 巴门尼德说：存在不能从非存在产生——变化是不可能的
- 阿那克萨戈拉说：变化是可能的——但不是从非存在到存在，而是**种子的分离和组合**
- 种子始终存在——只是从混合状态变为分离状态

**（3）与原子论的关系**
- 阿那克萨戈拉的"种子"与德谟克利特的"原子"有相似之处——都是万物的最小单位
- 但种子是**可分的**（包含一切），原子是**不可分的**
- 种子是**质的**（有肉、骨、血等），原子是**量的**（只有形状、大小、位置）

**学术争议**：
- 阿那克萨戈拉的"种子"是**物质的**，还是**非物质的**？
  - **物质说**：种子是物质的——是万物的最小物质单位
  - **非物质说**：种子是形式的——是事物的本质或属性
  - 大多数学者倾向于**物质说**——阿那克萨戈拉是唯物主义者
- "万物包含一切"是否导致**无限倒退**？
  - **倒退问题**：如果面包包含肉的种子，肉的种子又包含面包的种子——无限倒退
  - **解决方案**：阿那克萨戈拉可能认为：种子是**潜在的**——不是实际包含一切，而是潜在地包含一切
  - 大多数学者认为：阿那克萨戈拉没有解决这一问题——这是他的理论的缺陷

### 2.2 "心灵"（νοῦς）

**核心命题**：心灵是无限的、自主的，不与任何事物混合，而是独自存在。

**哲学问题**：

**（1）"心灵"的特征**
- 心灵是**无限的**——没有边界，没有限制
- 心灵是**自主的**——不受任何事物的影响
- 心灵是**不与任何事物混合的**——它是纯粹的，不受物质的污染
- 心灵是**运动的开始**——它启动宇宙的旋转
- 心灵是**知识的源泉**——它认识一切事物

**（2）心灵与物质的关系**
- 心灵是**超越物质的**——不与任何事物混合
- 但心灵**作用于物质**——启动宇宙的旋转，使种子分离
- 这一关系是西方哲学史上**第一次明确表达心灵与物质的二元论**

**（3）与柏拉图的关系**
- 柏拉图的"灵魂"（ψυχή）直接受阿那克萨戈拉的"心灵"影响
- 柏拉图的灵魂也是**超越物质的**——认识理念（非物质的对象）
- 但柏拉图的灵魂是**不朽的**——阿那克萨戈拉没有明确说心灵是不朽的

**学术争议**：
- 阿那克萨戈拉的"心灵"是**人格化的**，还是**非人格化的**？
  - **人格化说**：心灵是人格化的神——有意志、有知识、有目的
  - **非人格化说**：心灵是宇宙的秩序原则——不是人格化的神
  - 大多数学者倾向于**非人格化说**——阿那克萨戈拉的心灵是秩序原则，不是人格化的神
- 阿那克萨戈拉是否**真正引入了目的论**？
  - **目的论说**：心灵启动旋转是有目的的——为了使宇宙有序
  - **机械论说**：心灵启动旋转是机械的——只是启动了运动，没有目的
  - 大多数学者认为：阿那克萨戈拉**试图引入目的论**，但没有完全成功——他的理论仍然是机械论的

### 2.3 宇宙生成论

**核心命题**：最初，万物混合在一起；然后心灵启动了旋转，使万物分离。

**宇宙生成的过程**：

1. **最初**：万物混合在一起——一切事物的种子混沌地混合
2. **心灵启动旋转**：心灵启动了宇宙的旋转——旋转使种子分离
3. **种子分离**：重的种子（如土、石）向中心聚集——形成大地
4. **轻的种子分离**：轻的种子（如气、火）向边缘扩散——形成天体
5. **万物的形成**：种子继续分离和组合——形成万物

**哲学意义**：
- 阿那克萨戈拉的宇宙生成论是西方哲学史上**第一次用机械论和目的论结合来解释宇宙生成**
- 机械论：旋转使种子分离——这是物理过程
- 目的论：心灵启动旋转——这是目的因
- 这一思想影响了后来的**科学宇宙论**——从牛顿到爱因斯坦

**学术争议**：
- 阿那克萨戈拉的宇宙生成论是**科学的**，还是**神话的**？
  - **科学说**：阿那克萨戈拉试图用自然的原因来解释宇宙——这是科学的
  - **神话说**：阿那克萨戈拉的"心灵"类似于神话中的神——这是神话的
  - 大多数学者倾向于**科学说**——阿那克萨戈拉是科学家，不是神话学家

## 三、阿那克萨戈拉与后世哲学

### 3.1 对柏拉图的影响

- 柏拉图的"灵魂"（ψυχή）直接受阿那克萨戈拉的"心灵"影响
- 柏拉图在《斐多篇》中让苏格拉底批评阿那克萨戈拉——说阿那克萨戈拉没有真正使用心灵来解释宇宙
- 但柏拉图自己发展了"灵魂"的概念——灵魂是认识理念的能力

### 3.2 对亚里士多德的影响

- 亚里士多德的"不动的推动者"直接继承阿那克萨戈拉的"心灵"——神是纯粹的思想（νοῦς）
- 亚里士多德在《形而上学》中批评阿那克萨戈拉——说阿那克萨戈拉的心灵只是"机械的原因"，不是"目的因"
- 但亚里士多德自己也用"心灵"来解释宇宙——神是"对思考本身的思考"

### 3.3 对斯多葛学派的影响

- 斯多葛学派的"宇宙灵魂"（πνεῦμα）概念来自阿那克萨戈拉——心灵渗透万物
- 斯多葛学派也继承了阿那克萨戈拉的"命运"（εἱμαρμένη）概念——宇宙的运行遵循必然的规律

### 3.4 对基督教的影响

- 早期基督教教父（如查士丁、德尔图良）称阿那克萨戈拉为"**基督之前的基督徒**"
- 基督教的"上帝是心灵"的概念来自阿那克萨戈拉——上帝是宇宙的秩序原则
- 但基督教与阿那克萨戈拉有根本区别——基督教的人格化的上帝，阿那克萨戈拉的非人格化的心灵

## 四、文献考证

### 4.1 著作的保存

阿那克萨戈拉的著作《论自然》只有**残篇**保存下来——约20条（DK 59 B 1-21），主要来自：
- **辛普里丘**对亚里士多德《物理学》的评注
- **第欧根尼·拉尔修**的《名哲言行录》
- **斯托拜乌斯**的《文选》

### 4.2 主要版本

| 版本 | 特点 |
|------|------|
| Diels-Kranz, *Die Fragmente der Vorsokratiker* | 前苏格拉底哲学研究的标准文献 |
| Loeb Classical Library (Kirk, Raven, Schofield) | 希腊语-英语对照 |
| Cambridge (Curd) | 最权威的现代英译本和评注 |
| 商务印书馆（聂敏里） | 中文译本 |
