# 爱比克泰德 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 爱比克泰德 |
| **外文名** | Epictetus（Ἐπίκτητος） |
| **生卒年** | 约55年—约135年 |
| **国籍** | 罗马（出生于弗里吉亚，今土耳其境内） |
| **主要身份** | 斯多葛派哲学家、奴隶（早年）、自由人（后获释） |
| **师承/传统** | 斯多葛学派；师从穆索尼乌斯·鲁弗斯 |
| **主要著作** | *Enchiridion*（《手册》，约135年）；*Diatribai*（《论说集》，约108年，由学生阿里安记录） |
| **版本** | W. A. Oldfather, *Epictetus: The Discourses and Manual*, Loeb Classical Library, 1925-1928（标准英译本）；Robert Dobbin, *Epictetus: Discourses Book 1*, Oxford, 1998（学术注释版） |
| **中文译本** | 《爱比克泰德论说集》，商务印书馆；《手册》（《控制手册》），有多种中译本 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 斯多葛实践哲学 | 将斯多葛哲学浓缩为可操作的日常生活指南 |
| 控制二分法 | 区分能控制与不能控制之事，是斯多葛最著名的核心概念 |
| 奴隶哲学家 | 从奴隶成长为自由人，其身份本身就是对"自由"概念的最好诠释 |
| 哲学即生活方式 | 哲学不是理论知识，而是改变生活的实践技艺 |
| 罗马斯多葛 | 与塞涅卡、马可·奥勒留并称罗马斯多葛三大代表人物 |
| 认知行为疗法先驱 | 控制二分法和情感判断理论是现代CBT和ACT的直接前身 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 控制二分法（τὸ ἐφ' ἡμῖν / τὸ οὐκ ἐφ' ἡμῖν） | 区分在我们能力范围之内（判断、欲望、行动）和不在我们能力范围之内（身体、财产、名声）的事物；德性的核心在于将欲望集中在能力范围之内 | 《手册》第一章 |
| 冲动与回避（ὁρμή / ἀφορμή） | 朝向或远离某事物的行动倾向；应在正确判断的基础上引导冲动和回避 | 《手册》第一章 |
| 欲望与厌恶（ὄρεξις / ἔκκλισις） | 对善或看似善之物的欲求，对恶或看似恶之物的逃避；错误的欲望和厌恶导致情感（πάθη） | 《手册》第一章 |
| 情感（πάθη） | 错误的判断导致的扰乱性心理状态；愤怒、恐惧、焦虑等情感源于将欲望放在错误的事物上 | 《论说集》第一卷 |
| 自然（φύσις） | 宇宙理性秩序；人应顺应自然生活，即按照理性生活 | 贯穿全书 |
| 自由（ἐλευθερία） | 通过正确认识控制二分法获得的心灵状态；真正自由的人不被外在事物奴役 | 《论说集》第四卷 |
| 习惯/品质（ἕξις） | 通过反复练习培养的稳定心理状态；知道道理不等于拥有德性，必须练习 | 《论说集》第一卷第一章 |
| 使用（χρῆσις） | 对外在事物的正确利用，而非占有；身体、财产等只是被使用，不是"我们的" | 《手册》第五章 |
| 哲学家的标志（τὸ σημεῖον τοῦ φιλοσόφου） | 不是知道道理，而是按照道理生活；哲学是一种实践技艺，不是理论知识 | 《论说集》第一卷第一章 |
| 顺应命运（ἕπεσθαι τῇ φύσει） | 顺应自然（命运）的生活；不是被动屈服，而是主动接受 | 贯穿全书 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对罗马斯多葛的影响** | 爱比克泰德与塞涅卡、马可·奥勒留构成罗马斯多葛三大支柱；其《手册》是罗马斯多葛最简洁、最实用的文本 |
| **对马可·奥勒留的影响** | 马可·奥勒留的《沉思录》大量引用爱比克泰德，称其为最重要的哲学导师之一 |
| **对早期基督教的影响** | 爱比克泰德的伦理学对基督教修道传统有深刻影响；其"自由"概念与基督教的"灵魂自由"有相似之处 |
| **对文艺复兴的影响** | 其《手册》是文艺复兴时期最受欢迎的哲学文本之一，被翻译成多种欧洲语言 |
| **对蒙田的影响** | 蒙田大量引用爱比克泰德，继承了其"控制二分法"的思想 |
| **对斯宾诺莎的影响** | 斯宾诺莎的"自由是对必然性的认识"直接源于爱比克泰德的控制二分法 |
| **对现代哲学的影响** | 尼采称爱比克泰德为"古代最富有人情味的哲学家"；福柯和阿多关注其"自我技术" |
| **对现代心理学的影响** | 认知行为疗法（CBT）和接纳与承诺疗法（ACT）直接借鉴了爱比克泰德的控制二分法和情感判断理论；美国心理学家阿尔伯特·埃利斯的"理性情绪行为疗法"（REBT）直接源于爱比克泰德 |
| **主要批评** | 其"控制二分法"是否过于简化？是否忽视了人类与外在事物的深刻联系？现代社会心理学研究表明，将自我价值与外在成就完全分离可能导致社会孤立 |
| **学术评价** | 爱比克泰德不是原创性的理论家，而是杰出的实践哲学家；其贡献在于将复杂的斯多葛理论浓缩为简洁、可操作的日常生活指南，对后世的影响甚至超过了更"学术"的希腊斯多葛哲学家 |
