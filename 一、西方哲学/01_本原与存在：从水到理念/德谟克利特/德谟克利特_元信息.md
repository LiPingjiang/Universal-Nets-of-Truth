# 德谟克利特 元信息

## 基本信息
- **原名**：Δημόκριτος（Dēmokritos）
- **生卒年**：约前460—约前370
- **学派**：原子论
- **核心领域**：形而上学、认识论、伦理学、宇宙论

## 关键术语（多语言对照）
| 中文 | 希腊/拉丁原文 | 英文 | 备注 |
|------|-------------|------|------|
| 原子 | ἄτομος | atomos / atom | 不可分割的最小物质单位，字面意为"不可切分" |
| 虚空 | κενόν | kenon / void | 原子运动的场所，非存在但具有实在性 |
| 形状 | ῥυσμός | rhysmos / shape | 原子的形状差异决定事物的性质 |
| 影像 | εἴδωλον | eidōlon / image / effluence | 事物表面流射的影像，进入感官产生知觉 |
| 真正的认识 | γνησίη γνώμη | gnsiē gnōmē / genuine knowledge | 理性对原子和虚空的认识，超越感官的"暗昧认识" |

## 核心论点
- 万物由原子和虚空构成，原子是不可分割、不可变、无限多的微粒，在虚空中运动
- 原子仅有形状、大小、排列的差异，无性质差异；事物的不同源于原子的组合方式
- 不存在目的和神意，一切由原子的必然运动（涡旋）产生
- 感官认识是"暗昧的"（σκοτίη），理性才能把握原子和虚空的真相

## 与本书其他章节的关联
- 第二章"变化与永恒"：德谟克利特的原子论是对赫拉克利特-巴门尼德之争的最终回应——原子不变（存在），虚空中的运动产生变化（流变）
- 第三章"德性与知识"：其伦理学强调灵魂的安宁（εὐθυμία），与希腊化哲学的伦理目标相通
- 第六章"经验与理性"：原子论的认识论二分（暗昧认识与真正认识）预示了近代经验论与唯理论的争论

## 学术定位
- 原子论的集大成者，与留基伯共同创立了原子论哲学
- 其原子论是前苏格拉底自然哲学的顶峰和总结，将所有先前学说整合为一个统一的物质理论
- 原子论在近代科学革命中被重新发现并验证，伽桑狄、波义耳、牛顿均受其影响
- 其伦理学著作丰富，强调"灵魂安宁"（εὐθυμία）为人生目标，被称为"欢笑的哲学家"
- 马克思博士论文题为《德谟克利特自然哲学与伊壁鸠鲁自然哲学的差别》，可见其在哲学史中的关键地位
- 虚空概念的引入解决了巴门尼德"非存在不存在"的难题——虚空是"非存在"但具有实在性，为运动提供了可能性
