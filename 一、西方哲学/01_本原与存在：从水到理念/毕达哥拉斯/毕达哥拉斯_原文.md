# 毕达哥拉斯 · 原文

## 一、生平与著作

毕达哥拉斯（Πυθαγόρας，Pythagoras，约公元前570-495年）出生于小亚细亚的萨摩斯岛（Samos），后来移居意大利南部的克罗顿（Croton），在那里创立了**毕达哥拉斯学派**（Pythagorean School）。毕达哥拉斯学派是一个**宗教-哲学团体**——成员必须遵守严格的戒律，包括禁豆、不捡掉在地上的东西、不触碰白公鸡等。

毕达哥拉斯没有留下著作，他的思想主要通过**后世哲学家的引用**流传下来。主要来源包括：
- **亚里士多德**的《形而上学》
- **柏拉图**的对话录
- **第欧根尼·拉尔修**的《名哲言行录》
- **普罗克洛**的评注

## 二、核心文本

### 2.1 "数是万物的本原"（ἀριθμὸς ἀρχή）

**核心命题**：

> **"数是万物的本原"（τὸν ἀριθμὸν εἶναι τὴν ἀρχὴν τῶν πάντων）。**

**论证过程**：

**（1）数学的发现**
- 毕达哥拉斯发现了**音乐与数学的关系**：弦的长度比例决定音高——如2:1是八度，3:2是五度，4:3是四度
- 毕达哥拉斯发现了**几何与数学的关系**：直角三角形的边长关系（a² + b² = c²）
- 毕达哥拉斯发现了**天文与数学的关系**：天体运行遵循数学规律

**（2）数是万物的本原**
- 如果音乐、几何、天文都可以用数学来解释，那么**万物都可以用数学来解释**
- 数是**永恒的**、**不变的**、**完美的**——不像具体事物那样变化
- 数是**可理解的**——通过理性可以认识数
- 因此，数是万物的本原

**（3）数的分类**
- **奇数**（ἄρτιος）：代表有限、男性、光明、热
- **偶数**（περιττός）：代表无限、女性、黑暗、冷
- **完全数**（τέλειος）：等于其因数之和——如6 = 1+2+3
- **亲和数**（φίλοι）：两个数的因数之和等于对方——如220和284

### 2.2 "和谐"（ἁρμονία）

**核心命题**：

> **"和谐是宇宙的原则"（ἁρμονία τῶν πάντων）。**

**解释**：
- 毕达哥拉斯认为：宇宙是一个**和谐的整体**——万物按照数学比例排列
- 音乐是**和谐的**——因为音高遵循数学比例
- 天体是**和谐的**——因为天体运行遵循数学规律（"天体的音乐"）
- 灵魂是**和谐的**——因为灵魂是身体的数学比例

**哲学意义**：
- 毕达哥拉斯的"和谐"是西方哲学史上**第一次用数学来解释宇宙的秩序**
- 这一思想影响了柏拉图的理念论——理念是永恒的、完美的，类似于数
- 这一思想也影响了亚里士多德的目的论——宇宙是有目的的、和谐的

### 2.3 "灵魂轮回"（μετεμψύχωσις）

**核心命题**：

> **"灵魂是永恒的，通过轮回转世"（ψυχὴ ἀθάνατος, μετεμψυχούμενη）。**

**解释**：
- 毕达哥拉斯认为：灵魂是**永恒的**——不随身体死亡而消失
- 灵魂通过**轮回**（μετεμψύχωσις）转世——从一个人转到另一个人，或从人转到动物
- 灵魂的目的是**净化**（κάθαρσις）——通过哲学和宗教实践，灵魂可以摆脱轮回，达到永恒的幸福

**哲学意义**：
- 毕达哥拉斯的"灵魂轮回"是西方哲学史上**第一次明确表达灵魂不朽和轮回**
- 这一思想影响了柏拉图的"回忆说"——灵魂在前世已经认识了理念
- 这一思想也影响了基督教的"灵魂不朽"——但基督教反对轮回

### 2.4 "十对对立"（δέκα συστοιχίαι）

**核心命题**：

> **"万物由十对对立面构成"（δέκα συστοιχίαι τῶν ἐναντίων）。**

**十对对立面**：

| 有限 | 无限 |
|------|------|
| 一 | 多 |
| 右 | 左 |
| 男 | 女 |
| 静 | 动 |
| 直 | 曲 |
| 光明 | 黑暗 |
| 善 | 恶 |
| 正方 | 长方 |

**哲学意义**：
- 毕达哥拉斯的"十对对立"是西方哲学史上**第一次系统分类对立面**
- 这一思想影响了赫拉克利特的"对立统一"——但赫拉克利特强调对立面的统一，而毕达哥拉斯强调对立面的分离
- 这一思想也影响了柏拉图的"辩证法"——通过分析对立面来认识真理
