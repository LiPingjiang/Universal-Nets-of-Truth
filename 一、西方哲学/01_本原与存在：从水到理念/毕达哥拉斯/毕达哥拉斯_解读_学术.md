# 毕达哥拉斯 · 解读（学术版）

## 一、思想史定位

### 1.1 西方数学哲学的奠基人

毕达哥拉斯（约公元前570-495年）是西方哲学史上**第一位数学哲学家**。他发现了**音乐与数学的关系**（弦的长度比例决定音高）、**几何与数学的关系**（直角三角形的边长关系）、**天文与数学的关系**（天体运行遵循数学规律）。他的名言"万物皆数"（πάντα ἀριθμός）成为西方数学哲学的座右铭。

### 1.2 宗教-哲学团体的创始人

毕达哥拉斯在意大利南部的克罗顿创立了**毕达哥拉斯学派**——一个宗教-哲学团体。成员必须遵守严格的戒律，包括禁豆、不捡掉在地上的东西、不触碰白公鸡等。学派的核心教义包括：
- **灵魂轮回**（μετεμψύχωσις）：灵魂是永恒的，通过轮回转世
- **净化**（κάθαρσις）：通过哲学和宗教实践，灵魂可以摆脱轮回
- **和谐**（ἁρμονία）：宇宙是一个和谐的整体

### 1.3 对后世哲学的影响

毕达哥拉斯的影响贯穿整个西方哲学史：
- **柏拉图**：理念论直接受毕达哥拉斯影响——理念是永恒的、完美的，类似于数
- **亚里士多德**：批判了毕达哥拉斯的"数是万物的本原"，但继承了"和谐"的概念
- **新柏拉图主义**：普罗提诺将毕达哥拉斯的"一"（τὸ ἕν）发展为"太一"（τὸ ἕν）
- **近代科学**：哥白尼、开普勒、伽利略都受毕达哥拉斯影响——宇宙可以用数学来描述
- **现代物理学**：量子力学中的"对称性"概念直接来自毕达哥拉斯的"和谐"

## 二、核心命题分析

### 2.1 "数是万物的本原"

**核心命题**：数是万物的本原（τὸν ἀριθμὸν εἶναι τὴν ἀρχὴν τῶν πάντων）。

**哲学问题**：

**（1）为什么是数？**
- 毕达哥拉斯发现了**音乐与数学的关系**：弦的长度比例决定音高
- 毕达哥拉斯发现了**几何与数学的关系**：直角三角形的边长关系（a² + b² = c²）
- 毕达哥拉斯发现了**天文与数学的关系**：天体运行遵循数学规律
- 因此，如果音乐、几何、天文都可以用数学来解释，那么**万物都可以用数学来解释**

**（2）数的特征**
- 数是**永恒的**——不像具体事物那样变化
- 数是**不变的**——2永远是2，不会因为时间或地点而改变
- 数是**完美的**——完全数、亲和数等体现了数的完美性
- 数是**可理解的**——通过理性可以认识数

**（3）数的分类**
- **奇数**（ἄρτιος）：代表有限、男性、光明、热
- **偶数**（περιττός）：代表无限、女性、黑暗、冷
- **完全数**（τέλειος）：等于其因数之和——如6 = 1+2+3
- **亲和数**（φίλοι）：两个数的因数之和等于对方——如220和284

**学术争议**：
- 毕达哥拉斯的"数是万物的本原"是**字面意义的**，还是**象征意义的**？
  - **字面意义说**：毕达哥拉斯确实认为数是物理的本原——这是科学假说
  - **象征意义说**：毕达哥拉斯用数来象征**宇宙的秩序**——这是哲学隐喻
  - 大多数学者倾向于**字面意义说**——毕达哥拉斯是数学家和科学家，不是诗人
- 数是**物质的**，还是**非物质的**？
  - **物质说**：数是物质的一种形式——如点是"位置的单位"
  - **非物质说**：数是超越物质的——类似于柏拉图的理念
  - 大多数学者倾向于**非物质说**——毕达哥拉斯的数是抽象的、永恒的

### 2.2 "和谐"（ἁρμονία）

**核心命题**：和谐是宇宙的原则（ἁρμονία τῶν πάντων）。

**解释**：
- 毕达哥拉斯认为：宇宙是一个**和谐的整体**——万物按照数学比例排列
- 音乐是**和谐的**——因为音高遵循数学比例
- 天体是**和谐的**——因为天体运行遵循数学规律（"天体的音乐"）
- 灵魂是**和谐的**——因为灵魂是身体的数学比例

**哲学意义**：
- 毕达哥拉斯的"和谐"是西方哲学史上**第一次用数学来解释宇宙的秩序**
- 这一思想影响了柏拉图的理念论——理念是永恒的、完美的，类似于数
- 这一思想也影响了亚里士多德的目的论——宇宙是有目的的、和谐的

**学术争议**：
- "和谐"是**客观的**（宇宙本身有和谐），还是**主观的**（人类心灵投射的和谐）？
  - **客观说**：和谐是宇宙的客观属性——独立于人类意识
  - **主观说**：和谐是人类心灵的构造——是我们认识世界的方式
  - 大多数学者倾向于**客观说**——毕达哥拉斯认为和谐是宇宙的客观属性

### 2.3 "灵魂轮回"（μετεμψύχωσις）

**核心命题**：灵魂是永恒的，通过轮回转世（ψυχὴ ἀθάνατος, μετεμψυχούμενη）。

**解释**：
- 毕达哥拉斯认为：灵魂是**永恒的**——不随身体死亡而消失
- 灵魂通过**轮回**（μετεμψύχωσις）转世——从一个人转到另一个人，或从人转到动物
- 灵魂的目的是**净化**（κάθαρσις）——通过哲学和宗教实践，灵魂可以摆脱轮回，达到永恒的幸福

**哲学意义**：
- 毕达哥拉斯的"灵魂轮回"是西方哲学史上**第一次明确表达灵魂不朽和轮回**
- 这一思想影响了柏拉图的"回忆说"——灵魂在前世已经认识了理念
- 这一思想也影响了基督教的"灵魂不朽"——但基督教反对轮回

**学术争议**：
- 灵魂轮回是**字面意义的**，还是**比喻意义的**？
  - **字面意义说**：毕达哥拉斯确实相信灵魂轮回——这是宗教教义
  - **比喻意义说**：灵魂轮回是比喻——指灵魂的"转变"或"升华"
  - 大多数学者倾向于**字面意义说**——毕达哥拉斯学派是一个宗教团体，相信轮回

### 2.4 "十对对立"（δέκα συσστοιχίαι）

**核心命题**：万物由十对对立面构成（δέκα συσστοιχίαι τῶν ἐναντίων）。

**十对对立面**：

| 有限 | 无限 |
|------|------|
| 一 | 多 |
| 右 | 左 |
| 男 | 女 |
| 静 | 动 |
| 直 | 曲 |
| 光明 | 黑暗 |
| 善 | 恶 |
| 正方 | 长方 |

**哲学意义**：
- 毕达哥拉斯的"十对对立"是西方哲学史上**第一次系统分类对立面**
- 这一思想影响了赫拉克利特的"对立统一"——但赫拉克利特强调对立面的统一，而毕达哥拉斯强调对立面的分离
- 这一思想也影响了柏拉图的"辩证法"——通过分析对立面来认识真理

## 三、毕达哥拉斯与后世哲学

### 3.1 对柏拉图的影响

- 柏拉图的理念论直接受毕达哥拉斯影响——理念是永恒的、完美的，类似于数
- 柏拉图在《斐多篇》中让苏格拉底说："哲学家是追求死亡的人"——直接来自毕达哥拉斯的"净化"概念
- 柏拉图的"回忆说"也来自毕达哥拉斯的"灵魂轮回"——灵魂在前世已经认识了理念

### 3.2 对亚里士多德的影响

- 亚里士多德批判了毕达哥拉斯的"数是万物的本原"——数是抽象的，不能是物理的本原
- 但亚里士多德继承了毕达哥拉斯的"和谐"概念——宇宙是有目的的、和谐的
- 亚里士多德也继承了毕达哥拉斯的"对立"概念——潜能与现实就是对立统一的

### 3.3 对近代科学的影响

- **哥白尼**：日心说受毕达哥拉斯影响——天体运行遵循数学规律
- **开普勒**：行星运动三定律受毕达哥拉斯影响——行星轨道是椭圆的，但遵循数学规律
- **伽利略**："自然之书是用数学语言写成的"——直接来自毕达哥拉斯
- **牛顿**：万有引力定律受毕达哥拉斯影响——宇宙可以用数学方程来描述

### 3.4 对现代物理学的影响

- **量子力学**："对称性"概念直接来自毕达哥拉斯的"和谐"
- **弦理论**：宇宙的基本构成是"弦"——弦的振动产生不同的粒子——类似于毕达哥拉斯的"弦的比例产生不同的音高"
- **宇宙学**：宇宙的"微调"（fine-tuning）问题——宇宙的基本常数恰好适合生命存在——类似于毕达哥拉斯的"和谐"

## 四、文献考证

### 4.1 著作的保存

毕达哥拉斯没有留下著作，他的思想主要通过**后世哲学家的引用**流传下来：
- **亚里士多德**的《形而上学》
- **柏拉图**的对话录
- **第欧根尼·拉尔修**的《名哲言行录》
- **普罗克洛**的评注

### 4.2 主要版本

| 版本 | 特点 |
|------|------|
| Diels-Kranz, *Die Fragmente der Vorsokratiker* | 前苏格拉底哲学研究的标准文献 |
| Loeb Classical Library (Kirk, Raven, Schofield) | 希腊语-英语对照 |
| Cambridge (Kahn) | 最权威的现代英译本和评注 |
| 商务印书馆（聂敏里） | 中文译本 |
