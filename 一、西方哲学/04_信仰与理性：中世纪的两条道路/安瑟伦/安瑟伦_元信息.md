## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 经院哲学（Scholasticism）之父；奥古斯丁主义传统 |
| 核心方法 | 信仰寻求理解（fides quaerens intellectum）；理性论证神学真理 |
| 核心命题 | 本体论论证——"不能设想比它更大的东西"必然存在；信仰是理解的前提 |
| 神学立场 | 天主教正统；坎特伯雷大主教；反对罗塞林的三位一体论 |
| 认识论立场 | 理性主义神学——理性可以独立证明上帝存在，但需以信仰为前提 |
| 形而上学立场 | 极端实在论（Realism）——共相是真实存在的实体 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 直接传承 | 经院哲学传统的奠基者；影响了12-13世纪经院哲学家 |
| 哲学传承 | 奥古斯丁 → 安瑟伦 → 阿伯拉尔 → 阿奎那 → 司各脱 → 奥卡姆 |
| 神学影响 | 本体论论证成为基督教神学论证上帝存在的经典方式之一 |
| 哲学史影响 | 本体论论证引发了哲学史上关于"存在是否谓词"的最持久争论 |
| 批评者 | 高尼罗（Gaunilo）以"迷失岛"反驳；康德在《纯粹理性批判》中系统批评 |
| 支持者 | 笛卡尔、莱布尼茨、黑格尔、马尔科姆（Norman Malcolm）等从不同角度辩护 |
| 现代影响 | 本体论论证至今仍是分析哲学中上帝存在论证的核心议题之一 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 本体论论证的有效性 | 支持者：从概念分析推出存在是有效的；批评者：存在不是谓词，不能从概念推出存在 |
| 高尼罗的"迷失岛"反驳 | 高尼罗：如果安瑟伦的论证有效，那么"最完美的迷失岛"也必然存在——荒谬。安瑟伦回应：只有上帝是"不能设想比它更大的"，岛屿不适用 |
| 康德对本体论论证的批评 | 康德："存在"不是实在的谓词，添加"存在"不改变概念的内容；一百元实在的钱与一百元概念的钱概念相同 |
| 信仰与理性的关系 | 安瑟伦：信仰是理解的前提，但理性可以独立证明信仰的内容。批评者：这是否意味着信仰是不必要的？ |
| 安瑟伦是否"证明了"上帝 | 安瑟伦自称是为信仰寻求理解，但论证结构似乎试图从理性独立推出上帝存在——这是否矛盾？ |
