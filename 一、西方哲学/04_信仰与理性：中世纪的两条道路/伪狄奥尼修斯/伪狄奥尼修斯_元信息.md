# 伪狄奥尼修斯 · 元信息

## 基本信息

| 字段 | 内容 |
|------|------|
| 姓名（中文） | 伪狄奥尼修斯 |
| 姓名（希腊语） | Διονύσιος ὁ Ἀρεοπαγίτης |
| 生卒年 | 5-6世纪（具体年代不详） |
| 身份 | 匿名新柏拉图主义哲学家；伪托保罗在雅典的皈依者狄奥尼修斯 |
| 真实身份 | 不详；可能是叙利亚地区的一位修士或学者 |
| 师承 | 新柏拉图主义（普罗提诺、普罗克洛传统） |
| 主要影响 | 埃克哈特大师、托马斯·阿奎那、中世纪神秘主义传统 |
| 哲学传承 | 新柏拉图主义 → 伪狄奥尼修斯 → 基督教神秘主义 |

## 著作信息

| 著作 | 原文 | 性质 |
|------|------|------|
| 《神的名称》 | De Divinis Nominibus | 通过肯定之路言说上帝：上帝是善、是智慧、是爱 |
| 《神秘神学》 | De Mystica Theologia | 通过否定之路和超越之路言说上帝 |
| 《天阶序论》 | De Caelesti Hierarchia | 天使等级体系；宇宙作为神圣秩序的反映 |
| 《教阶序论》 | De Ecclesiastica Hierarchia | 教会等级体系与圣礼的神学 |
| 《书信集》 | Epistulae | 十封书信，包含神秘主义核心教导 |

## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 新柏拉图主义神秘主义；基督教否定神学 |
| 核心方法 | 否定神学（via negativa）；超越神学（via eminentiae）；肯定神学（via affirmativa） |
| 核心命题 | 上帝超越一切存在和一切被思想者；上帝不是任何存在者，而是超越一切存在者 |
| 认识论立场 | 对上帝的认识是"不知之知"（unknowing knowing）；超越理性与语言的界限 |
| 神学立场 | 上帝是"超存在"（hyperousios）；一切名称都是类比性的 |
| 宇宙论立场 | 宇宙是神圣光明的流溢；存在等级从上帝逐级下降 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 中世纪神秘主义 | 深刻影响埃克哈特大师、圣十字若望、圣女大德兰等神秘主义者 |
| 经院哲学 | 阿奎那在《神学大全》中大量引用伪狄奥尼修斯，尤其是关于上帝名称和天使等级的论述 |
| 东正教传统 | 东正教神秘主义（静修传统）深受伪狄奥尼修斯影响 |
| 近代哲学 | 康德的"本体"（noumenon）概念与否定神学有深刻关联；海德格尔对"存在"与"存在者"的区分 |
| 当代神学 | 后现代神学（如约翰·卡普托的"弱神学"）重新发掘否定神学的资源 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 真实身份 | 伪狄奥尼修斯的真实身份至今不明；学者推测可能是叙利亚修士，活跃于5世纪末至6世纪初 |
| 伪托的动机 | 为什么他要伪托保罗的皈依者？可能是为了赋予自己的著作权威性；也可能是某种"文学虚构" |
| 新柏拉图主义与基督教的关系 | 伪狄奥尼修斯是否只是将普罗克洛的新柏拉图主义基督教化？还是创造了一种全新的基督教神学？ |
| 否定神学的极限 | 否定神学是否最终导向不可知论或虚无主义？如果上帝超越一切言说，我们如何确定他存在？ |

## 关键年表

| 年份 | 事件 |
|------|------|
| 5世纪末-6世纪初 | 伪狄奥尼修斯著作的成书时期（推测） |
| 约 532 CE | 这些著作首次被提及；东罗马皇帝查士丁尼关闭雅典学园后，新柏拉图主义思想通过叙利亚渠道传入 |
| 9世纪 | 埃里金纳（John Scotus Eriugena）将伪狄奥尼修斯著作翻译成拉丁语，传入西方 |
| 13世纪 | 阿奎那在《神学大全》中大量引用伪狄奥尼修斯 |
| 14世纪 | 埃克哈特大师发展伪狄奥尼修斯的否定神学，提出"神性之深渊"（Gottheit）概念 |
