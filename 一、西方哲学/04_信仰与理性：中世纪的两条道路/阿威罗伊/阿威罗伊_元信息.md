## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 伊斯兰逍遥学派（Islamic Peripateticism）；亚里士多德主义正统；"注释者"（the Commentator） |
| 核心方法 | 亚里士多德注释学；理性与启示的调和；隐喻解释（taʾwīl） |
| 核心命题 | 真理不与真理矛盾；统一理智论（能动的理智是唯一的）；哲学与神学是同一真理的不同表达 |
| 神学立场 | 伊斯兰逊尼派；试图调和亚里士多德哲学与伊斯兰神学；受到保守派神学家批评，晚年被流放 |
| 认识论立场 | 理性主义——理性可以独立把握真理；启示是"大众的哲学"，哲学是"精英的神学" |
| 形而上学立场 | 亚里士多德正统——实体是首要的；共相存在于事物之中（温和实在论） |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 直接传承 | 伊斯兰逍遥学派的集大成者；影响了犹太哲学（迈蒙尼德）和拉丁经院哲学 |
| 拉丁西方 | 12-13世纪通过西班牙翻译运动传入；被称为"注释者"（the Commentator） |
| 阿威罗伊主义 | 13世纪巴黎大学形成"拉丁阿威罗伊主义"（Latin Averroism）——以布拉邦特的西格尔（Siger of Brabant）为代表 |
| 教会谴责 | 1270年和1277年，巴黎主教唐皮耶（Étienne Tempier）两次谴责阿威罗伊主义命题 |
| 对阿奎那的影响 | 阿奎那直接回应阿威罗伊的统一理智论，捍卫个体灵魂不朽 |
| 对文艺复兴的影响 | 阿威罗伊的亚里士多德注释成为文艺复兴时期亚里士多德研究的重要来源 |
| 对现代的影响 | 双重真理论预示了近代"信仰与理性分离"的思想 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 双重真理论的有效性 | 支持者：理性与启示不可能矛盾，因为它们来自同一真理来源；批评者：如果理性与启示矛盾，理性应该服从启示 |
| 统一理智论 | 支持者：能动的理智是唯一的，个体灵魂只是分有它；批评者：如果理智是统一的，那么个人灵魂不朽如何可能？ |
| 哲学与神学的关系 | 阿威罗伊：哲学是"精英的神学"，神学是"大众的哲学"。批评者：这贬低了启示的权威 |
| 阿威罗伊与伊斯兰神学的关系 | 支持者：阿威罗伊成功调和了哲学与神学；批评者：他的哲学与伊斯兰正统神学根本冲突（如世界永恒性、个体灵魂不朽） |
| 1277年谴责 | 支持者：教会的谴责是必要的，因为阿威罗伊主义威胁了基督教正统；批评者：谴责阻碍了理性与信仰的开放对话 |

## 关键年表

| 年份 | 事件 |
|------|------|
| 1126 | 生于科尔多瓦，后倭马亚王朝 |
| 约1140s | 学习伊斯兰法学、医学、哲学 |
| 约1153 | 成为塞维利亚（Seville）法官 |
| 1169 | 经伊本·图费勒推荐，进入穆瓦希德王朝宫廷，成为宫廷医生和哲学家 |
| 约1170s | 写作《决定论》和《检验》，回应安萨里对哲学家的批评 |
| 约1180s | 写作大量亚里士多德注释 |
| 1195 | 因保守派神学家的压力，被穆瓦希德王朝苏丹流放至卢塞纳（Lucena） |
| 约1198 | 被召回宫廷，但不久逝世于马拉喀什 |
| 1198 | 逝世 |
