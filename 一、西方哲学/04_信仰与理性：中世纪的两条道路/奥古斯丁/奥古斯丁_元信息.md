# 奥古斯丁 · 元信息

## 基本信息

| 字段 | 内容 |
|------|------|
| 姓名（中文） | 奥古斯丁 |
| 姓名（拉丁语） | Aurelius Augustinus |
| 生卒年 | 354 CE — 430 CE |
| 出生地 | 塔加斯特（Thagaste），今阿尔及利亚苏格艾赫拉斯 |
| 死亡地 | 希波（Hippo Regius），今阿尔及利亚安纳巴 |
| 身份 | 北非希波主教；教父哲学家 |
| 父亲 | 帕特里修斯（Patricius），非基督徒，后受洗 |
| 母亲 | 莫尼卡（Monica），虔诚基督徒 |
| 师承 | 安布罗斯（Ambrose of Milan）；西塞罗（阅读）；普罗提诺、波菲利（通过拉丁译本间接） |
| 主要学生 | 影响后世整个拉丁西方基督教思想传统 |
| 哲学传承 | 柏拉图主义 → 奥古斯丁 → 中世纪经院哲学、宗教改革、现代哲学 |

## 著作信息

| 著作 | 原文 | 成书时间 | 性质 |
|------|------|----------|------|
| 《忏悔录》 | Confessiones | 397-400 CE | 自传体神学哲学著作，西方自传文学的开山之作 |
| 《上帝之城》 | De Civitate Dei | 413-426 CE | 历史哲学与政治神学巨著，回应罗马陷落 |
| 《论三位一体》 | De Trinitate | 399-419 CE | 基督教三一论的哲学奠基 |
| 《论自由意志》 | De Libero Arbitrio | 388-395 CE | 自由意志与恶的问题 |
| 《论教师》 | De Magistro | 389 CE | 教育哲学；基督为内在之教师 |
| 《论真宗教》 | De Vera Religione | 390 CE | 基督教与柏拉图主义的综合 |

## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 教父哲学（Patristic Philosophy）；新柏拉图主义基督教化；拉丁西方神学传统 |
| 核心方法 | 内在转向（conversio ad interiora）；信仰寻求理解（fides quaerens intellectum） |
| 核心命题 | 信仰以便理解（crede, ut intelligas）；恶是善的缺乏（malum est privatio boni）；时间存在于灵魂中 |
| 认识论立场 | 光照论（divine illumination）：真理的最终来源是上帝之光 |
| 伦理学立场 | 爱（caritas）与欲（cupiditas）的区分；上帝之城 vs. 地上之城 |
| 政治立场 | 双城论：教权高于王权；和平是最终目的 |
| 神学立场 | 原罪论；预定论（后被加尔文极端化）；恩典论 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 直接传承 | 波爱修斯、伪狄奥尼修斯、安瑟伦、托马斯·阿奎那 |
| 宗教改革 | 马丁·路德深受奥古斯丁恩典论影响；加尔文继承其预定论 |
| 近代哲学 | 笛卡尔"我思"与奥古斯丁"内在转向"的深刻关联；笛卡尔在《第一哲学沉思集》中直接引用奥古斯丁 |
| 时间哲学 | 柏格森《时间与自由意志》、胡塞尔内时间意识现象学、海德格尔《存在与时间》均受奥古斯丁影响 |
| 政治哲学 | 双城论影响中世纪政教关系；近代政教分离思想亦是对奥古斯丁的回应 |
| 文学 | 《忏悔录》开创西方自传文学传统；卢梭《忏悔录》直接受其影响 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 奥古斯丁与柏拉图主义的关系 | 吉尔松（Gilson）认为奥古斯丁根本上超越了柏拉图主义；某些学者认为奥古斯丁只是基督教的柏拉图 |
| 恶的问题的解释力 | "恶是善的缺乏"是否足以解释痛苦的实在性？自由意志辩护 vs. 神义论问题 |
| 预定论与人文主义 | 奥古斯丁的预定论是否与人的自由和责任相容？ |
| 奥古斯丁与笛卡尔 | 笛卡尔的"我思"是否只是奥古斯丁"内在转向"的世俗化版本？ |
| 政治双城论的解读 | 双城论是支持教权至上，还是为近代政教分离提供思想资源？ |

## 关键年表

| 年份 | 事件 |
|------|------|
| 354 CE | 生于北非塔加斯特 |
| 370 CE | 赴迦太基求学，学习修辞学；与一女子同居并生子（阿德奥达图斯） |
| 373 CE | 阅读西塞罗《霍滕修斯》，立志追求智慧；加入摩尼教 |
| 383 CE | 赴罗马，后至米兰任教修辞学 |
| 384 CE | 聆听安布罗斯讲道，开始转向基督教 |
| 386 CE | "米兰花园"皈依经历（《忏悔录》第八卷）；受洗 |
| 388 CE | 返回非洲，建立修道团体 |
| 391 CE | 被按立为神父 |
| 395 CE | 被祝圣为希波主教 |
| 397-400 CE | 写作《忏悔录》 |
| 410 CE | 罗马被西哥特人攻陷；开始写作《上帝之城》 |
| 413-426 CE | 完成《上帝之城》 |
| 430 CE | 汪达尔人围攻希波时去世 |
