## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 经院哲学的最高峰；托马斯主义（Thomism）；综合亚里士多德主义与基督教神学 |
| 核心方法 | 综合（synthesis）：亚里士多德哲学与基督教神学的系统综合；五路证明（quinque viae） |
| 核心命题 | 存在是本质的现实化（esse est actus essentiae）；理性是信仰的仆人（ancilla fidei）；自然法是永恒法在理性受造物中的分有 |
| 神学立场 | 天主教正统；多明我会修士；"天使博士"（Doctor Angelicus）；1323年封圣；1879年利奥十三世确立为天主教官方哲学 |
| 认识论立场 | 温和实在论；感官经验是认识的起点，理智是认识的完成；"理智中没有什么不是先在感官中的"（nihil in intellectu quod non prius in sensu） |
| 形而上学立场 | 存在论（existentialist）形而上学——存在先于本质；上帝是纯粹的存在（ipsum esse subsistens）；被造物是存在与本质的复合 |
| 伦理学立场 | 德性伦理学；自然法理论；人的终极目的是认识上帝（beatitudo / 至福） |
| 政治哲学立场 | 自然法理论；不公正的法律不是法律（lex iniusta non est lex）；政教关系的温和立场 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 直接传承 | 经院哲学的最高峰；影响了司各脱、奥卡姆、苏亚雷斯等 |
| 托马斯主义 | 1323年封圣；1879年利奥十三世《永恒之父》通谕确立为天主教官方哲学；新托马斯主义（Neo-Thomism）在19-20世纪复兴 |
| 哲学史影响 | 经院哲学综合的典范；亚里士多德哲学与基督教神学综合的最高成就 |
| 对后世哲学的影响 | 影响了笛卡尔（我思与上帝存在的论证）、斯宾诺莎（实体学说）、莱布尼茨（单子论与充足理由律）、康德（对本体论论证的批评） |
| 对科学的影响 | 阿奎那的自然哲学为近代科学革命提供了概念框架（如因果性、潜能与现实） |
| 对政治哲学的影响 | 自然法理论影响了格劳秀斯、洛克、孟德斯鸠、美国独立宣言 |
| 对伦理学的影响 | 德性伦理学复兴（安斯康姆、麦金泰尔）以阿奎那为重要资源 |
| 现代影响 | 新托马斯主义在20世纪复兴（吉尔松、马利坦、拉纳）；分析托马斯主义（如约翰·菲尼斯） |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 五路证明的有效性 | 支持者：五路证明展示了理性如何走向信仰；批评者：五路依赖于因果链条的无限倒退不可能，但这一点需要证明 |
| 存在-本质理论的修正 | 支持者：阿奎那的修正（存在是本质的现实化）比阿维森纳更深刻；批评者：修正是否成功？存在与本质是否仍然可以区分？ |
| 自然法理论 | 支持者：自然法理论是政治哲学的基石；批评者：自然法是否只是特定文化的产物？是否存在"普遍"的自然法？ |
| 信仰与理性的关系 | 阿奎那：理性是信仰的仆人（ancilla fidei），但有其正当功能。批评者：这是否贬低了理性的独立性？ |
| 阿奎那与亚里士多德的关系 | 支持者：阿奎那成功综合了亚里士多德与基督教；批评者：阿奎那是否"扭曲"了亚里士多德来适应基督教？ |
| 阿奎那的封圣与官方地位 | 支持者：阿奎那的哲学与天主教教义完全一致；批评者：官方地位是否阻碍了哲学创新？ |

## 关键年表

| 年份 | 事件 |
|------|------|
| 约1225 | 生于意大利南部罗卡塞卡，那不勒斯王国贵族家庭 |
| 1230 | 被送往蒙特卡西诺（Monte Cassino）本笃会修道院学习 |
| 1239 | 因政治动荡离开蒙特卡西诺，进入那不勒斯大学 |
| 1244 | 加入多明我会（Order of Preachers），违背家族意愿 |
| 1245-1248 | 被派往巴黎，师从大阿尔伯特 |
| 1248-1252 | 随大阿尔伯特前往科隆，继续学习 |
| 1252 | 返回巴黎，开始神学学习 |
| 1256 | 成为巴黎大学神学硕士（magister）；开始写作《论存在与本质》 |
| 1259-1265 | 在意大利各地教学；写作《反异教大全》 |
| 1265-1268 | 返回巴黎，开始写作《神学大全》 |
| 1268-1272 | 再次在意大利教学；继续写作《神学大全》 |
| 1270 | 写作《论统一理智》，批评阿威罗伊主义 |
| 1272 | 返回那不勒斯，继续教学 |
| 1273年12月 | 在弥撒中经历神秘体验，此后停止写作，称"我所写的与我所见的相比，不过是稻草" |
| 1274年3月7日 | 在前往里昂公会议的途中病逝于福萨诺瓦修道院 |
| 1323 | 被教皇约翰二十二世封圣 |
| 1567 | 被教皇庇护五世授予"教会博士"（Doctor of the Church）称号 |
| 1879 | 教皇利奥十三世发布《永恒之父》（Aeterni Patris）通谕，确立阿奎那哲学为天主教官方哲学 |
