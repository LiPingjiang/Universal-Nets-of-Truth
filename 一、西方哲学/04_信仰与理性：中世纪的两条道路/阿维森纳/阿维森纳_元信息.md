## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 伊斯兰逍遥学派（Islamic Peripateticism）；亚里士多德主义传统；新柏拉图主义影响 |
| 核心方法 | 存在-本质区分（al-farq bayna l-māhiyya wa l-wujūd）；思想实验（漂浮人）；逻辑分析 |
| 核心命题 | 本质与存在是分离的；必然存在者（wājib al-wujūd）是唯一无需原因的存在；灵魂可以独立于身体被认识 |
| 神学立场 | 伊斯兰逊尼派；试图调和亚里士多德哲学与伊斯兰神学；受到保守派神学家批评 |
| 认识论立场 | 经验主义与理性主义结合；感官经验是认识的起点，但理性是认识的完成；自我意识的直接确定性 |
| 形而上学立场 | 存在论（existentialist）形而上学——存在先于本质；被造物需要原因，上帝不需要 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 直接传承 | 伊斯兰逍遥学派的高峰；影响了阿威罗伊、苏赫拉瓦迪（Suhrawardi）、伊本·图费勒（Ibn Tufayl） |
| 拉丁西方 | 12世纪通过西班牙翻译运动传入拉丁西方；深刻影响了经院哲学（尤其是阿奎那） |
| 医学影响 | 《医典》成为欧洲医学院标准教材，使用至17世纪；影响远超哲学著作 |
| 形而上学 | 存在-本质区分直接影响了阿奎那的形而上学；阿奎那称其为"哲学家"（the Philosopher） |
| 认识论 | "漂浮人"思想实验预示了笛卡尔的"我思"；自我意识作为认识论起点的先驱 |
| 科学方法论 | 对亚里士多德逻辑学的系统化整理影响了中世纪科学方法论 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 存在-本质区分的有效性 | 支持者：区分是形而上学的重大进步；批评者：存在与本质不可分离，阿维森纳的区分是概念上的而非实在的 |
| 必然存在者的论证 | 支持者：从存在-本质区分推出必然存在者是有效的；批评者：论证预设了"存在需要解释"，而这本身需要证明 |
| "漂浮人"与笛卡尔"我思"的关系 | 支持者：阿维森纳是笛卡尔的先驱；批评者：两者有本质差异——阿维森纳不怀疑感官世界，笛卡尔则系统怀疑 |
| 阿维森纳与伊斯兰神学的关系 | 支持者：阿维森纳成功调和了哲学与神学；批评者：他的哲学与伊斯兰正统神学有根本冲突（如世界永恒性、个体灵魂不朽） |
| 《医典》的科学价值 | 支持者：中世纪最先进的医学知识汇编；批评者：包含大量盖伦医学的错误理论，阻碍了现代医学发展 |

## 关键年表

| 年份 | 事件 |
|------|------|
| 980 | 生于布哈拉，萨曼王朝 |
| 约995 | 已熟练掌握伊斯兰法、逻辑学、医学、形而上学 |
| 约1000 | 父亲去世；开始为萨曼王朝宫廷服务 |
| 约1005 | 萨曼王朝灭亡，阿维森纳开始流亡生涯 |
| 约1010 | 在花剌子模宫廷服务，与比鲁尼（al-Biruni）进行学术通信 |
| 约1015 | 在伽色尼王朝服务；写作《治疗论》 |
| 约1020-1030 | 在哈马丹服务；完成《治疗论》和《医典》 |
| 约1030 | 写作《指要》，总结成熟思想 |
| 1037 | 逝世于哈马丹，因过度劳累和疾病 |
