# 第六章 经验与理性：近代认识论的两条路线

## 综述

**核心张力**：经验（experientia）与理性（ratio）——知识来源于感官经验，还是来源于理性先天原则？

**思维流变**：从"共相是否真实存在"（第五章）→ "知识的最终来源是什么"（本章）。奥卡姆的唯名论瓦解了经院哲学对"本质"的信念，促使哲学家重新追问：如果科学不研究本质，那么它研究什么？我们如何获得可靠的知识？

**本章位置**：紧接第五章"共相与个别"。奥卡姆的唯名论为近代经验主义铺平了道路，但另一方面，数学和自然科学的成功也促使哲学家寻求理性的先天基础。本章将展现这两条路线的对峙，以及它们如何为康德的认识论革命（第八章）奠定基础。

---

## 一、问题背景：近代认识论的兴起

### 1.1 从经院哲学到近代哲学

16-17世纪，欧洲经历了深刻的思想变革：

- **科学革命**：哥白尼、伽利略、牛顿用数学方法研究自然，取得了前所未有的成功。
- **宗教改革**：新教挑战天主教权威，个人与上帝的直接关系被强调。
- **地理大发现**：新世界和新文化的发现动摇了欧洲中心主义。
- **印刷术的普及**：知识传播加速，个人可以独立阅读圣经和古典文献。

这些变革共同促使哲学家追问：**知识的可靠基础是什么？** 如果传统的权威（教会、亚里士多德）受到质疑，那么什么能保证知识的确定性？

### 1.2 两条基本路线

| 路线 | 核心主张 | 知识来源 | 代表 |
|------|----------|----------|------|
| **经验主义** | 一切知识来源于感官经验 | 后天（a posteriori） | [培根](人物/培根/README.md)、[霍布斯](人物/霍布斯/README.md)、[洛克](人物/洛克/README.md)、[贝克莱](人物/贝克莱/README.md)、[休谟](人物/休谟/README.md) |
| **理性主义** | 知识来源于理性先天原则 | 先天（a priori） | [笛卡尔](人物/笛卡尔/README.md)、[斯宾诺莎](人物/斯宾诺莎/README.md)、[莱布尼茨](人物/莱布尼茨/README.md) |

## 二、核心思想与核心结论

### 2.1 经验主义路线：从培根到休谟

#### [培根](人物/培根/README.md)：经验主义方法论的先驱

- **核心思想**：四假相说（种族、洞穴、市场、剧场）系统分析了认识障碍；归纳法（存在表、缺乏表、程度表）建立了系统的经验收集方法。
- **核心结论**：知识就是力量；经验是知识的唯一来源；清除假相是获得真知的前提。
- **详细解读** → [培根专题](人物/培根/README.md)

#### [霍布斯](人物/霍布斯/README.md)：机械论唯物主义

- **核心思想**：一切存在归结为物体（自然物体、人造物体、国家）；用机械论解释心灵（思想是头脑中的运动）。
- **核心结论**：一切皆是物体；国家的合法性来自人民同意；绝对主权是和平的必要条件。
- **详细解读** → [霍布斯专题](人物/霍布斯/README.md)

#### [洛克](人物/洛克/README.md)：经验主义的系统化者

- **核心思想**：白板说（心灵生来空白，一切观念来源于经验）；简单观念与复杂观念的区分；第一性质（客观）与第二性质（主观）的区分。
- **核心结论**：心灵是白板；简单观念是知识的原子；自然权利（生命、自由、财产）不可剥夺。
- **详细解读** → [洛克专题](人物/洛克/README.md)

#### [贝克莱](人物/贝克莱/README.md)：经验主义的极端代表

- **核心思想**：存在即被感知（esse est percipi）；否定物质实体，认为物体只是"观念的集合"。
- **核心结论**：存在即被感知；物质实体不存在；上帝是最终的感知者。
- **详细解读** → [贝克莱专题](人物/贝克莱/README.md)

#### [休谟](人物/休谟/README.md)：经验主义的集大成者

- **核心思想**：印象与观念的区分（一切观念来源于印象）；因果关系的怀疑论（因果必然性是心理习惯）；归纳问题（从过去不能推出未来）。
- **核心结论**：一切观念来源于印象；因果必然性是主观的；自我不是实体（知觉的束）；理性是激情的奴隶；从"是"不能推出"应当"。
- **详细解读** → [休谟专题](人物/休谟/README.md)

### 2.2 理性主义路线：从笛卡尔到莱布尼茨

#### [笛卡尔](人物/笛卡尔/README.md)：近代哲学之父

- **核心思想**：系统怀疑法（三层：感官→梦境→恶魔）；我思故我在（cogito, ergo sum）；心物二元论（思想实体与广延实体）；上帝存在的证明（因果论证+本体论论证）。
- **核心结论**："我思"是不可怀疑的确定性基石；心灵和物质是两种独立的实体；上帝是认识论的保障；笛卡尔循环是理性主义的内在困境。
- **详细解读** → [笛卡尔专题](人物/笛卡尔/README.md)
- **与罗素对比** → [系统怀疑法详解](附录/附录B_核心概念详解.md#b1-系统怀疑法method-of-systematic-doubt)

#### [斯宾诺莎](人物/斯宾诺莎/README.md)：理性主义的极端代表

- **核心思想**：实体一元论（唯一的实体=神或自然）；几何学方法（用欧几里得方法写哲学）；情感理论（情感是心灵对身体变化的观念）。
- **核心结论**：实体是唯一的；心灵和身体是同一实体的两种属性；一切都被必然性决定；情感可以被理性控制。
- **详细解读** → [斯宾诺莎专题](人物/斯宾诺莎/README.md)

#### [莱布尼茨](人物/莱布尼茨/README.md)：理性主义的最后代表

- **核心思想**：单子论（简单的精神实体）；前定和谐（上帝预先安排的单子同步）；最好的可能世界（上帝选择了最好的世界）。
- **核心结论**：单子是简单的精神实体；物质世界只是现象；前定和谐解决心身问题；这是最好的可能世界。
- **详细解读** → [莱布尼茨专题](人物/莱布尼茨/README.md)

## 四、思维流向：从本章到下一章

### 4.1 核心遗留问题

1. **心身问题**：如果心灵和物质是两种不同的实体（笛卡尔），它们如何相互作用？如果只有物质（霍布斯），心灵如何存在？如果只有心灵（贝克莱），物质如何存在？
   - → [第七章：心灵与物质](../07_心灵与物质：笛卡尔难题及其后裔/README.md)

2. **因果问题**：如果因果关系只是心理习惯（休谟），那么科学如何可能？如果一切被必然性决定（斯宾诺莎），自由意志如何可能？
   - → [第九章：自由与必然](../09_自由与必然：从康德到法兰克福学派/README.md)

3. **知识的确定性**：如果一切来源于经验（洛克、休谟），那么数学和逻辑的必然性如何解释？如果理性是独立的（笛卡尔、莱布尼茨），那么它如何与经验世界联系？
   - → [第八章：主体与绝对](../08_主体与绝对：德国古典哲学的崛起/README.md)

4. **自由与必然**：如果上帝前定了一切（莱布尼茨），人的选择是否自由？如果一切被自然规律决定（霍布斯、斯宾诺莎），道德责任如何可能？
   - → [第九章：自由与必然](../09_自由与必然：从康德到法兰克福学派/README.md)

5. **怀疑的限度**：笛卡尔的系统怀疑法找到了"我思"这个不可怀疑的基石，但罗素在《哲学问题》中追问：即使"我思"确定，我们关于外部世界的知识仍然是不确定的——这种不确定性是否可以被接受？
   - → [第十章：语言与逻辑](../10_语言与逻辑：分析哲学的诞生/README.md)

### 4.2 与后续章节的关联

#### 流向第七章：心灵与物质

心身问题（mind-body problem）是近代哲学最核心的难题之一：

- **笛卡尔**：心灵和物质是两种独立的实体——但如何相互作用？
- **斯宾诺莎**：心灵和物质是同一实体的两种属性——但取消了自由意志。
- **莱布尼茨**：心灵和身体是独立的单子系列，通过前定和谐同步——但引入了上帝的持续干预。
- **霍布斯**：心灵是物质的运动——但无法解释主观体验。
- **贝克莱**：物质是心灵的观念——但无法解释科学的客观性。
- **休谟**：心灵和物质都是知觉的集合——但无法解释知识的稳定性。

→ [第七章详细展开](../07_心灵与物质：笛卡尔难题及其后裔/README.md)

#### 流向第十章：语言与逻辑——分析哲学的诞生（罗素《哲学问题》）

笛卡尔的系统怀疑法在20世纪初被罗素以新的方式继承和改造：

| 笛卡尔（1641） | 罗素（1912） |
|---------------|-------------|
| 怀疑一切，找到"我思"这个基石 | 分析具体问题，区分不同类型的确定性 |
| 通过上帝证明恢复外部世界 | 承认外部世界无法被"证明"，只能被"推断" |
| 追求绝对的、不可怀疑的知识 | 接受知识的有限性，区分亲知/描述/归纳 |
| 怀疑是**方法**，最终目的是**重建** | 怀疑是**分析**，最终目的是**澄清** |

**罗素在《哲学问题》中对笛卡尔的继承与改造**：
- **继承**：怀疑精神——不轻信常识，追问知识的可靠基础
- **改造**：放弃"重建"的野心——不再试图找到一个不可怀疑的基石，然后证明一切；而是分析不同类型的知识有不同的确定性程度
- **具体化**：笛卡尔的怀疑是"普遍的"（一切都可以被怀疑），罗素的怀疑是"局部的"（逐个分析感官、归纳、描述性知识的可靠性）

这一转变标志着从**近代认识论**到**分析哲学**的方法论转型：从"我能知道什么？"（康德式大问题）到"这种知识是什么类型？它有多确定？"（罗素式具体问题）。

→ [第十章详细展开](../10_语言与逻辑：分析哲学的诞生/README.md)
