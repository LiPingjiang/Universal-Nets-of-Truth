## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 大陆理性主义；实体一元论；泛神论（*Deus sive Natura*）；自由思想 |
| 核心方法 | 几何学方法（按欧几里得几何学的方式从定义和公理出发进行演绎）；理性直观 |
| 核心命题 | "上帝即自然"（*Deus sive Natura*）；实体唯一；自由是对必然性的认识；情感只能被更强的相反情感克服 |
| 认识论立场 | 理性主义认识论；知识的三个等级（意见、理性、直观）；真观念是真理的标准 |
| 形而上学立场 | 实体一元论；唯一的实体（上帝/自然）有无限多属性；思维和广延是同一实体的两种属性 |
| 伦理学立场 | 自由是对必然性的认识；从永恒的角度（*sub specie aeternitatis*）看待事物；情感的控制 |

## 核心概念

| 概念（中文） | 概念（拉丁文） | 核心文本 | 简要说明 |
|------------|--------------|---------|---------|
| 上帝即自然 | *Deus sive Natura* | 《伦理学》第一部分 | 上帝与自然是同一实体的两种表达，斯宾诺莎最著名的命题 |
| 实体 | *substantia* | 《伦理学》第一部分 定义三 | 在自身中存在、通过自身被理解的东西 |
| 属性 | *attributum* | 《伦理学》第一部分 定义四 | 理智在实体中感知的、构成实体本质的东西 |
| 样式 | *modus* | 《伦理学》第一部分 定义五 | 实体的变状，在他物中存在的东西 |
| 平行论 | *parallelismus* | 《伦理学》第二部分 命题七 | 观念的秩序与联系和事物的秩序与联系是同一的 |
| 前定和谐 | *harmonia praestabilita* | 莱布尼茨用语，但斯宾诺莎以平行论解决类似问题 | 心灵和身体不需要互动，因为它们是同一实体的两种表达 |
| 从永恒的角度 | *sub specie aeternitatis* | 《伦理学》第五部分 | 从永恒的角度看待事物，超越情感的束缚 |
| 自由的人 | *homo liber* | 《伦理学》第四部分 命题六十七 | 自由不是随心所欲，而是理解必然性并与之和谐共处 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 哲学史 | 黑格尔称斯宾诺莎为"近代哲学的重点"，其实体一元论影响德国观念论 |
| 伦理学 | 斯宾诺莎的伦理学以几何学方式展开，影响康德的道德哲学和当代德性伦理学 |
| 政治哲学 | 《神学政治论》被视为现代民主思想和宗教宽容论的奠基之作 |
| 宗教思想 | 斯宾诺莎因"异端"观点被犹太教会开除（1656年），但其思想影响了启蒙运动和自由神学 |
| 现代影响 | 斯宾诺莎的"情感只能被更强的相反情感克服"预示现代认知行为疗法；其平行论影响当代心身问题讨论 |
| 科学 | 斯宾诺莎作为磨镜师与惠更斯、莱布尼茨有学术交流，其光学研究具有科学价值 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 斯宾诺莎是无神论者还是泛神论者？ | 斯宾诺莎否认上帝是人格化的造物主，主张上帝即自然，因此当时被视为无神论者；但后世学者多称其为泛神论者 |
| 斯宾诺莎是否否定自由意志？ | 斯宾诺莎认为自由不是"为所欲为"，而是"对必然性的认识"；这是否是真正的自由？ |
| 几何学方法是否适合哲学？ | 斯宾诺莎以欧几里得几何学的方式写《伦理学》，但批评者认为哲学命题无法像几何命题那样被严格证明 |
| 斯宾诺莎的伦理学是否过于理性化？ | 斯宾诺莎主张通过理性理解情感来克服情感，但批评者认为这忽视了情感本身的独立价值 |

## 关键年表

| 年份 | 事件 |
|------|------|
| 1632 | 生于阿姆斯特丹的葡萄牙裔犹太家庭 |
| 1654 | 父亲去世，继承磨镜生意 |
| 1656 | 因异端观点被犹太教会开除（*cherem*，绝罚），此后以磨镜为生 |
| 1660 | 移居莱茵斯堡，开始撰写《知性改进论》和《伦理学》 |
| 1663 | 出版《笛卡尔哲学原理》（以几何学方式讲授笛卡尔哲学） |
| 1670 | 匿名出版《神学政治论》，主张圣经批判和宗教宽容 |
| 1673 | 拒绝海德堡大学哲学教席的邀请（因不愿放弃思想自由） |
| 1677 | 因肺病去世于海牙，年仅44岁；《伦理学》等遗作由朋友整理出版 |
