## 核心思想

### 1. 单子论（Monadologia）

> **原文**（法语）：
> la monade est une substance simple, qui n'entre dans les composés; elle n'a pas de parties.
> 
> **甲译**：
> **单子是简单的实体，不进入复合物；它没有部分。**

**单子的特征**：

| 特征 | 含义 |
|------|------|
| 没有部分 | 不可分，不可毁灭 |
| 没有窗户 | 不与外界发生作用 |
| 自身包含宇宙 | 每个单子都是整个宇宙的"镜子" |
| 知觉（perception） | 单子的基本活动 |
| 欲望（appetition） | 从一种知觉到另一种知觉的过渡 |

**核心结论**：莱布尼茨的单子是**精神性的实体**——没有广延，只有知觉和欲望。物质世界只是**现象**——真正的实在是由无限多的单子构成的。单子之间没有**因果作用**（没有窗户），但它们的和谐是上帝预先安排的（**前定和谐** / harmonie préétablie）。

### 2. 前定和谐（Harmonie Préétablie）

> **原文**（法语）：
> chaque substance simple est un monde à part, indépendant de toute autre chose, hors de Dieu.
> 
> **甲译**：
> **每个简单实体都是一个独立的世界，独立于一切其他事物，除了上帝。**

**核心结论**：莱布尼茨用前定和谐来解释心身关系：
- 心灵和身体是两个独立的单子系列
- 它们之间没有因果作用，但上帝在创造时就预定了它们的完美同步
- 就像两个钟表，被校准后永远同步，但彼此没有影响

这解决了笛卡尔的心身问题——但代价是引入了上帝的持续干预（虽然是一次性的"前定"）。

### 3. 最好的可能世界

> **原文**（法语）：
> Dieu a choisi celui qui est le meilleur parmi tous les mondes possibles.
> 
> **甲译**：
> **上帝在所有可能世界中选择了最好的一个。**

**核心结论**：莱布尼茨的《神正论》试图解决恶的问题：
- 上帝是全能、全知、全善的
- 但世界上存在恶
- 因此，这个世界必须是"所有可能世界中最好的"——恶的存在是为了更大的善（如自由意志）

这一思想被伏尔泰在《老实人》中讽刺：如果这是最好的世界，那么其他世界该有多糟？

---

## 核心结论

1. **单子是简单的精神实体**：没有广延，只有知觉和欲望。
2. **物质世界只是现象**：真正的实在由无限多的单子构成。
3. **前定和谐解决心身问题**：但引入了上帝的持续干预。
4. **这是最好的可能世界**：恶的存在是为了更大的善。
5. **通用字符的理想**：莱布尼茨梦想创造一种普遍的符号系统，所有推理都可以像数学一样精确。

---

## 与后续思想的关联

- **影响康德**：康德的先验哲学继承了莱布尼茨的理性主义传统。
- **影响现代逻辑**：莱布尼茨是符号逻辑和通用字符的先驱。
- **影响计算机科学**：通用字符的理想预示了编程语言和形式化系统。
- **影响罗素**：罗素对莱布尼茨的逻辑学有深入研究。
