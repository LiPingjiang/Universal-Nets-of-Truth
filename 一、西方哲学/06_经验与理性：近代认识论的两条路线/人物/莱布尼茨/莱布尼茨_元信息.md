## 核心概念

| 概念（中文） | 概念（法文/拉丁文） | 核心文本 | 简要说明 |
|------------|-------------------|---------|---------|
| 单子 | *monade* | 《单子论》§1 | 无部分的简单实体，宇宙的基本构成单位 |
| 单子没有窗户 | *Les Monades n'ont point de fenêtres* | 《单子论》§7 | 单子是完全封闭的，不接受来自外部的任何影响 |
| 感知 | *perception* | 《单子论》§14 | 单子在单一体中表现多样性的状态 |
| 欲求 | *appétition* | 《单子论》§15 | 单子从一种感知过渡到另一种感知的内在倾向 |
| 前定和谐 | *harmonie préétablie* | 《单子论》§78 | 上帝预先安排的各单子之间的和谐 |
| 充足理由律 | *principe de raison suffisante* | 《单子论》§32 | 任何事物都有其充足理由 |
| 矛盾律 | *principe de contradiction* | 《单子论》§31 | 不可能同时为真且为假 |
| 必然真理 | *vérités nécessaires* | 《新人类理解论》 | 不可能为假的真理（数学、逻辑） |
| 偶然真理 | *vérités contingentes* | 《新人类理解论》 | 可能为假的真理（经验事实） |
| 可能世界 | *mondes possibles* | 《神正论》 | 上帝在所有可能世界中选择了最好的一个 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 莱布尼茨与牛顿的微积分优先权之争 | 牛顿声称早在1665年就已发明微积分，但莱布尼茨于1684年率先发表；现代学界认为两人独立发明 |
| 前定和谐是否是"上帝的诡计"？ | 批评者认为前定和谐是上帝预先安排的"诡计"，使心灵和身体看起来在互动；莱布尼茨反驳说这是上帝智慧的体现 |
| 单子论是否导致"无窗的囚徒"？ | 每个单子都是封闭的，这是否意味着我们都被囚禁在自己的感知中？莱布尼茨认为单子通过上帝的安排而和谐 |
| "最好世界"理论是否为恶的辩护？ | 莱布尼茨认为恶的存在是有限性的必然结果，上帝已选择最好的世界；伏尔泰在《老实人》中讽刺这一观点 |
| 莱布尼茨的天赋观念论是否成立？ | 莱布尼茨用"大理石纹路"比喻修正天赋观念论，但当代经验主义者和认知科学家对此仍有争论 |

## 关键年表

| 年份 | 事件 |
|------|------|
| 1646 | 生于莱比锡，父亲是莱比锡大学道德哲学教授 |
| 1661 | 进入莱比锡大学，自学笛卡尔和经院哲学 |
| 1666 | 出版《论组合术》，提出普遍符号语言的构想 |
| 1672 | 前往巴黎，结识惠更斯等科学家，开始研究数学 |
| 1673 | 访问伦敦，被选为皇家学会会员 |
| 1676 | 访问斯宾诺莎（海牙），两人讨论哲学；发明微积分 |
| 1676 | 担任汉诺威公爵的图书馆馆长和顾问，此后终身在此任职 |
| 1684 | 发表微积分论文（早于牛顿的公开发表） |
| 1686 | 撰写《关于形而上学的对话》，提出前定和谐理论 |
| 1704 | 完成《新人类理解论》（回应洛克），但拒绝出版，直到1765年才问世 |
| 1710 | 出版《神正论》 |
| 1714 | 撰写《单子论》（致索菲公主的信） |
| 1716 | 去世于汉诺威，因新教信仰未能获得宫廷葬礼 |
