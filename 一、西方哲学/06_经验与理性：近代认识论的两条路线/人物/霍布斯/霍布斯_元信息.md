## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 英国经验主义；机械论唯物主义；社会契约论 |
| 核心方法 | 几何学方法（从定义出发的演绎）；机械论解释；经验观察 |
| 核心命题 | "一切人对一切人的战争"（*Bellum omnium contra omnes*）；"人的生活是孤独的、贫穷的、肮脏的、残忍的和短暂的"；社会契约论 |
| 认识论立场 | 经验主义认识论；一切观念来自感觉；反对天赋观念 |
| 形而上学立场 | 机械论唯物主义；一切现象可还原为物体的运动 |
| 政治哲学 | 绝对主权论；国家是"人造的人"（*Artificial Man*）；主权不可分 |

## 核心概念

| 概念（中文） | 概念（英文/拉丁文） | 核心文本 | 简要说明 |
|------------|-------------------|---------|---------|
| 自然状态 | *State of Nature* | 《利维坦》第十三章 | 没有政府的人类原始状态，"一切人对一切人的战争" |
| 社会契约 | *Social Contract* / *Covenant* | 《利维坦》第十七章 | 人们相互让渡权利、建立国家的协议 |
| 利维坦 | *Leviathan* | 《利维坦》书名 | 国家，取自《圣经》中的海怪，象征强大的主权 |
| 主权者 | *Sovereign* | 《利维坦》第十七章 | 通过社会契约获得权力的统治者或统治机构 |
| 自然权利 | *Natural Right* | 《利维坦》第十四章 | 人在自然状态下保护自己的权利 |
| 自然法 | *Law of Nature* | 《利维坦》第十四章 | 理性指导下的行为准则，要求寻求和平 |
| 感觉 | *Sense* / *Fancy* | 《利维坦》第一章 | 外部物体对感官的压力引起的表象 |
| 人造的人 | *Artificial Man* | 《利维坦》引言 | 国家是由人创造的"人造的人"，其灵魂是主权 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 政治哲学 | 霍布斯的社会契约论奠定了现代政治哲学的基础，影响洛克、卢梭、罗尔斯等 |
| 国际关系理论 | "一切人对一切人的战争"成为现实主义国际关系理论的核心概念 |
| 法学 | 霍布斯的主权论和守法义务论影响奥斯丁的法律实证主义 |
| 认识论 | 霍布斯将培根的经验主义与几何学方法结合，形成独特的机械论认识论 |
| 现代影响 | 霍布斯的"自然状态"思想在博弈论（如囚徒困境）和进化生物学中获得新解释 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 霍布斯是专制主义者还是自由主义者？ | 传统观点视霍布斯为专制主义辩护者；但施特劳斯等认为霍布斯为自由主义奠基，因为主权者的权力来自人民的同意 |
| 霍布斯是否是唯物主义者？ | 霍布斯明确主张一切存在都是物体，但他在宗教问题上态度暧昧，引发"无神论者"指控 |
| 自然状态是历史事实还是思想实验？ | 霍布斯本人称自然状态是"推理"而非历史，但后世学者对此仍有争议 |
| 霍布斯与笛卡尔的关系 | 霍布斯批评笛卡尔的心身二元论，但自己在心身关系问题上同样面临困难 |
