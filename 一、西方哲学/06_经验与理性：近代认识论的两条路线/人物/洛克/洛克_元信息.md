## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 英国经验主义；自由主义；自然权利理论 |
| 核心方法 | 经验观察；观念分析；从简单观念到复杂观念的建构 |
| 核心命题 | 白板说（*tabula rasa*）；一切观念来自经验；政府的目的是保护自然权利；宗教宽容 |
| 认识论立场 | 经验主义认识论；反对天赋观念；知识来源于感觉和反省 |
| 政治哲学 | 自由主义奠基者；自然权利论；社会契约论；有限政府；分权思想 |
| 教育哲学 | 白板说在教育中的应用；强调环境和教育对人格的塑造 |

## 核心概念

| 概念（中文） | 概念（英文/拉丁文） | 核心文本 | 简要说明 |
|------------|-------------------|---------|---------|
| 白板 | *tabula rasa* | 《人类理解论》第二卷第一章 | 心灵出生时的空白状态，无天赋观念，一切知识来自经验 |
| 感觉 | *sensation* | 《人类理解论》第二卷第一章 | 来自外部感官的观念来源 |
| 反省 | *reflection* | 《人类理解论》第二卷第一章 | 来自内部心灵活动的观念来源 |
| 简单观念 | *simple ideas* | 《人类理解论》第二卷 | 不可分析的基本观念，来自感觉或反省 |
| 复杂观念 | *complex ideas* | 《人类理解论》第二卷 | 由简单观念组合而成的观念 |
| 第一性质 | *primary qualities* | 《人类理解论》第二卷第八章 | 物体本身真实具有的性质（大小、形状、运动等） |
| 第二性质 | *secondary qualities* | 《人类理解论》第二卷第八章 | 物体在我们心中产生的感觉（颜色、声音、气味等） |
| 自然权利 | *natural rights* | 《政府论下篇》 | 生命、自由和财产是不可剥夺的自然权利 |
| 同意 | *consent* | 《政府论下篇》 | 政府的合法性来自被统治者的同意 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 认识论 | 洛克的《人类理解论》是近代经验主义认识论的奠基之作，影响贝克莱、休谟和整个英国经验主义传统 |
| 政治哲学 | 洛克的《政府论》是现代自由主义政治哲学的奠基之作，影响美国独立宣言和法国人权宣言 |
| 宗教宽容 | 洛克的《论宗教宽容》是现代宗教自由思想的奠基文献 |
| 教育哲学 | 洛克的《教育漫话》影响近代教育思想，强调环境和习惯对人格的塑造 |
| 经济学 | 洛克的劳动价值论（劳动赋予财产权）影响古典经济学和马克思主义 |
| 现代影响 | 洛克的自然权利论和有限政府思想仍是当代自由主义政治哲学的核心 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 白板说是否正确？ | 洛克认为心灵出生时没有任何观念；莱布尼茨反驳说心灵有潜在的天赋观念；当代认知科学（乔姆斯基）支持某种形式的天赋结构 |
| 第一性质/第二性质区分是否成立？ | 贝克莱认为第一性质和第二性质一样，都只存在于心灵中；当代物理学似乎支持洛克（基本粒子的属性独立于感知） |
| 洛克的财产权理论是否为资本主义辩护？ | 洛克的劳动价值论被马克思主义者批评为资产阶级意识形态；但也被自由主义者视为个人自由的基石 |
| 洛克与奴隶制的关系 | 洛克曾为奴隶贸易公司投资，其财产权理论是否隐含种族主义？当代学者对此有激烈争论 |
