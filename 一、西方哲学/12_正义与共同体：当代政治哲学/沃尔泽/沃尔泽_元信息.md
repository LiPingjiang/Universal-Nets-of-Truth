## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 复合平等 | 正义不是消除一切不平等，而是防止一种社会善的不平等蔓延到其他领域 | 《正义诸领域》（1983） |
| 社会意义理论 | 分配正义原则应从社会善在特定共同体中的共享意义中推导 | 《正义诸领域》（1983） |
| 阻断交换 | 通过制度设计阻止不当的跨领域交换，维持各分配领域的边界 | 《正义诸领域》（1983） |
| 垄断与支配之分 | 垄断是单一领域内的过度集中，支配是一种善的规则凌驾于所有领域 | 《正义诸领域》（1983） |
| 宽容的多元体制 | 宽容在不同政治体制中呈现为不同的制度形态，无单一模式 | 《论宽容》（1997） |
| 厚薄道德之分 | "薄"的普世道德约束"厚"的地方道德，兼顾普遍与特殊 | 《厚与薄》（1994） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系说明 |
|------|----------|
| 罗尔斯（John Rawls） | 沃尔泽的复合平等理论是对罗尔斯普世正义原则的直接回应与批评 |
| 亚里士多德（Aristotle） | 沃尔泽的"社会善"概念与亚里士多德的"分配正义"有思想渊源 |
| 马克思（Karl Marx） | 沃尔泽对市场逻辑边界的反思受到马克思异化理论的启发 |
| 以赛亚·伯林（Isaiah Berlin） | 伯林的价值多元主义为沃尔泽的多元正义观提供了哲学基础 |

### 4.2 同代

| 人物 | 关系说明 |
|------|----------|
| 罗尔斯（John Rawls） | 普世主义 vs. 特殊主义的核心分歧，持续对话关系 |
| 桑德尔（Michael Sandel） | 同为社群主义代表，共同批评自由主义的"无拘束自我" |
| 德沃金（Ronald Dworkin） | 德沃金的资源平等论与沃尔泽的复合平等形成对照 |
| 麦金太尔（Alasdair MacIntyre） | 同为社群主义阵营，但麦金太尔更激进，沃尔泽更务实 |

### 4.3 后启

| 人物/流派 | 关系说明 |
|-----------|----------|
| 多元主义正义理论 | 沃尔泽的多元正义观直接启发了后续的多元主义分配正义研究 |
| 米勒（David Miller） | 系统发展了沃尔泽的多元正义框架 |
| 全球正义讨论 | 沃尔泽的特殊主义立场成为全球正义辩论中的重要参照与批评对象 |
| 多元文化主义 | 沃尔泽的宽容理论为多元文化主义政策提供了理论资源 |
