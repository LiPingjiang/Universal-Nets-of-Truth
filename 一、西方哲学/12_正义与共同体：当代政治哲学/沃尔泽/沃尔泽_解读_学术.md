# 沃尔泽 · 解读（学术版）

## 一、思想史定位

### 1.1 复合平等与分配正义的多元转向

沃尔泽在当代政治哲学中的思想史意义主要体现在以下几个方面：

- **正义理论的多元主义转向**：沃尔泽的《正义诸领域》（1983）是20世纪后期分配正义理论最重要的著作之一。他以"社会善的社会意义"为出发点，挑战了罗尔斯和德沃金等人寻求普世正义原则的进路，主张正义原则应当是多元的、因领域而异的。这一"特殊主义"立场为正义理论开辟了新的问题空间。

- **社群主义对自由主义的核心批评**：作为社群主义阵营的重要代表（与桑德尔、麦金太尔并列），沃尔泽从"共同体"和"共享理解"的角度对自由主义的"无拘束自我"提出了系统批评。但与麦金太尔不同，沃尔泽并不否定自由主义的核心价值，而是试图在多元主义的框架内重新阐释这些价值。

- **宽容理论的制度维度**：沃尔泽的《论宽容》（1997）将宽容从抽象的道德原则落实为具体的制度安排。他区分了五种宽容体制，揭示了宽容作为一种政治实践的历史多样性和内在张力，为多元文化主义讨论提供了重要的理论资源。

- **战争伦理的规范性重建**：沃尔泽的《正义与非正义战争》（1977）是当代正义战争理论的奠基性著作，他重新阐释了正义战争的传统原则（开战正义和交战正义），为国际政治伦理提供了规范性框架。

### 1.2 思想发展的脉络

沃尔泽的思想发展可以分为三个阶段。第一阶段是1960—70年代，以《义务》（1970）和《正义与非正义战争》（1977）为代表，主要关注公民义务、战争伦理等规范性政治哲学问题。第二阶段是1980—90年代，以《正义诸领域》（1983）和《厚与薄》（1994）为代表，系统发展了多元主义正义理论和道德论证的"厚薄"之分。第三阶段是1990年代后期至今，以《论宽容》（1997）和《政治与激情》（2004）为代表，进一步将多元主义正义理论扩展到宽容、平等和民主政治领域。贯穿这三个阶段的核心关怀始终是：在承认差异和多元的前提下，如何实现正义与团结。

## 二、核心命题分析

### 2.1 复合平等

**核心命题：** 正义不是消除一切不平等，而是防止一种社会善的不平等蔓延到其他领域。

**哲学问题：** 在一个异质性的社会中，什么样的平等是既可欲又可行的？

1. 沃尔泽首先区分了"简单平等"与"复合平等"。简单平等要求所有人在一切方面都平等——无论是收入、权力还是地位。沃尔泽认为这种平等既不现实也不可取，因为它忽视了不同社会善的内在差异。

2. 复合平等的核心直觉是：不平等本身不是最大的恶，"暴政"才是。暴政指的是某种社会善的分配原则（如市场逻辑）渗透到所有领域，使一种优势转化为全面的优势。

3. 复合平等的制度实现是"阻断交换"——通过法律和制度阻止一种善被自由地兑换为另一种善。例如，禁止买卖政治权力、限制竞选资金、禁止出售学术学位。

4. 这一理论的深层哲学基础是"社会意义"概念：正义原则必须从共同体对善的共享理解中推导，而非从普世的抽象原则中推导。这使得沃尔泽的正义观具有鲜明的"阐释学"特征。

5. 复合平等面临的核心困难是：社会意义的多元性是否足以支撑一个稳定的正义理论？如果不同共同体对善的意义理解不同，是否意味着不存在跨文化的正义标准？

### 2.2 社会善的社会意义

**核心命题：** 分配正义的原则应当从社会善在特定共同体中的共享意义中推导出来。

**哲学问题：** 正义原则是普世的还是语境化的？

1. 沃尔泽提出，社会善不是自然物，而是文化建构物。每种善的"意义"——即它被追求、评价和分配的方式——是由共同体成员共享的理解所赋予的。

2. 社会意义因共同体不同而不同。例如，在传统社会中，土地被视为家族传承的神圣纽带；在现代社会中，土地被理解为可交易的商品。这种意义差异决定了不同的分配原则。

3. 意义的多元性意味着正义原则的多元性：金钱按市场交换分配，政治权力按民主参与分配，医疗按需要分配，教育按才能分配，荣誉按功绩分配。

4. 这一立场使沃尔泽与罗尔斯形成了根本对立：罗尔斯的"原初状态"假想一种超越具体共同体的正义原则；沃尔泽则坚持正义必须从共同体的内部理解出发。

5. 批评者指出，"社会意义"本身可能是含混的、冲突的——同一个共同体内部，不同群体对同一种善的意义可能有不同理解。沃尔泽对此的回应是：社会意义的形成和修正本身就是一个政治过程，需要通过民主审议来实现。

### 2.3 阻断交换与制度设计

**核心命题：** 正义的制度保障在于"阻断"不当的跨领域交换。

**哲学问题：** 如何设计制度来维持不同分配领域之间的边界？

1. 阻断交换是复合平等的操作性概念。它指的是通过法律和制度安排，阻止一种社会善的分配优势通过"兑换"蔓延到其他领域。

2. 沃尔泽列举了多种阻断交换的实例：反腐败法阻断政治权力向经济利益的转化；竞选资金法阻断经济力量向政治权力的转化；禁止出售学位阻断金钱向教育资格的转化；禁止出售勋章阻断金钱向荣誉的转化。

3. 阻断交换的核心思想是限制市场的范围，而非消灭市场。市场在其正当领域（商品交换）内是合理的分配机制，但当市场逻辑渗透到政治、教育、医疗等领域时，就构成了"暴政"。

4. 这一观念对当代政策辩论具有直接意义：医疗保障是否应当市场化？教育是否应当市场化？政治选举中的资金投入是否应当受限？沃尔泽的框架为这些辩论提供了规范性标准。

5. 但阻断交换也面临实践困难：在现代高度互联的经济中，各领域之间的边界越来越模糊。例如，教育既是公共品又是人力资本投资，医疗既是基本权利又是服务市场——如何在这种混合领域中确定恰当的分配原则？

### 2.4 宽容的多元体制

**核心命题：** 宽容不是一种单一原则，而是随政治体制不同而呈现为不同的制度形态。

**哲学问题：** 在什么样的政治条件下，文化差异可以和平共存？

1. 沃尔泽区分了五种宽容体制：多民族帝国（通过间接统治管理多元性）、国际社会（通过主权和互不干涉维持宽容）、联盟社会（通过群体间权力分享维持和平）、民族国家（通过同化或文化自治处理少数群体）、移民社会（面临融入与差异的持续张力）。

2. 每种体制面对不同的多元性挑战。多民族帝国的宽容是"管理性"的——帝国通过允许自治来降低治理成本；移民社会的宽容则是"构成性"的——差异本身就是社会的构成要素。

3. 沃尔泽特别关注移民社会的"双重拉力"：一方面，社会需要共同的文化基础来维持团结和信任；另一方面，社会需要尊重群体的文化差异来维持宽容。这两种力量之间的张力无法一劳永逸地解决，而需要持续的政治协商。

4. 宽容的限度问题由此产生：当差异威胁到宽容赖以存在的制度框架时，宽容必须设限。但限度的划定本身必须经过民主审议，而非由行政权力单方面决定。

5. 这一分析框架对当代多元文化主义辩论有重要启示：它表明不存在一种"普遍适用"的宽容模式，每个社会必须根据自身的历史条件和人口构成来设计适合的宽容制度。

## 三、学术争议与当代评价

### 3.1 特殊主义与普遍主义之争

沃尔泽的"社会意义"进路在学术界引发了关于正义之普遍性与特殊性的持久争论。罗尔斯和德沃金等自由主义平等派批评沃尔泽的特殊主义过于依赖共同体内部的"共识"，可能无法对不合理的社会意义（如种族隔离制度下的"意义"）提出批评。沃尔泽回应说，社会意义并非不可批判——共同体内部可以通过民主审议来修正不合理的意义，而且"薄"的普世道德（如基本人权）可以对"厚"的地方意义形成约束。这一"厚薄之分"是沃尔泽在《厚与薄》（1994）中对批评的重要回应。

### 3.2 复合平等的可操作性

复合平等理论面临的主要质疑是其可操作性。批评者指出，现代社会中不同领域之间的边界日益模糊，许多社会善具有多重意义。例如，高等教育既是公共品又是人力资本投资，医疗既是基本权利又是市场服务——在这种情况下，如何确定某一善的"正当领域"？此外，"阻断交换"的制度设计在实践中往往面临效率与公平的权衡。尽管如此，沃尔泽的框架仍然为反思市场逻辑的无限扩张提供了重要的规范性资源。

### 3.3 全球正义的挑战

沃尔泽的正义理论以民族国家为基本单位，这在全球化的背景下受到了挑战。森和博格等全球正义论者指出，全球范围内的经济不平等和制度性剥削要求超越民族国家的正义框架。沃尔泽在《厚与薄》中试图用"薄"的普世道德来回应这一挑战，但许多学者认为这一回应不够充分。在全球正义的讨论中，沃尔泽的"特殊主义"立场既是其理论的特色，也是其局限所在。
