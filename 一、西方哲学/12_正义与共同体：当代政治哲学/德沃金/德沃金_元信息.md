## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 权利作为王牌 | 个人权利是政治王牌，不能被集体利益压倒 | 《认真对待权利》 |
| 平等关切与尊重 | 政府对公民的两重义务——平等关切和平等尊重 | 《认真对待权利》 |
| 资源平等 | 正义要求资源平等分配而非福利平等分配 | 《至上的美德》 |
| 嫉妒检验 | 判断分配平等的标准——无人偏好他人资源组合时为平等 | 《至上的美德》 |
| 整全法 | 法律应被解释为连贯的原则体系，法律是解释性实践 | 《法律帝国》 |
| 解释主义 | 法律和道德不是被发现的事实而是被建构的解释 | 《法律帝国》《身披法袍的正义》 |
| 价值统一性 | 各种价值构成统一体系——真正的价值冲突源于概念混淆 | 《刺猬的正义》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 罗尔斯 | 德沃金受罗尔斯正义论启发，但在权利和平等的优先性上发展了独特理论 |
| H.L.A.哈特 | 德沃金在牛津师从哈特，后系统批评哈特的法律实证主义——规则模式无法捕捉原则 |
| 德沃金对功利主义的批判 | 继承了罗尔斯对功利主义的批评，但以"权利王牌"为核心提供了更强硬的反驳 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 罗尔斯 | 同为自由主义巨头——在平等和权利上有深层共鸣，但路径不同（罗尔斯：契约论；德沃金：权利论） |
| 诺齐克 | 德沃金批评诺齐克的自由意志主义——拒绝再分配违反平等关切义务 |
| 森 | 森批评德沃金的资源平等忽视了"能力"——同样的资源对不同能力的人意味着不同自由 |
| 桑德尔 | 桑德尔批评德沃金的"原则共同体"仍是自由主义框架——但两人都关注共同体的道德维度 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 当代权利理论 | 德沃金的"权利王牌"论是当代权利理论的核心范式 |
| 法理学解释主义 | 整全法理论开创了法理学的解释主义转向——影响了当代法律理论 |
| 分配正义理论 | 资源平等理论为分配正义开辟了"资源"路径——与森的"能力"路径形成对话 |
