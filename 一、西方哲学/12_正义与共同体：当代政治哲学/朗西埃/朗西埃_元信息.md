## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 感性的分配 | 同时决定什么可以被感知和什么可以参与共同的体系——划定可见/不可见的边界 | 《感性的分配》 |
| 歧义 | 不是利益分歧——而是关于"什么是对话"本身的分歧——一方"听不到"另一方 | 《歧义》 |
| 无份者的份额 | 被排除在感性分配之外的人对秩序的打断——这就是政治和民主 | 《歧义》 |
| 警察秩序 vs. 政治 | 警察=分配身体到位置的秩序；政治=对秩序的打断——两者对立 | 《歧义》 |
| 美学体制 | 艺术的第三个体制——脱离规则——建立感性的平等——与政治同构 | 《感性的分配》 |
| 智力的平等 | 每个人都能自己学习——不需要"解释者"——解放的前提 | 《无知的教师》 |
| 解放 | 不是"被教育"——而是"认识到智力与他人平等"——然后自己学习行动 | 《无知的教师》 |
| 政治主体化 | 政治主体不是预先存在的群体——而是通过无份者份额出现的创造 | 《歧义》 |

## 五、核心论点

1. 政治不是权力斗争或利益分配——而是"感性的分配"——划定可见/不可见、可听/不可听、可言/不可言的感性边界
2. 政治的本质是"歧义"——不是"不同意"——而是"听不到"——一方说的话另一方根本"听不到"——因为不共享同一种感性分配
3. 民主不是制度形式——而是"无份者的份额"——被排除者对既定秩序的打断——"不该算数的人"突然算数了
4. 政治与美学共享同一种"感性分配"结构——艺术的"美学体制"建立感性的平等——与政治民主同构
5. 智力是普遍的——每个人都能自己学习——"解释"不是帮助理解而是制造依赖——解放是"认识到智力与他人平等"

## 六、思想关系

### 6.1 前承

| 人物 | 关系 |
|------|------|
| 阿尔都塞 | 朗西埃的老师——后来决裂——决裂本身是朗西埃思想的出发点 |
| 马克思 | 阶级分析的影响——但朗西埃批评马克思将工人视为需要"觉醒"的"自在阶级" |
| 柏拉图 | 《理想国》中"谁该看、谁该做"的分配——朗西埃要打破这个分配 |
| 席勒 | "审美教育"中的感性平等——美学体制的先驱 |
| 康德 | "审美判断"的无功利性——美学体制的基础 |

### 6.2 同代

| 人物 | 关系 |
|------|------|
| 阿甘本 | 在政治与生命的关系上有交叉——但阿甘本关注主权——朗西埃关注感性分配 |
| 巴迪欧 | 在"政治事件"问题上有交叉——但巴迪欧走数学路线——朗西埃走感性路线 |
| 齐泽克 | 在意识形态和实在界问题上有交叉——但齐泽克走精神分析——朗西埃走感性分配 |
| 南希 | 在共通体问题上有交叉——但南希关注存在论——朗西埃关注感性分配 |
| 哈贝马斯 | 朗西埃批评哈贝马斯的"交往理性"——因为歧义意味着对话者不共享同一语言游戏 |
