## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 意义框架理论 | 自我认同在意义框架中形成——框架提供规范性背景 | 《自我的根源》 |
| 本真性伦理 | "对自己真实"包含独特性和对话性两个维度 | 《现代性之隐忧》 |
| 承认政治 | 身份在对话中通过承认关系形成——承认是基本需要 | 《承认的政治》 |
| 对程序中立的批评 | 自由主义的程序中立是幻觉——程序本身预设了道德立场 | 《承认的政治》 |
| 强评价理论 | 自我认同涉及强评价——判断什么"值得"追求 | 《自我的根源》 |
| 世俗时代研究 | 世俗化是信仰条件变迁——从"不可能不信"到"可能不信" | 《世俗时代》 |
| 社会想象 | 现代社会如何想象自身——从经济秩序到公共领域 | 《现代社会想象》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 黑格尔 | 泰勒的核心思想来源——承认的辩证法、历史性、精神现象学 |
| 海德格尔 | 存在与时间中的"此在"概念影响了泰勒的自我观 |
| 维特根斯坦 | 后期"语言游戏"和"生活形式"影响了泰勒的框架概念 |
| 以赛亚·伯林 | 泰勒的导师——消极自由与积极自由的区分 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 麦金太尔 | 同为社群主义代表——在自我认同和传统问题上有深刻共鸣 |
| 罗尔斯 | 泰勒批评罗尔斯的程序中立——但承认其贡献 |
| 桑德尔 | 同为社群主义——共同批评自由主义的原子个人主义 |
| 哈贝马斯 | 在承认和交往理性问题上有深入对话 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 霍耐特 | 承认理论的发展者——将泰勒的承认政治系统化为承认理论 |
| 当代多元文化主义 | 泰勒的承认政治深刻影响了多元文化主义理论 |
| 世俗化研究 | 《世俗时代》为理解宗教在现代社会提供了新框架 |
