## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 现代道德碎片化 | 现代道德语言是从不同传统拼凑的碎片——概念层面已分裂 | 《追寻美德》 |
| 实践理论 | 实践是复杂的社会性合作活动，通过它实现"内在善" | 《追寻美德》 |
| 美德伦理学 | 美德是在实践中培养的、追求内在善的品质 | 《追寻美德》 |
| 叙事自我 | 自我身份是叙事性的——嵌入在家庭、社区、传统中 | 《追寻美德》 |
| 合理性的传统依赖性 | 不存在中立的理性标准——理性相对于传统 | 《谁之正义？何种合理性？》 |
| 认识论危机 | 传统遇到无法解决的问题时的危机——可从其他传统学习 | 《谁之正义？何种合理性？》 |
| 对自由主义的批判 | 自由主义的"无拘无束的自我"和"中立国家"是幻觉 | 《追寻美德》《谁之正义？何种合理性？》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 亚里士多德 | 麦金太尔的核心灵感来源——美德、目的论、实践智慧 |
| 阿奎那 | 阿奎那对亚里士多德传统与基督教传统的综合是麦金太尔的范例 |
| 马克思 | 早年影响——社会结构分析和对资本主义的批判 |
| 维特根斯坦 | 后期维特根斯坦的"语言游戏"和"生活形式"影响了麦金太尔的传统概念 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 罗尔斯 | 麦金太尔批评罗尔斯的正义论——"原初状态"预设了自由主义的个人主义 |
| 查尔斯·泰勒 | 同为社群主义代表——在自我认同和传统问题上有深刻共鸣 |
| 桑德尔 | 同为社群主义——共同批评自由主义的"无拘无束的自我" |
| 威廉姆斯 | 在伦理学和古希腊哲学上有交叉——但路径不同 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 当代美德伦理学 | 麦金太尔是美德伦理学复兴的奠基人——影响了赫斯特豪斯、斯洛特等 |
| 社群主义运动 | 麦金太尔与泰勒、桑德尔共同构成了社群主义的核心 |
| 当代政治神学 | 麦金太尔晚期对神学的关注影响了当代政治神学 |
