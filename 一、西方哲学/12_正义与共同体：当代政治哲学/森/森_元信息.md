# 森 · 元信息

| 项目 | 内容 |
|------|------|
| 姓名 | 阿马蒂亚·森（Amartya Sen） |
| 生卒年 | 1933年— |
| 国籍 | 印度（后入籍英国） |
| 主要身份 | 经济学家、哲学家、1998年诺贝尔经济学奖获得者、哈佛大学经济学与哲学教授 |

## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 《集体选择与社会福利》 | *Collective Choice and Social Welfare* | 1970 | 社会选择理论、阿罗不可能性定理的拓展 |
| 《贫困与饥荒》 | *Poverty and Famines: An Essay on Entitlement and Deprivation* | 1981 | 饥荒的权利失败理论、1943年孟加拉大饥荒分析 |
| 《商品与能力》 | *Commodities and Capabilities* | 1985 | 能力方法的理论基础、功能与能力的区分 |
| 《再论不平等》 | *Inequality Reexamined* | 1992 | "什么的平等"问题、能力作为评价空间 |
| 《以自由看待发展》 | *Development as Freedom* | 1999 | 发展即自由、五种工具性自由 |
| 《正义的理念》 | *The Idea of Justice* | 2009 | 比较正义、公开理性、公正旁观者 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 能力方法 | 正义评价应以人的能力——能做什么和成为什么——为标准，而非资源或效用 | 《商品与能力》（1985）、《再论不平等》（1992） |
| 权利失败理论 | 饥荒的本质是特定群体获取粮食的权利集合崩溃，而非粮食总量不足 | 《贫困与饥荒》（1981） |
| 发展即自由 | 发展是扩展人们实质自由的过程，自由既是目的也是手段 | 《以自由看待发展》（1999） |
| 比较正义 | 正义追求应从设计完美制度转向在现实选项中减少明显不正义 | 《正义的理念》（2009） |
| 公开理性与公正旁观者 | 正义判断通过公开讨论和不偏不倚的第三方视角来推进 | 《正义的理念》（2009） |
| 行动力与福利之分 | 人不仅是福利的承受者，更是自主设定目标的行动主体 | 《以自由看待发展》（1999） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系说明 |
|------|----------|
| 亚当·斯密（Adam Smith） | 斯密的"公正旁观者"概念是森公开理性理论的直接灵感来源 |
| 罗尔斯（John Rawls） | 罗尔斯的正义论是森比较正义理论的批评对象和对话伙伴 |
| 马克思（Karl Marx） | 马克思对资本主义异化的批判启发了森对"商品中心"评价观的反思 |
| 阿罗（Kenneth Arrow） | 阿罗不可能性定理是森社会选择理论研究的起点 |

### 4.2 同代

| 人物 | 关系说明 |
|------|----------|
| 罗尔斯（John Rawls） | 超越正义 vs. 比较正义的核心分歧，持续的理论对话 |
| 努斯鲍姆（Martha Nussbaum） | 能力方法的共同发展者，但在能力清单的确定性上存在分歧 |
| 德沃金（Ronald Dworkin） | 德沃金的资源平等论与森的能力方法形成评价空间的对照 |
| 沃尔泽（Michael Walzer） | 两者都反对单一正义标准，沃尔泽主张领域多元，森主张评价空间转换 |

### 4.3 后启

| 人物/流派 | 关系说明 |
|-----------|----------|
| 发展伦理学 | 森的"发展即自由"为发展伦理学奠定了理论基础 |
| 人权理论 | 能力方法为人权理论提供了新的规范性框架 |
| 全球正义 | 比较正义和公开理性框架被广泛应用于全球正义讨论 |
| 人类发展指数（HDI） | 联合国开发计划署的HDI直接基于森的能力方法设计 |

## 五、本章定位

| 项目 | 内容 |
|------|------|
| 所属章节 | 一、西方哲学 → 12. 正义与共同体：当代政治哲学 |
| 章节小节 | 森：能力方法与比较正义 |
| 思维流向 | 从罗尔斯的超越正义转向比较正义，从资源/效用评价转向能力评价，从制度设计转向减少现实不正义 |
| 核心张力 | 超越正义 vs. 比较正义；资源 vs. 能力；完美制度 vs. 现实改善；普世原则 vs. 公开审议 |
| 与前后人物关系 | 前承罗尔斯正义论并以比较正义批评之，与沃尔泽的多元正义构成互补对照，后启发展伦理学和全球正义讨论 |
