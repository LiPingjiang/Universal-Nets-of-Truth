# 森 · 解读（学术版）

## 一、思想史定位

### 1.1 能力方法与正义理论的范式转换

森在当代政治哲学和经济学中的思想史意义主要体现在以下几个方面：

- **评价空间的范式转换**：森的"能力方法"从根本上改变了评价个人福祉和社会正义的"评价空间"。从边沁的效用空间到罗尔斯的"基本善"空间，再到森的"能力"空间，这一转换意味着我们不再以人们"拥有什么"或"感受什么"来衡量正义，而是以人们"能够做什么和成为什么"来衡量。这一转换深刻影响了发展经济学、福利经济学和社会政策。

- **饥荒理论的突破**：森在《贫困与饥荒》（1981）中对饥荒的"权利失败"分析颠覆了"饥荒源于粮食短缺"的传统认知。他的经验研究表明，许多大规模饥荒发生在粮食供给并未显著减少的时期，饥荒的真正原因是某些群体获取粮食的"权利"崩溃。这一发现对国际饥荒防治政策产生了深远影响。

- **发展观的重构**：森在《以自由看待发展》（1999）中将发展重新定义为自由的扩展，挑战了将发展等同于经济增长的主流观点。这一重构为"人类发展指数"（HDI）的创立提供了理论基础，深刻影响了联合国开发计划署的政策框架。

- **正义路径的重新定向**：森在《正义的理念》（2009）中提出了"比较正义"的概念，将正义的追求从"设计完美制度"转向"减少明显不正义"。这一转向对罗尔斯式的"超越正义"传统构成了系统挑战，为正义理论开辟了新的方向。

### 1.2 思想发展的脉络

森的思想发展始于对集体选择理论的数理研究。1970年的《集体选择与社会福利》以严密的形式分析揭示了阿罗不可能性定理的深层含义，为福利经济学提供了新的理论基础。1980年代，森转向对贫困、不平等和饥荒的经验研究，发展了"权利方法"和"能力方法"。1990年代，森进一步将能力方法扩展为一种发展观，在《以自由看待发展》中系统阐述了"发展即自由"的命题。2000年代，森在《正义的理念》中将能力方法与正义理论结合，提出了系统的"比较正义"框架。贯穿这一发展脉络的核心关怀是：如何以人的实际能力为中心来评价和改善社会安排。

## 二、核心命题分析

### 2.1 能力方法

**核心命题：** 正义的评价标准应当是人的能力——即人能够实际做到和成为的各种事物，而非资源或效用。

**哲学问题：** 评价个人处境和社会正义的恰当"空间"是什么？

1. 森首先批评了两种主流评价标准。资源标准（收入、财富、基本善）忽视了人与人之间将资源转化为实际能力的差异——一个残疾人和一个健全人拥有相同的收入，但前者的实际生活质量远低于后者。效用标准（快乐、满足）则受到"适应性偏好"的扭曲：长期受压迫的人可能已经降低了自己的期望，从而在不利条件下仍报告很高的满意度。

2. 能力方法的核心主张是：评价的恰当空间应当是"能力"——即一个人实际拥有的、选择不同功能组合的实质自由。能力关注的不是"你有多少"或"你高不高兴"，而是"你能够做什么和成为什么"。

3. 能力与功能的区分至关重要。功能是已实现的状态（如"营养良好"），能力是可选择的功能组合（如"能够选择营养良好或断食"）。这一区分保留了选择本身的价值——被迫营养良好和主动选择营养良好在功能上相同，但在能力上不同。

4. 能力方法的一个深层哲学意涵是：自由不仅是手段，也是目的。人的好生活不仅仅是满足某种功能清单，而是拥有选择自己珍视的生活方式的能力。这使得能力方法与自由主义的核心价值——自主和选择——产生了深层关联。

5. 能力方法面临的挑战包括：能力的清单如何确定？不同能力之间如何权衡？森拒绝给出一个固定的"核心能力清单"，认为这应当通过民主审议来确定。努斯鲍姆则试图提供一个确定的核心能力清单，这在两人之间产生了重要的学术分歧。

### 2.2 权利失败与饥荒

**核心命题：** 饥荒不是因为粮食总量不足，而是因为某些群体获取粮食的"权利"崩溃。

**哲学问题：** 贫困和饥荒的本质机制是什么？

1. 森的"权利"概念是指一个人在既定法律和经济制度下能够合法获取资源的途径。权利集合包括四个组成部分：自己生产的产品、市场购买力、劳动交换所得、社会保障提供的资源。

2. 森通过对1943年孟加拉大饥荒的经验分析表明，该年粮食总供给并未显著下降，但战争导致的通货膨胀使城市工人的实际工资购买力骤降，导致他们的"购买权利"崩溃。与此同时，农村稻农因粮价上涨反而获利。饥荒的本质是特定群体权利集合的萎缩，而非总供给的不足。

3. 这一分析的政策含义深远：防止饥荒的关键不在于增加粮食总量，而在于保障每个人的权利集合——通过就业保障计划、价格干预、社会保障网络、信息透明等手段来防止权利崩溃。

4. 更广泛地说，森发现民主国家从未发生过大规模饥荒。这是因为政治自由（言论自由、新闻自由、选举问责）为政府提供了及时响应饥荒预警的激励——在民主制度下，忽视饥荒的政府会被选票淘汰。

5. 这一发现的哲学意涵在于：政治自由不仅具有内在价值（是自由的一部分），还具有工具价值（防止灾难性后果）。自由与发展之间存在内在的因果联系——这正是"发展即自由"命题的经验基础。

### 2.3 发展即自由

**核心命题：** 发展是扩展人们享有的实质自由的过程；自由既是发展的首要目的，也是发展的主要手段。

**哲学问题：** 发展的本质目的是什么？

1. 森对传统发展观的批评是双重的。一方面，将发展等同于GNP增长或人均收入提高过于狭隘——收入只是工具，不是目的。另一方面，将发展等同于工业化或技术进步也偏离了目标——这些只是手段，最终目的是人的自由。

2. "发展即自由"命题的两个维度是：自由作为目的（发展就是要让人更自由）和自由作为手段（自由的人更能促进发展）。五种工具性自由——政治自由、经济条件、社会机会、透明性保障、防护性保障——既是自由的构成要素，也是彼此增强的手段。

3. 这一发展观的一个核心洞见是：不同种类的自由之间存在互补性。政治自由通过问责机制防止饥荒和经济崩溃；经济自由通过市场参与提供激励；社会机会（教育、医疗）通过提高人的能力扩展其他自由；透明性保障通过信息公开减少腐败；防护性保障通过社会安全网防止堕入深渊。

4. 这一框架对发展政策的启示是深远的：发展不仅仅是经济政策，更是政治、社会、制度的综合工程。忽视政治自由和社会机会的发展战略——即使短期内实现了经济增长——在长期是不可持续的。

5. "发展即自由"还揭示了一个重要的规范性洞见：发展评价不应只看宏观数据，而应看个人的实际自由是否得到了扩展。一个GDP高增长但政治压迫、教育匮乏的社会，按照森的标准，并不能算是真正"发展"了。

### 2.4 比较正义

**核心命题：** 正义的追求不应是设计完美制度，而是在现实选项中减少明显的不正义。

**哲学问题：** 正义理论应当追求超越性理想还是现实的比较改善？

1. 森将罗尔斯式的正义理论称为"超越制度主义"（transcendental institutionalism），其特征是追问"什么是完全正义的制度"并通过假想契约（原初状态）来确定正义原则。森认为这种路径有两个根本缺陷。

2. 第一个缺陷是"超越性共识"的不必要性：识别完全正义的制度并不等于能够在现实选项之间做出选择。森以选画为喻：三个人可能无法就"世界上最美的画"达成一致，但完全可能一致认为某幅画比另一幅更好。比较判断不需要超越性共识。

3. 第二个缺陷是对"现实不正义"的忽视：超越制度主义关注的是"完美制度"的设计，而非"现实中的不正义"的消除。森认为，后者才是正义理论应当面对的核心问题——世界上存在大量明显的不正义（饥饿、可预防疾病、性别歧视、政治压迫），我们不需要先就"完美制度"达成共识才能着手减少这些不正义。

4. 比较正义的实现机制是"公开理性"——通过开放的、有信息的公共讨论来推进正义判断。森借鉴亚当·斯密的"公正旁观者"概念，主张正义判断需要引入不偏不倚的第三方视角，但这种视角不是假想的"无知之幕"，而是实际的公共辩论。

5. 比较正义路径的一个重要优势是它能够处理全球正义问题。罗尔斯的原初状态局限于单一社会内部的成员，而公开理性天然具有开放性——它允许跨国界的视角交流和公共讨论。这使得森的框架比罗尔斯的更适于处理全球范围内的正义问题。

## 三、学术争议与当代评价

### 3.1 能力清单的确定问题

森拒绝提供一份固定的"核心能力清单"，认为能力清单的确定应当通过民主审议和公开理性来实现。这一立场受到了努斯鲍姆的挑战——努斯鲍姆在《女性与人类发展》和《创造能力》中提出了一份包含十项核心能力的确定清单，认为这对于宪法权利的设计和政策评估是必要的。森与努斯鲍姆之间的这一分歧反映了一个更深层的问题：正义原则的确定究竟应当是哲学家的理论任务，还是民主社会的政治任务？

### 3.2 比较正义与罗尔斯之争

森对罗尔斯"超越正义"的批评在学术界引发了广泛讨论。一些学者支持森的立场，认为比较路径更符合现实政治的需要；另一些学者则指出，罗尔斯的"原初状态"并非只是设计完美制度的工具，而是提供了一种检验正义原则的规范性标准，这种标准对于比较判断本身也是必要的——如果不参照某种正义理想，我们如何判断哪个选项"更少不正义"？这场争论至今仍是正义理论中最活跃的辩论之一。

### 3.3 能力方法的理论地位

能力方法的理论地位也存在争议。一些学者将能力方法视为一种完整的正义理论，与罗尔斯的正义论并列；另一些学者则认为能力方法只是一种评价框架，其本身不足以构成完整的正义理论——它告诉我们"评价什么"，但没有告诉我们"如何分配"。森本人倾向于后者：能力方法提供的是评价空间，而分配原则需要通过公开理性来确定。这一立场使能力方法保持了开放性和灵活性，但也使它面临着"规范性不足"的批评。
