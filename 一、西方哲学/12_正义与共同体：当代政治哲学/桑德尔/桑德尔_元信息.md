## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 无拘束的自我 | 自由主义预设的先于目的和归属的自我——哲学上不可成立 | 《自由主义与正义的局限》 |
| 构成性社群 | 塑造自我身份的社群——归属不是选择而是构成 | 《自由主义与正义的局限》 |
| 正义的判断性 | 正义不可避免地涉及对善的实质判断——不能保持中立 | 《正义：该如何做是好？》 |
| 正义的局限 | 正义是补救性美德——全靠正义运转的社会是贫困的 | 《自由主义与正义的局限》 |
| 公共哲学 | 鼓励公民就公共善进行道德辩论的政治——反对程序中立 | 《公共哲学》 |
| 公共善 | 政治应追求共同体共享之善——不是个人利益总和 | 《民主的不满》《公共哲学》 |
| 市场道德边界 | 某些领域不应被市场入侵——商品化会腐蚀价值 | 《金钱不能买什么》 |
| 绩效主义批判 | "成功者应得一切"制造傲慢与屈辱——忽视运气和共同体 | 《绩效主义的暴政》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 亚里士多德 | 桑德尔的核心灵感来源——政治应关注人的善和美德，正义与善不可分离 |
| 黑格尔 | 自我在伦理生活中被塑造——反对原子个人主义 |
| 麦金太尔 | 同为社群主义先驱——麦金太尔的《追寻美德》为桑德尔提供了方法论基础 |
| 泰勒 | 桑德尔在牛津的博士导师——泰勒的"根源自我"影响了桑德尔的"构成性社群" |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 罗尔斯 | 桑德尔最重要的批评对象——"无拘束的自我"和"程序中立"是桑德尔批评的核心 |
| 查尔斯·泰勒 | 同为社群主义核心——在自我认同和传统问题上有深刻共鸣 |
| 沃尔泽 | 同为社群主义——在"复杂平等"和"公共善"问题上有交叉 |
| 德沃金 | 在权利理论上存在张力——德沃金的"原则共同体"与桑德尔的"构成性社群"路径不同 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 当代社群主义 | 桑德尔是社群主义运动的核心——影响了后来的政治理论 |
| 公共哲学运动 | 桑德尔的"正义"课程和公共写作影响了全球公共讨论 |
| 民主理论 | 对"程序共和国"的批评影响了参与式民主和审议民主理论 |
