## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 数学本体论 | 以集合论为本体论基础——"存在即多"——一不是根本的 | 《存在与事件》 |
| 事件哲学 | 事件是对"计数为一"的根本断裂——不可预测的新可能性涌现 | 《存在与事件》 |
| 真理程序 | 真理是事件之后忠于事件的过程——四个领域：科学、艺术、政治、爱 | 《存在与事件》 |
| 事件性主体 | 主体不是先验的"我思"——而是在忠于事件的过程中产生的 | 《存在与事件》 |
| 恶的三种形态 | 模拟事件、背叛事件、使真理变成总体 | 《伦理学》 |
| 普遍性概念 | 真理是普遍的——不是"大家都同意"——而是"对所有人有效" | 《存在与事件》《伦理学》 |
| 共产主义假设 | 共产主义作为一种政治真理程序——平等政治的持续 | 《共产主义假设》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 柏拉图 | 巴迪欧自称为"柏拉图主义者"——理念论和数学本体论的灵感来源 |
| 康托尔 | 集合论为巴迪欧的数学本体论提供了数学基础 |
| 阿尔都塞 | "断裂"概念影响了巴迪欧的事件哲学——"认识论断裂"→"事件断裂" |
| 拉康 | 精神分析理论影响了巴迪欧的主体概念——主体作为"缺口" |
| 马克思 | 政治哲学的出发点——平等政治和"共产主义假设" |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 阿甘本 | 同为当代法国/意大利政治哲学核心——在"弥赛亚"和"事件"问题上有交叉 |
| 朗西埃 | 在政治和平等问题上有对话——但路径不同（朗西埃关注"感性分配"） |
| 德勒兹 | 巴迪欧写了《德勒兹》——既欣赏又批评——在"多"的问题上有深层对话 |
| 齐泽克 | 在拉康-黑格尔-马克思主义的交叉地带——与巴迪欧有激烈辩论 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 当代大陆哲学 | 巴迪欧的体系性哲学在"反体系"时代是独特的——影响了新一代哲学家 |
| 当代马克思主义 | "共产主义假设"为当代马克思主义复兴提供了理论资源 |
| 数学与哲学交叉 | 巴迪欧的数学本体论推动了数学与哲学的交叉研究 |
