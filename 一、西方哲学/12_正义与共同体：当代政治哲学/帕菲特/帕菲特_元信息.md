# 帕菲特 · 元信息

## 一、人物基本信息

| 属性 | 内容 |
|------|------|
| 姓名 | 德里克·帕菲特（Derek Parfit） |
| 生卒年 | 1942年—2017年 |
| 国籍 | 英国（出生于中国成都） |
| 主要身份 | 哲学家、牛津大学万灵学院研究员 |
| 学派归属 | 分析哲学（伦理学、形而上学、元伦理学） |

## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 理与人 | *Reasons and Persons* | 1984 | 个人同一性的还原论、非同一性问题、令人厌恶的结论、人口伦理学悖论 |
| 论重要之事 | *On What Matters* | 2011-2017（两卷本） | 规范实在论、三重理论（康德主义/后果主义/契约主义收敛）、伦理客观性 |
| 个人同一性 | "Personal Identity" | 1971（论文） | 个人同一性问题的早期奠基性论证，心理连续性理论的雏形 |
| 同一性的非重要性 | "The Unimportance of Identity" | 1995（论文） | 对还原论自我观的进一步精炼，身份与重要性的区分 |
| 如何避免令人厌恶的结论 | "How We Can Avoid the Repugnant Conclusion?" | 2016（论文） | 晚年对人口伦理学问题的持续回应 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 还原论自我观 | 个人不是超越于身体与心理事件之上的"进一步实体"，自我只是这些事件的总和；同一性不是重要的，心理连续性才是 | 《理与人》（1984） |
| 非同一性问题 | 当行为影响"谁会存在"而非"同一个存在者的福祉"时，传统"伤害"概念失效；对代际伦理与人口政策具有深远影响 | 《理与人》（1984） |
| 令人厌恶的结论 | 总体功利主义在人口伦理学中必然推出反直觉结论：数量可弥补质量的不足；至今未获满意解决 | 《理与人》（1984） |
| 三重理论（Triple Theory） | 康德主义、后果主义与契约主义在各自最合理版本中收敛于同一组结论；试图终结伦理学传统三派之争 | 《论重要之事》（2011-2017） |
| 规范实在论 | 存在不可还原的规范性真理，独立于主体心理状态与社会习俗；道德判断具有客观真假 | 《论重要之事》（2011-2017） |
| 理性收敛 | 充分理性的主体面对同一情境将拥有相同行动理由；规范客观性的核心论证 | 《论重要之事》（2011-2017） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系类型 | 核心关联 |
|------|----------|----------|
| 康德（Immanuel Kant） | 哲学基础 | 帕菲特的"三重理论"直接重释康德的绝对命令；规范实在论继承了康德的理性主义传统 |
| 边沁（Jeremy Bentham） | 哲学基础 | 功利主义传统是帕菲特人口伦理学分析的对象；总体功利主义是"令人厌恶的结论"的来源 |
| 罗尔斯（John Rawls） | 同代影响与对话 | 罗尔斯的契约主义与正义论是帕菲特三重理论和规范伦理学对话的重要对象 |
| 洛克（John Locke） | 哲学史对话 | 洛克的记忆理论是帕菲特心理连续性理论的历史先驱；帕菲特将洛克的记忆标准发展为更系统的还原论框架 |
| 休谟（David Hume） | 哲学史对话 | 休谟对自我实体性的怀疑是帕菲特还原论自我观的哲学史先驱；帕菲特以分析哲学语言重述了休谟式的自我怀疑论 |
| 西季威克（Henry Sidgwick） | 间接前承 | 西季威克的实践理性二元论与功利主义为帕菲特的伦理学分析提供了历史语境 |

### 4.2 同代

| 人物 | 关系类型 | 核心关联 |
|------|----------|----------|
| 内格尔（Thomas Nagel） | 同代对话 | 两人在规范伦理学、客观性与个人同一性问题上存在相互影响与对话；共享分析哲学的方法论但走向不同形而上学结论 |
| 德沃金（Ronald Dworkin） | 同代对话 | 德沃金的资源平等与帕菲特的规范实在论在自由主义传统中形成平行竞争；德沃金对帕菲特的三重理论提出内部批评 |
| 斯坎伦（T.M. Scanlon） | 理论对话与批评 | 斯坎伦的契约主义是帕菲特三重理论的重要组成部分；斯坎伦本人对帕菲特的"收敛"诠释提出批评 |
| 辛格（Peter Singer） | 同代对话与支持 | 辛格作为功利主义传统的代表人物支持帕菲特的"三重理论"，认为其展示了伦理学综合的可能性 |
| 威廉姆斯（Bernard Williams） | 早期影响与后期分歧 | 威廉姆斯早期关于个人同一性的论文对帕菲特有直接影响；但两人在"后果主义"与"内在理由"问题上存在根本分歧 |
| 诺齐克（Robert Nozick） | 理论对话 | 诺齐克的"最接近连续者理论"是帕菲特还原论自我观的直接批评对象；两人在个人同一性问题上处于对立立场 |
| 麦凯（J.L. Mackie） | 间接对话 | 麦凯的"错误理论"（error theory）是帕菲特规范实在论的直接对立面；帕菲特在《论重要之事》中系统回应了错误理论 |

### 4.3 后启

| 人物/领域 | 关系类型 | 核心关联 |
|-----------|----------|----------|
| 卡根（Shelly Kagan） | 直接后启 | 卡根的后果主义伦理学以内格尔和帕菲特的问题框架为基础，系统发展极端后果主义的论证 |
| 当代人口伦理学 | 问题框架提供 | 帕菲特的"非同一性问题"与"令人厌恶的结论"成为当代人口伦理学研究的中心议题；阿伯萨尼的不可能定理直接建立在其问题框架上 |
| 当代规范伦理学 | 理论框架提供 | 帕菲特的"三重理论"为当代伦理学三大传统的综合研究提供了基础框架；尽管争议巨大，但已成为不可回避的议题 |
| 当代元伦理学 | 理论框架提供 | 帕菲特的规范实在论与"理性收敛"论证成为当代元伦理学中认知主义与反认知主义之争的核心参照 |
| 阿伯萨尼（Gustaf Arrhenius） | 直接后启 | 阿伯萨尼在帕菲特的问题框架中证明了一系列"不可能定理"，将人口伦理学问题数学化 |
| 当代神经伦理学 | 问题推动 | 帕菲特的还原论自我观与心理连续性理论为当代神经伦理学（如脑机接口、人格上传）提供了哲学基础 |
| 实验哲学 | 方法批判与对话 | 帕菲特对思想实验直觉的依赖成为实验哲学批判的对象；温伯格、尼科尔斯和斯蒂克等人的跨文化直觉研究直接以其论证为靶子 |
