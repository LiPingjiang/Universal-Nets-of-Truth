## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 正义即公平 | 正义原则应通过一种公平的选择程序（原初状态）来确定，而非诉诸功利计算或直觉 | 《正义论》 |
| 原初状态与无知之幕 | 一种假设性选择情境：各方在无知之幕的遮蔽下选择正义原则，保证选择的中立性和公平性 | 《正义论》 |
| 两个正义原则 | （1）平等的基本自由原则；（2）差别原则与公平的机会平等原则。两原则间有词典式优先关系 | 《正义论》 |
| 差别原则 | 社会和经济的不平等必须对最不利者最有利；自然天赋视为"共同资产" | 《正义论》 |
| 反思平衡 | 在深思熟虑的判断与原则之间双向调整，直到两者达成一致的方法论 | 《正义论》 |
| 政治的而非形而上学的 | 正义观念是"独立站立的"，不依赖任何特定形而上学或整全学说 | 《政治自由主义》 |
| 重叠共识 | 不同整全学说出于各自理由支持同一政治正义观念，提供基于正当理由的稳定性 | 《政治自由主义》 |
| 公共理性 | 公民在讨论基本政治问题时应提供彼此可以合理接受的公共理由 | 《政治自由主义》 |
| 合理多元论 | 自由制度下理性运作的自然结果，表现为多种合理但互不相容的整全学说并存 | 《政治自由主义》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 | 具体影响 |
|------|------|----------|
| 康德（Immanuel Kant, 1724—1804） | 核心理论先驱 | 罗尔斯的正义即公平深受康德道德哲学影响：人作为自由平等的理性存在者、目的自身、自律等概念是原初状态设计的深层依据。罗尔斯自述其理论是"康德式建构主义"的一种形式 |
| 卢梭（Jean-Jacques Rousseau, 1712—1778） | 社会契约传统先驱 | 卢梭的"公意"（volonté générale）和"社会契约"概念为罗尔斯的原初状态提供了社会契约论的方法论框架。罗尔斯在《道德哲学史讲义》中对卢梭有专门讨论 |
| 洛克（John Locke, 1632—1704） | 自然权利传统先驱 | 洛克的自然权利理论和社会契约论为罗尔斯第一正义原则（平等的基本自由）提供了历史根源。罗尔斯将自己置于洛克式的"自然自由体系"的批评者位置 |
| 密尔（J. S. Mill, 1806—1873） | 功利主义的对话者 | 罗尔斯与密尔的功利主义进行持续对话。他承认密尔的贡献，但系统批评功利主义允许为总体福利牺牲个体的缺陷。密尔的自由原则也被罗尔斯整合进第一正义原则 |
| 亚里士多德（Aristotle, 384—322 BCE） | 远程理论背景 | 罗尔斯在《正义论》中讨论了"亚里士多德原则"——人们偏好运用复杂能力而非简单能力的倾向——作为其善理论的一部分 |

### 4.2 同代

| 人物 | 关系 | 具体互动 |
|------|------|----------|
| 诺齐克（Robert Nozick, 1938—2002） | 最主要的论敌 | 诺齐克在《无政府、国家与乌托邦》（1974）中从自由意志论角度批评差别原则侵犯个人权利。两人的辩论构成了20世纪政治哲学最著名的交锋 |
| 德沃金（Ronald Dworkkin, 1931—2013） | 同盟与批评者 | 德沃金在资源平等理论方面与罗尔斯既有共识也有分歧。他支持罗尔斯对功利主义的批评，但在平等的具体实现上提出了不同的"拍卖"模型 |
| 桑德尔（Michael Sandel, 1953— ） | 共同体主义批评者 | 桑德尔在《自由主义与正义的局限》（1982）中批评罗尔斯的"无约束的自我"预设，引发了自由主义与共同体主义的著名论战 |
| 沃尔泽（Michael Walzer, 1935— ） | 复合平等论者 | 沃尔泽在《正义诸领域》（1983）中提出"复合平等"理论，批评罗尔斯寻求普遍正义原则的企图，主张正义应区分不同社会善的分配领域 |
| 森（Amartya Sen, 1933— ） | 能力方法批评者 | 森批评罗尔斯的"基本善"框架忽视了人们将资源转化为实际功能的能力差异，并提出"能力方法"作为替代。两人的争论是当代正义理论的核心议题之一 |

### 4.3 后启

| 人物/流派 | 关系 | 具体影响 |
|-----------|------|----------|
| 当代政治哲学整体 | 理论框架的设定者 | 罗尔斯的框架成为当代几乎所有政治哲学讨论的出发点：无论是自由主义内部还是外部，学者都在回应罗尔斯提出的问题 |
| 科恩（G. A. Cohen, 1941—2009） | 马克思主义批评者 | 科恩在《拯救正义与平等》（2008）中批评差别原则允许激励性不平等，主张更激进的平等主义 |
| 努斯鲍姆（Martha Nussbaum, 1947— ） | 能力方法的发展者 | 努斯鲍姆发展了森的能力方法，并将其与罗尔斯框架系统比较，形成了当代正义理论中最富成果的对话之一 |
| 波格（Thomas Pogge, 1953— ） | 全球正义理论家 | 波格纳将罗尔斯的正义理论推向全球正义领域，批评罗尔斯在《万民法》中的立场过于保守 |
| 斯坎伦（T. M. Scanlon, 1940— ） | 契约论的发展者 | 斯坎伦发展了一种不同于罗尔斯的契约论——基于"不可合理拒绝的原则"的道德契约论 |
| 当代共和主义 | 对话者 | 佩迪特（Philip Pettit）等人的共和主义理论在与罗尔斯自由主义的对话中发展起来 |

## 五、本章定位

| 字段 | 内容 |
|------|------|
| 所属章节 | 第十二章：正义与共同体——当代政治哲学 |
| 章节小节 | 第一节：正义即公平——罗尔斯的正义理论 |
| 思维流向 | 从"什么是正义的社会基本结构"出发，经由原初状态的设计和两个正义原则的推导，到合理多元论条件下的政治自由主义，最终延伸至国际正义 |
| 核心张力 | 自由与平等的张力；普遍正义与多元论实的张力；个人权利与社会合作的张力 |
| 与前人物关系 | 继承康德、卢梭、洛克的社会契约传统，同时批判功利主义的分配逻辑 |
| 与后人物关系 | 为诺齐克（自由意志论）、桑德尔（共同体主义）、德沃金（资源平等）、森（能力方法）等人的理论提供了回应的靶心 |
| 章节内角色 | 本章的"坐标原点"——后续人物的理论几乎都是对罗尔斯的回应、修正或反对 |
