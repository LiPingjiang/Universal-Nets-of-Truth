# 普兰丁格 · 元信息

## 一、人物基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 阿尔文·卡尔·普兰丁格（Alvin Carl Plantinga） |
| 生卒年 | 1932年11月15日— |
| 国籍 | 美国 |
| 主要身份 | 哲学家、宗教哲学家、模态逻辑学家、知识论家 |
| 学派归属 | 分析哲学、改革宗知识论（Reformed Epistemology）、基督教分析神学 |

## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 《上帝与其他心灵》 | God and Other Minds | 1967 | 以他心问题为类比，论证对上帝存在的信念与他心信念具有同等认识论地位 |
| 《必然性的本质》 | The Nature of Necessity | 1974 | 模态逻辑与形而上学，可能世界语义学，本体论论证的模态重构 |
| 《上帝、自由与恶》 | God, Freedom, and Evil | 1977 | 自由意志辩护，对恶的逻辑问题的系统回应 |
| 《保证：当前的争论》 | Warrant: The Current Debate | 1993 | 保证理论的系统阐述，区分保证与辩护 |
| 《保证与恰当功能》 | Warrant and Proper Function | 1993 | 以恰当功能为核心的知识论框架 |
| 《保证的基督教信仰》 | Warranted Christian Belief | 2000 | 改革宗知识论的完整表述，扩展的A/C模型 |
| 《冲突的真正所在》 | Where the Conflict Really Lies: Science, Religion, and Naturalism | 2011 | 反自然主义的进化论证（EAAN），科学与宗教关系的重新定义 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 改革宗知识论 | 对上帝的信仰可以是"恰当地基本的"，无需证据或论证即可理性持有，由神圣感应和圣灵工作产生 | 《保证的基督教信仰》（2000） |
| 自由意志辩护 | 恶的存在与上帝全能全善之间不存在逻辑矛盾；自由意志的宝贵价值为上帝创造可能产生恶的世界提供了充分理由 | 《上帝、自由与恶》（1977） |
| 反自然主义的进化论证（EAAN） | 自然主义与进化论的结合产生自我挫败的认识论问题；认知官能在自然主义框架下可靠性低 | 《冲突的真正所在》（2011） |
| 可能世界语义学与本体论论证 | 利用模态逻辑和可能世界语义学重构传统本体论论证，证明上帝必然存在 | 《必然性的本质》（1974） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系说明 |
|------|----------|
| 加尔文（John Calvin, 1509-1564） | 神圣感应（sensus divinitatis）概念的直接来源；加尔文在《基督教要义》中提出上帝赋予人认识上帝的内在官能 |
| 阿奎那（Thomas Aquinas, 1225-1274） | 自然神学传统，上帝存在的五路证明；普兰丁格扩展的A/C模型以阿奎那和加尔文命名 |
| 弗雷格（Gottlob Frege, 1848-1925） | 分析哲学与逻辑学的奠基人；普兰丁格继承了弗雷格以来的分析哲学方法 |
| 克里普克（Saul Kripke, 1940-） | 模态逻辑与可能世界语义学的先驱；普兰丁格将克里普克的模态逻辑工具应用于宗教哲学 |

### 4.2 同代

| 人物 | 关系说明 |
|------|----------|
| 斯温伯恩（Richard Swinburne, 1934-） | 同为当代分析宗教哲学的领军人物，但斯温伯恩更强调证据主义的有神论证明，与普兰丁格的基础主义形成互补 |
| 蒂利希（Paul Tillich, 1886-1965） | 存在主义神学传统，对宗教语言的分析深刻影响了普兰丁格对信仰与理性关系的思考 |
| 玛丽莲·亚当斯（Marilyn McCord Adams, 1943-2017） | 分析宗教哲学家，对恶的问题提出了基督论解决方案；与普兰丁格在恶的问题上既有互补也有分歧 |
| 沃尔特斯托夫（Nicholas Wolterstorff, 1932-） | 同为改革宗知识论的代表人物，与普兰丁格合编《信仰与理性》（1983） |

### 4.3 后启

| 人物/运动 | 关系说明 |
|------|----------|
| 当代分析宗教哲学 | 普兰丁格的工作奠定了当代分析宗教哲学的方法论基础，深刻影响了1970年代以来的宗教哲学研究范式 |
| 改革宗知识论运动 | 普兰丁格是改革宗知识论（1983年命名）的奠基人，该运动影响了包括威廉·阿尔斯通（William Alston）等一大批哲学家 |
| 保证理论的发展 | 普兰丁格的"保证"（warrant）概念成为当代认识论的核心议题之一，影响超出宗教哲学领域 |
| 科学与宗教对话 | 普兰丁格的EAAN论证重新定义了科学与宗教对话的框架，影响了当代科学哲学中的自然主义辩论 |
