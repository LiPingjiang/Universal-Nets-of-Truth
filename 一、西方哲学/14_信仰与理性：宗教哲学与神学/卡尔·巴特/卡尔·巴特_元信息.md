# 卡尔·巴特 · 元信息

## 一、人物基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 卡尔·巴特（Karl Barth） |
| 生卒年 | 1886—1968 |
| 国籍 | 瑞士 |
| 主要身份 | 新教神学家、系统神学家、牧师、教授 |
| 学派归属 | 新正统神学（Neo-orthodoxy）/ 辩证神学（Dialectical theology） |

## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 《罗马书释义》 | Der Römerbrief | 1919（第一版）/ 1922（第二版） | 上帝的超越性、无限的质的区别、辩证神学方法 |
| 《基督教教义学纲要》 | Die christliche Dogmatik im Entwurf | 1927 | 从辩证神学向系统教义学的过渡 |
| 《教会教义学》 | Kirchliche Dogmatik | 1932—1967（共十三卷半） | 三位一体、上帝之道、拣选、创世、和解——系统神学全面建构 |
| 《巴门神学宣言》 | Theologische Erklärung von Barmen | 1934 | 反纳粹、拒绝自然神学、基督为唯一启示 |
| 《新教神学：19世纪》 | Die protestantische Theologie im 19. Jahrhundert | 1947 | 对施莱尔马赫以降自由派神学的批判性回顾 |
| 《上帝的人性》 | Die Menschlichkeit Gottes | 1956 | 上帝的神性包含人性，基督中心论的辩证深化 |
| 《福音派神学：导论》 | Einführung in die evangelische Theologie | 1963 | 神学方法论、神学家的使命与责任 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 新正统神学 | 反对自由派神学将上帝内化于人类经验，主张上帝是"全然他者"，启示是上帝主动的自我显明 | 《罗马书释义》 |
| 辩证神学 | 上帝的启示既是显现又是隐藏，上帝之道既是"是"也是"否"，人类关于上帝的陈述始终是辩证的 | 《罗马书释义》 |
| 上帝之道三重形态 | 启示的道（基督）、成文的道（圣经）、宣讲的道（教会传讲），三者构成动态循环 | 《教会教义学》I/1 |
| 基督中心论的拣选论 | 拣选的主体是耶稣基督而非"某些人"，基督既是拣选的上帝也是被拣选的人 | 《教会教义学》II/2 |
| 反自然神学 | 拒绝analogia entis（存在之类比），主张analogia fidei（信仰之类比），否定人可从自然推论上帝 | 《巴门宣言》、《教会教义学》II/1 |
| 上帝的人性 | 上帝的神性包含其"人性"，上帝越是上帝就越与人同在 | 《上帝的人性》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 | 影响要点 |
|------|------|----------|
| 克尔凯郭尔（Søren Kierkegaard） | 方法论先驱 | "无限的质的区别"概念、间接沟通、悖论性信仰 |
| 路德（Martin Luther） | 神学传统渊源 | "隐藏的上帝"（Deus absconditus）、唯独恩典、唯独基督 |
| 加尔文（John Calvin） | 改革宗传统渊源 | 上帝的主权与荣耀、启示的优先性（巴特重构了加尔文的拣选论） |
| 施莱尔马赫（Friedrich Schleiermacher） | 批判对象 | 巴特全面反对其将上帝内化于宗教意识的神学进路 |

### 4.2 同代

| 人物 | 关系 | 互动要点 |
|------|------|----------|
| 布尔特曼（Rudolf Bultmann） | 早期同袍，后期分歧 | 共同发起辩证神学运动，后在神话解构与历史耶稣问题上分道扬镳 |
| 蒂利希（Paul Tillich） | 平行与竞争 | 蒂利希的"存在之类比"与巴特的"信仰之类比"形成根本对峙 |
| 布鲁纳（Emil Brunner） | 论战对手 | 1934年自然神学论战，巴特以"Nein!"全面拒绝布鲁纳的"接触点"理论 |
| 朋霍费尔（Dietrich Bonhoeffer） | 同道与批评者 | 共同抵抗纳粹，但朋霍费尔批评巴特过于强调启示而忽视"此世"的责任 |

### 4.3 后启

| 人物/流派 | 关系 | 影响要点 |
|-----------|------|----------|
| 后自由主义神学（Lindbeck, Frei） | 继承与转化 | 将巴特的基督中心论与文化-语言学进路结合，发展"教义-语言"模型 |
| 当代系统神学（Jüngel, Webster, McCormack） | 直接继承 | 在拣选论、三位一体论、神学方法论上深化巴特遗产 |
| 莫尔特曼（Jürgen Moltmann） | 批判性继承 | 接受基督中心论但转向"十字架神学"和"希望神学"，强调历史中的能动性 |
| 激进正统（Radical Orthodoxy, Milbank） | 部分继承与批评 | 赞赏其反世俗化立场，但批评其排斥自然神学导致神学与公共理性隔绝 |

## 五、本章定位

| 项目 | 内容 |
|------|------|
| 所属章节 | 第十四章：信仰与理性——宗教哲学与神学 |
