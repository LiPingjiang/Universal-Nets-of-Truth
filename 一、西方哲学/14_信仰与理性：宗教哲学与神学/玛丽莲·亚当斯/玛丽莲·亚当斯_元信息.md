## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 恶的崇高理论 | 恶的崇高指使受害者生命在表面上失去一切积极意义的极端恶，对传统神义论的补偿策略构成不可逾越的挑战 | 《恶的崇高与上帝之善》（1999） |
| 参与式神义论 | 上帝对恶的态度不是超然地"允许"，而是亲身地"参与"；通过基督的道成肉身和十字架，上帝亲自进入人类苦难 | 《基督与恶的崇高》（2006） |
| 基督论解决方案 | 查尔西顿基督论（神人二性）为恶的崇高提供具体解决方案：基督既承受人类苦难（人性），又赋予其神圣意义（神性） | 《基督与恶的崇高》（2006） |
| 非比性的上帝之善 | 上帝之善不是道德善，而是超越人类道德范畴的"非比性的善"——善本身的源泉，在于与受造物的爱的关系和转化恶的能力 | 《恶的崇高与上帝之善》（1999） |
| 转化（defeat）理论 | 恶不是被"补偿"（用其他善抵消），而是被"转化"（恶本身被整合到积极的关系或意义中） | 《恶的崇高与上帝之善》（1999） |
| 与上帝的亲密关系 | 转化恶的崇高的终极方式不是解释、补偿或正义，而是上帝与受害者之间的亲密关系，使苦难成为关系的途径 | 《基督与恶的崇高》（2006） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系说明 |
|------|----------|
| 奥古斯丁（Augustine of Hippo, 354-430） | 恶作为"善的缺乏"（privatio boni）的理论；亚当斯的"转化"概念受奥古斯丁"恶被善克服"思想的影响，但超越了奥古斯丁的"补偿"框架 |
| 阿奎那（Thomas Aquinas, 1225-1274） | 自然神学传统和存在论框架；阿奎那对上帝属性的分析（全能、全善、全知）是亚当斯讨论上帝之善的前承 |
| 普兰丁格（Alvin Plantinga, 1932-） | 自由意志辩护对恶的逻辑问题的回应；亚当斯接受了普兰丁格的逻辑成果，但认为恶的问题不仅是逻辑问题，更是存在论和情感问题 |
| 奥卡姆（William of Ockham, c.1287-1347） | 亚当斯中世纪哲学研究的专精对象；奥卡姆的唯名论、逻辑学和基督论对亚当斯的哲学方法产生了深远影响 |
| 加尔文（John Calvin, 1509-1564） | 改革宗神学传统中的苦难观和神佑论；亚当斯作为圣公会牧师，继承了改革宗和天主教传统的神学资源 |

### 4.2 同代

| 人物 | 关系说明 |
|------|----------|
| 斯温伯恩（Richard Swinburne, 1934-） | 同为分析宗教哲学家，但斯温伯恩从概率和证据角度论证上帝存在，亚当斯从存在论和情感角度回应恶的问题；二者在恶的问题上有互补关系 |
| 范胡尔斯特（Henri Blocher, 1939-） | 改革宗神学家，对恶的问题和神佑论有深入研究；与亚当斯在改革宗传统中有神学对话 |
| 希克（John Hick, 1922-2012） | 宗教多元论和"灵魂塑造"神义论（soul-making theodicy）的代表人物；亚当斯批判了希克的"补偿式"灵魂塑造理论，认为其不足以回应恶的崇高 |
| 罗伯特·亚当斯（Robert Merrihew Adams, 1937-） | 玛丽莲·亚当斯的丈夫，著名哲学家，在莱布尼茨哲学和道德哲学方面有深入研究；二人合编了《恶的问题》论文集（1990） |

### 4.3 后启

| 人物/运动 | 关系说明 |
|------|----------|
| 当代恶的问题研究 | 亚当斯的"恶的崇高"概念已成为恶的问题研究的标准术语，深刻影响了后来的分析宗教哲学家和神学家 |
| 女性主义神学 | 作为分析宗教哲学中最杰出的女性学者，亚当斯为女性主义神学在分析哲学中的发展做出了重要贡献，影响了后来的埃琳·温德尔、萨拉·科克利等学者 |
| 苦难神学与创伤神学 | 亚当斯的参与式神义论深刻影响了当代苦难神学（pastoral theology）、创伤疗愈神学和政治神学中的苦难观 |
| 政治神学（解放神学） | 亚当斯对上帝与受害者"同在"的强调，与解放神学对上帝与穷人、被压迫者同在的强调产生了共鸣和对话 |
| 基督论研究 | 亚当斯的《基督与恶的崇高》推动了当代分析神学中基督论与恶的问题的交叉研究，影响了后来的大卫·本特雷·哈特、托马斯·温格尔等学者 |
