## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 言语行为理论 | 说话就是做事——语言不只描述世界还改变世界 | 《言语行为》 |
| 言外行为分类 | 断言类、指令类、承诺类、表达类、宣告类 | 《表达式与意义》 |
| 构成性规则 | 创造活动本身的规则——"X counts as Y in C" | 《言语行为》 |
| 制度性事实 | 依赖集体意向性的事实——货币、婚姻、所有权 | 《社会实在的建构》 |
| 集体意向性 | "我们"做某事的原始能力——不可还原为个体意向性 | 《社会实在的建构》 |
| 中文屋论证 | 语法不等于语义——程序本身不产生理解 | "心灵、大脑与程序"（1980） |
| 生物自然主义 | 意识是大脑的生物学性质——不可还原为计算 | 《心灵的再发现》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 奥斯汀 | 塞尔的博士导师——言语行为理论直接继承自奥斯汀 |
| 维特根斯坦 | 后期维特根斯坦的"语言游戏"影响了塞尔的言语行为理论 |
| 格赖斯 | 格赖斯的"说话者意义"理论是塞尔言语行为理论的前驱 |
| 普特南 | 在心灵哲学和指称理论上有交叉——塞尔批评功能主义，普特南后来也放弃了 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 戴维森 | 在行动哲学和语言哲学上有交叉但路径不同 |
| 普特南 | 在心灵哲学上——塞尔的中文屋论证与普特南的功能主义形成对照 |
| 丹尼特 | 在意识哲学上——丹尼特代表"计算主义"，塞尔代表"生物自然主义" |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 吉尔伯特 | 发展了集体意向性理论——但在还原vs非还原问题上有分歧 |
| 图勒加尔 | 社会本体论的重要发展者——受塞尔影响 |
| 当代AI哲学 | 中文屋论证仍是AI意识讨论的核心论证 |
