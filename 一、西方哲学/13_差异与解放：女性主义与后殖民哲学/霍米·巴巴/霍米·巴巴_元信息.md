## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 模拟 | 被殖民者模仿殖民者文化产生"几乎是但不完全是"的暧昧效果 | 《文化的定位》 |
| 混杂性 | 殖民接触产生文化混合，颠覆文化纯粹性的虚构 | 《文化的定位》 |
| 第三空间 | 殖民者与被殖民者话语交汇的暧昧区域，二元对立在此瓦解 | 《文化的定位》 |
| 殖民话语的暧昧性 | 殖民权力同时具有支撑与威胁自身的内在矛盾 | 《文化的定位》 |
| 文化翻译 | 后殖民主体通过挪用和变形殖民话语来行使能动性 | 《文化的定位》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 萨义德 | 巴巴继承萨义德的殖民话语分析，但批评其忽视话语内部的矛盾 |
| 福柯 | 话语/权力理论为巴巴的殖民话语分析提供方法论资源 |
| 德里达 | 解构主义的延异和踪迹概念为巴巴的混杂性和第三空间提供灵感 |
| 拉康 | 精神分析的"他者"和"面具"概念影响巴巴的模拟理论 |
| 法侬 | 巴巴重新解读法侬的后殖民主体理论，强调文化策略而非暴力革命 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 斯皮瓦克 | 同为后殖民理论核心人物，但斯皮瓦克更关注底层和认识论问题 |
| 萨义德 | 巴巴对萨义德的东方主义提出了重要的内部修正 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 当代后殖民研究 | 混杂性和第三空间成为后殖民文学和文化研究的核心分析工具 |
| 全球化研究 | 巴巴的文化翻译概念被运用于全球化语境中的文化流动分析 |
