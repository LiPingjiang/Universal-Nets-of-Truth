## 五、核心论点

1. "女人不是天生的，而是后天形成的"——"女性气质"是社会建构——不是自然事实——建构如此深——深到被误认为"自然"
2. 女性是"他者"——男性是"主体"——但女性的"他者"地位是被迫的——不是自由选择的
3. 女性被困在"内在性"——男性被推向"超越性"——解放是恢复"超越性"——重新获得介入世界的能力
4. 自由不是"绝对的"——是"在处境中的"——处境限制自由但也使自由有意义——对萨特"绝对自由"的修正
5. 解放不是"变成男人"——而是"互为主体"——两性共同走向"自由-自由"——前提是经济独立

## 六、思想关系

### 6.1 前承

| 人物 | 关系 |
|------|------|
| 萨特 | 终身伴侣和思想伙伴——存在主义是波伏瓦的哲学基础——但波伏瓦修正了萨特的"绝对自由" |
| 黑格尔 | "主-奴"辩证法——波伏瓦借用但修正——女性的"他者"处境比奴隶更复杂 |
| 海德格尔 | 存在论——"在世存在""筹划"等概念影响了存在主义框架 |
| 梅洛-庞蒂 | 身体现象学——影响了波伏瓦对"身体处境"的分析 |

### 6.2 同代

| 人物 | 关系 |
|------|------|
| 萨特 | 思想伙伴——但波伏瓦的哲学贡献是独立的——"处境中的自由"修正了"绝对自由" |
| 梅洛-庞蒂 | 身体现象学的对话——两人都关注"身体"与"处境"的关系 |
| 加缪 | 存在主义内部的分歧——加缪的"荒谬"与波伏瓦的"模糊性"有交叉 |

### 6.3 后启

| 人物/思潮 | 关系 |
|------|------|
| 巴特勒 | 巴特勒批评波伏瓦的"性别建构"论——但在此基础上发展了"性别操演"理论 |
| 斯皮瓦克 | 斯皮瓦克批评波伏瓦忽视了种族和阶级——但"他者"概念影响了后殖民女性主义 |
| 当代女性主义 | 波伏瓦是女性主义哲学的奠基者——所有后续女性主义都在她的基础上发展 |
| 交叉性理论 | 波伏瓦对种族和阶级的忽视推动了交叉性理论的发展 |
