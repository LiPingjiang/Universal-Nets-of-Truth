# 斯宾诺莎 · 解读（白话版）

## 一、这个人是谁？

巴鲁赫·斯宾诺莎（1632-1677）是一个荷兰人，出生在葡萄牙裔犹太商人家庭。他从小接受犹太教教育，学习了希伯来文、拉丁文和经院哲学。但他很快开始对犹太教教义产生怀疑，24岁时被阿姆斯特丹犹太教会**革出教门**——这是犹太教最严厉的惩罚，相当于被整个社会抛弃。

被开除后，斯宾诺莎拒绝了德国海德堡大学提供的教授职位（因为条件是要保持"宗教克制"），也拒绝了法国路易十四提供的丰厚年金（因为条件是要将下一部著作献给这位国王）。他选择靠**磨制光学镜片**为生，住在海牙附近的一个小房间里，过着极其简朴的生活。

他终身未婚，没有财产，朋友很少。他44岁时因吸入过多镜片打磨粉尘患上肺结核去世。他生前只出版了两部著作，死后朋友们秘密出版了他的最重要的著作——**《伦理学》**——一部用几何学方法证明的哲学体系。

| 人生阶段 | 事件 |
|----------|------|
| 1632 | 出生于阿姆斯特丹，葡萄牙裔犹太商人家庭 |
| 1654 | 父亲去世，继承家业但不久放弃 |
| 1656 | 被犹太教会革出教门（herem），终身不得与犹太人交往 |
| 1661-1675 | 撰写《伦理学》，反复修改，生前未出版 |
| 1662-1663 | 发表《知性改进论》和《笛卡尔哲学原理》 |
| 1670 | 匿名出版《神学政治论》，主张宗教宽容和民主共和 |
| 1673 | 拒绝海德堡大学哲学教授职位 |
| 1677 | 因肺病去世，享年44岁；朋友们秘密出版《遗著》 |

## 二、他发现了什么问题？

斯宾诺莎生活的时代，欧洲正被宗教战争撕裂。荷兰虽然相对宽容，但宗教和政治的冲突依然激烈。斯宾诺莎发现了两个根本问题：

**第一，笛卡尔的哲学是自相矛盾的。** 笛卡尔说世界有三种实体：上帝（无限的）、心灵（能思考的）、物质（有广延的）。但斯宾诺莎追问：如果心灵和物质是完全不同的实体，它们怎么能相互作用？笛卡尔说通过松果腺，但斯宾诺莎认为这根本不能解决问题——一个非物质的东西怎么可能推动一个物质的东西？

**第二，宗教成了政治冲突的根源。** 斯宾诺莎看到，欧洲数十年的宗教战争（如三十年战争）造成了巨大灾难。他认为，宗教冲突的根本原因是：人们把《圣经》当作真理的绝对来源，而不同的人对《圣经》有不同的解释。如果哲学和科学不能与宗教权威分离，和平和理性就不可能实现。

## 三、他说了什么？

### 3.1 只有一个东西真正存在——上帝就是自然

斯宾诺莎说：**世界上只有一个实体，这个实体就是上帝，也就是自然。** 这不是说"上帝创造了自然"，而是说"上帝就是自然本身"。

这听起来像是"无神论"，但斯宾诺莎认为自己不是无神论者。他只是重新定义了上帝：上帝不是在自然规律之外干预人事的人格神，而就是**自然本身的内在法则和力量**。就像重力不是某个神在拉东西，而是自然本身的规律一样。

斯宾诺莎打了一个比方：人们通常把上帝想象成一个国王，可以按照自己的意愿改变法律。但斯宾诺莎说，上帝更像是一个**自然法则本身**——国王可以改变法律，但自然法则不能改变。这个"上帝"是无限的、必然的、永恒的，但不是人格化的。

| 传统宗教观 | 斯宾诺莎的"上帝" |
|------------|------------------|
| 上帝在自然之外 | 上帝就是自然 |
| 上帝可以干预人事 | 上帝即自然法则，不可干预 |
| 上帝有意志和目的 | 上帝没有意志，只有必然性 |
| 上帝创造了世界 | 世界是上帝/自然的必然表现 |
| 人可以通过祈祷改变上帝 | 人只能通过理解自然来适应它 |

### 3.2 心灵和身体不是两个东西，而是同一东西的两种表现

斯宾诺莎对笛卡尔心身问题的解决方案非常独特。他说：**心灵和身体不是两个独立的实体，而是同一实体（上帝/自然）的两种属性。**

这就像一个硬币的两面：一面是"思维"（心灵），一面是"广延"（身体）。当你看到"正面"时，你同时也在看到"反面"——只是你只看了一面。同样，当你思考时，你的大脑也在发生物理过程；当你大脑发生物理过程时，你也在经历某种思维。两者之间没有因果关系，而是**同一过程的两种描述**。

这消解了笛卡尔的问题：不需要解释心灵如何影响身体，因为它们本来就是同一回事的两种说法。

### 3.3 一切事物都是必然的，自由就是理解必然

斯宾诺莎说：**世界上没有偶然，一切都是必然的。** 你的每一个想法、每一次呼吸、每一个决定，都是必然的——就像抛出的石头必然下落一样。

这听起来很可怕——"那我岂不是没有任何自由？" 但斯宾诺莎说，**自由不是"能够做不同的事"，而是"理解为什么只能做这件事"**。

打个比方：你掉进河里，被水流冲走。如果你不知道水流的方向，你会挣扎、恐惧、被淹死——这是"不自由"。但如果你理解了水流的方向和规律，你可以顺着水流游泳，最终到达安全的地方——这是"自由"。同样，当我们理解了情感、欲望、行为的必然原因，我们就从被动转为主动，从恐惧转为理解。

斯宾诺莎说："**自由人最少想到死，而他的智慧不是对死的沉思，而是对生的沉思。**"[^freedom]

### 3.4 情感是可以被理解的，理解情感就是解放

斯宾诺莎认为，我们的情感——爱、恨、恐惧、欲望——不是心灵的"自由行动"，而是**力量的变化**。每一事物（包括人）都有一种内在的努力：保存自己的存在。当这种力量增加时，我们感到快乐；当这种力量减少时，我们感到悲伤。

| 情感类型 | 例子 | 原因 |
|----------|------|------|
| 被动情感 | 恐惧、愤怒、嫉妒、悲伤 | 力量减少，由外部原因引起 |
| 主动情感 | 快乐、爱、欲望、宁静 | 力量增加，由内部理解引起 |

斯宾诺莎说，情感本身不是坏的。坏的是**不理解情感的原因**。如果我们知道为什么我们会恐惧，恐惧就会减少。如果我们知道为什么我们会嫉妒，嫉妒就会转化为理解。"**对情感的原因有充分知识的心灵，就是自由的。**"[^affect]

这听起来很像现代心理学——特别是认知行为疗法（CBT）的核心理念。

### 3.5 政治：思想自由不可剥夺

斯宾诺莎在《神学政治论》中提出了一个激进的政治观点：**在民主国家中，思想自由不仅不可剥夺，而且剥夺思想自由会危害国家本身。**

他的论证是这样的：

（1）国家的目的是保护人民的权利和安全。

（2）思想是内在的、无法被控制的东西——即使一个人被迫说某种观点，他内心的真实想法仍然无法被改变。

（3）如果国家试图强迫人们接受某种观点，人们会成为"被迫的仆人"，产生仇恨和反抗，最终导致国家的不稳定。

（4）因此，国家最安全的做法是让每个人自由思想和表达。

斯宾诺莎说：**"剥夺人们的思想自由，比剥夺他们的财产更危险。"** 这是启蒙运动和自由主义的重要先驱。

## 四、他做了什么综合？

斯宾诺莎的哲学是一个**综合的体系**，它试图将多个对立统一起来：

| 对立 | 斯宾诺莎的综合 |
|------|----------------|
| 上帝 vs. 自然 | 上帝就是自然（Deus sive Natura） |
| 心灵 vs. 身体 | 同一实体的两种属性（平行论） |
| 自由 vs. 必然 | 自由即理解必然 |
| 理性 vs. 情感 | 理性不是压抑情感，而是理解情感 |
| 个人 vs. 社会 | 个人权利不可让渡，国家必须保护思想自由 |
| 宗教 vs. 哲学 | 哲学高于宗教，但宗教可以作为大众的道德教育工具 |

斯宾诺莎用**几何学方法**来构建这个体系：就像欧几里得从公理出发推导出所有几何定理一样，斯宾诺莎从定义和公理出发，推导出关于上帝、心灵、情感和自由的全部命题。每本书都有"定义"、"公理"、"命题"、"证明"、"附释"和"绎理"。

这种写法很奇特，但斯宾诺莎的意图很明确：**哲学应该像数学一样确定和严密。** 他不希望读者"相信"他的观点，他希望读者"看到"这些观点必然为真——就像看到三角形内角和为180度一样。

## 五、为什么重要？

### 5.1 对哲学的影响

斯宾诺莎是**黑格尔之前最系统的哲学家**。他的实体论影响了整个德国古典哲学——黑格尔、谢林、费希特都必须在斯宾诺莎的实体论基础上建构自己的体系。马克思恩格斯称他为"近代辩证法之父"和"唯物主义的先驱"。

### 5.2 对科学的影响

爱因斯坦明确表示受到斯宾诺莎的影响。爱因斯坦说："**我信仰斯宾诺莎的上帝——即一个在自然法则的和谐中显现的上帝，而不是一个干预人类命运和行为的上帝。**" 斯宾诺莎的必然性观念与现代科学的因果性观念有深刻的亲缘性。

### 5.3 对心理学的影响

弗洛伊德称斯宾诺莎为"情感心理学的先驱"。现代认知行为疗法（CBT）的核心理念——"理解情感即改变情感"——可以直接追溯到斯宾诺莎。

### 5.4 对政治的影响

斯宾诺莎的宗教宽容论和民主共和论直接影响了启蒙运动。美国宪法第一修正案（保护言论和宗教自由）和法国人权宣言的思想渊源之一，就是斯宾诺莎在《神学政治论》中论证的思想自由不可剥夺。

### 5.5 对文学和艺术的影响

歌德、诺瓦利斯、乔治·艾略特、豪尔赫·路易斯·博尔赫斯等文学家和艺术家都深受斯宾诺莎影响。歌德说斯宾诺莎给了他"宁静和安详"。博尔赫斯在《斯宾诺莎》一诗中写道："斯宾诺莎磨制他的镜片，他磨制的不是镜片，而是上帝的明镜。"

---

## 六、一句话总结

**斯宾诺莎说：世界上只有一个实体——上帝即自然；心灵和身体是同一实体的两种表现；一切事物都按必然性运行，而自由不在于摆脱必然，而在于理解必然；通过理解情感的原因，我们将从被动转为主动，获得对上帝/自然的理智的爱——这种爱是最高和永恒的幸福。**

---

## 脚注

[^freedom]: **自由即理解必然**：斯宾诺莎《伦理学》第四部分命题67。斯宾诺莎认为自由不是意志自由（选择不同可能性的能力），而是对必然性的理解和接受。"自由人最少想到死，而他的智慧不是对死的沉思，而是对生的沉思。"（第四部分命题67附释）

[^affect]: **理解情感即解放**：斯宾诺莎《伦理学》第五部分命题3-4。斯宾诺莎认为，如果我们对情感的原因有充分的知识，情感就不再是被动的。"一个被决定去做某事的被动心灵，如果它对其必然性有充分的知识，则它的痛苦就会减少。"（第五部分命题3）
