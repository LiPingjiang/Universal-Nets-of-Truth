## 二、思想标签

| 标签 | 说明 |
|------|------|
| 理性主义 | 与笛卡尔、莱布尼茨并称17世纪理性主义三杰，主张以几何学方法建立哲学体系 |
| 泛神论/一元论 | 提出"上帝即自然"（Deus sive Natura），认为上帝与自然是同一实体 |
| 实体论 | 主张只有一个实体（上帝/自然），心灵与物质是同一实体的两种属性 |
| 几何学方法 | 以欧几里得几何学的公理化方法来构建哲学体系 |
| 情感理论 | 将情感（affectus）理解为力量增减的变化，提出通过理解情感来获得自由 |
| 决定论 | 一切事物都按必然性存在和运行，自由不在于摆脱必然性，而在于理解必然性 |
| 政治自由主义 | 主张宗教宽容、思想自由、民主共和 |
| 犹太哲学批判 | 从理性角度批判犹太教经典和教会权威，导致被犹太教会革出教门（1656） |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 实体（substantia） | 在自身内并通过自身被认识的存在；不依赖于他物而存在的终极实在；只有一个实体——上帝/自然 | 《伦理学》第一部分，命题1-15 |
| 属性（attributum） | 实体的本质或内在结构；每个实体有无限多的属性，但人类只能认识两种：广延（思维）和思维（cogitatio） | 《伦理学》第一部分，定义4-6 |
| 样式（modus） | 实体的变形或状态；一切个别事物都是实体的样式；样式被实体决定，通过其他样式被认识 | 《伦理学》第一部分，定义5 |
| 自因（causa sui） | 自身即是自己存在的原因；上帝/实体是自因 | 《伦理学》第一部分，定义1 |
| 自由因（causa libera） | 只按照自己的本质的必然性而存在、不为他物决定的原因；上帝是唯一的自由因 | 《伦理学》第一部分，命题17 |
| 能动的自然/被分的自然（natura naturans / natura naturata） | 能动的自然：作为创造者的自然（上帝）；被分的自然：作为被造物的自然（一切样式） | 《伦理学》第一部分，命题29附释 |
| 自类因（causa sui） | 自身存在的原因；实体的定义性特征 | 《伦理学》第一部分，定义1 |
| 力量（potentia） | 每一事物保持自身存在的努力；力量的增减产生情感 | 《伦理学》第三部分，命题6-7 |
| 努力（conatus） | 每一事物保持自身存在的自然倾向；是情感和行为的基础 | 《伦理学》第三部分，命题6 |
| 情感（affectus） | 力量增减、身体变化的心灵知觉；分为被动情感（passiones）和主动情感（actiones） | 《伦理学》第三部分，定义3 |
| 对上帝的理智的爱（amor intellectualis Dei） | 从对上帝/自然的永恒必然性的理解中产生的最高的、最持久的情感；心灵的最高幸福 | 《伦理学》第五部分，命题32-36 |
| 永恒（aeternitas） | 不是时间的无限延续，而是存在本身的无时间性；心灵从永恒的观点（sub specie aeternitatis）理解事物 | 《伦理学》第五部分，命题23 |

## 四、多语言术语表

| 中文术语 | 拉丁文 | 英文 | 希腊文 | 备注 |
|----------|--------|------|--------|------|
| 实体 | substantia | substance | οὐσία (ousia) | 亚里士多德以来本体论核心概念，斯宾诺莎将其唯一化 |
| 属性 | attributum | attribute | - | 实体的本质结构，笛卡尔用此词表示广延和思维，斯宾诺莎认为实体有无限多属性 |
| 样式 | modus | mode / affection | τρόπος (tropos) | 实体的状态或变形，对应亚里士多德的accidens但含决定论含义 |
| 自因 | causa sui | self-caused | αἴτιον ἑαυτοῦ | 斯宾诺莎定义实体的核心概念，中世纪已有此概念 |
| 上帝即自然 | Deus sive Natura | God or Nature | - | 斯宾诺莎最著名的命题，表达泛神论/一元论立场 |
| 能动的自然 | natura naturans | naturing nature | - | 作为创造力量、自由因的自然（上帝本身） |
| 被分的自然 | natura naturata | natured nature | - | 作为被创造物的自然（一切样式和个体） |
| 努力 | conatus | striving / endeavour | ὁρμή (hormē) | 斯宾诺莎伦理学核心概念，每一事物保存自身存在的内在力量 |
| 情感 | affectus | affect / emotion | πάθος (pathos) | 区别于笛卡尔的"passions"，斯宾诺莎将其纳入力学决定论框架 |
| 对上帝的理智的爱 | amor intellectualis Dei | intellectual love of God | - | 斯宾诺莎伦理学的最高目标，一种永恒的、免于被动情感的宁静 |
| 从永恒的观点 | sub specie aeternitatis | under the aspect of eternity | - | 从必然性的整体视角理解事物，而非从有限时间的偶然视角 |
| 自由 | libertas | freedom / liberty | ἐλευθερία (eleutheria) | 不在于意志自由，而在于对必然性的理解 |
| 观念 | idea | idea | ἰδέα (idea) | 斯宾诺莎认为观念是思维属性的样式，观念与对象对应 |
| 真观念 | idea vera | true idea | - | 与对象符合的观念，但斯宾诺莎强调真观念的标准在于其内在 clarity |

| 充足理由 | ratio | reason | λόγος (logos) | 每一事物都有存在或必然存在的充足理由 |
| 必然性 | necessitas | necessity | ἀνάγκη (anankē) | 斯宾诺莎的决定论核心：一切事物都按必然性存在 |

## 五、核心命题

| 命题 | 内容 | 出处 |
|------|------|------|
| 实体唯一 | 除了上帝之外，不存在任何其他实体，也不能设想任何其他实体 | 《伦理学》第一部分，命题14 |
| 上帝即自然 | 上帝即自然（Deus sive Natura） | 《伦理学》第四部分，序言；《伦理学》多处 |
| 属性平行 | 思维属性的秩序和联系与广延属性的秩序和联系是同一的 | 《伦理学》第二部分，命题7 |
| 决定论 | 一切事物都由上帝的无限本性的必然性所决定，按永恒和必然的方式存在和运行 | 《伦理学》第一部分，命题29 |
| 努力即力量 | 每一事物只要是在其自身内，只要不被他物决定，它必然尽力保持其存在 | 《伦理学》第三部分，命题6 |
| 情感决定论 | 情感的主动和被动不在于情感的本质，而在于其力量的外部或内部原因 | 《伦理学》第三部分，命题1-3 |
| 自由即理解 | 一个被决定去做某事的被动心灵，如果它对其必然性有充分的知识，则它的痛苦就会减少 | 《伦理学》第五部分，命题3-4 |
| 理智的爱 | 对上帝的理智的爱是上帝用以爱自己的那种爱 | 《伦理学》第五部分，命题36 |
| 思想自由 | 思想自由不仅不可剥夺，而且剥夺思想自由会危害国家 | 《神学政治论》第20章 |

## 六、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 巴门尼德/赫拉克利特（01本原与存在） | 斯宾诺莎的实体论是对巴门尼德"存在者存在"命题的近代化重述；斯宾诺莎的样式变化论吸收了赫拉克利特的流变思想，但将其纳入必然性框架 | 第一章：本原与存在 |
| 柏拉图/亚里士多德（01本原与存在） | 斯宾诺莎的实体概念融合了柏拉图的理念（永恒不变）与亚里士多德的实体（ousia），但否定亚里士多德的第一实体与第二实体的区分，将实体唯一化 | 第一章 |
| 笛卡尔（03理性与经验） | 斯宾诺莎接受笛卡尔的广延与思维的二分，但否定笛卡尔的三种实体说（心灵、物质、上帝），将其还原为同一实体的两种属性；批判笛卡尔的心身二元论和意志自由论 | 第三章（本章） |
| 马勒伯朗士（03理性与经验） | 马勒伯朗士的偶因论与斯宾诺莎的属性平行论都否定心身之间的直接因果互动，但马勒伯朗士以上帝的持续干预解释同步，斯宾诺莎以同一实体的两种属性消解问题 | 第三章（本章） |
| 莱布尼茨（03理性与经验） | 莱布尼茨的单子论是对斯宾诺莎一元论的反动，试图恢复个体实体的多元性，但保留了前定和谐的决定论框架 | 第三章（本章） |
| 霍布斯（03理性与经验） | 霍布斯的社会契约论和权力国家论影响了斯宾诺莎的政治哲学；斯宾诺莎在《神学政治论》中接受了霍布斯的权力让渡逻辑，但强调思想自由的不可让渡性 | 第三章（本章） |
| 奥古斯丁/托马斯·阿奎那（04信仰与理性） | 斯宾诺莎在《神学政治论》中直接批判圣经神学和教会权威，但他对上帝作为唯一实体的论证在结构上与托马斯·阿奎那的上帝存在证明有对话关系；斯宾诺莎的"对上帝的理智的爱"与奥古斯丁的"安享上帝"有亲缘性 | 第四章：信仰与理性 |
| 笛卡尔/马勒伯朗士（07心灵与物质） | 斯宾诺莎的属性平行论是笛卡尔心身问题的第三种解决方案（不同于偶因论和前定和谐）：心灵与身体是同一实体的两种属性，其秩序是同一的，不存在因果互动问题 | 第七章：心灵与物质：笛卡尔难题及其后裔 |
| 黑格尔（08主体与绝对） | 黑格尔明确将斯宾诺莎视为近代哲学中"实体"阶段的顶点，认为斯宾诺莎的实体缺乏主体性（自为存在）；黑格尔的"绝对精神"是对斯宾诺莎实体的辩证超越 | 第八章：主体与绝对 |
| 谢林（08主体与绝对） | 谢林的自然哲学直接受斯宾诺莎影响，试图将斯宾诺莎的"能动的自然"与费希特的"自我"相结合 | 第八章 |
| 叔本华/尼采（09自由与必然） | 叔本华将斯宾诺莎的"conatus"发展为"生命意志"（Will zum Leben）；尼采将斯宾诺莎的"力量"（potentia）和"自我保存"转化为权力意志（Wille zur Macht） | 第九章：自由与必然 |
| 弗洛伊德（09自由与必然） | 斯宾诺莎的情感决定论和对情感机制的分析被弗洛伊德视为精神分析的先驱；弗洛伊德称斯宾诺莎为"情感心理学的先驱" | 第九章 |
| 马克思（09自由与必然） | 马克思恩格斯将斯宾诺莎的实体论视为唯物主义的哲学表达；恩格斯称斯宾诺莎为"近代辩证法之父" | 第九章 |
| 爱因斯坦（09自由与必然） | 爱因斯坦明确表示受到斯宾诺莎的泛神论影响，称"我信仰斯宾诺莎的上帝"——即一个在自然法则中显现的上帝，而非干预人事的上帝 | 第九章 |

## 七、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对莱布尼茨的影响** | 莱布尼茨的单子论是对斯宾诺莎一元论的反动，试图恢复个体实体的多元性，但保留了前定和谐的决定论框架；莱布尼茨称斯宾诺莎的体系"辉煌但可怕" |
| **对德国古典哲学的影响** | 黑格尔将斯宾诺莎视为近代哲学中"实体"阶段的顶点；谢林的自然哲学直接受斯宾诺莎影响；费希特、谢林、黑格尔都必须在斯宾诺莎的实体论基础上建构自己的体系 |
| **对马克思主义的影响** | 马克思恩格斯将斯宾诺莎的实体论视为唯物主义的哲学表达；恩格斯称斯宾诺莎为"近代辩证法之父"；列宁称《伦理学》为"唯物主义的杰出著作" |
| **对浪漫主义的影响** | 歌德、诺瓦利斯、施莱格尔等浪漫主义者深受斯宾诺莎影响；歌德说斯宾诺莎给了他"宁静和安详"；诺瓦利斯称斯宾诺莎为"陶醉于上帝的人" |
| **对心理学的影响** | 斯宾诺莎的情感理论被弗洛伊德视为精神分析的先驱；弗洛伊德称斯宾诺莎为"情感心理学的先驱"；现代认知行为疗法（CBT）的某些理念可追溯至斯宾诺莎的"理解情感即解放" |
| **对政治哲学的影响** | 斯宾诺莎的宗教宽容论、思想自由论和民主共和论直接影响了启蒙运动；美国宪法第一修正案的思想渊源之一；对以赛亚·伯林的消极自由概念有影响 |
| **对科学思想的影响** | 爱因斯坦明确表示受到斯宾诺莎的泛神论影响；斯宾诺莎的决定论框架与现代科学的因果性观念有亲缘性 |
| **主要批评** | 黑格尔批评斯宾诺莎的实体"缺乏主体性"（"没有光的黑夜"）；莱布尼茨批评其否定个体实体和自由意志；雅各比批评其为"无神论"；教会将其著作列为禁书 |
| **学术评价** | 斯宾诺莎的哲学体系是西方哲学史上最具严密性和一致性的体系之一；其几何学方法虽然在哲学中难以完全实现，但体现了理性主义对知识确定性的追求；其政治哲学在自由主义传统中占有重要地位 |
