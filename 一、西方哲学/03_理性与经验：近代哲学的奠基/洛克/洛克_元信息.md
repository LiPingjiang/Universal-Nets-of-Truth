## 二、思想标签

| 标签 | 说明 |
|------|------|
| 经验主义奠基人 | 与培根、霍布斯并称英国经验主义传统，但洛克是系统化经验主义认识论的第一人 |
| 白板说 | 心灵出生时是一张白板（tabula rasa），一切知识来源于经验 |
| 自然权利理论 | 人人生而具有生命、自由和财产的自然权利，政府的存在是为了保护这些权利 |
| 社会契约论 | 政府的合法性来源于人民的同意（consent），人民有权推翻违背契约的政府 |
| 宗教宽容 | 主张宗教自由，但排除无神论和天主教（认为后者服从外国权力） |
| 自由主义先驱 | 对古典自由主义、宪政民主、人权理论产生了决定性影响 |
| 科学方法论 | 与波义耳、牛顿等科学家密切交往，主张以实验和观察为基础的科学方法 |
| 教育哲学 | 提出"白板说"基础上的教育理论，强调环境和教育对人的塑造作用 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 观念（idea） | 心灵在思维时的直接对象；观念是知识的唯一材料；分为简单观念和复杂观念 | 《人类理解论》第一卷，第一章 |
| 简单观念（simple idea） | 由感觉或反省直接获得的、不可再分的观念，如颜色、声音、硬度、运动等 | 《人类理解论》第二卷，第一章-第八章 |
| 复杂观念（complex idea） | 由心灵将简单观念组合、比较、抽象而形成的观念，如实体、模式、关系等 | 《人类理解论》第二卷，第十二章-第二十四章 |
| 第一性质（primary quality） | 物体本身固有的性质，不依赖于观察者；如广延、形状、运动、数目、不可入性 | 《人类理解论》第二卷，第八章 |
| 第二性质（secondary quality） | 物体借助于其微小部分的能力在我们心中产生感觉的能力；如颜色、声音、气味、味道 | 《人类理解论》第二卷，第八章 |
| 实体（substance/substratum） | 观念所依托的"某种东西"（something-I-know-not-what）；我们对实体没有清晰的观念 | 《人类理解论》第二卷，第二十三章 |
| 名义本质（nominal essence） | 我们用名称来标记的一组属性；我们对事物的分类依据的是名义本质 | 《人类理解论》第三卷，第三章 |
| 实在本质（real essence） | 事物内部真正存在的、产生其可感性质的内在结构；我们对实在本质的知识有限 | 《人类理解论》第三卷，第三章 |
| 白板（tabula rasa） | 心灵出生时是一张空白白板，没有任何天赋观念，一切知识来源于经验 | 《人类理解论》第一卷，第二章 |
| 自然状态（state of nature） | 没有政府的状态；人们根据自然法享有平等和自由；但自然状态缺乏公正的执行者 | 《政府论》下篇，第二章-第三章 |
| 自然权利（natural rights） | 人人生而具有的权利：生命、自由和财产；这些权利不可剥夺 | 《政府论》下篇，第二章 |
| 同意（consent） | 政府的合法性来源于人民的同意；未经同意的统治是不合法的 | 《政府论》下篇，第八章 |
| 财产权（property） | 人对自身劳动及其产物拥有权利；劳动是财产权的原始来源 | 《政府论》下篇，第五章 |
| 分权（separation of powers） | 立法权和执行权应当分离，以防止权力滥用 | 《政府论》下篇，第十二章-第十三章 |
| 宽容（toleration） | 国家不应干预个人宗教信仰，但宽容不应扩展到无神论和天主教 | 《论宗教宽容》 |

## 五、核心命题

| 命题 | 内容 | 出处 |
|------|------|------|
| 白板说 | 心灵出生时是一张白板，没有任何天赋观念，一切知识来源于经验 | 《人类理解论》第一卷，第二章 |
| 观念来源于感觉和反省 | 我们的一切观念要么来源于外部感觉，要么来源于内部反省；不存在天赋观念 | 《人类理解论》第二卷，第一章 |
| 第一性质与第二性质 | 第一性质是物体本身固有的；第二性质是物体产生感觉的能力，不是物体本身的性质 | 《人类理解论》第二卷，第八章 |
| 实体不可知 | 我们对实体只有模糊的观念，称之为"某种不知为何物的东西" | 《人类理解论》第二卷，第二十三章 |
| 自然权利 | 人人生而具有生命、自由和财产的自然权利，这些权利先于政府而存在 | 《政府论》下篇，第二章 |
| 劳动创造财产 | 人对自己的人身拥有所有权，因此对自己身体的劳动及其产物也拥有所有权 | 《政府论》下篇，第五章 |
| 同意是政府合法性的基础 | 政府的合法性来源于人民的同意；未经同意的统治是不合法的 | 《政府论》下篇，第八章 |
| 人民有权反抗暴政 | 当政府违背其保护人民权利的契约时，人民有权反抗甚至推翻政府 | 《政府论》下篇，第十九章 |
| 宗教宽容 | 国家不应干预个人宗教信仰，因为宗教事务的本质是良心事务，不可强制 | 《论宗教宽容》 |
| 思想自由 | 思想自由是不可剥夺的；没有人可以把自己的思想交给他人控制 | 《论宗教宽容》 |

## 六、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 培根（01本原与存在） | 洛克继承了培根的经验主义传统，主张以观察和实验为知识的基础；但洛克将培根的经验主义系统化，建立了完整的认识论体系 | 第一章：本原与存在 |
| 霍布斯（03理性与经验） | 洛克与霍布斯都主张社会契约论，但洛克认为自然状态不是战争状态，而是和平但不安全的状态；洛克反对霍布斯的绝对主权论，主张政府的权力应当有限 | 第三章（本章） |
| 斯宾诺莎（03理性与经验） | 洛克与斯宾诺莎都主张宗教宽容，但斯宾诺莎从理性主义出发（思想自由不可剥夺），洛克从经验主义出发（良心不可强制）；斯宾诺莎是决定论者，洛克强调自由意志 | 第三章（本章） |
| 笛卡尔（03理性与经验） | 洛克直接批判笛卡尔的天赋观念论，主张心灵出生时是一张白板；但洛克接受了笛卡尔的心灵-身体二元论（虽然在《人类理解论》中对实体持怀疑态度） | 第三章（本章） |
| 莱布尼茨（03理性与经验） | 莱布尼茨在《人类理解新论》中系统批判了洛克的《人类理解论》，主张天赋观念的存在；莱布尼茨认为心灵不是白板，而是"有纹理的大理石" | 第三章（本章） |
| 马勒伯朗士（03理性与经验） | 洛克与马勒伯朗士在观念论上有对话关系；洛克反对马勒伯朗士的"在上帝中看见"理论，认为观念来源于经验而非上帝 | 第三章（本章） |
| 贝克莱（03理性与经验） | 贝克莱的经验主义是对洛克的激进化：如果一切知识来源于经验，那么物质实体（substratum）就是不可知的，贝克莱进一步否认物质的存在 | 第三章（本章） |
| 休谟（03理性与经验） | 休谟将洛克的经验主义推向怀疑论的极端：如果一切知识来源于经验，那么因果关系、自我实体、外部世界都不能被证明存在；休谟的怀疑论是对洛克经验主义的逻辑后果 | 第三章（本章） |
| 托马斯·阿奎那（04信仰与理性） | 洛克的自然法理论继承了阿奎那的自然法传统，但将自然法世俗化，不再以上帝的启示为基础，而以人的理性为基础 | 第四章：信仰与理性 |
| 格劳秀斯（04信仰与理性） | 洛克的自然权利理论直接受格劳秀斯国际法理论的影响；格劳秀斯的自然法世俗化为洛克的政治哲学奠定了基础 | 第四章 |
| 孟德斯鸠（03理性与经验） | 孟德斯鸠的分权理论（立法、行政、司法三权分立）直接受到洛克分权思想（立法权和执行权分离）的影响 | 第三章（本章） |
| 卢梭（03理性与经验） | 卢梭的社会契约论受洛克影响，但卢梭认为自然状态是平等和自由的，而洛克认为自然状态已有财产不平等；卢梭主张"公意"，洛克主张个人权利 | 第三章（本章） |
| 康德（08主体与绝对） | 康德的"先天综合判断"可以被视为对洛克经验主义的回应：康德试图证明存在不来源于经验的知识（先天知识），以纠正洛克的极端经验主义 | 第八章：主体与绝对 |
| 美国独立宣言/法国人权宣言（09自由与必然） | 美国独立宣言和法国人权宣言直接受到洛克自然权利思想的影响；"生命、自由和追求幸福"是洛克"生命、自由和财产"的变体 | 第九章：自由与必然 |
| 以赛亚·伯林（09自由与必然） | 伯林的消极自由概念（免于干涉的自由）直接来源于洛克；伯林将洛克视为自由主义传统的奠基人之一 | 第九章 |
| 罗尔斯（09自由与必然） | 罗尔斯《正义论》中的"基本自由"概念和"原初状态"概念与洛克的自然权利和社会契约论有深刻渊源 | 第九章 |

## 七、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对英国光荣革命的影响** | 《政府论》下篇直接为1688年光荣革命提供了理论辩护；洛克与沙夫茨伯里伯爵密切合作，参与了反对斯图亚特王朝的政治活动 |
| **对美国独立和宪政的影响** | 美国独立宣言（1776）直接引用洛克的自然权利理论；美国宪法中的分权制衡和权利保护条款深受洛克影响；杰斐逊将洛克视为"美国民主之父"之一 |
| **对法国大革命的影响** | 法国人权宣言（1789）中的"自由、平等、财产"直接来源于洛克的自然权利理论；伏尔泰、狄德罗等启蒙思想家将洛克视为英国自由主义的典范 |
| **对经验主义的影响** | 洛克是系统化经验主义认识论的第一人；贝克莱和休谟都是洛克经验主义的直接继承者；休谟将洛克的经验主义推向怀疑论的极端 |
| **对政治哲学的影响** | 洛克被视为古典自由主义传统的奠基人；以赛亚·伯林、哈耶克、诺齐克等自由主义思想家都受洛克影响；罗尔斯的"基本自由"概念与洛克有深刻渊源 |
| **对教育哲学的影响** | 《教育漫话》强调环境和教育对人的塑造作用，对西方教育理论产生了深远影响；"白板说"成为环境决定论的理论基础 |
| **对宗教哲学的影响** | 《论宗教宽容》和《基督教的合理性》主张理性宗教，反对教会权威；对自然神论和英国国教自由主义的发展有重要影响 |
| **主要批评** | 莱布尼茨批判其否定天赋观念；贝克莱批判其保留不可知的物质实体；休谟批判其保留不可知的因果性和自我；马克思主义批判其资产阶级自由主义立场；女性主义者批判其"家长制"自由主义 |
| **学术评价** | 洛克是西方哲学史上最具影响力的哲学家之一；其经验主义认识论虽被后世批判，但确立了经验主义作为哲学方法的重要地位；其政治哲学是自由主义传统的奠基之作，至今仍有现实意义 |

## 八、研究文献

- Peter H. Nidditch 编，*An Essay Concerning Human Understanding*，Clarendon Press，1975. —— 《人类理解论》标准学术版，附详细的校勘和注释。
- Peter Laslett 编，*Two Treatises of Government*，Cambridge University Press，1960. —— 《政府论》标准学术版，含重要的历史导论和版本考证。
- John W. Yolton, *Locke and the Compass of Human Understanding*，Cambridge University Press, 1970. —— 洛克认识论的系统研究。
- John W. Yolton, *Locke: An Introduction*，Blackwell, 1985. —— 洛克哲学的入门导读。
- Michael Ayers, *Locke: Epistemology and Ontology*，Routledge, 1991，2卷. —— 分析哲学视角对洛克最全面的研究之一。
- John Dunn, *The Political Thought of John Locke*，Cambridge University Press, 1969. —— 洛克政治思想的历史语境研究。
- Richard Ashcraft, *Revolutionary Politics and Locke's Two Treatises of Government*，Princeton University Press, 1986. —— 从政治史角度重新解读《政府论》。
- Patrick Mackenzie, *Locke*，Oxford University Press, 1991. —— 牛津通识读本系列中的洛克导读。
- 关文运译，《人类理解论》，商务印书馆，1981. —— 中文世界最经典的译本。
- 叶启芳、瞿菊农译，《政府论》上下篇，商务印书馆，1982. —— 中文世界最经典的译本。
- 赵敦华，《西方哲学简史》，北京大学出版社，2001. —— 洛克专节，适合作为入门参考。
- 邓晓芒，《西方哲学史》，高等教育出版社，2005. —— 洛克专节。
