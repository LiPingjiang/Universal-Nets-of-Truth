# 洛克 · 解读（白话版）

## 一、这个人是谁？

约翰·洛克（1632-1704）是英国人，他的人生经历非常丰富多彩——他不仅是哲学家，还是医生、政治家和教育家。

洛克出生于英格兰萨默塞特郡的一个清教徒家庭。他父亲是乡村律师，曾在英国内战中为议会派作战。洛克在牛津大学学习，最初接受的是传统的经院哲学教育，但他对医学和科学更感兴趣。后来，他通过牛津大学的实验哲学圈子认识了化学家波义耳，并成为了波义耳的学生和朋友。

洛克最重要的政治关系是**沙夫茨伯里伯爵**（Anthony Ashley Cooper, Earl of Shaftesbury）。沙夫茨伯里是英国辉格党领袖，洛克担任他的私人医生、秘书和政治顾问。通过沙夫茨伯里，洛克参与了英国最激烈的政治斗争——反对斯图亚特王朝的绝对君主制、主张议会权力和宗教宽容。

1683年，沙夫茨伯里伯爵因参与反对国王的阴谋而被迫流亡荷兰，洛克随行。在荷兰流亡期间，洛克完成了他最重要的著作——《人类理解论》和《政府论》。1688年光荣革命后，洛克随奥兰治的威廉返回英国，他的著作在1689年同时出版，为革命提供了理论辩护。

| 人生阶段 | 事件 |
|----------|------|
| 1632 | 出生于英格兰萨默塞特郡 |
| 1652-1667 | 在牛津大学学习，获得学士和硕士学位；对医学产生兴趣 |
| 1667 | 成为沙夫茨伯里伯爵的私人医生和秘书 |
| 1671 | 开始撰写《人类理解论》，起因是与几位朋友的讨论 |
| 1675-1679 | 在法国旅居，研究笛卡尔哲学和法国科学 |
| 1683-1689 | 因政治迫害流亡荷兰，完成《人类理解论》和《政府论》 |
| 1688 | 光荣革命，威廉和玛丽登上英国王位 |
| 1689 | 返回英国，发表《人类理解论》、《政府论》和《论宗教宽容》 |
| 1696-1700 | 担任英国贸易和殖民事务大臣 |
| 1704 | 去世，享年72岁 |

## 二、他说了什么？

### 2.1 心灵是一张白纸——没有天赋观念

洛克最著名的主张是：**我们的心灵出生时是一张白纸（白板），没有任何与生俱来的观念。一切知识都来源于经验。**

在洛克之前，笛卡尔和许多神学家认为，人类天生就有某些"天赋观念"——比如上帝的存在、数学真理和道德法则。这些观念是上帝在我们心灵中刻下的，不需要经验就能知道。

洛克说：不对。如果真有天赋观念，那么**所有人——包括婴儿、白痴和未开化民族——都应该知道这些观念**。但事实是，一个婴儿不知道"同一事物不能同时存在又不存在"；一个从未接触过数学的土著人不知道几何定理。如果这些观念是"天赋的"，为什么不是所有人都能立刻知道？

洛克说，这一切都是因为**教育和经验**。我们之所以知道"同一事物不能同时存在又不存在"，是因为我们从小被教导了这条逻辑规则；我们之所以知道数学，是因为我们学习了数学。如果没有经验，我们什么都不知道。

**心灵的白纸是怎么被填满的？** 洛克说，有两个来源：

| 来源 | 内容 | 例子 |
|------|------|------|
| 外部感觉（sensation） | 通过眼、耳、鼻、舌、身接触外部世界 | 看到红色、听到音乐、闻到花香、感到疼痛 |
| 内部反省（reflection） | 观察自己的心灵活动 | 思考、怀疑、相信、推理、记忆、欲望 |

洛克打比方说：心灵就像一间空房间，经验和反省是两扇窗户，光线通过窗户照进来，房间里才有了东西。

### 2.2 颜色真的在物体上吗？第一性质和第二性质

洛克提出了一个非常有影响力的区分：**第一性质和第二性质**。

他说，物体的性质分为两种：

| 类型 | 例子 | 是否存在于物体本身 | 是否随观察者变化 |
|------|------|-------------------|------------------|
| 第一性质 | 形状、大小、运动、数目、重量 | 是 | 否 |
| 第二性质 | 颜色、声音、气味、味道、温度 | 否（只是产生感觉的能力） | 是 |

这是什么意思？

**第一性质**是物体本身真正具有的。比如一个苹果，不管你怎么看它，它都有形状、大小、重量。即使你把它切成两半，每一半仍然有形状和大小。即使你把它切到最小，这些性质仍然存在。

**第二性质**不是物体本身具有的。比如苹果是红色的，但"红色"本身并不存在于苹果中——苹果本身只有一些微观结构，这些结构能够反射特定波长的光线，当光线进入我们的眼睛，我们的大脑产生了"红色"的感觉。如果世界上没有眼睛，"红色"就不存在。同样，"声音"只是空气振动在我们耳中产生的知觉；"气味"只是分子刺激我们嗅觉神经的结果。

洛克打了一个比方：第一性质就像物体的"骨架"，第二性质就像物体的"衣服"——骨架是真实的，衣服只是我们看起来的样子。

但这个区分后来引发了大问题。贝克莱追问：如果一切知识来源于经验，而我们只能经验第二性质（颜色、声音等），那我们怎么知道第一性质（形状、大小）真的存在于物体中？也许"形状"和"大小"也是我们的感觉？贝克莱最终说：**物质可能根本不存在！**

### 2.3 实体是什么？——某种不知为何物的东西

洛克对"实体"的概念提出了一个非常有趣的怀疑。他说，当我们感知到各种性质——比如一个东西是黄色的、有延展性的、重的、可熔的——我们自然会想：这些东西背后一定有一个"支撑"，让这些性质得以存在。

但洛克问：**这个"支撑"到底是什么？**

他的回答是：**我们不知道。我们只知道它"存在"，但不知道它"是什么"。** 洛克用了一个非常著名的短语来形容实体：**"某种不知为何物的东西"**（something-I-know-not-what）。

打个比方：你看到一个舞台，上面有各种灯光、布景和演员在表演。你知道一定有舞台在背后支撑着这一切，但你从来没有看到舞台本身——你看到的只是舞台上的表演。同样，我们知道一定有"某种东西"在支撑物体的各种性质，但我们从来没有直接感知到这个"支撑"本身。

这个怀疑后来被贝克莱和休谟大大发展了。贝克莱说：既然物质实体不可知，那么也许只存在精神（心灵和上帝）。休谟则更进一步说：不仅物质实体不可知，连"自我"这个精神实体也不可知——我们拥有的只是一束不断变化的知觉。

### 2.4 人人生而自由平等——自然权利与政府

洛克最重要的政治理论是**自然权利论**。他说：

**在自然状态（没有政府的状态）中，人人生而自由、平等和独立。**

这不是说自然状态是完美的。在自然状态中，人们享有生命、自由和财产的权利，但有一个大问题：如果我的权利被侵犯了，谁来惩罚侵犯者？在自然状态中，每个人都是自己案件的裁判者，这会导致混乱和不公正。因此，人们同意建立一个政府，来更好地保护他们的权利。

**政府的存在必须基于人民的同意。** 如果没有同意，政府就是不合法的。洛克说：即使一个人从未明确说"我同意"，只要他在某个国家居住、享受该国家的保护，他就是"默示同意"了该政府的统治。

| 自然状态 | 公民社会（政府状态） |
|----------|----------------------|
| 没有政府 | 有政府 |
| 人人自由平等 | 人人自由平等，但受法律约束 |
| 缺乏公正的裁判者 | 有公正的裁判者 |
| 缺乏有效的执法者 | 有有效的执法者 |
| 不安全，但享有自然权利 | 更安全，权利受法律保护 |

**最重要的：人民有权反抗暴政。** 当政府违背其保护人民权利的契约时，人民有权反抗甚至推翻它。洛克说，这不是"叛乱"，而是政府自己把自己变成了"与人民处于战争状态"的敌人。这一思想为美国革命和法国大革命提供了理论武器。

### 2.5 劳动创造财产

洛克提出了一个著名的财产理论：**人对自己的人身拥有所有权，因此对自己身体的劳动及其产物也拥有所有权。**

虽然土地和自然资源为人类共有，但当你将劳动与自然资源结合时，你就获得了财产权。比如，一片森林属于所有人，但当你砍下一棵树并把它做成椅子，这把椅子就是你的财产，因为你加入了你的劳动。

但洛克加了一个限制条件：**你必须留下"足够多、同样好的东西"给其他人。** 如果一片土地的所有资源都被一个人占有了，而其他人没有剩余，这种占有就是不公正的。

这个理论听起来很合理，但后来的批评者指出：欧洲殖民者对美洲土地的占有，并不符合"留下足够多、同样好"的条件。马克思主义者也批评说，洛克的劳动理论掩盖了资本主义原始积累的不平等——工人的劳动成果被资本家占有，而工人只得到工资。

### 2.6 宗教宽容：国家不能管灵魂

洛克在《论宗教宽容》中提出了一个在当时非常激进的观点：**国家不应该干预个人的宗教信仰。**

他的论证很简单：

（1）国家和教会有不同的目的。国家的目的是保护公民的生命、自由和财产。教会的目的是灵魂救赎。

（2）灵魂的救赎是良心的事务，而良心不能被强迫。你可以强迫一个人参加宗教仪式，但你不能强迫他真正相信。

（3）因此，国家干预宗教是无效的，而且是有害的——它会导致宗教冲突和内战。

洛克说：**"剥夺人们的思想自由，比剥夺他们的财产更危险。"**

但洛克也有他的局限性。他说，宽容不应扩展到**无神论者**（因为他们不承认上帝，就没有遵守道德的基础）和**天主教徒**（因为他们服从外国权力——教皇）。尽管如此，洛克仍然是宗教宽容理论最重要的近代奠基人之一。

## 三、他做了什么综合？

洛克的哲学是一个**综合的体系**，它将多个领域统一在一个经验主义的框架下：

| 领域 | 洛克的综合 |
|------|------------|
| 认识论 | 一切知识来源于经验（感觉+反省），否定天赋观念 |
| 形而上学 | 物体有第一性质（真实的）和第二性质（感觉的），实体不可知 |
| 政治哲学 | 自然权利（生命、自由、财产）→ 社会契约 → 同意 → 有限政府 |
| 教育哲学 | 心灵是白板，教育塑造人；强调环境和习惯 |
| 宗教哲学 | 理性宗教，宗教宽容，但排除无神论和天主教 |
| 科学方法论 | 以观察和实验为基础，但承认人类认识的局限性 |

洛克的综合有一个核心特征：**经验主义与自由主义的结合**。他不仅在认识论上是经验主义者，在政治哲学上也主张从经验出发——从人们实际的权利和需要出发，而不是从抽象的神学或形而上学出发来论证政府的合法性。

## 四、为什么重要？

### 4.1 对哲学的影响

洛克是**系统化经验主义的第一人**。他的"白板说"成为英国经验主义的基石，贝克莱和休谟都是他的直接继承者。虽然休谟后来将经验主义推向了怀疑论的极端，但洛克确立了一个基本原则：**一切知识必须接受经验的检验。**

康德称自己的哲学是对休谟怀疑论的回应，但休谟的怀疑论本身是对洛克经验主义的逻辑发展。因此，整个德国古典哲学（康德、费希特、谢林、黑格尔）都可以被理解为对洛克所确立的问题框架的回应。

### 4.2 对科学的影响

洛克的认识论与牛顿物理学有深刻的方法论亲缘性。牛顿说"我不做假设"（Hypotheses non fingo），洛克说我们只拥有来源于经验的观念。两者都强调以观察和实验为基础，反对经院哲学的抽象思辨。洛克的"简单观念"概念与牛顿的"经验归纳"方法共同奠定了近代科学方法论的哲学基础。

### 4.3 对政治的影响

洛克是**古典自由主义之父**。他的自然权利理论直接影响了：

- **美国独立宣言**（1776）："生命、自由和追求幸福"是洛克"生命、自由和财产"的变体
- **美国宪法**：权利保护条款和分权制衡设计深受洛克影响
- **法国人权宣言**（1789）："自由、平等、财产"直接来源于洛克
- **现代民主宪政**：同意理论、反抗权理论至今仍是民主理论的基石

### 4.4 对教育的影响

洛克的《教育漫话》强调环境和教育对人的塑造作用。他说："**人的心灵就像一块白板，教育可以在上面写下任何字迹。**" 这一思想成为近代教育理论的基石，也影响了后来的环境决定论和社会改革思想。

### 4.5 对宗教的影响

洛克的宗教宽容论是启蒙运动最重要的文献之一。他的核心原则——国家与教会分离、良心不可强制——被后来的自由主义思想家（如密尔、罗尔斯）发展为更普遍的宽容理论。美国宪法第一修正案（保护言论和宗教自由）的思想渊源之一，就是洛克在《论宗教宽容》中论证的宗教自由不可剥夺。

---

## 五、一句话总结

**洛克说：我们的心灵出生时是一张白板，一切知识来源于经验；物体有真实的第一性质（如形状、大小）和仅存在于感觉中的第二性质（如颜色、声音）；人人生而具有生命、自由和财产的自然权利，政府的合法性来源于人民的同意，当政府违背保护人民权利的契约时，人民有权反抗；国家不应干预个人宗教信仰，因为灵魂的事务属于良心，不可强制。**

## 脚注

[^tabula_rasa]: **白板说**：洛克《人类理解论》第一卷第二章的核心命题。"白板"（tabula rasa）是拉丁文，原意为"被擦净的蜡板"——古罗马人用这种蜡板来写字，写完后可以擦净重新使用。洛克用它来表示心灵出生时没有任何内容，一切知识都来源于经验。这一理论直接反对笛卡尔的天赋观念论，成为英国经验主义的基石。

[^primary_secondary]: **第一性质与第二性质**：洛克《人类理解论》第二卷第八章的核心区分。第一性质是物体本身固有的、不依赖于观察者的性质（如广延、形状、运动）；第二性质不是物体本身的性质，而是物体产生感觉的能力（如颜色、声音、气味）。这一区分直接来源于伽利略和笛卡尔，但洛克将其系统化为认识论框架。贝克莱后来利用这一区分否认物质实体的存在，休谟则怀疑我们是否能知道任何物体本身。

[^substance]: **实体不可知**：洛克《人类理解论》第二卷第二十三章。洛克认为，我们对实体只有模糊的观念，称之为"某种不知为何物的东西"（something-I-know-not-what）。我们只是将某些简单观念组合在一起，并假定为它们背后有一个支撑。贝克莱和休谟后来大大发展了这种怀疑。

[^state_of_nature]: **自然状态**：洛克《政府论》下篇第二章的核心概念。洛克认为自然状态不是战争状态（如霍布斯所说），而是和平但不安全的状态。人们享有自然权利，但缺乏公正的裁判者。因此，人们同意建立政府以更好地保护自己的权利。

[^property]: **劳动创造财产**：洛克《政府论》下篇第五章的核心命题。虽然土地和自然资源为人类共有，但每个人对自己的人身拥有所有权，因此对自己身体的劳动及其产物也拥有所有权。但洛克加了限制：必须留下"足够多、同样好的东西"给其他人。后来的批评者指出，这一条件在资本主义原始积累中很少被满足。

[^consent]: **同意**：洛克《政府论》下篇第八章的核心概念。洛克区分了明示同意（直接参与契约）和默示同意（通过居住和享受保护而间接同意）。这一理论是古典自由主义社会契约论的基石，但也引发了关于"默示同意"是否真正自由的持续争论。

[^resistance]: **反抗权**：洛克《政府论》下篇第十九章的核心命题。当政府严重和持续地违背其信托时，人民有权反抗。但洛克强调，只有当政府"严重和持续地"违背信托时，反抗才是正当的。他警告不要轻易使用反抗权，因为社会稳定也很重要。

[^toleration]: **宗教宽容**：洛克《论宗教宽容》的核心命题。国家与教会有不同的目的，国家无权干预宗教。但洛克的宽容有界限：不宽容无神论者和天主教徒。尽管如此，他仍然是宗教宽容理论最重要的近代奠基人之一。
