## 二、思想标签

| 标签 | 说明 |
|------|------|
| 经验主义激进化 | 将洛克的经验主义推向主观唯心主义 |
| 存在即被感知 | 物质事物的存在就是被感知（esse est percipi） |
| 非物质主义 | 否认物质实体的存在 |
| 观念论 | 一切存在物要么是心灵，要么是心灵中的观念 |
| 自然符号论 | 自然规律是上帝设定的符号系统 |
| 反怀疑主义 | 认为唯心主义比唯物论更接近常识，能避免怀疑主义 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 存在即被感知（esse est percipi） | 物质事物的存在就是被感知；没有被感知的物质事物不存在 | 《人类知识原理》第一部分§3-6，1710 |
| 观念（idea） | 心灵中的感知内容；一切知识对象都是观念 | 《人类知识原理》导言§1 |
| 精神（spirit/mind） | 感知观念的主体；包括有限精神（人类）和无限精神（上帝） | 《人类知识原理》第一部分§2 |
| 物质（matter） | 巴克莱否认其存在的假设性实体；洛克称之为"我不知道什么" | 《人类知识原理》第一部分§9-15 |
| 第一性质/第二性质 | 巴克莱批判洛克的区分，认为两者都只存在于心灵中 | 《人类知识原理》第一部分§9-15 |
| 自然符号论 | 自然规律不是因果机制，而是上帝设定的符号语言 | 《阿尔西弗隆》，1732 |
| 主观唯心主义 | 认为只有心灵和观念存在的哲学立场 | 巴克莱哲学的整体定位 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对休谟的影响** | 休谟继承巴克莱的经验主义，但将其推向更彻底的怀疑主义——不仅怀疑物质实体，也怀疑心灵实体和因果关系 |
| **对康德的影响** | 康德将巴克莱的唯心主义称为"独断的唯心主义"，提出先验唯心主义作为替代；巴克莱促使康德重新思考"物自体"问题 |
| **对现代认识论的影响** | 巴克莱的"存在即被感知"在当代认识论中仍然是重要的讨论对象，与现象主义、直接实在论等立场对话 |
| **对物理学的影响** | 量子力学中观测问题的讨论与巴克莱的"存在即被感知"有某种相似之处（虽然这种类比并不严格） |
| **主要批评** | 约翰逊博士的"踢石头"反驳（哲学上无效）；休谟认为上帝解决方案是特设性的；康德认为巴克莱混淆了现象和物自体 |
| **学术评价** | 巴克莱是经验主义传统中的关键环节，将洛克的经验主义推向了唯心主义，为休谟的怀疑主义和康德的先验哲学铺平了道路 |
