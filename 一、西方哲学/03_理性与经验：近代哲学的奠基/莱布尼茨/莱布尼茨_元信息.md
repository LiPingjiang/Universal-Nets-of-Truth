# 莱布尼茨 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 戈特弗里德·威廉·莱布尼茨 |
| **外文名** | Gottfried Wilhelm Leibniz |
| **生卒年** | 1646—1716 |
| **国籍** | 德国（神圣罗马帝国） |
| **主要身份** | 哲学家、数学家、逻辑学家、法学家、外交官、历史学家、图书馆学家 |
| **师承/传统** | 受笛卡尔、斯宾诺莎、霍布斯影响；独立发展理性主义体系 |
| **主要著作** | *Discours de métaphysique*（《形而上学论》，1686）；*Monadologie*（《单子论》，1714）；*Essais de théodicée*（《神正论》，1710）；*Nouveaux essais sur l'entendement humain*（《人类理智新论》，1704）；*Principes de la nature et de la grâce fondés en raison*（《自然与恩惠的原理》，1714）；*Principia philosophiae*（《哲学原理》未发表） |
| **版本** | G. W. Leibniz, *Sämtliche Schriften und Briefe*, Akademie Verlag, 1923-（标准学术版，仍在进行中）；C. I. Gerhardt 编，*Die Philosophischen Schriften*, 7卷，1875-1890 |
| **中文译本** | 《人类理智新论》，商务印书馆；《单子论》，商务印书馆；《神正论》，三联书店；多种论文集 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 理性主义 | 与笛卡尔、斯宾诺莎并称"大陆理性主义三杰"，主张理性是知识的主要来源 |
| 单子论 | 以单子（无部分的简单实体）为终极实在，构建独特的形而上学体系 |
| 前定和谐 | 提出心物关系的独特解决方案——上帝预先安排所有单子之间的完美和谐 |
| 最好的世界 | 主张现存世界是所有可能世界中最好的，这是《神正论》的核心论点 |
| 神正论 | 为《神正论》自创词，意为"上帝的正义"，试图证明上帝与恶之共存 |
| 充足理由原则 | 一切存在和发生都有充足理由，是莱布尼茨哲学的基础原则 |
| 微积分发明者 | 独立于牛顿发明微积分，引发了著名的"优先权之争" |
| 逻辑学家 | 发明了二进制算术，设计了逻辑演算，被誉为"数理逻辑的先驱" |
| 百科全书式学者 | 涉猎哲学、数学、法学、外交、历史、图书馆学等众多领域，是最后一位"文艺复兴式学者" |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 单子（monade） | 没有部分的简单实体，不可分、不可灭，具有知觉和欲望，是构成世界的最终元素 | 《单子论》§1 |
| 前定和谐（harmonie préétablie） | 上帝预先安排的和谐，使没有因果关系的单子之间产生完美的对应；是莱布尼茨对心物关系问题的独特解决方案 | 《单子论》§78-80 |
| 最好的世界（le meilleur des mondes possibles） | 上帝在所有可能世界中选择了最完美的一个，即包含最多善和最少的必然之恶的世界 | 《神正论》§10 |
| 充足理由原则（principe de raison suffisante） | 一切存在和发生都有充足理由；这一原则是莱布尼茨哲学的基础，也是沃尔夫体系化的核心 | 《单子论》§32 |
| 天赋观念（idées innées） | 灵魂中固有的观念，不是以现实命题的方式，而是以倾向、习惯或自然能力的方式；回应洛克的批判 | 《人类理智新论》I.1 |
| 不可分辨者的同一性（identité des indiscernables） | 没有两个实体在所有方面完全相同；每个单子都是独特的，反映宇宙的视角不同 | 《单子论》§9 |
| 连续律（loi de continuité） | 自然中没有跳跃，从最小到最大的一切都是连续变化的；是莱布尼茨哲学的重要原则 | 《单子论》§56 |
| 知觉（perception） | 单子的基本属性，单子通过知觉反映整个宇宙；单子的知觉有清晰与模糊之分 | 《单子论》§14 |
| 欲望（appétition） | 单子的趋向性，从当前知觉过渡到下一知觉的内在冲动；是单子变化的内在动力 | 《单子论》§15 |
| 可能世界（mondes possibles） | 在逻辑上不自相矛盾的所有可能世界，上帝从中选择了最好的一个 | 《神正论》§8 |
| 反省（réflexion） | 对我们内部发生的事情的感受，是观念的来源之一（与感觉并列）；区别于洛克的感觉主义 | 《人类理智新论》I.2 |
| 神正论（théodicée） | 莱布尼茨为《神正论》自创的词，意为"上帝的正义"，试图证明上帝与恶之共存 | 《神正论》标题 |

## 四、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 笛卡尔（理性主义） | 莱布尼茨继承了笛卡尔的理性主义，但批判了笛卡尔的实体论和心物二元论；用单子论替代了笛卡尔的实体二元论 | 第七章：心灵与物质 |
| 斯宾诺莎（一元论） | 莱布尼茨与斯宾诺莎曾会面，深受其影响，但批判了斯宾诺莎的一元论和实体论；莱布尼茨的单子论可以看作是对斯宾诺莎的"修正" | 第七章：心灵与物质 |
| 洛克（经验主义） | 莱布尼茨在《人类理智新论》中回应了洛克的《人类理解论》；同意洛克反对"铭刻的真理"，但主张"天赋观念"以自然能力的方式存在 | 第六章：经验与理性 |
| 牛顿（物理学） | 莱布尼茨与牛顿独立发明微积分，引发"优先权之争"；莱布尼茨批判了牛顿的绝对时空观，主张相对时空观 | 第七章：心灵与物质 |
| 沃尔夫（体系化） | 沃尔夫是莱布尼茨哲学的系统化者，将莱布尼茨的思想整理成严密的逻辑体系；莱布尼茨本人反对过度的体系化 | 第三章：理性与经验 |
| 培尔（怀疑论） | 培尔在《历史批判辞典》中批判了莱布尼茨的《神正论》，认为"恶的问题"从未被解决；莱布尼茨在《神正论》中回应了培尔的批评 | 第三章：理性与经验 |
| 伏尔泰（讽刺） | 伏尔泰在《老实人》中讽刺了莱布尼茨的"最好的世界"论，创造了"潘格洛斯博士"（Dr. Pangloss）这一形象 | 第九章：自由与必然 |
| 康德（批判哲学） | 康德的先验哲学深受莱布尼茨影响，但批判了莱布尼茨的独断论；康德在《纯粹理性批判》中专门讨论了莱布尼茨的"充足理由原则" | 第八章：主体与绝对 |
| 黑格尔（辩证法） | 黑格尔在《逻辑学》中讨论了莱布尼茨的单子论，将其作为"主观性"哲学的典范；黑格尔批判了莱布尼茨的"封闭单子"概念 | 第八章：主体与绝对 |
| 罗素（逻辑分析） | 罗素在《莱布尼茨哲学》中分析了莱布尼茨的逻辑学，认为莱布尼茨的哲学本质上是一种逻辑学；罗素的逻辑分析哲学受到莱布尼茨的启发 | 第十章：语言与逻辑 |
| 现代计算机科学 | 莱布尼茨发明的二进制算术是现代计算机科学的基础；他的逻辑演算思想对数理逻辑和人工智能的发展有深远影响 | 当代应用 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对理性主义的影响** | 莱布尼茨与笛卡尔、斯宾诺莎并称"大陆理性主义三杰"，但他的理性主义比笛卡尔更复杂、更开放——他承认理性无法证明一切，但对理性的信任超过经验 |
| **对形而上学的影响** | 莱布尼茨的单子论是西方哲学史上最独特的形而上学体系之一。它以"无部分的简单实体"为基础，构建了一个动态的、有机的宇宙图景，影响了后来的浪漫主义有机论和黑格尔的辩证法 |
| **对认识论的影响** | 莱布尼茨在《人类理智新论》中回应了洛克，提出了"天赋观念"以自然能力方式存在的理论，对康德的先验哲学有直接影响 |
| **对宗教哲学的影响** | 莱布尼茨的《神正论》是西方哲学史上最重要的宗教哲学著作之一。他试图证明上帝与恶之共存，但伏尔泰在《老实人》中讽刺了"最好的世界"论，使其成为哲学史上最大的讽刺之一 |
| **对数学和科学的影响** | 莱布尼茨独立发明微积分，设计二进制算术，提出逻辑演算思想，被誉为"数理逻辑的先驱"。他的"充足理由原则"对后来的科学方法论有深远影响 |
| **对康德的影响** | 康德的先验哲学深受莱布尼茨影响，但批判了莱布尼茨的独断论。康德在《纯粹理性批判》中专门讨论了莱布尼茨的"充足理由原则"，并提出了自己的"先验逻辑" |
| **对现代哲学的影响** | 罗素在《莱布尼茨哲学》中分析了莱布尼茨的逻辑学，认为莱布尼茨的哲学本质上是一种逻辑学。罗素的逻辑分析哲学受到莱布尼茨的启发。海德格尔也讨论了莱布尼茨的存在论思想 |
| **对现代计算机科学的影响** | 莱布尼茨发明的二进制算术是现代计算机科学的基础。他的逻辑演算思想对数理逻辑和人工智能的发展有深远影响 |
| **主要批评** | 莱布尼茨的"最好的世界"论被伏尔泰讽刺为"潘格洛斯主义"——无论发生什么灾难，都是"最好的世界"的必要部分。这一批评揭示了莱布尼茨乐观主义的脆弱性。此外，单子论的前定和谐是否过于依赖"上帝的全能"？如果没有上帝，前定和谐是否仍然成立？ |
| **学术评价** | 莱布尼茨是西方哲学史上最多才多艺的哲学家之一。他不仅是哲学家，还是数学家、逻辑学家、法学家、外交家和历史学家。他的哲学体系（单子论、前定和谐、最好的世界）是西方哲学史上最精致的理性主义体系之一。但他也是"最后一位文艺复兴式学者"——随着学科专业化的发展，再也没有人能像他一样涉猎如此广泛的领域。他的未发表手稿（约20万页）至今仍在整理中，标准学术版（Akademie Verlag）已经出版了一百多年，仍未完成。 |

## 六、研究文献

- Bertrand Russell, *A Critical Exposition of the Philosophy of Leibniz*, Cambridge University Press, 1900. —— 罗素的经典分析，认为莱布尼茨的哲学本质上是逻辑学。
- G. H. R. Parkinson, *Logic and Reality in Leibniz's Metaphysics*, Oxford University Press, 1965. —— 莱布尼茨的逻辑学与形而上学。
- Robert Merrihew Adams, *Leibniz: Determinist, Theist, Idealist*, Oxford University Press, 1994. —— 莱布尼茨哲学综合研究。
- Daniel Garber, *Leibniz: Body, Substance, Monad*, Oxford University Press, 2009. —— 莱布尼茨的物理学与形而上学。
- 陈修斋，《莱布尼茨哲学研究》，人民出版社，1993. —— 莱布尼茨哲学中文研究。
