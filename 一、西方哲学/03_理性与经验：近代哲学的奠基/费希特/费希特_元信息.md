# 费希特 · 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 约翰·戈特利布·费希特 |
| **外文名** | Johann Gottlieb Fichte |
| **生卒年** | 1762–1814 |
| **国籍** | 德国（萨克森选侯国/普鲁士王国） |
| **主要身份** | 哲学家、耶拿大学教授、柏林大学创始人之一 |
| **师承/传统** | 康德批判哲学 + 德国古典唯心主义 |
| **主要著作** | *Versuch einer Kritik aller Offenbarung*（《试评一切天启》，1792）；*Grundlage der gesamten Wissenschaftslehre*（《全部知识学的基础》，1794/1795）；*Grundlage des Naturrechts*（《自然法权基础》，1796/1797）；*Die Bestimmung des Menschen*（《人的使命》，1800）；*Die Anweisung zum seligen Leben*（《至福生活指南》，1806）；*Reden an die deutsche Nation*（《对德意志民族的演讲》，1808） |
| **版本** | Gesamtausgabe der Bayerischen Akademie der Wissenschaften, Erich Fuchs 等编，Frommann-Holzboog，1964-2012，42卷（标准学术版） |
| **中文译本** | 梁志学、沈真译，《全部知识学的基础》，商务印书馆，1986；梁志学译，《人的使命》，商务印书馆，1982 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 德国古典哲学 | 德国古典哲学最重要的代表人物之一 |
| 知识学 | 知识学（Wissenschaftslehre）的创始人 |
| 主观唯心主义 | 一切存在的根据都是自我 |
| 本原行动 | 自我设定自己的原始行动，同时是行动和存在 |
| 绝对自我 | 不受限制的自我，是一切存在的根据 |
| 有限自我 | 受非我限制的自我，是经验意识的主体 |
| 实践哲学 | 自我的本质是活动，必须不断克服限制实现自由 |
| 民族主义 | 《对德意志民族的演讲》对德国民族主义的影响 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 知识学（Wissenschaftslehre） | 费希特的哲学体系，意为"关于知识的学说"；从自我出发推导出一切存在 | 《全部知识学的基础》，1794/1795 |
| 自我（Ich） | 绝对的主体，一切知识的出发点；自我设定自己 | 《全部知识学的基础》第一原理，1794/1795 |
| 非我（Nicht-Ich） | 与自我相对立的客体世界；自我设定的对立面 | 《全部知识学的基础》第二原理，1794/1795 |
| 本原行动（Tathandlung） | 自我设定自己的原始行动，同时是行动（活动）和事实（存在） | 《全部知识学的基础》§1-2，1794/1795 |
| 设定（setzen） | 自我的基本活动，既是行动也是存在 | 《全部知识学的基础》第一原理，1794/1795 |
| 绝对自我（absolutes Ich） | 不受限制的自我，是一切存在的根据 | 《全部知识学的基础》§5-6，1794/1795 |
| 有限自我（endliches Ich） | 受非我限制的自我，是经验意识的主体 | 《全部知识学的基础》§5-6，1794/1795 |
| 相互作用（Wechselwirkung） | 自我与非我之间的辩证关系，通过它产生意识中的所有表象 | 《全部知识学的基础》第三原理，1794/1795 |
| 原始权利（Urrecht） | 每个人生来就有权利要求他人承认自己是自由的 | 《自然法权基础》，1796/1797 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对谢林的影响** | 谢林继承费希特的自我哲学，但提出了自然哲学，认为自然和自我都是绝对者的表现 |
| **对黑格尔的影响** | 黑格尔批判费希特的"主观唯心主义"，提出了更全面的"绝对精神"体系 |
| **对马克思的影响** | 马克思批判费希特的唯心主义，但继承了费希特的实践哲学传统 |
| **对存在主义的影响** | 费希特的"自我设定"概念对后来的存在主义有影响 |
| **对德国民族主义的影响** | 费希特的《对德意志民族的演讲》在拿破仑占领期间激发了德国民族主义，对德国历史产生了深远影响 |
| **主要批评** | 谢林认为费希特忽视了自然界的独立存在；黑格尔认为费希特的体系是"反思的哲学"；马克思认为费希特忽视了物质条件；叔本华认为费希特的自我哲学是"独断的唯心主义" |
| **学术评价** | 费希特是德国古典哲学从康德到黑格尔的关键环节，他的知识学将康德的批判哲学推向了绝对唯心主义，对后来的德国唯心主义、马克思主义和存在主义都产生了重要影响 |
