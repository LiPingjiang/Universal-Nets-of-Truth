# 霍布斯 · 白话解读

---

## 这个人是谁？

想象一个1588年出生的人，活到了91岁——在那个平均寿命不到40岁的年代，这本身就是一件了不起的事。这个人就是托马斯·霍布斯。

霍布斯出生的年份恰好是西班牙无敌舰队入侵英格兰的那一年。据说他母亲因为听到无敌舰队的消息受了惊吓，提前生下了他。霍布斯后来开玩笑说："恐惧和我是一对孪生兄弟。"这句话听起来像是玩笑，但如果你读完他的哲学，就会发现"恐惧"这个词几乎是他整个思想体系的情感底色——正是因为恐惧死亡、恐惧暴力、恐惧混乱，人类才愿意放弃自由，建立一个拥有绝对权力的国家。

霍布斯的家境并不富裕。他父亲是个小教区牧师，还因为在教堂门口打架闹事跑路了，把他丢给了叔父抚养。但霍布斯聪明好学，靠着奖学金上了牛津大学。毕业后他给一个贵族家庭当家庭教师——这份工作他干了一辈子，也因此获得了接触上层社会和游历欧洲的机会。

他在意大利见到了伽利略，在巴黎结识了一批一流的数学家和哲学家。伽利略对他的影响特别大：如果整个自然界都可以用物体的运动来解释，为什么不能把人也当作运动的物体来研究？为什么不能把国家当作一个人造的机器来分析？这就是霍布斯哲学的基本出发点——用物理学的方法来研究人和社会。

英国内战（1642—1651）是霍布斯政治思想的直接催化剂。他亲眼目睹了国王和议会之间的权力斗争如何导致国家分裂、社会崩溃。这段经历让他深信：没有绝对的权威，就没有和平；没有和平，就没有文明。

---

## 他说了什么？

### 1. 自然状态：没有政府的可怕世界

霍布斯问了一个很根本的问题：如果没有任何政府、法律、警察，人类会怎样？

他的回答是：一场"每个人对每个人的战争"。

这不是说人人天天在打架，而是说人人都活在随时可能被打的恐惧中。因为没有警察抓贼，没有法院判案，所以每个人都得自己保护自己。为了自保，你可能需要先发制人——不是因为你想害人，而是因为你怕别人害你。别人也是这么想的。结果就是人人自危，谁也不敢放下武器。

霍布斯对这种状态有一段经典描述，说那种生活是"孤独、贫困、污秽、野蛮和短暂的"。没有农业（因为你种的东西会被抢），没有贸易（因为没有人会信任对方），没有建筑（因为建了也会被毁），没有科学，没有艺术——什么文明都没有。

| 自然状态的特征 | 具体表现 |
|----------------|----------|
| 没有公共权力 | 没有政府、没有警察、没有法院 |
| 人人平等 | 谁也不能保证自己比所有人都强 |
| 持续恐惧 | 时刻担心被他人攻击 |
| 无正义可言 | 没有法律，就没有"对"和"错" |
| 文明无法存在 | 没有"你的""我的"之分 |

### 2. 自然法：理性告诉我们该怎么做

但霍布斯不是悲观主义者。他认为人虽然天生自私，但人有理性。理性告诉我们一个最基本的道理：活着比死了好，和平比战争好。

由此，霍布斯提出了他的"自然法"。这里的"法"不是政府颁布的法律，而是理性发现的生活准则。头几条特别重要：

- **第一条**：尽量寻求和平。能和谈就和谈，别动不动就打。
- **第二条**：如果别人也愿意和平，那你就得放弃一部分自由。你不能又要和平又要随心所欲，天下没这好事。
- **第三条**：说了就要算数。既然大家说好了和平，就得守信用。

霍布斯一共列了十九条自然法，但核心就是这三条。他甚至说，这些自然法可以浓缩成一句话："己所不欲，勿施于人。"

但问题来了：光有理性知道该怎么做还不够。因为在自然状态中，即使你信守承诺，别人也可能背叛你。谁先放下武器谁就是傻瓜——这就是"囚徒困境"。理性告诉你该合作，但自利告诉你别当第一个。所以，自然法虽然好，但在自然状态中根本执行不了。

### 3. 社会契约：怎么走出这个困局？

怎么办？霍布斯给出的答案是：找一个人（或一群人），把大家自我保护的权利交给他，让他来当裁判，来维持秩序。

具体怎么做呢？所有人达成一个协议："我把我管自己的权利交给这个人，条件是你也一样。"这个协议就是"社会契约"。

注意，这里有个关键设计：被授权的那个人（主权者）不在协议里面。这意味着什么？意味着主权者不需要遵守什么约定，他的权力是绝对的。你把权利交出去之后就不能收回来，因为收回来等于回到自然状态——也就是回到战争。

这就像一群人在暴风雨中决定造一艘大船。大家把各自的木板拼在一起，选了一个船长。船造好了之后，你不能说"我觉得船长开得不好，我要把我的木板拿回去"——因为你一拆，整艘船就散了，大家都得淹死。

### 4. 利维坦：人造的巨人

霍布斯给这个由契约产生的国家取了一个很酷的名字——"利维坦"。利维坦是《圣经·约伯记》里提到的一头巨大海兽，力大无穷，不可征服。霍布斯用它来比喻国家：

| 利维坦的"身体部位" | 对应的国家要素 |
|---------------------|---------------|
| 灵魂 | 主权者 |
| 关节 | 官员和公务员 |
| 神经 | 赏罚机制 |
| 力量 | 财富 |
| 意志 | 法律 |
| 健康 | 和平 |
| 疾病 | 内乱 |
| 死亡 | 国家解体 |

霍布斯说这个"利维坦"是一个"有死的上帝"——我们在"不朽的上帝"之下，靠这个"有死的上帝"来获得和平与安全。

### 5. 绝对主权：权力不能分割

霍布斯对主权者的权力做了最极端的赋予：绝对、不可分割、不可转让。

主权者拥有什么权力？立法、司法、宣战、讲和、收税、审查言论——你能想到的一切国家权力，全都在主权者手里。

为什么不能分权？霍布斯的逻辑很简单：权力一分为二，就等于两个"最高权威"在打架。国王和议会争权，结果就是内战。教会和国家争权，结果还是内战。所以他坚决反对分权——不是因为他喜欢独裁，而是因为他怕内战。

但这里有一个微妙的限制：主权者虽然权力绝对，但他的权力存在理由只有一个——保护臣民。如果主权者保护不了你了（比如国家被外敌征服了），你对他的服从义务就自动解除了。用霍布斯的话说，"保护和服从是相互的"。这就像你雇了个保镖，如果他连自己都保护不了，你当然没义务继续听他的。

### 6. 唯物主义：一切都是物质运动

除了政治哲学，霍布斯还有一个宏大的哲学野心：用物质和运动来解释一切。

在他看来，世界上只有一种东西——物体。你看到的、摸到的、听到的，全是物体。那"精神"呢？霍布斯说精神也是物质的运动——你的思想是脑子里物质在运动，你的感觉是外部物体挤压你的感官产生的，你的意志不过是欲望和厌恶最后的"投票结果"。

推理是什么？霍布斯有个特别现代的说法：推理就是"计算"——概念的加法和减法。比如"人"加"会说话"等于"会说话的人"，"三角形"减去"三条边"就不再"是三角形"了。这种把思维当作计算的想法，在20世纪计算机出现之后看起来简直有预言性。

他甚至把国家也当作一个"人造的人"来看待——就像你造一台机器一样，国家是人造出来的，有灵魂（主权者）、有意志（法律）、有神经（奖惩机制）。这在当时是非常激进的想法，因为以前的人都认为国家是"自然形成"的或者"上帝安排"的。

## 他做了什么综合？

霍布斯的伟大之处在于他把几个重大转变汇聚在了一起：

| 综合领域 | 传统观点 | 霍布斯的综合 |
|----------|----------|-------------|
| 政治合法性 | 来自上帝或自然秩序 | 来自人民的契约授权 |
| 人性观 | 人天然是社会/政治动物 | 人天然是自私的个体，社会是人造的 |
| 哲学方法 | 目的论（一切皆有目的） | 机械论（一切皆可用因果运动解释） |
| 国家本质 | 自然有机体或神意产物 | 人造机械装置 |
| 权力来源 | 神授或世袭 | 人民的权利转让 |

简单说，霍布斯做了三件事：

**第一，他发明了"个人"这个政治哲学的起点。** 在他之前，政治哲学的出发点是城邦、家庭或等级。霍布斯把个人——一个有欲望、有恐惧、有理性的个体——放在了一切政治思考的起点上。所有的政治制度都是从这个个体出发来设计的。

**第二，他用契约论替代了神权论和自然等级论。** 国家不再需要上帝来背书，也不需要"有些人天生就该统治"的说法。国家的权威来自被统治者的同意——即便这种同意在霍布斯那里是一次性的、不可撤回的。

**第三，他把机械唯物主义的方法推广到了社会领域。** 伽利略用运动解释了天上的星星，霍布斯要用运动解释人和社会。国家不是有机体，是机器；人不是政治动物，是被恐惧和欲望驱动的物质存在。

## 为什么重要？

### 1. 他是现代政治哲学的起点

在霍布斯之前，政治哲学讨论的核心问题是"什么样的政体最好"（柏拉图的理想国、亚里士多德的政体分类）。霍布斯把问题变成了"为什么需要国家"和"国家的权力从哪里来"。这个问题的转换是革命性的——此后洛克、卢梭、康德的政治哲学都在霍布斯的框架里展开，无论是赞同还是反对。

### 2. 他提出了"自然状态"这个思想工具

"自然状态"可能是政治哲学史上最有用的思想实验之一。通过假设一个没有政府的状态，我们可以清楚地看到政府存在的理由和限度。今天我们讨论"政府该管什么""个人有哪些不可侵犯的权利"这些问题时，本质上还在使用霍布斯的框架。

### 3. 他为现代主权国家理论奠基

"主权"这个概念虽然在博丹那里已经出现，但霍布斯是第一个系统论证主权为什么必须是绝对的、不可分割的思想家。现代民族国家的理论框架——一个拥有最高权威、对领土内一切事务有最终决定权的政治共同体——在很大程度上就是霍布斯式的。

### 4. 他开启了世俗化政治的潮流

霍布斯把政治从神学中剥离出来。国家不需要上帝来授权，不需要教会来监督。政治的根基是人性和理性，不是神意和启示。这一转向为后来的启蒙运动和现代世俗国家铺平了道路。

### 5. 他的唯物主义和机械论预示了现代科学精神

霍布斯坚持一切现象都可以用物质运动来解释，思维是大脑的活动，推理是计算。这些观点在今天看来已经非常接近认知科学和人工智能的基本假设。虽然他的具体科学结论很多是错的，但他坚持用自然原因解释自然现象的方法论精神是正确的。

---

## 一句话总结

**霍布斯告诉我们：没有政府的地方就是地狱，而为了逃离地狱，人类愿意交出自由换取安全——这就是国家诞生的逻辑，也是现代政治的起点。**
