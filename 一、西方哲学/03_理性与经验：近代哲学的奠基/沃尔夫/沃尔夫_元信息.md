## 二、思想标签

| 标签 | 说明 |
|------|------|
| 理性主义体系化 | 将莱布尼茨的哲学思想系统化，建立百科全书式的哲学体系 |
| 德国启蒙哲学 | 德国启蒙运动的哲学基础，影响了整个18世纪德国知识界 |
| 充足理由律 | 将莱布尼茨的充足理由原则系统化，作为哲学体系的基本原则 |
| 数学方法论 | 主张哲学应该用数学的方法（定义、公理、定理、证明）来进行 |
| 大学哲学教育 | 建立了现代大学哲学教育的模式（逻辑、形而上学、伦理学、政治学） |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 充足理由律（Satz vom zureichenden Grund） | 一切存在者或发生者都有其存在的或发生的充足理由；是沃尔夫整个哲学体系的基础 | 《关于人类行为的理性思想》，1720 |
| 理性思想（Vernünftige Gedanken） | 沃尔夫哲学体系的总称，以理性为最高原则，涵盖逻辑、形而上学、伦理学、政治学等 | 《理性思想》系列，1713-1724 |
| 认识等级（Erkenntnisstufen） | 认识从模糊到充分的五个等级：模糊→清晰→明确→完整→充分 | 《人类理解力的理性思想》，1713 |
| 可能事物（mögliche Dinge） | 在逻辑上不自相矛盾的事物；哲学研究的对象之一 | 《人类理解力的理性思想》 |
| 必然存在者（notwendiges Wesen） | 其存在不依赖于任何其他事物的存在者；即上帝 | 《关于上帝、世界和人类灵魂的理性思想》，1719 |
| 自然法（Naturgesetz） | 理性所认识的、普遍有效的道德法则 | 《关于人类社会生活的理性思想》，1721 |
| 义务（Pflicht） | 源于自然法或实证法的道德约束 | 《关于人类社会生活的理性思想》 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对康德的影响** | 康德早年是沃尔夫主义者，其哲学体系沿用了沃尔夫的体系化方法；康德后来在《纯粹理性批判》中批判了沃尔夫的"独断论" |
| **对德国启蒙运动的影响** | 沃尔夫是德国启蒙运动的哲学基础，其理性主义影响了整个18世纪德国知识界 |
| **对大学哲学教育的影响** | 沃尔夫建立了现代大学哲学教育的模式（逻辑学、形而上学、伦理学、政治学） |
| **对哈曼和赫尔德的批判** | 哈曼批判沃尔夫的理性主义忽视了信仰和情感；赫尔德批判沃尔夫的体系是僵化的、抽象的 |
| **主要批评** | 康德的"独断论"批判：沃尔夫未经批判就假设理性可以解决一切哲学问题；哲学问题是否真的能像数学问题一样被严格证明？ |
| **学术评价** | 沃尔夫是杰出的系统化者而非原创性思想家；他的贡献在于建立了一种做哲学的方式，而非提出新的哲学命题 |
