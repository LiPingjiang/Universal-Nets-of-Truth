## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 让·勒朗·达朗贝尔 |
| **外文名** | Jean le Rond d'Alembert |
| **生卒年** | 1717—1783 |
| **国籍** | 法国 |
| **主要身份** | 数学家、物理学家、哲学家、百科全书编纂家、法兰西学院院士 |
| **师承/传统** | 启蒙运动，受笛卡尔、牛顿、洛克影响；与狄德罗合作主编《百科全书》 |

| **版本** | Jean le Rond d'Alembert, *Œuvres complètes*, Paris, 1805（标准版）；G. J. Hollister-Short 编，*Selected Writings*, Cambridge, 2018 |
| **中文译本** | 目前尚无完整中文译本，序言部分有选译见于哲学史著作 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 百科全书派 | 与狄德罗合作主编《百科全书》，是启蒙运动的旗舰项目和核心人物 |
| 温和理性主义 | 推崇理性方法，但认识到理性方法的局限，不盲目崇拜数学方法 |
| 科学哲学家 | 既是数学家又是哲学家，致力于将科学方法应用于哲学 |
| 知识分类学家 | 以人类心灵的三种能力（记忆、理性、想象）为基础构建了系统的知识分类体系 |
| 牛顿主义者 | 在物理学上追随牛顿，但反对牛顿的绝对时空观，提出自己的运动定律 |
| 启蒙宣言者 | 《百科全书编辑者序言》被称为"启蒙运动的宣言"，阐述了启蒙运动的核心理念 |
| 法兰西学院院士 | 1772年当选为法兰西学院终身秘书，是法国学术界的权威人物 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 百科全书（Encyclopédie） | 达朗贝尔与狄德罗主编的《百科全书》，共28卷（1751-1772），是启蒙运动的旗舰项目，旨在汇集和传播人类知识 | 《百科全书编辑者序言》 |
| 编辑者序言（Discours préliminaire） | 达朗贝尔撰写的《百科全书》序言，被称为"启蒙运动的宣言"，阐述了知识分类体系和哲学方法 | 《百科全书编辑者序言》 |
| 知识的三个分支（trois branches de la connaissance） | 来自记忆的知识（历史）、来自理性的知识（哲学/科学）、来自想象的知识（艺术）；对应人类心灵的三种能力：记忆、理性、想象 | 《百科全书编辑者序言》第一部分 |
| 思辨哲学（philosophie spéculative） | 以认识真理为目的的哲学，包括形而上学、逻辑学、自然哲学等；与实践哲学相对 | 《百科全书编辑者序言》第二部分 |
| 实践哲学（philosophie pratique） | 以指导行为为目的的哲学，包括伦理学、政治学等；与思辨哲学相对 | 《百科全书编辑者序言》第二部分 |
| 理性方法（méthode de la raison） | 以数学方法为模型的理性推理方法：精确定义、严格推论、清晰术语；是达朗贝尔哲学方法论的核心 | 《哲学要素论》 |
| 理性与经验（raison et expérience） | 经验是一切知识的基础，但理性是阐明和组织知识的唯一手段；达朗贝尔试图综合理性主义和经验主义 | 《百科全书编辑者序言》第二部分 |
| 数学家方法（méthode des mathématiciens） | 以数学的精确性和严格性为模型的哲学方法；达朗贝尔主张哲学应学习数学家的方法，但也认识到其局限 | 《哲学要素论》第一章 |
| 知识的界限（bornes de nos connoissances） | 哲学应认识自身知识的界限，不超出理性所能及的范围；这是达朗贝尔批判哲学的萌芽 | 《百科全书编辑者序言》第二部分 |
| 启蒙（les Lumières） | 以理性、科学和自由为核心的思想运动，百科全书是启蒙运动的旗舰项目；达朗贝尔的序言是启蒙运动理念的集中表达 | 《百科全书编辑者序言》导论 |

## 四、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 狄德罗（百科全书） | 达朗贝尔与狄德罗合作主编《百科全书》，但两人后来因宗教和政治观点分歧而疏远；达朗贝尔更保守，狄德罗更激进 | 第三章：理性与经验 |
| 伏尔泰（启蒙运动） | 达朗贝尔与伏尔泰是启蒙运动的核心人物，但达朗贝尔更学术化，伏尔泰更大众化；伏尔泰称达朗贝尔为"我们中最有学问的人" | 第九章：自由与必然 |
| 洛克（经验主义） | 达朗贝尔继承了洛克的经验主义，但试图将其与理性主义综合；他认为经验是基础，理性是阐明和组织的手段 | 第六章：经验与理性 |
| 笛卡尔（理性主义） | 达朗贝尔继承了笛卡尔的理性主义方法，但批判了笛卡尔的形而上学独断论；达朗贝尔主张理性应认识自身的界限 | 第七章：心灵与物质 |
| 牛顿（科学） | 达朗贝尔在物理学上追随牛顿，但反对牛顿的绝对时空观，提出自己的运动定律；他试图将牛顿力学推广到哲学 | 第七章：心灵与物质 |
| 培根（知识分类） | 达朗贝尔的知识分类体系继承了培根的《新工具》和《学术的进展》，但达朗贝尔以人类心灵的三种能力为基础，更为系统 | 第三章：理性与经验 |
| 孔狄亚克（感觉主义） | 达朗贝尔与孔狄亚克同为百科全书派，但达朗贝尔更强调理性的作用，孔狄亚克更强调感觉的作用 | 第三章：理性与经验 |
| 康德（批判哲学） | 达朗贝尔对理性界限的认识是康德批判哲学的先驱；康德可能受到达朗贝尔的启发，但更系统地发展了批判哲学 | 第八章：主体与绝对 |
| 沃尔夫（体系哲学） | 达朗贝尔继承了沃尔夫的知识分类传统，但批判了沃尔夫的过度体系化；达朗贝尔更强调经验的角色 | 第三章：理性与经验 |
| 孟德斯鸠（政治哲学） | 达朗贝尔与孟德斯鸠是同时代人，都参与了《百科全书》的编纂；孟德斯鸠的政治哲学影响了达朗贝尔的实践哲学观 | 第三章：理性与经验 |
| 杜尔哥（经济改革） | 达朗贝尔与杜尔哥是好友和同道，都支持经济改革和重农学派；杜尔哥曾任命达朗贝尔为《百科全书》的"保护人" | 第三章：理性与经验 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对启蒙运动的影响** | 达朗贝尔是启蒙运动最重要的组织者之一。他与狄德罗合作主编的《百科全书》是启蒙运动的旗舰项目，其序言被称为"启蒙运动的宣言"，直接影响了整个欧洲的知识界 |
| **对知识分类学的影响** | 达朗贝尔以人类心灵的三种能力（记忆、理性、想象）为基础构建的知识分类体系，是近代知识分类学最重要的成果之一。它影响了后来的图书馆学、学科分类和大学课程设置 |
| **对哲学方法论的影响** | 达朗贝尔主张以数学方法改造哲学，要求精确定义、严格推论和清晰术语。同时，他也认识到理性方法的局限——在以道德和物理事物为对象的科学中，不能完全应用数学的精确性。这种"审慎理性主义"对后来的哲学方法论有重要影响 |
| **对科学哲学的影响** | 达朗贝尔作为数学家和物理学家，致力于将科学方法应用于哲学。他的"数学家方法"思想影响了后来的科学哲学，从孔德的实证主义到维也纳学派的逻辑实证主义，都可以追溯到达朗贝尔 |
| **对康德的影响** | 达朗贝尔对理性界限的认识是康德批判哲学的先驱。虽然达朗贝尔没有系统地发展批判哲学，但他对"理性方法的局限"的认识——"并不总是可能将道德和物理事物还原为精确的定义"——预示了康德对理性独断论的批判 |
| **对法兰西学院的影响** | 达朗贝尔1772年当选为法兰西学院终身秘书，是法国学术界的权威人物。他利用这一地位推广启蒙思想，影响了法国的教育改革和学术发展 |
| **对《百科全书》的影响** | 达朗贝尔与狄德罗的合作使《百科全书》成为启蒙运动最重要的传播工具。虽然两人后来因宗教和政治观点分歧而疏远，但《百科全书》的影响力已经超越了他们个人的分歧 |
| **主要批评** | 达朗贝尔被批评为"温和的改革者"而非"革命者"——他的保守态度（如对宗教的妥协）被更激进的启蒙思想家（如狄德罗、霍尔巴赫）批评。此外，他的"数学家方法"是否过于简化？道德和物理事物是否真的可以部分还原为数学方法？这些问题至今仍在争论 |
| **学术评价** | 达朗贝尔是启蒙运动最重要的"桥梁人物"之一——他既是科学家又是哲学家，既是理性主义者又是经验主义者，既是改革者又是保守者。他的贡献不在于原创性的理论建构，而在于将科学方法、理性主义和启蒙理念综合成一种可操作的哲学实践。他的《百科全书编辑者序言》是启蒙运动理念的集中表达，至今仍是研究启蒙运动的必读文献。 |
