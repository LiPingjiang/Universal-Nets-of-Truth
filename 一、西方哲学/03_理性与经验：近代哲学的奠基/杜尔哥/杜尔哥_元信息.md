# 杜尔哥 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 安·罗伯尔·雅克·杜尔哥 |
| **外文名** | Anne Robert Jacques Turgot, Baron de Laune |
| **生卒年** | 1727—1781 |
| **国籍** | 法国 |
| **主要身份** | 经济学家、政治家、哲学家、重农学派代表人物、路易十六时期财政总监 |
| **师承/传统** | 启蒙运动，重农学派（physiocratie），受魁奈、孔狄亚克影响；与伏尔泰、达朗贝尔交好 |
| **主要著作** | *Tableau philosophique des progrès successifs de l'esprit humain*（《关于人类精神持续进步的历史哲学》，1750年）；*Réflexions sur la formation et la distribution des richesses*（《关于财富的形成和分配的反思》，1766年）；*Valeurs et monnaies*（《价值与货币》，1749年）；*Lettres sur le commerce des grains*（《关于谷物贸易的信件》，1770年） |
| **版本** | *Œuvres de Turgot*, éd. G. Schelle, 5 vols., Paris, 1913-1923（标准学术版）；D. Dakin, *Turgot and the Ancien Régime in France*, London, 1939（英译本选集） |
| **中文译本** | 目前尚无完整中文译本，部分内容见于经济学史和哲学史著作 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 进步史观 | 提出"人类精神持续进步"的历史哲学，是近代进步史观最重要的奠基人之一 |
| 历史哲学 | 将哲学方法应用于历史研究，试图揭示历史发展的普遍规律；影响了后来的黑格尔和马克思 |
| 重农学派 | 重农学派（physiocratie）最重要的代表人物，主张自然秩序和自由放任经济政策 |
| 经济自由主义 | 主张贸易自由，认为市场机制使个人利益与公共利益一致；是古典经济学的先驱 |
| 财政改革家 | 1774-1776年担任路易十六财政总监，推行激进的经济改革（自由贸易、废除行会、土地税改革），但失败被免职 |
| 启蒙思想家 | 与伏尔泰、达朗贝尔、孔狄亚克等启蒙运动核心人物交好，参与百科全书编纂 |
| 需要动力论 | 认为"需要"（besoin）是人类行为和历史进步的根本动力 |
| 知识革命论 | 认为历史是一系列知识革命，每一次革命为下一次革命做准备 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 进步史观（histoire des progrès / progrès de l'esprit humain） | 人类历史是精神持续进步的必然过程，推动进步的原因在于精神本身的本性和社会的本性 | 《关于人类精神持续进步的历史哲学》导论 |
| 历史哲学（philosophie de l'histoire） | 将哲学方法应用于历史研究，试图揭示历史发展的普遍规律；区别于编年史和叙事史 | 《关于人类精神持续进步的历史哲学》 |
| 人类精神（esprit humain） | 人类理性、知识和文明的总和，是历史进步的主体；通过经验学习和反思完善知识 | 《关于人类精神持续进步的历史哲学》导论 |
| 知识的阶段（étapes de la connaissance / révolutions） | 人类精神进步的阶段性革命，每一阶段由前一阶段准备，又为下一阶段做准备；这里的"革命"指知识革命而非政治革命 | 《关于人类精神持续进步的历史哲学》§10 |
| 需要与动力（besoin et moteur） | 需要是人类行为和历史进步的根本动力，真实的需要和想象的需要都推动进步；社会创造新需要，新需要产生新知识 | 《关于人类精神持续进步的历史哲学》§10 |
| 科学与艺术（sciences et arts） | 人类精神进步的两大主要动力，互相促进，推动社会完善；科学完善艺术，艺术完善科学 | 《关于人类精神持续进步的历史哲学》§12 |
| 贸易自由（liberté du commerce） | 经济自由主义的核心主张，认为自由贸易使个人利益与公共利益一致；通过市场机制使财富在社会各阶级传播 | 《关于财富的形成和分配的反思》§33 |
| 土地与财富（terre et richesse） | 土地是一切财富的最初和唯一来源，劳动在土地上创造财富；这是重农学派的核心命题 | 《关于财富的形成和分配的反思》§1 |
| 重农学派（physiocratie） | 主张自然秩序（ordre naturel）和自由放任（laissez-faire）经济政策，认为农业是唯一生产财富的部门 | 重农学派传统 |
| 政治革命（révolution politique） | 政治革命是人类精神进步的结果，而非原因；自由和理性并肩前进，政治革命是精神进步的外在表现 | 《关于人类精神持续进步的历史哲学》§2 |
| 知识的传播（répandre les connoissances） | 社会传播知识的本性，推动人类精神的进步；知识传播是进步的关键机制 | 《关于人类精神持续进步的历史哲学》导论 |
| 经济改革（réforme économique） | 杜尔哥在路易十六时期推行激进经济改革：自由贸易、废除行会、土地税改革、取消强制劳役；改革失败反映了旧制度改革的困境 | 历史实践 |

| 孔狄亚克（感觉主义） | 杜尔哥与孔狄亚克是好友，两人都受洛克影响；杜尔哥的经济哲学部分借鉴了孔狄亚克的价值理论 | 第三章：理性与经验 |
| 魁奈（重农学派） | 魁奈是重农学派创始人，杜尔哥是其最重要的继承者和改革者；杜尔哥的经济改革实践了重农学派理论 | 第三章：理性与经验 |
| 亚当·斯密（古典经济学） | 斯密在《国富论》中借鉴了杜尔哥的经济思想，尤其是劳动分工和自由贸易理论；两人有过直接交流 | 第十二章：正义与共同体 |
| 黑格尔（历史哲学） | 黑格尔的历史哲学在方法论上受杜尔哥影响——将哲学应用于历史，揭示历史发展的规律；但黑格尔更强调辩证法，杜尔哥更强调渐进进步 | 第八章：主体与绝对 |
| 马克思（历史唯物主义） | 马克思的历史唯物主义在方法论上受杜尔哥影响——将经济因素作为历史发展的动力；但马克思更强调阶级斗争，杜尔哥更强调知识进步 | 第九章：自由与必然 |
| 康德（历史哲学） | 康德的《世界公民观点下的普遍历史观念》同样主张历史有普遍规律，但康德更强调"自然意图"而非"精神进步" | 第八章：主体与绝对 |
| 孟德斯鸠（政治哲学） | 杜尔哥继承了孟德斯鸠的历史方法，但超越了孟德斯鸠的"地理环境决定论"，强调精神和知识的进步 | 第三章：理性与经验 |
| 洛克（经验主义） | 杜尔哥的经济哲学受洛克影响，尤其是财产权和劳动价值理论；但杜尔哥更强调社会整体而非个人权利 | 第六章：经验与理性 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对进步史观的影响** | 杜尔哥是近代进步史观最重要的奠基人之一。他的"人类精神持续进步"的历史哲学直接影响了孔多塞（《人类精神进步史表纲要》）、黑格尔（历史哲学）和马克思（历史唯物主义）。虽然后来的进步史观更加复杂（如增加了"阶级斗争"、"辩证法"等），但杜尔哥的基本框架——历史是精神进步的必然过程——是近代历史哲学的起点 |
| **对历史哲学的影响** | 杜尔哥将哲学方法应用于历史研究，试图揭示历史发展的普遍规律。这一方法论影响了后来的黑格尔和马克思，但杜尔哥本人没有系统地发展历史哲学——他的历史哲学只是一篇论文（1750年），而黑格尔和马克思则写了专著 |
| **对古典经济学的影响** | 杜尔哥在《关于财富的形成和分配的反思》中提出了古典经济学的核心思想：自由贸易、劳动分工、价值理论。他直接影响了亚当·斯密（《国富论》），斯密在《国富论》中借鉴了杜尔哥的思想。杜尔哥被认为是古典经济学的先驱之一，与魁奈、斯密并列 |
| **对重农学派的影响** | 杜尔哥是重农学派最重要的代表人物和改革者。他不仅在理论上发展了重农学派，而且在实践上推行了激进的经济改革（1774-1776年担任财政总监）。但改革失败反映了旧制度改革的困境——既得利益者的阻力使改革无法成功。杜尔哥的改革失败预示了法国大革命的不可避免 |
| **对启蒙运动的影响** | 杜尔哥是启蒙运动最重要的经济思想家之一。他与伏尔泰、达朗贝尔、孔狄亚克等核心人物交好，参与百科全书编纂，其进步史观和经济自由主义是启蒙运动核心理念的重要组成部分 |
| **对孔多塞的影响** | 杜尔哥是孔多塞的导师和好友。孔多塞在《人类精神进步史表纲要》中直接发展了杜尔哥的进步史观，将历史分为十个阶段，并预言了人类未来的无限进步。孔多塞的乐观主义直接源于杜尔哥 |
| **对黑格尔和马克思的影响** | 黑格尔和马克思的历史哲学在方法论上受杜尔哥影响——将哲学应用于历史，揭示历史发展的规律。但黑格尔更强调辩证法，马克思更强调阶级斗争，而杜尔哥更强调知识进步和需要动力 |
| **对现代经济学的影响** | 杜尔哥的经济思想——自由贸易、劳动分工、价值理论——是现代经济学的重要先驱。他的"贸易自由使个人利益与公共利益一致"的思想是现代自由市场经济学的基础。但他的重农主义（土地是唯一财富来源）已被现代经济学超越 |
| **主要批评** | 杜尔哥的进步史观是否过于乐观？20世纪的两次世界大战和极权主义是否挑战了"进步"的必然性？杜尔哥的经济改革是否过于激进，忽视了社会改革的渐进性？他的重农主义是否过于简化，忽视了工业生产和资本的作用？这些问题至今仍在争论 |
| **学术评价** | 杜尔哥是18世纪最杰出的思想家之一——他既是理论家（进步史观、经济哲学），又是实践家（财政改革）。他的进步史观是近代历史哲学的起点，他的经济思想是古典经济学的先驱。但他的改革失败也反映了理论与实践的差距——在旧制度下，激进改革往往无法成功。杜尔哥的思想对后来的历史哲学、经济学和政治学都有深远影响，但他本人在哲学史上的地位不如伏尔泰、卢梭或康德那样显赫，因为他的思想更多地被后来的思想家所吸收和超越。 |
