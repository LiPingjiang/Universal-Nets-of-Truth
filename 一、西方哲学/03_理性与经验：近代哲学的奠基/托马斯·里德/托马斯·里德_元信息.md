# 托马斯·里德 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 托马斯·里德 |
| **外文名** | Thomas Reid |
| **生卒年** | 1710—1796 |
| **国籍** | 苏格兰（英国） |
| **主要身份** | 哲学家、苏格兰常识学派创始人、道德学家、阿伯丁大学校长、格拉斯哥大学道德哲学教授 |
| **师承/传统** | 苏格兰启蒙运动，受牛顿、洛克影响；继承并发展了常识哲学传统 |
| **主要著作** | *An Inquiry into the Human Mind on the Principles of Common Sense*（《根据常识原则对人类心灵的研究》，1764年）；*Essays on the Intellectual Powers of Man*（《论人类理智能力》，1785年）；*Essays on the Active Powers of Man*（《论人类行动能力》，1788年） |
| **版本** | D. Brookes 编，*Thomas Reid: An Inquiry into the Human Mind on the Principles of Common Sense*, Edinburgh University Press, 1997（学术校勘版）；D. R. Brookes 编，*Thomas Reid: Essays on the Intellectual Powers of Man*, Edinburgh University Press, 2002 |
| **中文译本** | 目前尚无完整中文译本，部分内容见《西方哲学史》相关章节 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 常识哲学 | 苏格兰常识学派创始人，主张常识是一切知识和推理的基础，对怀疑论和观念论进行批判 |
| 直接实在论 | 主张我们直接感知外部对象本身，而非对象的观念或图像；反对从笛卡尔到贝克莱的观念论传统 |
| 苏格兰启蒙运动 | 苏格兰启蒙运动最重要的哲学家之一，与休谟、哈奇森、亚当·斯密等同时代 |
| 观念论批判者 | 对笛卡尔、洛克、贝克莱的观念论传统进行系统批判，认为观念论是"哲学家们的虚构" |
| 怀疑论批判者 | 认为怀疑论违背常识，是最荒谬的哲学；主张自然本能引导我们信任感官和意识 |
| 道德哲学家 | 在格拉斯哥大学担任道德哲学教授，著有《论人类行动能力》，发展了自然法道德哲学 |
| 第一原则论 | 主张常识有不可否定的"第一原则"，否认它们等于否认人类知识的整个结构 |
| 心灵官能论 | 主张心灵有各种官能（知觉、记忆、判断、推理等），虽然可能犯错，但大部分判断是可靠的 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 常识（common sense） | 人类本性赋予的基本信念和判断能力，是一切知识和推理的基础；不是理性的产物，而是本性的指令 | 《根据常识原则对人类心灵的研究》第一章 |
| 直接实在论（direct realism） | 我们直接感知外部对象本身，而非对象的观念或图像；感官对象是外在于心灵并独立于心灵的 | 《根据常识原则对人类心灵的研究》第六章 |
| 观念论（idealism / ideism） | 里德所批判的哲学——认为我们感知的不是外部对象，而是心灵中的观念或图像；里德认为观念论是"哲学家们的虚构" | 《根据常识原则对人类心灵的研究》第六章 |
| 第一原则（first principles） | 常识的基本原则，不依赖推理而自明，是一切知识的基础；否认它们等于否认人类知识的整个结构 | 《论人类理智能力》第六论文 |
| 自然本能（natural instinct） | 人类本性引导我们信任感官、记忆和意识的原始本能；不是理性的产物，而是本性的指令 | 《根据常识原则对人类心灵的研究》第一章 |
| 怀疑论（scepticism） | 里德主要批判对象——怀疑感官、记忆和意识的可靠性；里德认为怀疑论违背常识，是最荒谬的哲学 | 《根据常识原则对人类心灵的研究》第五章 |
| 心灵的官能（faculties of the mind） | 心灵的各种能力——知觉、记忆、判断、推理等；里德认为这些官能虽然可能犯错，但大部分判断是可靠的 | 《论人类理智能力》 |
| 感官的证词（testimony of our senses） | 感官提供的知识证据；里德认为我们应该信任感官的证词，因为这是自然本能的引导 | 《根据常识原则对人类心灵的研究》第一章 |
| 知觉的直接性（immediacy of perception） | 我们直接感知外部对象，没有观念或图像的介入；这是直接实在论的核心主张 | 《根据常识原则对人类心灵的研究》第六章 |
| 苏格兰常识学派（Scottish school of common sense） | 以里德为代表的苏格兰哲学学派，主张常识哲学、直接实在论和对怀疑论的批判；其他代表人物包括詹姆斯·贝蒂、杜格尔德·斯图尔特等 | 苏格兰启蒙运动 |
| 本性的构造（constitution of our nature） | 人类本性的自然结构，决定了我们的基本信念和判断方式；常识原则源于本性的构造，而非理性建构 | 《论人类理智能力》第六论文 |
| 自明性（self-evidence） | 常识第一原则的特征——不依赖推理而自明，被人类常识普遍认可；否认自明原则等于自相矛盾 | 《论人类理智能力》第六论文 |
| 道德能力（active powers） | 人类行动和道德判断的能力，包括意志、道德感等；里德在《论人类行动能力》中发展了自然法道德哲学 | 《论人类行动能力》 |
| 自由意志（liberty / free will） | 里德主张自由意志是常识的一部分——我们自然地相信自己有选择的自由；否认自由意志等于否认常识 | 《论人类行动能力》 |

## 四、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 休谟（怀疑论） | 里德是休谟最重要的批判者之一。里德认为休谟的怀疑论是观念论的必然结果，而观念论本身是荒谬的；但休谟和里德都受苏格兰启蒙运动影响 | 第十章：经验与理性 |
| 洛克（观念论） | 里德批判了洛克的观念论——洛克认为我们感知的是观念而非对象；里德认为这一理论导致怀疑论 | 第六章：经验与理性 |
| 贝克莱（主观观念论） | 里德批判了贝克莱的主观观念论——贝克莱认为只有心灵和观念存在；里德认为这完全违背了常识 | 第六章：经验与理性 |
| 笛卡尔（心物二元论） | 里德批判了笛卡尔的心物二元论和观念论——笛卡尔认为我们感知的不是外部对象，而是心灵的观念；里德认为这导致怀疑论 | 第七章：心灵与物质 |
| 哈奇森（道德感） | 里德继承了哈奇森的道德感理论，但将"道德感"转化为"常识原则"——道德判断是常识的一部分 | 第三章：理性与经验 |
| 亚当·斯密（苏格兰启蒙） | 里德与斯密是同时代人，都受苏格兰启蒙运动影响；斯密的道德哲学部分受里德影响 | 第十二章：正义与共同体 |
| 康德（批判哲学） | 里德的常识哲学是康德批判哲学的某种先驱——两者都试图为知识提供"不可怀疑"的基础；但康德的基础是"先验范畴"，里德的基础是"常识" | 第八章：主体与绝对 |
| 摩尔（常识反驳） | 20世纪英国哲学家摩尔继承了里德的常识哲学，在"外部世界的证明"中直接使用了里德的常识论证 | 第十章：语言与逻辑 |
| 奥斯汀（日常语言哲学） | 奥斯汀的日常语言哲学继承了里德对"观念论"的批判，认为哲学问题源于语言的误用 | 第十章：语言与逻辑 |
| 维特根斯坦（语言游戏） | 维特根斯坦的"语言游戏"和"生活形式"概念与里德的"常识"有某种呼应——两者都强调实践和日常信念的优先性 | 第十章：语言与逻辑 |
| 普特南（直接实在论） | 当代美国哲学家普特南在晚年转向了直接实在论，承认里德的影响 | 第十章：语言与逻辑 |
| 麦克道威尔（直接实在论） | 当代分析哲学家麦克道威尔发展了精致的直接实在论，承认里德的影响 | 第十章：语言与逻辑 |
| 培尔（怀疑论） | 里德对怀疑论的批判与培尔的怀疑论形成对比——培尔认为怀疑论是理性的必然结果，里德认为怀疑论是理性的滥用 | 第三章：理性与经验 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对苏格兰常识学派的影响** | 里德是苏格兰常识学派的创始人。该学派在18世纪末19世纪初的苏格兰、英国和美国产生了巨大影响，成为英国哲学的主流之一。其他代表人物包括詹姆斯·贝蒂、杜格尔德·斯图尔特、托马斯·布朗等。常识学派在19世纪的美国大学中占据主导地位，影响了美国哲学的早期发展 |
| **对休谟怀疑论的批判** | 里德对休谟的批判是哲学史上最重要的批判之一。他认为，休谟的怀疑论不是理性的结果，而是观念论的必然结果。因为观念论认为我们感知的不是外部对象，而是观念——那么我们如何知道外部对象存在？休谟的回答是：我们不知道。里德认为，这一怀疑论是荒谬的，因为它违背了常识。里德的批判对后来的认识论有深远影响——从摩尔的"常识反驳"到当代的直接实在论复兴，都可以看到里德的影子 |
| **对观念论的批判** | 里德对笛卡尔-洛克-贝克莱观念论传统的批判是哲学史上最重要的认识论批判之一。他认为，观念论是"哲学家们的虚构"——在自然中没有任何对应物。普通人的世界观是正确的：我们直接感知外部对象。这一批判直接影响了后来的"日常语言哲学"（奥斯汀）和"直接实在论复兴"（普特南、麦克道威尔） |
| **对20世纪分析哲学的影响** | 里德对20世纪分析哲学的影响是间接但深远的。摩尔的"常识反驳"（"这是一个手，因此外部世界存在"）直接源于里德。奥斯汀的日常语言哲学继承了里德对观念论的批判。维特根斯坦的"生活形式"和普特南的"直接实在论"都与里德的常识哲学有某种呼应。里德被认为是分析哲学最重要的"被遗忘的前辈"之一 |
| **对美国哲学的影响** | 常识学派在19世纪的美国大学中占据主导地位，影响了美国哲学的早期发展。从普林斯顿大学到耶鲁大学，常识哲学都是标准课程。直到19世纪末，美国哲学才逐渐被德国观念论（尤其是黑格尔）所取代。里德对美国实用主义（尤其是皮尔士）也有间接影响——皮尔士的"常识实在论"与里德的"常识哲学"有某种呼应 |
| **对道德哲学的影响** | 里德在格拉斯哥大学担任道德哲学教授，著有《论人类行动能力》。他发展了自然法道德哲学，主张道德判断是常识的一部分。他的道德哲学对后来的苏格兰道德哲学和英国道德哲学有深远影响，但不如他的认识论那样著名 |
| **对康德的影响** | 里德的常识哲学是康德批判哲学的某种先驱。两者都试图为知识提供"不可怀疑"的基础——但康德的基础是"先验范畴"（理性的建构），而里德的基础是"常识"（本性的直觉）。康德可能受到里德的影响，但康德更加系统地论证了知识的基础。里德对康德的批评是：康德过于复杂化了常识——我们不需要先验论证来证明外部世界存在，因为常识本身就够了 |
| **主要批评** | 里德的常识哲学受到多方面的批评：（1）他是否过于依赖"常识"？常识是否总是可靠的？科学发现（如地球围绕太阳转）是否挑战了常识？（2）他是否回避了真正的哲学问题？哲学家追问"我们如何知道外部世界存在？"——里德的回答是"常识告诉我们"——这是否过于简单？（3）他是否低估了理性的作用？常识是否是"非理性"的？里德对这些问题的回答是：常识的可靠性是程度性的，不是绝对性的；我们信任常识，但承认它可能犯错；科学发现不是对常识的否定，而是对常识的修正和深化 |
| **学术评价** | 里德是18世纪最重要的认识论哲学家之一，也是苏格兰启蒙运动最重要的代表人物之一。他的贡献不在于提出新的哲学理论，而在于对从笛卡尔到休谟的近代认识论传统进行了系统的批判。他的常识哲学、直接实在论和对怀疑论的批判，对后来的哲学产生了深远影响——从19世纪的苏格兰常识学派到20世纪的日常语言哲学和直接实在论复兴。但里德在哲学史上的地位长期被低估——他被视为"休谟的批判者"而非独立的哲学家。直到20世纪后半叶，随着分析哲学对"常识"和"直接实在论"的重新关注，里德的地位才得到重新评估。当代哲学家（如普特南、麦克道威尔）承认里德是他们最重要的先驱之一。 |
