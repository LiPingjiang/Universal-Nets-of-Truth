## 三、核心论点列表

1. **快乐与痛苦是人类行为的主宰**：人的一切行为由趋乐避苦驱动，这是行为的因果必然性，也是道德判断的基础。
2. **功利原则即最大幸福原则**：行为的善恶取决于其增进或减少利益相关者幸福的倾向；最大多数人的最大幸福是检验一切的标准。
3. **快乐可量化**：快乐和痛苦可通过强度、持续时间、确定性、邻近性、丰度、纯度、广度七个维度进行计算比较。
4. **快乐的平等主义**：每个人的快乐同等重要，"每个人都算一个，没有人算多于一个"，反对等级特权。
5. **权利是法律的产物**：自然权利是胡言乱语，权利只能由法律创造；义务在先，权利在后。
6. **全景监狱原理**：通过可见性不对称实现权力运作的自动化和不可见化；被监视者内化监视后实现自我约束。
7. **制度改革的功利主义路径**：法律和制度应以最大幸福为目标进行理性设计，通过制度改革引导个人利益与公共利益一致。

## 四、章节关联

| 关联人物/章节 | 关系说明 |
|--------------|----------|
| 休谟（英国经验主义） | 边沁的经验主义和情感主义伦理学源头 |
| 洛克（自然权利论） | 边沁反对洛克的自然权利说，主张权利来自法律 |
| 密尔（修正功利主义） | 边沁的学生，修正了快乐量化的立场，引入"质的差异" |
| 康德（义务论） | 边沁的功利主义与康德义务论构成伦理学基本对立 |
| 福柯（规训与惩罚） | 对边沁全景监狱的经典哲学分析，揭示规训社会机制 |
| 伏尔泰（启蒙理性） | 同为启蒙精神，关注社会改革与实践 |
| 恩格斯（历史唯物主义） | 功利主义与唯物史观在社会改革取向上存在对话可能 |
| 怀特海（过程哲学） | 边沁的"快乐计算"与怀特海对量化思维的批评形成对照 |
| 伽达默尔（诠释学） | 边沁的实证主义与伽达默尔对实证主义的批判形成对话 |
