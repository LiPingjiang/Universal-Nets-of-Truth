## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 苏格拉底学派（Σωκρατικοί）；前苏格拉底哲学的终结者；柏拉图主义的奠基者 |
| 核心方法 | 诘问法（ἔλεγχος / elenchus）；助产术（μαιευτική / maieutics） |
| 核心命题 | 德性即知识（ἀρετὴ ἐπιστήμη ἐστίν）；无人自愿作恶（οὐδεὶς ἑκὼν κακός）；未经检验的生活不值得过（ὁ ἀνεξέταστος βίος οὐ βιωτός） |
| 伦理学立场 | 理性主义伦理学；德性伦理学的先驱；知识主义（intellectualism） |
| 政治立场 | 守法主义（legalism）；公民义务论；对民主制度的批判性态度 |
| 认识论立场 | 无知之知（knowing that one does not know）；定义主义（definitionism） |
| 宗教立场 | 个人神灵（δαιμόνιον）；对传统多神教的批判性态度 |

| 教育哲学 | 助产术影响现代启发式教学法（Socratic method） |
| 现代影响 | 批判性思维教育；苏格拉底式提问法（Socratic questioning）广泛用于法学、医学、商学教育 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 苏格拉底问题（Socratic Problem） | 历史的苏格拉底与柏拉图笔下的苏格拉底有多大差异？Vlastos 提出"两个苏格拉底"理论；学界尚无定论 |
| 德性即知识的解释 | 苏格拉底是否真的认为知识是德性的充分条件？还是只是必要条件？ |
| 意志软弱（ἀκρασία）问题 | 苏格拉底否认意志软弱的可能性，但这与日常经验相悖；亚里士多德对此提出批评 |
| 《美诺篇》98a 的矛盾 | 苏格拉底在《美诺篇》结尾说德性是"神圣的命运"而非知识，这与"德性即知识"命题矛盾吗？ |
| 苏格拉底的政治立场 | 他是民主制度的批评者（因为他认为政治需要专业知识）还是守法主义者（《克里托篇》）？ |
| 临终遗言的解读 | "我们欠阿斯克勒庇俄斯一只公鸡"是对生命的否定（尼采的解读）还是对日常义务的坚守？ |

## 关键年表

| 年份 | 事件 |
|------|------|
| 约 470 BCE | 生于雅典 |
| 约 450 BCE | 青年时期，可能接受过雕刻训练 |
| 约 440 BCE | 开始哲学活动，在雅典广场与人对话 |
| 432 BCE | 参加波提狄亚战役（Potidaea），表现出色 |
| 424 BCE | 参加德利翁战役（Delium），在撤退中保持镇定 |
| 422 BCE | 参加安菲波利斯战役（Amphipolis） |
| 约 420 BCE | 与柏拉图相识（柏拉图约生于 428 BCE） |
| 406 BCE | 担任议员（βουλευτής），拒绝参与对将领的非法审判 |
| 404 BCE | 雅典三十僭主统治期间，拒绝执行非法命令 |
| 399 BCE | 被控"不敬神"和"腐蚀青年"，被判处死刑，饮鸩而死 |
