## 著作信息

| 文献 | 类型 | 卷数 | 主题 | 保存状况 |
|------|------|------|------|----------|
| 《物性论》（*De Rerum Natura*） | 哲学长诗（六音步诗体） | 六卷 | 原子论、宇宙论、灵魂论、伦理学、宗教批判、文明史 | 完整保存（通过中世纪抄本传承，1417 年重新发现） |

### 各卷主题概览

| 卷次 | 主题 |
|------|------|
| 卷一 | 原子与虚空的基本原理；物质不灭；批判宗教 |
| 卷二 | 原子的运动与偏斜（clinamen）；原子的性质；世界的多样性 |
| 卷三 | 灵魂的物质性与可朽性；死亡论证；对死亡恐惧的消解 |
| 卷四 | 感觉理论；视觉、听觉、味觉；梦；性爱 |
| 卷五 | 宇宙的生灭；反目的论；人类文明的起源与发展 |
| 卷六 | 自然现象的解释（雷电、地震、火山、瘟疫）；雅典瘟疫的描写 |

## 思想标签

| 标签类型 | 标签内容 |
|----------|----------|
| 哲学流派 | 伊壁鸠鲁主义（Epicureanism）；罗马自然主义 |
| 核心方法 | 理性分析（ratio）；自然主义解释；诗歌论证 |
| 核心命题 | 物质不灭（nil posse creari de nilo）；原子偏斜（clinamen）；死亡与我们无关（nil igitur mors est ad nos）；宗教是恐惧的根源（tantum religio potuit suadere malorum） |
| 自然哲学立场 | 原子论；唯物主义；反目的论；决定论的有限突破 |
| 伦理学立场 | 伊壁鸠鲁式快乐主义；消除恐惧为最高目标；无扰动（ataraxia）为理想状态 |
| 宗教立场 | 批判传统宗教迷信；承认神的存在但否认神的干预；自然神论的先驱 |
| 认识论立场 | 感觉主义；理性对感觉的补充 |

## 历史影响

| 影响领域 | 具体影响 |
|----------|----------|
| 古代接受 | 西塞罗赞赏其诗艺；维吉尔《农事诗》有致敬段落；奥维德预言其不朽 |
| 中世纪 | 几乎完全失传；仅少数修道院保存抄本 |
| 文艺复兴 | 1417 年波焦重新发现；影响伽桑狄的原子论复兴；影响布鲁诺的无限宇宙论 |
| 科学革命 | 通过伽桑狄影响牛顿的原子论；为机械论世界观提供哲学基础 |
| 启蒙时代 | 影响法国唯物主义（霍尔巴赫、拉美特利）；影响休谟的宗教批判 |
| 近现代哲学 | 死亡哲学中的"对称论证"；马克思博士论文对原子偏斜的分析 |
| 文学影响 | 影响维吉尔、奥维德；影响但丁（《神曲》中提及伊壁鸠鲁）；影响蒙田、斯宾诺莎 |

## 重要争议

| 争议问题 | 主要立场 |
|----------|----------|
| 生平的可靠性 | 圣哲罗姆关于卢克莱修发疯和自杀的记载是否可信？现代学者多持怀疑态度，认为这是基督教作者的污名化叙述 |
| 《物性论》是否完成 | 全诗是否经过卢克莱修本人的最终修订？西塞罗的信件暗示他在卢克莱修死后对诗稿进行了编辑 |
| clinamen 与自由意志 | 原子偏斜引入的是随机性还是真正的自由意志？随机性是否等于自由？西塞罗和现代学者对此均有批评 |
| 对称论证的有效性 | 出生前的不存在与死后的不存在真的是对称的吗？托马斯·内格尔认为两者不对称：死亡剥夺了未来的好处，而出生前的不存在没有剥夺任何东西 |
| 卢克莱修的宗教立场 | 他是无神论者还是自然神论者？他承认神的存在，但否认神的干预，这与无神论有本质区别 |

## 关键年表

| 年份 | 事件 |
|------|------|
| 约 99 BCE | 生于罗马（或意大利某地） |
| 约 70 BCE | 开始写作《物性论》（推测） |
| 约 55 BCE | 去世；西塞罗在信中提到《物性论》 |
| 1417 CE | 波焦·布拉乔利尼在德国修道院重新发现《物性论》抄本 |
| 1473 CE | 《物性论》首次印刷出版（布雷西亚版） |
| 1841 CE | 马克思博士论文《德谟克利特的自然哲学和伊壁鸠鲁的自然哲学的差别》深入分析 clinamen |
| 2011 CE | 斯蒂芬·格林布拉特《大转向》（*The Swerve*）获普利策奖，重述《物性论》重发现的文化史意义 |

## 参考文献（元信息专用）

4. Monica Gale (ed.), *Oxford Readings in Classical Studies: Lucretius* (Oxford University Press, 2007)——卢克莱修研究论文集。
5. 方书春译，《物性论》（商务印书馆，1981）——中文世界最重要的卢克莱修译本。
