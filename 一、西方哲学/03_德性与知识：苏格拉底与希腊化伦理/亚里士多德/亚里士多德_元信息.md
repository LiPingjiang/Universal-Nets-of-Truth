## 伦理学论证结构

### 功能论证（ergon argument）

```
1. 每种事物都有其特有功能（ἔργον）
2. 人的功能不是生长（植物），也不是感觉（动物）
3. 人的特有功能是"灵魂按照理性的活动"
4. 善的功能就是完满地实现其功能
5. 因此，人的善 = 灵魂按德性的完满活动
```

### 德性定义的结构

```
德性 = 选择的倾向（ἕξις προαιρετική）
     + 相对于我们的中庸（μεσότης πρὸς ἡμᾶς）
     + 由理性规定（ὡρισμένῃ λόγῳ）
     + 正如明智的人会规定的那样（ᾧ ἂν ὁ φρόνιμος ὁρίσειεν）
```

### 习惯化的机制

```
重复行动 → 稳定倾向（ἕξις） → 第二自然 → 自发愉悦 → 德性品格
```

### 实践智慧与德性的关系

```
伦理德性（方向） + 实践智慧（手段） = 完满的德性行动
  ↓ 没有实践智慧 → 只是自然倾向
  ↓ 没有伦理德性 → 只是聪明（可能用于恶）
```

## 两种幸福的层次

| 维度 | 实践幸福（πρακτική εὐδαιμονία） | 沉思幸福（θεωρητική εὐδαιμονία） |
|------|-------------------------------|--------------------------------|
| 实现途径 | 伦理德性 + 政治活动 | 理论智慧 + 沉思 |
| 核心活动 | 行动（πρᾶξις） | 思考（θεωρία） |
| 外在依赖 | 需要一定外在条件 | 最自足 |
| 持久性 | 有限（受生命限制） | 最持久 |
| 愉悦程度 | 高 | 最高 |
| 神性程度 | 人的幸福 | 最接近神的幸福 |
| 适用人群 | 大多数人 | 哲学家/思想家 |
| 亚里士多德评价 | 好 | 最好 |

## 与同时代及后世伦理学的比较

### 与苏格拉底-柏拉图伦理学

| 维度 | 苏格拉底-柏拉图 | 亚里士多德 |
|------|----------------|-----------|
| 善的位置 | 超验的理念世界 | 内在的人类生活 |
| 德性基础 | 知识（回忆说） | 习惯（ἔθος） |
| 方法论 | 辩证法（问答） | 系统分析（十卷结构） |
| 目标 | 灵魂转向 | 灵魂按德性的活动 |
| 政治观 | 哲学家-王统治 | 法治与公民教育 |

### 与希腊化伦理学

| 维度 | 亚里士多德 | 伊壁鸠鲁主义 | 斯多葛主义 |
|------|-----------|-------------|-----------|
| 最高善 | 幸福（德性活动） | 快乐（无扰动） | 德性（符合自然） |
| 情感态度 | 中庸化 | 消除不必要的欲望 | 完全不动心（ἀπάθεια） |
| 外在善 | 需要但非本质 | 极简主义 | 完全无关（" indifferent"） |
| 社会维度 | 城邦政治 | 花园友谊 | 世界公民 |
| 神观 | 不动的推动者 | 不干预的神 | 宇宙理性（λόγος） |
| 自由意志 | 选择（προαίρεσις） | 原子偏斜 | 命运（εἱμαρμένη） |

### 与现代伦理学三大传统

| 传统 | 核心问题 | 代表 | 与亚里士多德的关系 |
|------|----------|------|-------------------|
| 德性伦理学 | 什么样的人？ | 亚里士多德、麦金太尔 | 直接继承 |
| 义务论 | 什么是对的？ | 康德 | 批判继承（从品格转向规则） |
| 功利主义 | 什么产生最大幸福？ | 边沁、密尔 | 批判继承（从完满转向效用） |

## 思想史影响

### 古代
- **逍遥学派**：泰奥弗拉斯托斯、斯特拉波等继承发展
- **希腊化时期**：与伊壁鸠鲁主义、斯多葛主义形成三足鼎立
- **罗马时期**：西塞罗、塞涅卡等吸收亚里士多德伦理学

### 中世纪
- **伊斯兰哲学**：阿维森纳、阿威罗伊评注《尼各马可伦理学》
- **基督教经院哲学**：托马斯·阿奎那将亚里士多德伦理学与基督教结合
- **但丁**：《神曲》中的道德结构深受亚里士多德影响

### 近代
- **文艺复兴**：亚里士多德伦理学重新被研究
- **宗教改革**：新教伦理与亚里士多德德性伦理的张力
- **启蒙运动**：功利主义和义务论兴起，德性伦理学被边缘化

### 现代复兴
- **1958年**：G. E. M. 安斯康姆《现代道德哲学》呼吁回归德性伦理学
- **1981年**：阿拉斯代尔·麦金太尔《追寻德性》系统复兴亚里士多德传统
- **1986年**：玛莎·努斯鲍姆《善的脆弱性》从情感与运气角度重新解读
- **当代**：德性伦理学成为与分析伦理学、大陆伦理学并列的第三大传统
