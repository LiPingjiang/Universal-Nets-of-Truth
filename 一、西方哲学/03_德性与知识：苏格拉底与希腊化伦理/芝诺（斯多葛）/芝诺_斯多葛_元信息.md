## 思想定位

| 字段 | 内容 |
|------|------|
| 历史时期 | 希腊化时期（Hellenistic Period） |
| 哲学部门 | 伦理学（核心）、物理学、逻辑学 |
| 核心命题 | 顺应自然而生活（ὁμολογουμένως τῇ φύσει ζῆν） |
| 对立学派 | 伊壁鸠鲁学派（Epicureanism） |
| 后继发展 | 罗马斯多葛（塞涅卡、爱比克泰德、马可·奥勒留） |
| 现代影响 | 认知行为疗法（CBT）、世界主义政治哲学、自然法理论 |

## 核心概念

| 概念 | 希腊语/拉丁语 | 含义 | 重要性 |
|------|--------------|------|--------|
| 顺应自然 | ὁμολογουμένως τῇ φύσει ζῆν | 使意志与宇宙理性一致 | ★★★★★ |
| 德性 | ἀρετή (aretē) | 唯一真正的善 | ★★★★★ |
| 控制二分法 | τὰ ὑφ' ἡμῖν / ἐφ' ἡμῖν | 区分可控与不可控 | ★★★★★ |
| 世界公民 | κοσμοπολίτης (kosmopolitēs) | 超越城邦的普遍公民身份 | ★★★★☆ |
| 自然法 | lex / λόγος | 根植于自然的理性法则 | ★★★★☆ |
| 激情 | πάθος (pathos) | 违背理性的情感 | ★★★★☆ |
| 命运 | εἱμαρμένη (heimarmenē) | 宇宙理性的必然秩序 | ★★★★☆ |
| 选择 | προαίρεσις (prohairesis) | 人唯一真正自由的领域 | ★★★★☆ |
| 无差别物 | ἀδιάφορα (adiaphora) | 外物本身无所谓善恶 | ★★★★☆ |
| 无激情 | ἀπάθεια (apatheia) | 消除激情后的内心平静 | ★★★☆☆ |
