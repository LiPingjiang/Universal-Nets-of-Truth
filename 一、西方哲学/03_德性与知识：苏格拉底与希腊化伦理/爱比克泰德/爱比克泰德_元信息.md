| 生卒年 | 约 55 CE — 约 135 CE |

## 思想定位

| 字段 | 内容 |
|------|------|
| 历史时期 | 罗马早期帝国（Early Roman Empire） |
| 哲学部门 | 伦理学（核心）、实践哲学 |
| 核心命题 | 控制二分法（τὰ ἐφ' ἡμῖν / ἐφ' ἡμῖν）；德性是唯一真正的善；自由即内心的自主 |
| 对立学派 | 伊壁鸠鲁学派（Epicureanism）、怀疑论学派（Pyrrhonism） |
| 后继发展 | 马可·奥勒留《沉思录》、新斯多葛主义、认知行为疗法（CBT） |
| 现代影响 | 认知行为疗法（Albert Ellis、Aaron Beck）、现代斯多葛复兴运动（Stoic Week、Daily Stoic） |

## 核心概念

| 概念 | 希腊语 | 含义 | 重要性 |
|------|--------|------|--------|
| 在我们能力范围内 | ἐφ' ἡμῖν | 判断、冲动、欲望、厌恶——一切我们自己的行为 | ★★★★★ |
| 不在我们能力范围内 | οὐκ ἐφ' ἡμῖν | 身体、财产、名誉、官职——一切非我们行为的东西 | ★★★★★ |
| 判断/信念 | δόγμα (dogma) | 对事物的看法，扰乱人的根源 | ★★★★★ |
| 冲动 | ὁρμή (hormē) | 行动的内在驱动力 | ★★★★☆ |
| 欲望 | ὄρεξις (orexis) | 对某物的追求 | ★★★★☆ |
| 厌恶 | ἔκκλισις (ekklisis) | 对某物的回避 | ★★★★☆ |
| 自由 | ἐλευθερία (eleutheria) | 按自己意愿生活，不受外部强制 | ★★★★★ |
| 必然 | ἀνάγκη (anankē) | 宇宙的必然秩序，应当接受 | ★★★★☆ |
| 选择 | προαίρεσις (prohairesis) | 人唯一真正自由的领域，道德责任的根基 | ★★★★★ |
| 角色 | πρόσωπον (prosōpon) | 人在宇宙戏剧中扮演的角色 | ★★★☆☆ |
