# 一、西方哲学 · 古希腊哲学

## 时间跨度

约公元前6世纪 — 公元前322年（亚里士多德去世）

## 核心问题

万物的本原（arche）是什么？什么是善的生活？知识如何可能？

## 1.3 希腊化-罗马哲学（Hellenistic & Roman Philosophy）

> **时间跨度**：公元前4世纪末 — 公元5世纪
> **核心问题**：如何在不确定的世界中获得内心的宁静（ataraxia）？
