# 费耶阿本德 · 元信息

## 一、人物基本信息

| 字段 | 内容 |
|------|------|
| 姓名 | 保罗·卡尔·费耶阿本德（Paul Karl Feyerabend） |
| 生卒年 | 1924年1月13日 — 1994年2月11日 |
| 国籍 | 奥地利（后移居美国） |
| 出生地 | 维也纳（Vienna, Austria） |
| 主要身份 | 科学哲学家、科学史家 |
| 学派归属 | 无政府主义认识论（Epistemological Anarchism）；科学哲学中的历史主义；多元主义 |
| 教育背景 | 维也纳大学（历史、社会学，1947）、伦敦大学（博士，1951） |
| 任教经历 | 布里斯托大学（1955—1956）、加州大学伯克利分校（1958—1989）、苏黎世联邦理工学院（1980—1990） |
| 核心称号 | "科学哲学界的坏孩子"；"无政府主义认识论的代言人"；"方法论多元主义的激进代表" |

## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 《反对方法》 | *Against Method* | 1975（修订版1988、1993） | 科学中不存在普遍有效的方法论；"怎么都行"（anything goes）是无政府主义认识论的核心口号 |
| 《自由社会中的科学》 | *Science in a Free Society* | 1978 | 科学与非科学之间没有明确界限；科学不应享有特殊认识论地位；民主社会中科学应与巫术、神话平等竞争 |
| 《实在论、理性主义与科学方法》 | *Realism, Rationalism and Scientific Method* | 1981 | 科学哲学论文集，系统批判科学实在论和理性主义方法论 |
| 《经验主义问题》 | *Problems of Empiricism* | 1981 | 对经验主义认识论的批判 |
| 《告别理性》 | *Farewell to Reason* | 1987 | 对理性主义的系统告别；传统与多元主义的辩护 |
| 《知识、科学与相对主义》 | *Knowledge, Science and Relativism* | 1999（遗著论文集） | 后期论文汇编，进一步深化相对主义和多元主义立场 |
| 《征服丰富性》 | *Conquest of Abundance* | 1999（遗著） | 晚期遗著，对科学文化抽象化的哲学反思 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 无政府主义认识论 | 科学中不存在、也不应该存在普遍约束性的方法论规则。"唯一不禁止的方法就是怎么都行"（anything goes）。科学家在实践中的成功恰恰来自于违反既定方法论规则 | 《反对方法》 |
| 反方法 | 所有方法论规则（如观察优于理论、一致性与解释、可证伪性、理论选择应基于证据）在科学史上都有大量反例。科学家在任何特定情况下都应该有权选择任何方法 | 《反对方法》 |
| 理论增生 | 为了促进科学的进步，需要不断引入与主流理论竞争的新理论。单一理论垄断会阻碍科学发现。新理论不需要等待现有理论失败后再提出 | 《反对方法》 |
| 不可通约性 | 不同理论（特别是"范式"）使用不可翻译的概念框架，因此它们之间不存在中立的比较基础。理论选择不是基于客观证据的理性决定，而是基于审美、社会、心理等因素 | 《反对方法》《自由社会中的科学》 |
| 科学与非科学的界限 | 科学与非科学（如宗教、神话、占星术、巫术）之间不存在明确的认识论界限。如果科学被赋予特殊地位，那是社会权力运作的结果，而非理性的胜利 | 《自由社会中的科学》 |
| 认识论无政府主义的社会论 | 在一个民主社会中，科学不应被赋予高于其他知识形式（如传统智慧、神话、艺术）的特权。所有知识形式都应平等竞争，由民主过程决定其社会地位 | 《自由社会中的科学》 |
| 反实在论多元主义 | 实在论（特别是科学实在论）是一种武断的形而上学立场。科学的描述不应被视为"真实的"，而只是众多可能描述中的一种。多元主义承认不同描述方式的同等合法性 | 《告别理性》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 | 具体影响 |
|------|------|----------|
| 马赫（Ernst Mach, 1838—1916） | 现象主义先驱 | 马赫的反形而上学和多元主义科学观为费耶阿本德的激进多元主义提供了先例 |
| 波普尔（Karl Popper, 1902—1994） | 直接导师与批判对象 | 费耶阿本德是波普尔的学生，早期接受批判理性主义，但后来系统批判波普尔的证伪主义方法过于僵化 |
| 拉卡托斯（Imre Lakatos, 1922—1974） | 直接对话者与论敌 | 两人在伦敦政治经济学院有密切互动。拉卡托斯的MSRP是费耶阿本德"反方法"的主要批判靶心之一。费耶阿本德认为任何方法论规则都是可违反的 |
| 库恩（Thomas Kuhn, 1922—1996） | 历史主义先驱 | 库恩的范式理论和不可通约性概念为费耶阿本德提供了直接的思想资源。费耶阿本德将库恩的"常规科学"批判为"思想暴政" |
| 维特根斯坦（后期） | 语言哲学先驱 | 后期维特根斯坦的语言游戏和家族相似概念，为费耶阿本德的多元主义认识论和反本质主义提供了哲学基础 |

### 4.2 同代

| 人物 | 关系 | 具体互动 |
|------|------|----------|
| 拉卡托斯（Imre Lakatos, 1922—1974） | 论敌与对话者 | 两人在LSE有密切互动。费耶阿本德的《反对方法》部分是对拉卡托斯MSRP的激进回应。拉卡托斯曾计划与费耶阿本德合著对话录，但因病逝未果 |
| 库恩（Thomas Kuhn, 1922—1996） | 同盟与批判者 | 费耶阿本德热烈拥护库恩的历史主义，但批评库恩对"常规科学"过于宽容。费耶阿本德认为常规科学本身就是对思想的压制 |

### 4.3 后启

| 人物/流派 | 关系 | 具体影响 |
|-----------|------|----------|
| 科学知识社会学（SSK） | 思想资源 | 布鲁尔、巴恩斯等人的强纲领将费耶阿本德的认识论无政府主义推向社会学方向，主张社会因素决定理论选择 |
| 后现代科学哲学 | 思想先驱 | 费耶阿本德的反理性主义、反基础主义和多元主义为后现代科学哲学提供了核心资源 |
| 女性主义科学哲学 | 思想资源 | 女性主义科学哲学家借鉴费耶阿本德对科学中立性的批判，揭示科学中的性别偏见和权力运作 |
| 后殖民科学研究 | 思想资源 | 费耶阿本德对科学霸权的批判为后殖民科学研究提供了认识论武器 |
| 当代科学传播与公众参与 | 实践影响 | 费耶阿本德关于科学民主化的主张，影响了当代科学传播和公众参与科学决策的理论 |
| 科学实在论/反实在论辩论 | 持续对话 | 费耶阿本德的反实在论至今是科学实在论辩论中反实在论一方的核心资源 |

## 五、本章定位

| 字段 | 内容 |
|------|------|
| 所属章节 | 第十五章：科学、知识与实在——科学哲学 |
| 章节小节 | 第五节：无政府主义认识论与反方法——费耶阿本德 |
| 思维流向 | 从"科学方法是否普遍有效"出发，经由科学史中方法论规则被不断违反的事实，到无政府主义认识论对一切方法论规则的否定，最终反思科学在民主社会中的正当地位 |
| 核心张力 | 方法论规则与科学实践的张力；理性与多元的张力；科学权威与民主平等的张力 |
| 与前人物关系 | 继承波普尔的批判理性主义但将其推向反方法的极端；继承库恩的历史主义但批判其"常规科学"概念；受后期维特根斯坦语言游戏理论影响 |
| 与后人物关系 | 为科学知识社会学（SSK）、后现代主义科学哲学、女性主义科学哲学、后殖民科学研究提供了核心资源 |
| 章节内角色 | 本章的"激进否定者"——费耶阿本德以极端的方式否定了从逻辑实证主义到历史主义的所有方法论希望，将科学哲学推向认识论无政府主义 |
