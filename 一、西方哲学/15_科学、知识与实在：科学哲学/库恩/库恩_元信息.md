## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 范式（Paradigm） | 科学共同体在某一时期内共享的"学科基质"——包括定律、理论、方法、世界观、标准问题及解决方案。范式为科学家提供了"看问题的方式"，决定了什么是值得研究的问题、什么是合理的解决方案 | 《科学革命的结构》 |
| 常规科学（Normal Science） | 在某一范式确立之后，科学家在范式框架内进行"解谜"活动——解决范式所设定的问题，完善范式的细节，扩展范式的应用范围。常规科学不是批判性的，而是累积性的、保守性的 | 《科学革命的结构》 |
| 科学革命（Scientific Revolution） | 当范式在解谜过程中积累足够多的"反常"（anomalies）而无法被消化时，范式进入危机。危机可能导致新范式的出现，新范式取代旧范式即为科学革命。革命不是渐进积累，而是"世界观的改变" | 《科学革命的结构》 |
| 不可通约性（Incommensurability） | 新旧范式之间不存在中立的标准来评判优劣。因为（1）不同范式使用不同的概念词汇，概念之间不可完全翻译；（2）不同范式的标准不同；（3）不同范式的科学家"生活在不同的世界"中。理论选择不是纯粹理性的决定 | 《科学革命的结构》 |
| 科学发展的非累积模式 | 科学发展不是线性累积的真理发现过程，而是"前科学→常规科学→危机→革命→新常规科学"的循环。旧范式的理论在革命中被抛弃，而非被整合进新理论。科学进步不是朝向一个固定真理的逼近 | 《科学革命的结构》 |
| 分类学不可通约性（晚期） | 库恩晚年将不可通约性深化为"分类学"层面的概念：不同范式的概念分类体系不同，概念之间的对应关系不是一一映射，而是"局部整体论"的。这类似于语言学的概念差异 | 《结构之后的路》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 | 具体影响 |
|------|------|----------|
| 迪昂（Pierre Duhem, 1861—1916） | 方法论先驱 | 迪昂的整体论和理论不可证实论为库恩的范式不可通约性提供了直接先例。库恩明确引用迪昂 |
| 彭加勒（Henri Poincaré, 1854—1912） | 约定主义先驱 | 彭加勒的约定主义为库恩的范式选择非理性论提供了方法论背景：理论框架的选择包含约定成分 |
| 弗莱克（Ludwik Fleck, 1896—1961） | 直接先驱 | 弗莱克的《科学事实的起源与发展》（1935）是库恩的直接先驱。弗莱克提出的"思想集体"（thought collective）和"思想风格"（thought style）概念直接启发了库恩的"范式"概念 |
| 汉森（N. R. Hanson, 1924—1967） | 观察理论论先驱 | 汉森的"观察渗透理论"（理论负载的观察）为库恩的范式决定观察提供了直接支持 |
| 维特根斯坦（后期） | 语言哲学先驱 | 后期维特根斯坦的语言游戏、家族相似和不可翻译性概念，为库恩的不可通约性理论提供了哲学基础 |
| 波兰尼（Michael Polanyi, 1891—1976） | 个人知识论先驱 | 波兰尼的"个人知识"和"隐性知识"概念，为库恩强调科学实践中的默会维度提供了支持 |

### 4.2 同代

| 人物 | 关系 | 具体互动 |
|------|------|----------|
| 波普尔（Karl Popper, 1902—1994） | 主要论敌 | 1965年伦敦会议上的波普尔-库恩辩论是20世纪科学哲学最著名的论战。波普尔主张科学的本质是批判和证伪，而库恩主张科学的常态是保守的"解谜"。波普尔批评库恩的"常规科学"是蒙昧主义 |
| 拉卡托斯（Imre Lakatos, 1922—1974） | 批判者与对话者 | 拉卡托斯试图用"研究纲领"取代"范式"，保留科学进步的理性标准。拉卡托斯批评库恩的不可通约性导致非理性主义，但吸收了库恩的历史主义视角 |
| 费耶阿本德（Paul Feyerabend, 1924—1994） | 激进同盟者 | 费耶阿本德热烈拥护库恩的历史主义，但批评库恩对"常规科学"过于宽容。费耶阿本德将库恩的理论推向更激进的无政府主义方向 |
| 波普尔学派 | 批判对象 | 波普尔的学生（如拉卡托斯、沃特金斯）系统批判库恩的历史主义，认为其威胁了科学理性主义 |

### 4.3 后启

| 人物/流派 | 关系 | 具体影响 |
|-----------|------|----------|
| 科学知识社会学（SSK） | 直接继承 | 布鲁尔、巴恩斯、柯林斯等人的强纲领直接继承了库恩的社会历史视角，将社会因素作为科学理论选择的核心解释变量 |
| 社会建构论（SCOT） | 应用领域 | 技术社会建构论将库恩的范式概念应用于技术史和技术社会学研究 |
| 后现代主义 | 思想资源 | 库恩的不可通约性和相对主义倾向为后现代主义认识论提供了核心资源 |
| 女性主义科学哲学 | 思想资源 | 女性主义科学哲学家借鉴库恩对科学中立性的批判，揭示科学中的性别偏见 |
| 后殖民科学研究 | 思想资源 | 库恩对科学普遍性的质疑为后殖民科学研究提供了认识论武器 |
| 科学传播与公众理解科学 | 实践影响 | 库恩关于科学共同体的理论，影响了科学传播和公众参与科学研究的理论框架 |
| 当代科学哲学 | 核心框架 | 库恩的范式、革命、不可通约性至今是科学哲学、科学史、科学社会学讨论的核心概念 |

## 五、本章定位

| 字段 | 内容 |
|------|------|
| 所属章节 | 第十五章：科学、知识与实在——科学哲学 |
| 章节小节 | 第六节：范式、科学革命与不可通约性——库恩 |
| 思维流向 | 从"科学是如何发展的"出发，经由常规科学中范式的保守作用，到危机和革命中范式的断裂性转换，最终反思不可通约性对科学理性主义的基本挑战 |
| 核心张力 | 传统与创新的张力；累积与断裂的张力；理性与社会的张力；普遍性与特殊性的张力 |
| 与前人物关系 | 继承迪昂的整体论和彭加勒的约定主义，将其发展为历史主义科学观；受弗莱克、汉森、波兰尼、维特根斯坦影响 |
| 与后人物关系 | 为科学知识社会学（SSK）、后现代主义、女性主义科学哲学、后殖民科学研究提供了核心框架；与拉卡托斯的MSRP、费耶阿本德的无政府主义形成科学哲学内部的三种历史主义模型 |
| 章节内角色 | 本章的"核心枢纽"——库恩是20世纪科学哲学从逻辑主义到历史主义转向的标志人物，其影响远超科学哲学本身，波及科学史、社会学、政治学和文化研究 |
