## 二、核心著作

| 著作 | 原文标题 | 出版年份 | 核心主题 |
|------|----------|----------|----------|
| 《形式化语言中的真理概念》 | *Der Wahrheitsbegriff in den formalisierten Sprachen* | 1933/1936 | 真理语义学，T-等式，对象语言与元语言的区分 |

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 语义真理概念 | 真理是语句与事实之间的语义关系，通过T-等式精确表述，与亚里士多德符合论一脉相承 | 《形式化语言中的真理概念》 |
| T-等式/T-schema | "'S'是真的当且仅当p"——真理定义必须满足的形式条件，体现实质充分性 | 同上 |
| 元语言与对象语言的区分 | 真理只能在比对象语言更丰富的元语言中定义，不能在被定义的语言自身内部定义 | 同上 |
| 真理不可定义性定理 | 对于包含基本算术的一致形式语言，真理谓词在该语言内部不可定义 | 同上 |
| 形式正确性与实质充分性 | 满意的真理定义必须同时满足逻辑一致性（形式正确性）和推出所有T-等式（实质充分性） | 《真理的语义概念》 |
| 模型论奠基 | 通过对满足关系的递归定义，开创了现代模型论 | 《逻辑、语义学与元数学》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 亚里士多德 | 塔斯基的语义真理概念是对亚里士多德符合论（"说是者说是，非者说非"）的形式化重构 |
| 罗素 | 类型论和悖论研究为塔斯基的语言分层提供了技术背景；罗素的逻辑原子主义影响了分析哲学的语言转向 |
| 哥德尔 | 哥德尔的不完全性定理（1931）与塔斯基定理使用相同的对角线论证技术，揭示了形式系统的内在限制 |
| 列斯涅夫斯基（Leśniewski） | 塔斯基在华沙大学的导师之一，华沙学派的领袖，影响了塔斯基对形式系统的严谨态度 |
| 卢卡西维茨（Łukasiewicz） | 华沙学派的逻辑学家，塔斯基在华沙大学的另一位导师，多值逻辑的先驱 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 卡尔纳普 | 维也纳学派核心成员，塔斯基的语义学直接启发了卡尔纳普从句法主义转向语义学；两人在逻辑实证主义鼎盛期有密切学术交流 |
| 哥德尔 | 与塔斯基同在1930年代做出逻辑学的革命性贡献，两人使用相似的对角线论证，揭示形式系统的根本限制 |
| 列斯涅夫斯基 | 华沙学派同事，塔斯基的博士导师之一，共同推动了波兰数理逻辑的国际化 |
| 希尔伯特学派 | 塔斯基的工作与希尔伯特的形式主义纲领形成对话——塔斯基证明了形式主义的某些内在限制 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 戴维森 | 将塔斯基的真理论改造为意义理论——用T-等式作为意义理论的形式框架，提出"真理论语义学" |
| 克里普克 | 可能世界语义学继承了塔斯基的形式化语义学方法；克里普克的"真值 gaps"理论是对塔斯基二值真理概念的修正 |
| 蒙塔古（Montague） | 蒙塔古语法将塔斯基的语义学方法应用于自然语言，发展出形式语义学 |
| 普特南 | 在指称理论和对"真"的哲学理解上与塔斯基的语义学有深刻对话 |
| 当代模型论 | 塔斯基的满足定义和真理定义是现代模型论的基础，影响了数学、计算机科学和语言学的形式化研究 |
