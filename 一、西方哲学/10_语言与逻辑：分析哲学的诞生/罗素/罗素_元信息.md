## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 逻辑原子主义 | 世界由逻辑原子（简单事实）构成，复杂命题可分析为原子命题的真值函项 | 《逻辑原子主义哲学》 |
| 摹状词理论 | "当今法国国王是秃头"并非指称不存在之物，而是可分析为存在性陈述 | 《论指称》（1905） |
| 类型论 | 解决集合论悖论，区分不同逻辑类型，避免自指 | 《数学原理》 |
| 逻辑构造 | 用逻辑工具重构世界知识，拒斥形而上学实体 | 《我们关于外间世界的知识》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 弗雷格 | 罗素通过阅读《算术基础》发现弗雷格，继承并推进逻辑主义纲领 |
| 莱布尼茨 | 罗素的数学哲学深受莱布尼茨理性主义影响 |
| 休谟 | 罗素的经验主义倾向受休谟影响，但用逻辑工具来修正之 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 怀特海 | 与罗素合著《数学原理》，后转向过程哲学 |
| 维特根斯坦（早期） | 罗素是维特根斯坦的老师，两人相互影响 |
| 摩尔 | 剑桥同事，共同推动分析哲学的"反观念论"运动 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 维特根斯坦（早期） | 《逻辑哲学论》直接继承并推进罗素的逻辑原子主义 |
| 卡尔纳普 | 罗素的逻辑构造方法深刻影响卡尔纳普的《世界的逻辑构造》 |
| 维也纳学派 | 罗素的逻辑分析方法被维也纳学派吸收 |
| 斯特劳森 | 斯特劳森批评罗素的摹状词理论，提出预设理论 |
