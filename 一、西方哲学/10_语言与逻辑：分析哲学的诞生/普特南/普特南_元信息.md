## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 意义的外在主义 | 意义不在头脑中——自然种类词的指称取决于物质实际结构 | "意义的意义"（1975） |
| 孪生地球论证 | 地球与孪生地球的说话者内心状态相同但"水"指称不同 | "意义的意义"（1975） |
| 语义劳动分工 | 语词意义部分"外包"给专家共同体——意义是社会分布的 | "意义的意义"（1975） |
| 功能主义 | 心理状态是功能状态——心灵像软件可在不同硬件上运行 | "心理状态的本质"（1967） |
| 缸中之脑论证 | "我是缸中之脑"要么自指矛盾要么为假——从语义学消解怀疑论 | 《理性、真理与历史》 |
| 内在实在论 | 真理相对于概念框架但受世界制约——无"上帝之眼"视角 | 《理性、真理与历史》 |
| 事实与价值的纠缠 | 事实判断和价值判断不是截然分离的——"残忍"既是描述也是评价 | 《事实/价值二分的崩溃》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 奎因 | 普特南在哈佛的同事——奎因的整体论和自然主义影响了普特南的哲学路径 |
| 赖辛巴赫 | 普特南的博士导师——影响了他的科学哲学取向 |
| 克里普克 | 两人独立提出了指称的外在主义——克里普克论专名，普特南论自然种类词 |
| 维特根斯坦 | 后期维特根斯坦影响了普特南的实用主义转向 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 戴维森 | 哈佛同事——在意义理论、真理论和指称理论上有深刻对话 |
| 塞尔 | 在心灵哲学和语言哲学上有交叉——塞尔的中文屋与普特南的功能主义形成对照 |
| 罗蒂 | 罗蒂将普特南的某些洞见引向更激进的实用主义方向 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 布劳恩 | 发展了普特南的外在主义——提出"社会外在主义" |
| 伯格 | 继承并扩展了语义劳动分工概念 |
| 麦克道威尔 | 在普特南的内在实在论基础上发展了"松弛的自然主义" |
