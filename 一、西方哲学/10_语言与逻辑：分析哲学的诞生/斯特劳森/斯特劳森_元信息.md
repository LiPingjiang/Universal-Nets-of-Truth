## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 预设理论 | "当今法国国王是秃头"预设了法国国王存在，而非简单为假 | "论指称"（1950） |
| 描述形而上学 | 描绘我们实际思维中的概念结构——而非建构新的形而上学体系 | 《个体》 |
| 基本殊相 | 物质物体是我们的概念图式中最基本的个别事物——而非感觉材料或事件 | 《个体》 |
| 识别与再识别 | 同一性判断依赖于在公共空间时间框架中的识别和再识别能力 | 《个体》 |
| 对罗素摹状词理论的批判 | 罗素的摹状词理论歪曲了日常语言的实际用法 | "论指称" |
| 非修正的自然主义 | 人类生活的自然态度足以回应怀疑论，不需要理论辩护 | 《怀疑论与自然主义》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 罗素 | 斯特劳森批判罗素的摹状词理论——认为它歪曲了日常语言 |
| 维特根斯坦（后期） | 日常语言分析方法深刻影响斯特劳森 |
| 康德 | 斯特劳森在《感觉的界限》中重新审视康德，提出"描述形而上学"概念 |
| 亚里士多德 | 斯特劳森的"描述形而上学"具有亚里士多德式概念分析的特征 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 赖尔 | 牛津日常语言学派同仁 |
| 奥斯汀 | 牛津同事，共同推动日常语言学派；斯特劳森批评奥斯汀的以言行事/以言取效区分 |
| 奎因 | 斯特劳森批评奎因的自然主义认识论过于"修正主义" |
| 格莱斯 | 格莱斯与斯特劳森在意义理论上有深入对话 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 塞尔（John Searle） | 继承斯特劳森对预设和指称的分析 |
| 威廉姆森（Timothy Williamson） | 在知识论和模态形而上学中继承斯特劳森的分析方法 |
| 当代描述形而上学 | 斯特劳森开创的"描述形而上学"在当代分析形而上学中仍有影响 |
