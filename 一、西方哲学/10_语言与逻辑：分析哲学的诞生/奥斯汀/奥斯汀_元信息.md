## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 施行语 | 很多陈述句不是在描述事实，而是在做某种行为——说话本身就是行动 | 《如何以言行事》 |
| 言语行为三重区分 | 以言表意（locutionary）、以言行事（illocutionary）、以言取效（perlocutionary） | 《如何以言行事》 |
| 恰当条件 | 施行语要"成功"必须满足的条件：权力条件、情境条件、诚意条件 | 《如何以言行事》 |
| 言语行为分类 | 五种类型：判定式、行使式、承诺式、表态式、阐明式 | 《如何以言行事》 |
| 对感觉材料理论的批判 | "感觉材料"是对日常语言的误解——"看起来""似乎"不是在描述神秘对象 | 《感觉与可感物》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 维特根斯坦（后期） | 日常语言分析方法和"语言游戏"概念深刻影响奥斯汀 |
| 摩尔 | 摩尔对日常语言概念的分析是奥斯汀方法论的先驱 |
| 赖尔 | 牛津同事，共同推动日常语言学派的形成 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 赖尔 | 牛津日常语言学派同仁，方法相似但关注不同（赖尔关注心灵，奥斯汀关注语言行动） |
| 斯特劳森 | 牛津同事，斯特劳森批评奥斯汀的以言行事/以言取效区分不够清晰 |
| 维特根斯坦（后期） | 共同强调日常语言分析，但维特根斯坦更关注"语言游戏"的多样性，奥斯汀更关注系统分类 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 塞尔（John Searle） | 系统化和形式化言语行为理论，提出更精确的恰当条件分类 |
| 格莱斯（H.P. Grice） | 发展"会话含义"理论，与言语行为理论互补 |
| 哈贝马斯 | 吸收言语行为理论，置于交往理性框架中 |
| 朱迪斯·巴特勒 | 将言语行为理论应用于性别建构——"性别是被施行的" |

## 五、本章定位

| 项目 | 内容 |
|------|------|
