# 石里克 · 元信息

## 一、人物基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 莫里茨·石里克（Moritz Schlick） |
| 生卒 | 1882—1936 |
| 国籍 | 德国 / 奥地利（工作时） |
| 身份 | 哲学家、物理学家 |
| 学术身份 | 维也纳学派创始人、逻辑实证主义代表人物 |
| 教职 | 维也纳大学自然哲学讲席教授（1922—1936） |
| 导师 | 马克斯·普朗克（Max Planck，物理学博士导师） |

## 二、核心著作

| 年份 | 德文原名 | 中文译名 | 重要性 |
|------|----------|----------|--------|
| 1918/1925 | Allgemeine Erkenntnislehre | 普通认识论 | 早期认识论代表作，提出协调理论与Erleben/Erkennen区分 |
| 1925 | "Naturphilosophische Betrachtungen" | 自然哲学考察 | 过渡期论文，从实在论转向实证主义立场 |
| 1930 | "Die Wende der Philosophie" | 哲学的转折点 | 宣告哲学的本质转向：哲学作为澄清活动 |
| 1932 | "Positivismus und Realismus" | 实证主义与实在论 | 阐明实证主义与实在论的关系，论争基础 |
| 1934 | "Über das Fundament der Erkenntnis" | 论认识的基础 | 与卡尔纳普论争：真理vs确认 |
| 1936 | "Meaning and Verification" | 意义与证实 | 证实原则的经典表述，石里克最后重要论文 |
| 1938 | Form and Content | 形式与内容 | 遗作，哲学思维导论 |

## 三、核心学说

| 学说 | 核心内容 | 关键概念 |
|------|----------|----------|
| 证实原则与意义理论 | 一个命题的意义在于其验证方法；无法验证的命题无意义 | Verifikationsprinzip, 意义即验证方法 |
| 哲学作为澄清活动 | 哲学不是知识体系，而是通过逻辑分析澄清命题意义的活动 | 哲学即活动（activity, not doctrine） |
| 直觉与概念知识的区分 | 体验（Erleben）是私人的不可言说的，认识（Erkennen）是概念性的可交流的 | Erleben vs Erkennen |
| 协调理论 | 知识是概念系统与经验事实之间的协调对应关系，非对实在的复制 | Zuordnung |
| 拒斥综合先天判断 | 不存在既独立于经验又扩展知识的判断；一切知识要么分析要么综合 | 拒斥 synthetisches Urteil a priori |
| 强证实与弱证实的区分 | 主张弱证实标准：只需存在可能的经验检验程序即可 | 强证实 vs 弱证实 |

## 四、思想关系

### 4.1 前承（受影响于）

| 人物 | 影响内容 |
|------|----------|
| 马赫（Ernst Mach） | 感觉一元论、反形而上学立场、要素论对石里克早期思想有直接影响 |
| 罗素（Bertrand Russell） | 逻辑分析方法、类型论、摹状词理论为石里克提供了分析工具 |
| 维特根斯坦（早期） | 《逻辑哲学论》深刻影响石里克关于哲学本质和命题意义的思考 |
| 卡西尔（Ernst Cassirer） | 新康德主义背景，石里克在《普通认识论》中与之对话 |
| 彭加勒（Henri Poincaré） | 约定论思想影响石里克对综合先天判断的批判 |
| 希尔伯特（David Hilbert） | 公理化方法与形式化思想对石里克认识论有影响 |

### 4.2 同代（合作与论争）

| 人物 | 关系 |
|------|------|
| 卡尔纳普（Rudolf Carnap） | 维也纳学派核心成员，与石里克在真理vs确认、物理主义等问题上有重要分歧 |
| 纽拉特（Otto Neurath） | 维也纳学派左翼，主张物理主义和连贯论，与石里克基础主义立场对立 |
| 哈恩（Hans Hahn） | 维也纳学派创始成员，数学家，1929年与石里克、纽拉特合著学派宣言 |
| 魏斯曼（Friedrich Waismann） | 石里克的学生与合作者，长期担任维特根斯坦与石里克之间的沟通桥梁 |
| 维特根斯坦（后期） | 1927—1932年间与石里克密切交往，后期思想对石里克有重要影响 |
| 波普尔（Karl Popper） | 批评证实原则，主张可证伪性（falsifiability）为科学划界标准 |

### 4.3 后启（影响者）

| 人物 | 影响内容 |
|------|----------|
| 艾耶尔（A. J. Ayer） | 将逻辑实证主义引入英语世界，《语言、真理与逻辑》直接受石里克证实原则影响 |
| 亨普尔（Carl Hempel） | 确认逻辑理论发展了石里克的证实思想 |
| 费格尔（Herbert Feigel） | 石里克学生，将维也纳学派思想传播至美国 |
| 克拉夫特（Victor Kraft） | 维也纳学派成员，在石里克死后整理学派遗产 |
| 奎因（W. J. O. Quine） | 虽批评逻辑实证主义，但其经验论传统可追溯至石里克 |
