## 二、核心著作

|------|----------|----------|----------|

## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 可证伪性 | 科学理论的标准不是可证实性而是可证伪性——理论必须对经验反驳保持开放 | 《科学发现的逻辑》 |
| 证伪主义 | 科学方法不是归纳证实而是猜想与反驳——大胆猜想，严格检验 | 《猜想与反驳》 |
| 反归纳主义 | 归纳逻辑不存在——科学不是从观察中归纳出理论，而是先有理论后有观察 | 《科学发现的逻辑》 |
| 开放社会 | 个人有权做出自己的决定，社会制度是人为的、可改进的——与封闭社会相对 | 《开放社会及其敌人》 |
| 历史决定论批判 | 历史没有可发现的规律，历史预测在逻辑上不可能——因为新知识不可预测 | 《历史决定论的贫困》 |
| 零星社会工程 | 通过逐步的、可修正的改良来消除社会弊端——反对整体性乌托邦改造 | 《开放社会及其敌人》 |
| 三个世界理论 | 世界1（物理）、世界2（精神）、世界3（客观知识）——世界3具有自主性 | 《客观知识》 |
| 逼真性 | 一个理论即使不是真的，也可以比另一个理论更接近真理 | 《猜想与反驳》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 休谟 | 休谟的归纳问题直接启发了波普尔的证伪主义——如果归纳不可能，科学方法就不是归纳 |
| 康德 | 康德对理性的批判精神影响了波普尔，虽然波普尔拒绝了康德的先验综合判断 |
| 维也纳学派 | 波普尔与维也纳学派有密切交流但从未正式加入——他拒绝了证实原则而提出证伪原则 |
| 爱因斯坦 | 爱因斯坦对波普尔影响巨大——相对论"敢赌"的态度是波普尔科学观的典范 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 卡尔纳普 | 维也纳学派的核心人物——与波普尔在划界标准上根本分歧（证实 vs 证伪） |
| 维特根斯坦 | 与波普尔在剑桥有一次著名的"拨火棍事件"——两人哲学取向根本不同 |
| 哈耶克 | 伦敦经济学院同事——共同捍卫自由主义，但方法论取向不同（哈耶克更偏奥地利学派经济学） |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 拉卡托斯 | 波普尔的学生——提出"研究纲领"理论修正朴素证伪主义 |
| 库恩 | 科学哲学历史学派的代表——批评波普尔忽视了"常规科学"的积累性 |
| 费耶阿本德 | 更激进的批评者——认为不存在唯一的科学方法（"怎么都行"） |
| 索罗斯 | 波普尔在伦敦经济学院的学生——将开放社会理念转化为金融实践和慈善事业 |
