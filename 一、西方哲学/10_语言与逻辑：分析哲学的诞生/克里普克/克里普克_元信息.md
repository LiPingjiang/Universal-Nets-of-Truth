## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 刚性指称 | 专名在所有可能世界中指称同一对象——专名不是缩略的摹状词 | 《命名与必然性》 |
| 因果指称理论 | 专名通过因果历史链条指称对象，而非通过描述性内容 | 《命名与必然性》 |
| 必然的后验真理 | "水是H₂O"是后验发现的，但在所有可能世界中为真 | 《命名与必然性》 |
| 先验的偶然真理 | "巴黎的标准米尺是一米"是先验知道的，但不是必然的 | 《命名与必然性》 |
| 本质主义 | 对象有跨可能世界的本质属性（起源必然性、结构必然性） | 《命名与必然性》 |
| 可能世界语义学 | 为模态逻辑提供形式化语义基础——克里普克语义 | 模态逻辑论文（1959-1963） |
| 规则遵循悖论 | 不存在能确定"遵循规则"的事实——怀疑论解决方案 | 《维特根斯坦论规则与私人语言》 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 弗雷格 | 克里普克批判性地继承弗雷格的涵义/指称区分，提出刚性指称理论 |
| 罗素 | 克里普克批判罗素的摹状词理论——专名不是缩略的摹状词 |
| 卡尔纳普 | 克里普克继承卡尔纳普的内涵/外延区分和可能世界概念，发展出可能世界语义学 |
| 维特根斯坦（后期） | 克里普克重新解读维特根斯坦的规则遵循问题，提出怀疑论解决方案 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 奎因 | 克里普克与奎因形成根本对照——奎因拒斥必然性，克里普克复兴必然性 |
| 唐内兰（Keith Donnellan） | 同为直接指称论的倡导者，在指称的语用方面与克里普克互补 |
| 普特南 | 提出"孪生地球"论证，与克里普克的"必然的后验真理"形成共振 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 索姆斯（Scott Soames） | 系统化克里普克的直接指称论，发展两维语义学 |
| 萨蒙（Nathan Salmon） | 发展克里普克式的本质主义和指称理论 |
| 威廉姆森（Timothy Williamson） | 在模态认识论和知识论中继承克里普克的模态形而上学方法 |
