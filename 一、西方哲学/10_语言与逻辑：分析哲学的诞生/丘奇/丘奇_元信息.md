## 二、核心著作

| 年份 | 著作/论文 | 核心内容 |
|------|-----------|----------|
| 1932-1933 | "A Set of Postulates for the Foundation of Logic" | 提出λ演算雏形，尝试以函数为基础建立逻辑公理系统 |
| 1936a | "An Unsolvable Problem of Elementary Number Theory" | 系统引入λ演算，证明存在不可判定的初等数论问题 |
| 1936b | "A Note on the Entscheidungsproblem" | 证明一阶逻辑判定问题不可解（丘奇定理） |
| 1936c | "A Note on the Entscheidungsproblem" (Correction) | 对前文的修正与补充 |
| 1956 | *Introduction to Mathematical Logic* | 数理逻辑标准教材，系统阐述一阶逻辑与高阶逻辑 |

## 三、核心学说

| 学说 | 核心主张 | 意义 |
|------|----------|------|
| λ演算（Lambda Calculus） | 一切可计算函数均可由函数的抽象与应用表达，仅需变量、λ抽象、函数应用三种构造 | 为计算提供了最简洁的形式化模型，成为函数式编程的理论基础 |
| 判定问题不可解性（丘奇定理） | 一阶逻辑的判定问题（Entscheidungsproblem）不可解：不存在算法判定任一一阶公式是否可证 | 确立了逻辑推理的根本限度，终结了希尔伯特的判定纲领 |
| 丘奇-图灵论题 | 每一可有效计算的函数都是λ可定义的（等价于图灵机可计算） | 将直觉性的"可计算"概念精确化，奠定理论计算机科学的哲学基础 |
| 丘奇-罗瑟定理 | λ演算具有汇合性：若表达式可归约到两个范式，则二者必可归约到同一范式 | 保证λ演算计算结果的唯一性，确立系统的良好性质 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 大卫·希尔伯特（David Hilbert） | 判定问题（Entscheidungsproblem）的提出者；丘奇的工作直接回应了希尔伯特纲领中的判定问题 |
| 库尔特·哥德尔（Kurt Gödel） | 哥德尔不完备性定理（1931）为丘奇的工作奠定了基础；哥德尔的一般递归函数定义与丘奇的λ可定义函数等价 |
| 伯特兰·罗素（Bertrand Russell） | 罗素的类型论与《数学原理》对丘奇的形式化方法有深刻影响 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 艾伦·图灵（Alan Turing） | 丘奇的博士生；独立提出图灵机模型，与丘奇的λ演算等价；二人共同提出丘奇-图灵论题 |
| 斯蒂芬·克林（Stephen Kleene） | 丘奇的博士生；系统发展了λ演算与递归函数论，证明λ可定义函数与一般递归函数的等价性 |
| 巴克利·罗瑟（J. Barkley Rosser） | 丘奇的博士生；与丘奇共同证明丘奇-罗瑟定理，发展了组合逻辑 |
| 阿尔弗雷德·塔斯基（Alfred Tarski） | 与丘奇同属20世纪逻辑学的核心人物；二人在可判定性与真理理论方面有学术交流 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 约翰·麦卡锡（John McCarthy） | 丘奇的学生（普林斯顿）；基于λ演算发明了Lisp语言，开创函数式编程范式 |
| 亨德里克·巴柯里（Hendrik Barendregt） | 系统发展了无类型λ演算的理论，著有λ演算的标准参考书 |
| 达纳·斯科特（Dana Scott） | 丘奇的学生；为λ演算建立了指称语义模型（Scott域），解决了λ演算的数学基础问题 |
