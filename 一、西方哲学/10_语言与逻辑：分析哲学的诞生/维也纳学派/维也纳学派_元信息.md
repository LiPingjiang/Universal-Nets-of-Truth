# 维也纳学派 · 元信息

## 一、学派基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 维也纳学派（Wiener Kreis） |
| 活跃时期 | 1920s–1930s |
| 所在地 | 奥地利维也纳大学 |
| 核心立场 | 逻辑实证主义（Logical Positivism）/ 逻辑经验主义（Logical Empiricism） |
| 学派性质 | 哲学团体，非单一哲学家 |

---

| 成员 | 生卒年 | 核心角色 | 代表著作 |
|------|--------|----------|----------|

| 鲁道夫·卡尔纳普（Rudolf Carnap） | 1891–1970 | 技术核心 | 《世界的逻辑构造》（1928）、《语言的逻辑句法》（1934） |

## 三、核心学说

| 学说 | 简要说明 | 代表人物 |
|------|----------|----------|
| 证实原则 | 命题的意义在于其证实方法；形而上学命题不可证实故无意义 | 石里克、卡尔纳普 |
| 统一科学纲领 | 所有科学语言可还原为物理语言 | 纽拉特、卡尔纳普 |
| 拒斥形而上学 | 传统哲学问题（上帝、灵魂、本体）是语言误用的产物 | 全体成员 |
| 物理主义 | 物理语言是统一科学的基础语言 | 纽拉特、卡尔纳普 |
| 句法分析 | 哲学问题可通过语言的逻辑句法分析来解决 | 卡尔纳普 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 弗雷格 | 形式逻辑分析方法的基础 |
| 罗素 | 逻辑原子主义和逻辑构造方法 |
| 维特根斯坦（早期） | 《逻辑哲学论》启发"哲学问题可通过语言分析消解" |
| 马赫 | 经验主义和反形而上学传统的先驱 |
| 彭加勒 | 约定论对卡尔纳普的宽容原则有影响 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 维特根斯坦（后期） | 与维也纳学派关系复杂，既有影响又有分歧 |
| 波普尔 | 批评证实原则，提出证伪主义 |
| 赖辛巴赫 | 柏林学派，与维也纳学派相近但独立 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 卡尔纳普 | 作为学派核心成员移居美国，将逻辑实证主义传播到英语世界 |
| 奎因 | 从内部批判逻辑实证主义，拒斥分析-综合二分 |
| 艾耶尔 | 将逻辑实证主义传播到英国和英语世界 |
| 亨普尔 | 继承并修正逻辑实证主义的科学哲学 |
