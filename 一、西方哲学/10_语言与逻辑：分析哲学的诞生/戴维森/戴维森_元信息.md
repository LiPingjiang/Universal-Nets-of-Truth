## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 理由作为原因 | 行动的理由解释是一种因果解释——主要理由既是合理化解释也是原因 | "行动、理由与原因"（1963） |
| 真理论语义学 | 用塔斯基的真理论作为意义理论的形式框架——知道真值条件就是知道意义 | "彻底解释"（1973） |
| 彻底解释 | 在无预先共享语言的情况下构造意义理论——同时确定信念和意义 | "彻底解释"（1973） |
| 善意原则 | 解释者必须假设说话者大部分信念为真——解释的逻辑前提 | "论概念图式的思想"（1974） |
| 无概念图式的经验 | 概念图式与经验内容的区分不可理解——不存在不可通约的图式 | "论概念图式的思想"（1974） |
| 异常一元论 | 心理事件与物理事件是同一事件，但心理领域不服从严格定律 | "心理事件"（1970） |
| 真理与知识融贯论 | 信念只能被其他信念证成——感知是原因但不是证成理由 | "真理与知识的融贯论"（1986） |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 奎因 | 戴维森的博士导师——翻译的不确定性启发了彻底解释，整体论影响了融贯论 |
| 塔斯基 | 塔斯基的真理论为戴维森的意义理论提供了形式框架 |
| 维特根斯坦 | 维特根斯坦的"语言游戏"和"意义即使用"启发了戴维森对意义的思考 |
| 安斯康姆 | 安斯康姆的"意图"研究是戴维森行动哲学的直接对话对象 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 普特南 | 哈佛同事——在意义理论、指称理论上有深刻对话 |
| 塞尔 | 在行动哲学和言语行为理论上有交叉但路径不同 |
| 罗蒂 | 罗蒂将戴维森的某些洞读引向实用主义方向，戴维森不完全认同 |
| 库恩 | 戴维森的"无概念图式"论证直接挑战了库恩的"不可通约性"论题 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 麦克道威尔 | 继承并修正了戴维森——试图在"理由的空间"和"法则的空间"之间架起桥梁 |
| 布兰顿 | 发展了戴维森的意义理论——走向推论主义语义学 |
| 金在权 | 对异常一元论提出"排除论证"——当代心灵哲学的核心争论之一 |
| 拉姆伯格-汉森 | 发展了戴维森的彻底解释理论 |
