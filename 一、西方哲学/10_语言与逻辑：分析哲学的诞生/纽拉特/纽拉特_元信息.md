# 纽拉特 · 元信息

## 一、人物基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 奥托·纽拉特（Otto Neurath） |
| 生卒 | 1882–1945 |
| 国籍 | 奥地利（后流亡英国） |
| 身份 | 科学哲学家 / 社会学家 / 政治经济学家 |
| 学派归属 | 维也纳学派（Vienna Circle）核心成员 |
| 核心标签 | 物理主义 / 统一科学运动 / 反基础主义 / 船喻 |
| 学术活动地 | 维也纳 → 海牙 → 牛津 |
| 政治立场 | 社会民主主义、科学乌托邦主义 |
| 其他贡献 | ISOTYPE 视觉统计系统创始人 |

## 二、核心著作

| 年份 | 著作/论文 | 主题 |
|------|-----------|------|
| 1921 | 《反施宾格勒》（*Anti-Spengler*） | 批判历史决定论，倡导科学理性与社会规划 |
| 1931 | 《物理主义》（*Physicalism*） | 系统阐述物理主义纲领，主张科学统一 |
| 1932 | "协议语句"（*Protocol Sentences*） | 船喻的首次提出，反驳石里克的基础主义 |
| 1937 | 《作为百科全书式整合的统一科学》 | 区分百科全书与系统，阐述反基础主义 |
| 1938– | 《统一科学国际百科全书》（主编） | 统一科学运动的制度化项目 |
| 1944 | 《社会科学的基础》 | 将物理主义应用于社会科学的方法论 |
| 1973 | 《经验主义与社会学》（遗著合集） | 纽拉特哲学全景式呈现 |

## 三、核心学说

| 学说 | 核心主张 | 关键概念 |
|------|----------|----------|
| 物理主义 | 一切科学陈述可翻译为物理语言，科学本质上是统一的 | 物理语言、主体间性、还原论 |
| 统一科学/百科全书计划 | 科学知识应以百科全书而非系统的模式整合 | 百科全书vs系统、反基础主义、开放性 |
| 船喻/反基础主义 | 知识修正只能在既有信念网络中局部进行，不存在绝对可靠的基础 | 船喻、融贯论、协议语句之争 |
| 协议语句之争 | 否认石里克的"确证"为知识基础，主张融贯检验 | 协议语句、确证（Konstatierungen） |
| 反形而上学 | 拒斥一切不可验证的形而上学命题 | 可验证性、逻辑分析、反神学 |
| ISOTYPE与科学传播 | 用标准化视觉图标普及统计数据，实现知识民主化 | 视觉教育、社会改革 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 马赫（Ernst Mach） | 经验主义传统，反形而上学，感觉要素论；纽拉特继承其经验主义但反对其现象主义 |
| 狄尔泰（Wilhelm Dilthey） | 精神科学与自然科学之分；纽拉特反对这一区分，主张科学统一 |
| 孔德（Auguste Comte） | 实证主义传统，科学分类与社会学构想；纽拉特的统一科学受其实证精神启发 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 石里克（Moritz Schlick） | 维也纳学派领袖，协议语句之争的主要对手；石里克持基础主义，纽拉特持融贯论 |
| 卡尔纳普（Rudolf Carnap） | 维也纳学派核心成员，物理主义的同盟者，但在协议语句问题上立场摇摆，最终走向温和立场 |
| 哈恩（Hans Hahn） | 维也纳学派成员，数学家，支持纽拉特的物理主义纲领 |
| 弗兰克（Philipp Frank） | 物理学家，统一科学运动的合作者，共同编辑《百科全书》 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 奎因（W. V. O. Quine） | "信念之网"（web of belief）直接继承纽拉特的船喻；自然化认识论、反对分析-综合区分均与纽拉特一致 |
| 普特南（Hilary Putnam） | 内在实在论与纽拉特的反基础主义和融贯倾向有深层呼应 |
| 古德曼（Nelson Goodman） | 反基础主义、整体论倾向与纽拉特相通；对"构造世界"的多元主义可追溯至纽拉特的百科全书理念 |
