## 三、核心学说

| 学说 | 简要说明 | 代表著作 |
|------|----------|----------|
| 范畴错误 | 将概念错误地归入不适当的范畴——如把"心灵"当作与"身体"并列的实体 | 《心的概念》 |
| 机器中的幽灵 | 对笛卡尔心物二元论的著名批评——二元论是范畴错误 | 《心的概念》 |
| 倾向性分析 | 心理词汇不是内在状态的名称，而是行为倾向的名称 | 《心的概念》 |
| 智力行为 | 智力不是内在过程的产物，而是行为本身的品质 | 《心的概念》 |
| 知道如何 vs 知道那是 | 实践能力与命题知识是两种独立的知识形式 | "Knowing How and Knowing That" |
| 概念分析 | 哲学的任务是澄清概念的使用，而非发现新事实 | 贯穿全部著作 |

## 四、思想关系

### 4.1 前承

| 人物 | 关系 |
|------|------|
| 维特根斯坦（后期） | "治疗性哲学"传统深刻影响赖尔的概念分析方法 |
| 摩尔 | 摩尔对概念分析的关注是赖尔方法论的先驱 |
| 罗素 | 早期受罗素逻辑分析影响，后转向日常语言分析 |

### 4.2 同代

| 人物 | 关系 |
|------|------|
| 奥斯汀 | 牛津日常语言学派同仁，方法相似但关注不同 |
| 斯特劳森 | 牛津同事，斯特劳森更关注描述性形而上学 |
| 维特根斯坦（后期） | 赖尔受其影响但更强调系统性概念分析 |

### 4.3 后启

| 人物 | 关系 |
|------|------|
| 塞尔 | 批评赖尔的分析忽视意识主观性 |
| 丹尼特 | 功能主义吸收赖尔倾向性分析的洞见 |
| 阿姆斯特朗 | 批评赖尔过于行为主义，提出因果理论 |
| 斯坦利 | 试图将"知道如何"还原为"知道那是" |

| 项目 | 内容 |
