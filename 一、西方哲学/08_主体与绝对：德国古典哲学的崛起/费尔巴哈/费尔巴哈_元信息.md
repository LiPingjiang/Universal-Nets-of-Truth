## 核心命题与论证结构

### 命题一：神学的秘密是人类学

**论证结构**：分析上帝的各项属性（全知、全能、全善、永恒等）→ 论证每一项都是人的某种能力的理想化 → 上帝是人的本质的投射 → 神学在不知不觉中研究的是人 → 神学的秘密是人类学。

**关键文本**：《基督教的本质》第一部分"宗教之真正的即人本学的本质"。

### 命题二：宗教是人的自我异化

**论证结构**：人将自己的本质对象化为上帝 → 上帝成为与人对立的、异己的力量 → 人将自身的好品质归于上帝后自身变得贫乏 → 人在上帝面前感到渺小和有罪 → 异化的扬弃在于认识到上帝的本质即人的本质 → 将被投射的品质回归人自身。

**关键文本**：《基督教的本质》导论及第二部分"宗教之不真的即神学的本质"。

### 命题三：新哲学以感性的人为出发点

**论证结构**：黑格尔哲学将思维绝对化、将存在概念化 → 这是对现实的根本性遗忘 → 新哲学必须从感性存在出发 → "存在是思维的原则"而非相反 → 感性的、具体的人是新哲学的主体 → 哲学应成为"生活的事业"。

**关键文本**：《未来哲学原理》§1–§65。

### 命题四："我"的秘密是"你"

**论证结构**：孤立的个人不具有人的本质 → 人的本质在共同体中展开 → 自我意识依赖他者的承认 → "真理只在于并通过'你'而存在" → 人的本质是社会性的、关系性的。

**关键文本**：《未来哲学原理》§57–§65；《基督教的本质》"宗教之根本观点"附录。

### 命题五：宗教的基础是对自然的依赖感

**论证结构**：宗教最初的对象是自然而非上帝 → 人意识到自身对自然的依赖 → 依赖感产生敬畏和崇拜 → 自然宗教是宗教的原始形态 → 人是自然的意识，自然先于人而存在。

**关键文本**：《宗教的本质》讲演一至讲演三。

## 关键概念辨析

### 异化（Entfremdung）vs 对象化（Vergegenständlichung）

费尔巴哈区分了对象化和异化两个层次。对象化是人的基本活动——人将自己的本质表现在对象中，这是正常的、必要的。异化则是对象化的一种特殊形式——当人将自己的本质投射到一个异己的对象（上帝）上，并反过来受这个对象支配时，对象化就变成了异化。异化是对象化的"病变"形式。

### 感性（Sinnlichkeit）vs 感觉主义（Sensualismus）

费尔巴哈的"感性"不等同于18世纪法国唯物主义的感觉主义。感觉主义将一切认识还原为感官知觉，而费尔巴哈的"感性"指的是人的整个现实存在——包括但不限于感官知觉，还包括情感、意志、身体性、社会性。感性是与抽象思维对立的，但不是与理性对立的。

### 类本质（Gattungswesen）vs 个体本质

费尔巴哈认为人的本质是"类本质"——即人类共同具有的、将人与其他存在者区分开来的本质属性（理性、意志、爱）。个体的人是类本质的特殊体现，但类本质不等同于任何具体个体的属性集合。这一概念后来受到马克思的批评——马克思认为不存在固定的"类本质"，人的本质是历史地形成的"社会关系的总和"。
