# 斯宾诺莎（属性平行论）· 元信息

---

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **中文名** | 巴鲁赫·德·斯宾诺莎（后改名本尼迪克特·德·斯宾诺莎） |
| **外文名** | Baruch de Spinoza / Benedictus de Spinoza |
| **生卒年** | 1632–1677 |
| **国籍** | 荷兰（出生于葡萄牙犹太家庭，后被犹太教会驱逐） |
| **主要身份** | 哲学家、磨镜师（以磨制光学镜片为生） |
| **师承/传统** | 笛卡尔主义（批判性继承）、犹太教喀巴拉传统、文艺复兴新柏拉图主义 |
| **主要著作** | *Ethica Ordine Geometrico Demonstrata*（《伦理学》，1677，遗作）；*Tractatus Theologico-Politicus*（《神学政治论》，1670）；*Tractatus de Intellectus Emendatione*（《知性改进论》，遗作）；*Principia Philosophiae Cartesianae*（《笛卡尔哲学原理》，1663） |
| **版本** | Carl Gebhardt 编，*Spinoza Opera*，Carl Winters，1925（标准学术版）；Edwin Curley 译，*The Collected Works of Spinoza*，Princeton University Press，1985（英译） |
| **中文译本** | 贺麟译，《伦理学》，商务印书馆，1958；温锡增译，《神学政治论》，商务印书馆，1963 |

## 二、思想标签

| 标签 | 说明 |
|------|------|
| 实体一元论 | 只存在一个实体，即上帝或自然（Deus sive Natura） |
| 属性平行论 | 心灵与身体是同一实体在思维属性和广延属性下的两种表达 |
| 双面论先驱 | 心灵和身体是同一事物的两个面，就像硬币的正面和反面 |
| 理性主义 | 以几何学方法（定义、公理、命题、证明）构建哲学体系 |
| 泛神论（被指控） | "上帝即自然"的立场使其被同时代人视为无神论者 |
| 情感理论 | 情感既有心理方面也有身体方面，是同一事件的两种描述 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 属性平行论（parallelismus attributorum） | 观念的秩序和联系与事物的秩序和联系是同一的；思维属性下的样式序列与广延属性下的样式序列是同一实体的两种表达 | 《伦理学》第二部分命题七 |
| 同一实体（una eademque substantia） | 能思的实体和广延的实体是同一个实体，只是现在从这个属性、现在从那个属性来理解 | 《伦理学》第二部分命题七推论 |
| 心灵是身体的观念（Mens est idea Corporis） | 构成人类心灵的观念的对象是身体，即某种实际存在的广延样式；心灵就是身体在思维属性下的表达 | 《伦理学》第二部分命题十三 |
| 心身不互动 | 身体不能决定心灵去思考，心灵也不能决定身体去运动；心灵事件和身体事件是同一事件的两种描述，不是因果关系 | 《伦理学》第三部分命题二 |
| 上帝即自然（Deus sive Natura） | 上帝和自然指同一个实体；不是普通宗教意义上的人格创造者，而是无限的、自我存在的实体 | 《伦理学》第一部分；第四部分序言 |
| 自由即自我决定 | 自由不是"不受决定"，而是"由自身本性决定"；真正的自由是成为自身行动的充分原因 | 《伦理学》第一部分定义七；第四部分序言 |
| 三种认知 | 感觉想象（第一类）、理性（第二类）、直觉知识（第三类，最高目标） | 《伦理学》第二部分命题四十附注二 |

## 四、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 笛卡尔（心身关系） | 斯宾诺莎批判继承笛卡尔，否定其实体二元论，将心灵和身体重新理解为同一实体的两种属性 | 第七章：心灵与物质：笛卡尔难题及其后裔 |
| 马勒伯朗士（偶因论） | 两者都否定心身之间的真正因果互动，但斯宾诺莎以同一实体的两种属性消解问题，马勒伯朗士以上帝的持续干预解释同步 | 第七章 |
| 莱布尼茨（前定和谐） | 莱布尼茨拒绝斯宾诺莎的一元论，认为它消除了个体实体的真实性，导致泛神论和决定论；两者在心身不互动上有一致性，但形而上学基础完全不同 | 第七章 |
| 拉美特利（人是机器） | 两者都否定笛卡尔二元论，但斯宾诺莎以实体一元论（上帝/自然）为基础，拉美特利以物质一元论为基础 | 第七章 |
| 康德（批判哲学） | 康德的先验哲学保留了理性主义传统，但对斯宾诺莎的独断论形而上学进行了批判 | 后续章节（如批判哲学章节） |
| 黑格尔（辩证法） | 黑格尔高度推崇斯宾诺莎，称其哲学为"现代哲学的真正开端"；斯宾诺莎的实体概念对黑格尔的绝对精神有重要影响 | 后续章节（如德国观念论章节） |
| 达马西奥（神经科学） | 现代神经科学家安东尼奥·达马西奥在《寻找斯宾诺莎》中认为，斯宾诺莎的情感理论与现代神经科学的发现高度吻合 | 跨章关联（现代） |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对德国观念论的影响** | 黑格尔称斯宾诺莎哲学为"现代哲学的真正开端"；谢林、黑格尔的绝对观念论深受斯宾诺莎实体概念的影响 |
| **对现代心灵哲学的影响** | 属性平行论被视为"双面论"（double-aspect theory）和"中性一元论"（neutral monism）的历史先驱；与现代非还原物理主义有相似之处 |
| **对神经科学的影响** | 安东尼奥·达马西奥在《寻找斯宾诺莎》中论证，斯宾诺莎的情感理论（情感既有心理方面也有身体方面）与现代神经科学对情感的理解高度吻合 |
| **对伦理学的影响** | 《伦理学》的情感理论（从被动情感到主动情感的转化）和自由的自我决定概念对后世伦理学有深远影响 |
| **主要批评** | 被同时代人视为无神论者；决定论框架似乎消除了自由意志；属性的本体论地位（客观还是主观）长期存在解释争议 |
| **学术评价** | 斯宾诺莎的属性平行论是西方哲学史上最彻底的心身一元论解决方案：不是通过找到心灵和身体之间的"桥梁"来解决问题，而是通过证明心灵和身体根本就是同一件事来消解问题 |
