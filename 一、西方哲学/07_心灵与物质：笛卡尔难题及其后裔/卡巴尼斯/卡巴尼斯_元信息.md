## 二、思想标签

| 标签 | 说明 |
|------|------|
| 医学唯物主义 | 以医学观察为基础，主张心灵完全依赖于身体的物质状态 |
| 大脑分泌思想 | 著名命题：大脑产生思想就像器官分泌体液一样 |
| 反笛卡尔二元论 | 彻底否定非物质心灵的存在，是笛卡尔二元论最激进的批判者 |
| 道德自然化 | 道德情感是物质过程的产物，不是神圣命令或理性法则的结果 |
| 社会改革医学化 | 社会改革应建立在医学科学的基础上 |
| 神经科学先驱 | 核心主张——心灵是大脑的功能——已成为当代神经科学共识 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对现代神经科学的影响** | 卡巴尼斯的核心主张——心灵是大脑的功能——已成为当代神经科学和认知神经科学的基本共识 |
| **对精神病学的影响** | 卡巴尼斯关于“精神疾病即身体疾病”的洞见预示了现代精神病学，影响了精神药物学的发展 |
| **对道德哲学的影响** | 卡巴尼斯对道德的自然化为后来的功利主义、进化伦理学和道德心理学奠定了基础 |
| **对社会改革的影响** | 卡巴尼斯关于社会改革应建立在医学基础上的观点，影响了公共卫生、教育改革和社会福利制度 |
| **主要批评** | 面临“解释鸿沟”和“困难问题”的批评：即使知道大脑的所有物理过程，仍无法解释主观意识体验；还原论忽视了心灵和文化的复杂性 |
| **当代回响** | 当代神经科学家（如达马西奥）在研究大脑与情感关系时验证了卡巴尼斯的直觉；卡巴尼斯的方向已成为当代科学和哲学的主流共识 |
