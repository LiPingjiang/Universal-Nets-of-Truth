## 二、思想标签

| 标签 | 说明 |
|------|------|
| 机械唯物主义 | 将人体视为自动机器，心灵是物质的功能 |
| 物质一元论 | 否定笛卡尔二元论，主张只存在一种物质实体 |
| 自然主义 | 道德情感和道德行为是物质过程的产物 |
| 启蒙激进主义 | 明确否定非物质灵魂的存在，批判宗教偏见 |
| 医学经验主义 | 以医学观察（发烧、饮食、药物对心灵的影响）为论证基础 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 人是机器（l'homme machine） | 人体是一台自动机器，心灵是其功能，不存在独立的非物质灵魂 | 《人是机器》，1747 |
| 物质一元论（monisme matérialiste） | 整个宇宙中只存在一种实体，以不同方式被修改；心灵是物质的功能 | 《人是机器》 |
| 物质的灵魂（âme matérielle） | 灵魂不是独立的非物质实体，而是大脑的功能或运动的原则 | 《人是机器》 |
| 人与动物的连续性 | 人类和动物在心理能力上没有本质区别，只有程度上的差异 | 《人是机器》 |
| 运动的原则（principe de mouvement） | 生命和思维的物质基础，存在于大脑的感觉部分 | 《人是机器》 |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对法国唯物主义的影响** | 拉美特利的物质主义对狄德罗、霍尔巴赫、爱尔维修等启蒙思想家产生了重要影响，奠定了法国唯物主义传统的思想基础 |
| **对现代心灵哲学的影响** | 其核心主张——心灵是物质的功能，不是独立的非物质实体——与现代功能主义（functionalism）和物理主义（physicalism）有重要的思想联系 |
| **对神经科学的影响** | 拉美特利的医学背景使他特别关注大脑与心灵的关系，其物质主义为后来的神经科学提供了思想基础：研究大脑就是研究心灵 |
| **对道德哲学的影响** | 认为道德情感和道德行为是物质过程的产物，为功利主义和自然主义伦理学提供了思想基础 |
| **主要批评** | 面临"解释鸿沟"（explanatory gap）和"困难问题"（hard problem of consciousness）：即使知道大脑的所有物理过程，仍无法解释为什么会产生主观意识体验 |
| **当代回响** | 安东尼奥·达马西奥（Antonio Damasio）在《笛卡尔的错误》中呼应了拉美特利的物质主义立场，认为情感与身体状态不可分割 |
