## 二、思想标签

| 标签 | 说明 |
|------|------|
| 笛卡尔主义神学化 | 将笛卡尔哲学与奥古斯丁神学传统相结合 |
| 偶因论 | 被造物之间没有真正的因果关系，只有上帝才是真正的原因 |
| 在上帝中看见 | 认识论理论：我们通过上帝心灵中的观念来认识事物 |
| 普遍意志与特殊意志 | 上帝主要通过普遍意志（按普遍规律）行动，只在特殊情况下通过特殊意志（直接干预）行动 |
| 理性主义神学 | 试图用理性证明上帝在世界中的持续在场 |

## 三、核心概念

| 概念 | 含义 | 出处 |
|------|------|------|
| 偶因论（occasionnalisme） | 被造物（心灵和身体）之间没有真正的因果关系；每次"互动"都是上帝直接干预的结果；被造物只是"偶然原因"（causes occasionnelles） | 《真理的探求》第六卷第二部分第三章，1674–1675 |
| 偶然原因（cause occasionnelle） | 触发上帝行动的机会，不是真正的原因；被造物的意志和运动只是触发上帝普遍意志的偶然原因 | 《真理的探求》；《关于形而上学和宗教的对话》第七篇 |
| 真正原因（cause véritable） | 真正产生效果的原因；只有上帝的意志和其效果之间有必然联系，因此只有上帝才是真正的原因 | 《真理的探求》 |
| 在上帝中看见（vision en Dieu） | 我们认识事物不是通过直接感知，而是通过上帝心灵中的观念；上帝的理性是精神的场所，就像空间是物体的场所一样 | 《真理的探求》第三卷第二部分第六章 |
| 普遍意志（volonté générale） | 上帝按照普遍规律行动的意志；自然规律是上帝普遍意志的表达 | 《关于形而上学和宗教的对话》第七篇 |
| 特殊意志（volonté particulière） | 上帝针对特定情况的直接干预，如奇迹 | 《关于形而上学和宗教的对话》第七篇 |

## 四、跨章链接

| 链接目标 | 关系说明 | 所在章节 |
|----------|----------|----------|
| 笛卡尔（心身关系） | 马勒伯朗士接受笛卡尔的心身二元论，但拒绝其松果腺互动理论，认为两种完全不同的实体不可能真正互动 | 第七章：心灵与物质：笛卡尔难题及其后裔 |
| 莱布尼茨（前定和谐） | 莱布尼茨的前定和谐论是对马勒伯朗士偶因论的直接替代方案；莱布尼茨批评偶因论使上帝变成不断修理时钟的工匠 | 第七章 |
| 斯宾诺莎（属性平行论） | 两者都否定心身之间的真正因果互动，但马勒伯朗士以上帝的持续干预解释同步，斯宾诺莎以同一实体的两种属性消解问题 | 第七章 |
| 贝克莱（唯心主义） | 贝克莱的唯心主义与马勒伯朗士的"在上帝中看见"有相似之处，但马勒伯朗士保留了物质世界的存在 | 后续章节（如存在论相关章节） |
| 休谟（因果关系） | 休谟对因果关系的批判与马勒伯朗士对"真正原因"的分析有关联 | 后续章节（如因果性章节） |
| 阿尔诺（Antoine Arnauld） | 与马勒伯朗士有著名的哲学争论，批判其"在上帝中看见"理论 | 第七章（相关） |

## 五、历史影响

| 影响领域 | 具体内容 |
|----------|----------|
| **对莱布尼茨的影响** | 莱布尼茨的前定和谐论直接回应了马勒伯朗士的偶因论，试图用"一次性的完美安排"替代"持续的神圣干预" |
| **对贝克莱的影响** | 贝克莱的唯心主义与马勒伯朗士的"在上帝中看见"有相似之处，但贝克莱进一步否认了物质世界的存在 |
| **对休谟的影响** | 休谟对因果关系的怀疑论与马勒伯朗士对有限实体之间"必然联系"的否定有关联 |
| **对当代心灵哲学的影响** | 偶因论被视为"副现象论"（epiphenomenalism）和"因果封闭性"论证的历史先驱 |
| **主要批评** | 莱布尼茨批评偶因论使上帝变成不断干预的工匠，不够优雅；休谟批评被造物与上帝之间的"必然联系"同样无法通过经验证明 |
| **学术评价** | 马勒伯朗士的偶因论在逻辑上是一致的，但将上帝变成了不断干预的工匠，引发了神学上的困难；其"在上帝中看见"的认识论理论是17世纪观念论的重要发展 |
