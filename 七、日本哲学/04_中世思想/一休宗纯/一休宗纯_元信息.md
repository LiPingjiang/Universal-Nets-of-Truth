# 一休宗纯（Ikkyū Sōjun）元信息

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 一休宗纯（いっきゅう そうじゅん），幼名千菊丸，后名周建，法号一休斋，谥号一休宗纯 |
| 生卒年 | 1394-1481（应永元年至文明十三年） |
| 出生地 | 日本京都，据云为后小松天皇之私生子（一说为小仓氏之子） |
| 宗派/学派 | 临济宗（禅宗）→ 其修行方式自创「狂禅」一派；曾嗣法华叟宗昙、嗣薪之虎关师炼，后脱离寺院体系 |
| 主要领域 | 禅宗修行、汉诗、和歌、书法、茶道（据说一休为村田珠光之师，日本茶道之先驱） |
| 代表作 | 《狂云集》（Kyōunshū，约800首汉诗与和歌）、《假名法语》（口语体法语）、《一休法语》 |
| 历史定位 | 日本室町时代最具个性的禅宗僧侣；日本「狂禅」之代表人物；日本茶道之精神先驱；日本禅宗文学与破戒禅学之集大成者；日本中世精神史上之反叛象征 |
| 学术背景 | 幼年入安国寺为侍童，十一岁从华叟宗昙学禅；曾在醍醐寺、建仁寺、妙心寺等名刹修行；曾因顿悟而被华叟印可，但拒绝嗣位；后长期游离于寺院体系之外，与庶民、妓女、艺人交往；晚年任大德寺第四十七代住持 |

## 核心论点列表

1. **破戒即真戒**：佛教戒律是接引初学的方便法门，而非究竟真理。执着于持戒之相，反而成为悟道的障碍。真正的大戒不在身行，而在心性的自由。一休以饮酒、近色等行为故意破戒，意在打破修行者对「戒」的执着。

2. **呵佛骂祖以破相**：禅宗传统中有呵佛骂祖的公案，一休将其发挥到极致。他宣称要「杀佛」「杀祖」，不是真的否定佛陀，而是要用最极端的方式破除修行者对佛相、祖相的执着——因为任何执着都是悟道的障碍。

3. **狂禅即真禅**：社会所定义的「正常」与「疯狂」标准，本身是无明的产物。一休以佯狂（装疯）的方式，打破社会对僧侣行为的期待，以此显示精神自由的真正境界。狂不是目的，而是手段——通过打破常规，让常规所束缚的心灵获得解放。

4. **声色即道**：一休反对将修行与世俗生活对立。他认为，男女之情、饮食之乐、酒色之欢，本身不是罪恶，而是「道」的显现。只要心无执着，任何境遇都可以是修行的道场。般若波罗蜜多（智慧）不在深山，而在市井、在酒肆、在闺房。

5. **游戏三昧**：真正的修行应当像游戏一样自在，不应被修行的形式所束缚。一休写诗、饮酒、与妓女交往，都不是在堕落，而是在「游戏三昧」——以不被行为所束缚的心态来行动。

6. **顿悟不依赖次第**：一休继承了临济宗顿悟的传统，反对通过渐修、持戒、坐禅等次第来求悟。他认为，悟可以在任何瞬间、任何境遇中发生，不需要特定的修行方式。坐禅可以悟，喝酒也可以悟；读经可以悟，骂人也可以悟。

7. **社会批判与庶民立场**：一休对室町时代寺院制度的腐败、僧侣的伪善、权贵的虚荣进行了尖锐的批判。他不依附于任何权力机构，长期与庶民、艺人、妓女为伍，代表了日本禅宗中一种独特的反体制精神。

8. **茶道之精神源头**：传说一休将宋茶之茶碗与饮茶法传给村田珠光，并影响了珠光「茶禅一味」思想的形成。一休的侘寂审美、一期一会精神、简朴茶室的理念，被认为是日本草庵茶道的精神源头。
