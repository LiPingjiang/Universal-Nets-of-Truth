## 核心论点列表

1. **「粹」之哲学系统化**：九鬼将日本日常语汇「粹」（iki）提升为哲学范畴，以亚里士多德「四因说」之结构分析法，揭示其「媚态—意气地—谛观」之三重语义结构，开创了日本美学概念之哲学化先河。
2. **粹作为存在样式**：粹不仅是审美趣味，更是一种存在论层面的存在方式——不支配人、亦不受人支配；不占有物、亦不为物所役；在否定中肯定，在离脱中成就。
3. **偶然之形而上学独立地位**：九鬼批判西方形而上学以必然性为基底的传统，主张偶然性是与必然性对等的独立形而上学范畴，偶然之世界为开放之场域，内含不可预测之「出会」。
4. **模态理论之四元重构**：在传统的必然—可能—不可能三模态基础上，补入「偶然」以成四元体系。偶然之特征在于「已然而理由不存于自身」，即「他律性」之暴露。
5. **偶然与存在之敞开**：人的存在本质上为「偶然之存在」，被抛状态（Geworfenheit）非徒然之不幸，而是存在之敞开（Offenheit），偶然之相遇撕开自我封闭之存在织体，令光得以射入。
6. **东西方哲学之对话**：九鬼以现象学方法（海德格尔）与柏格森之生命哲学分析日本文化范畴，同时以日本美学之独特范畴（粹、幽玄、物哀）补充西方美学之不足，体现京都学派「东西方对话」之学术自觉。
7. **色道之存在论阐释**：将江户时代游里文化（色道）从道德贬抑中解放出来，以现象学方法揭示其作为「存在之场」的深层结构——以离为入、在沉浸中保持游离的存在辩证法。
8. **非英雄主义之存在姿态**：粹之存在姿态与西方英雄主义之「征服—支配」逻辑不同，以「非支配」为特征，体现日本文化中独特的「无为而无不为」之精神。

| 七、日本哲学/03_近代日本哲学/西田几多郎 | 九鬼为西田几多郎之学生，其哲学深受西田「绝对无」之存在论影响，粹之「谛观」与西田之「绝对矛盾的自我同一」有密切关联 |
| 七、日本哲学/03_近代日本哲学/西谷启治 | 西谷与九鬼同属京都学派，二者皆关注「虚无」与「存在」之张力，西谷之「虚无主义」分析与九鬼之「偶然性」存在论可形成对话 |
| 七、日本哲学/03_近代日本哲学/铃木大拙 | 铃木从禅学角度、九鬼从美学与存在论角度，共同阐释日本文化之独特性，二者构成日本文化哲学的两大支柱 |
| 七、日本哲学/02_日本美学 | 九鬼是近代日本美学研究之奠基人，《粹的构造》为日本美学范畴（粹、幽玄、物哀）之哲学化提供了方法论范本 |
| 三、西方哲学/04_现象学/海德格尔 | 九鬼直接师事海德格尔，其现象学方法、存在论分析、「被抛状态」概念皆受海德格尔深刻影响 |
| 三、西方哲学/06_存在主义 | 九鬼之存在论与法国存在主义（萨特、加缪）关注相近，其「偶然性」哲学与萨特之「偶然性」概念可形成比较研究 |
| 五、比较哲学/东西方比较 | 九鬼是东西方比较哲学之典型范例，以现象学方法分析日本文化范畴，同时以日本范畴补充西方哲学之不足 |
