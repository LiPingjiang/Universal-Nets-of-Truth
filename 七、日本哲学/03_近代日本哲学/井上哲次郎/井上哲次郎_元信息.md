## 核心论点列表

1. **现象即实在论**：现象即实在，实在即现象，离现象无实在，离实在无现象。现象非实在之影乃实在之自现，如水与波不二。试图超越西方"现象—实在"二元对立，构建日本独特本体论。

2. **东西哲学融合论**：西方哲学好分析重分别，东方哲学好综合重圆融，二者各有所偏。当融合东西之长构建新哲学体系，以西方观念论为框架、东方非二元论为内容，使精神与物相即相入。

3. **日本主义实践论**：日本哲学之特质在于实践，非空谈性理乃身体力行。大和魂为实践之力、行动之勇、忠诚之心。日本主义非排斥外来思想，乃以日本精神为根底融摄一切有益思想。

4. **国民道德论**：教育之根本在道德养成，道德之根本在国民精神发扬。忠孝为道德大本，《教育敕语》示道德正鹄。德育为先，知识无道德则危、道德无知识则弱。为天皇制教育提供哲学辩护。

5. **哲学史方法论三义**：治日本哲学史当明三义——源流之辨（追溯思想渊源流变）、比较之法（与中国印度西洋哲学比较）、实践之眼（关注社会实践面向）。此为日本哲学史研究奠基性方法论。

## 章节关联

| 关联哲学家 | 关联领域 | 关联说明 |
|-----------|---------|---------|
| 西田几多郎 | 现象即实在论、场所逻辑 | 井上"现象即实在"为西田"场所"论提供理论先例，但西田的体系更为成熟和原创 |
| 大西祝 | 日本学院哲学、认识论 | 同为日本哲学学科创立期人物，大西在京都大学，井上在东京大学，分别代表两条路径 |
| 加藤弘之 | 国家主义、明治体制理论 | 同为明治体制理论辩护者，加藤从政治学角度，井上从哲学伦理学角度 |
| 黑格尔 | 观念论、辩证法 | 井上现象即实在论受黑格尔"实体即主体"思辨逻辑影响 |
| 华严宗 | 相即相入、圆融无碍 | 井上融合论借用华严宗"相即相入"概念框架 |
| 阳明学 | 知行合一、实践哲学 | 井上研究日本阳明学派，以其为日本实践哲学之代表 |
| 福泽谕吉 | 明治启蒙、教育思想 | 井上与福泽同为明治教育思想重要人物，但井上更侧重德育和国家主义 |
| 丸山真男 | 日本政治思想史 | 丸山批判性继承井上的哲学史方法论，但对其国家主义倾向持批判态度 |
