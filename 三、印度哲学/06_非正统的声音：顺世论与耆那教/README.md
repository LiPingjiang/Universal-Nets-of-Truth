# 06 非正统的声音：顺世论与耆那教

## 核心张力

**正统(āstika) vs 非正统(nāstika)**

- 正统(āstika)：承认吠陀权威——六派哲学
- 非正统(nāstika)：否认吠陀权威——佛教、顺世论、耆那教

## 思维流向

从第五章"空与有"的佛教内部论争，到第六章"非正统的声音"：
- 顺世论(Lokāyata/Cārvāka)：彻底唯物论，反对一切超验
- 耆那教(Jaina)：极端多元实在论，非一端论
- 佛教：中道，非有非无
- 三非正统与正统六派的持续论争

## 人物与文献

### 顺世论(Lokāyata/Cārvāka)
- **文献**：《摄一切见论》(Sarvadarśanasaṃgraha)中的顺世论章、残篇
- **核心学说**：
  - 四大(地水火风)为唯一实在
  - 意识是物质的属性，身死识灭
  - 反对祭祀、轮回、解脱
  - 快乐主义伦理：现世的快乐是唯一目的
- **关键概念**：
  - 四大(mahābhūta)：地(pṛthivī)、水(āpas)、火(tejas)、风(vāyu)
  - 识即身(caitanyam dehasamhatim)：意识是身体属性的组合

### 耆那教(Jaina)创始人：摩诃毗罗(Mahāvīra, 约前599-前527)
- **文献**：《阿含经》(Āgama)、图尔达尔帕(Tattvārtha Sūtra)
- **核心学说**：
  - 七谛说(sapta-tattva)：命(jīva)、非命(ajīva)、漏(āsrava)、缚(bandha)、遮(saṃvara)、灭(nirjarā)、解脱(mokṣa)
  - 非一端论(anekāntavāda)：一切判断都有条件，非绝对
  - 或然论(syādvāda)：七支论法(saptabhaṅgī)
- **关键概念**：
  - 命(jīva)：灵魂，有知，受业力束缚
  - 非命(ajīva)：物质(dharma, adharma, ākāśa, pudgala, kāla)
  - 非一端论(anekāntavāda)：实体有无限属性，任何命题只表达部分真理

### 佛教 vs 顺世论论争
- 佛教批评顺世论：断见(ucchedavāda)，否定因果业报
- 顺世论批评佛教：虚构轮回，无证据支持

### 佛教 vs 耆那教论争
- 佛教批评耆那教：极端苦行无意义，命我论(ātman)与无我(anātman)对立
- 耆那教批评佛教：中道是虚无，否定实体导致道德虚无

## 核心问题

1. **唯物论在印度何以可能？** 顺世论：四大为实在，识为物性；为何在印度思想中边缘化？
2. **非一端论的逻辑结构：七支论法如何运作？** 或然论：syaṇ nāsti（或许不存在）、syaṇ asti（或许存在）等七支
3. **非正统与正统的边界：佛教为何被归为非正统？** 佛教否认吠陀权威，但体系最完整；耆那教同样否认，但保留苦行传统
