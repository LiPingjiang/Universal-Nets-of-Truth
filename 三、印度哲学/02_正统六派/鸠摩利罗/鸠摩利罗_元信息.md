# 鸠摩利罗（弥曼差派）

## 基本信息

| 项目 | 内容 |
|------|------|
| 生卒年 | 约7世纪（650-700年） |
| 梵名 | Kumārila Bhaṭṭa |
| 学派 | 弥曼差派（Mīmāṃsā），跋陀支派（Bhaṭṭa） |
| 代表著作 | 《颂疏》（Ślokavārttika）、《简疏》（Ṭupṭīkā）、《广疏》（Bṛhaṭṭīkā） |
| 历史地位 | 弥曼差派最重要的哲学家，印度教复兴的重要推手，最有力的佛教批判者之一 |

## 核心论点

1. **吠陀权威论**：吠陀是非人格的（非任何人所作）、永恒的，因此具有绝对权威
2. **声常住论**：吠陀的声音是永恒的，我们听到的是永恒声音的"显现"，非声音本身
3. **自证论**：一切知识的正确性是自明的，不需要外部验证；不正确性才需要外部原因
4. **有我论**：坚持永恒神我的存在，批判佛教无我论无法解释记忆、责任和认识的连续性
5. **反刹那灭论**：批判佛教的刹那灭论无法解释因果关系和认识的连续性
6. **行动路**：解脱通过正确履行吠陀规定的义务（祭祀、仪式）实现，而非单纯的知识
