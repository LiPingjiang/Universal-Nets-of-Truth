# 02 解脱之道：正统六派的系统建构

## 核心张力

**解脱(mokṣa) vs 束缚(bandha)**

- 解脱(mokṣa)：从轮回中解放，梵我合一，苦的止息
- 束缚(bandha)：业力轮回，无明覆盖，灵魂受困于生死

## 思维流向

从奥义书的梵我合一直觉，到六派的系统哲学建构：
- 弥曼差：吠陀永恒，声是常——最保守的正统，祭祀即目的
- 吠檀多：梵是唯一实在，世界如幻——奥义书的哲学系统化
- 数论：原初物质与神我二元——演化论而非创造论
- 瑜伽：控制心识的八支实践——数论的修行补充
- 正理：四量与五支论式——印度最系统的逻辑学
- 胜论：六句义与原子论——印度版的范畴学与原子论

## 人物与文献

### 弥曼差派(Mīmāṃsā)

#### 阇弥尼(Jaimini, 约前4世纪)
- **原文**：《弥曼差经》(Mīmāṃsā Sūtra)
- **核心学说**：吠陀永恒，声是常(śabda-nityatā)，祭祀是目的本身
- **关键概念**：
  - 无前(apūrva)：祭祀产生的不可见力量，连接祭祀与果报
  - 声常论(śabda-nityatā)：词与意义的关系永恒，吠陀非人造

#### 枯马立拉(Kumārila Bhaṭṭa, 约7世纪)
- **原文**：《颂释补》(Ślokavārttika)
- **核心学说**：批判佛教，为吠陀仪式主义辩护，自证论
- **关键概念**：
  - 自证论(svataḥ-prāmāṇya)：知识的自明性，无需外部验证

### 吠檀多派(Vedānta)

#### 商羯罗(Śaṅkara, 约788-820)
- **原文**：《梵经注》(Brahma Sūtra Bhāṣya)、《奥义书注》、《示教千则》(Upadeśasāhasrī)
- **核心学说**：不二论(Advaita)——梵是唯一实在，世界如幻(māyā)
- **关键概念**：
  - 上梵(nirguṇa Brahman)：无属性之梵，唯一实在
  - 下梵(saguṇa Brahman)：有属性之梵，为修行者方便施设
  - 幻(māyā)：世界的表象性，非真实非虚无

#### 罗摩奴阇(Rāmānuja, 1017-1137)
- **原文**：《梵经注》(Śrī Bhāṣya)、《薄伽梵歌注》
- **核心学说**：限定不二论(Viśiṣṭādvaita)——梵是有人格的，世界和灵魂是梵的属性
- **关键概念**：
  - 限定不二(viśiṣṭādvaita)：梵与灵魂既同一又有差异，如身体与灵魂
  - 虔信(bhakti)：对神的爱与奉献是解脱之道

#### 摩陀婆(Madhvācārya, 1238-1317)
- **原文**：《梵经注》、《薄伽梵歌注》
- **核心学说**：二元论(Dvaita)——梵、灵魂、物质三者根本不同
- **关键概念**：
  - 五种差异(pañca-bheda)：神与灵魂、神与物质、灵魂与灵魂、灵魂与物质、物质与物质之间皆有差异

### 数论派(Sāṃkhya)

#### 自在黑(Īśvara Kṛṣṇa, 约3-4世纪)
- **原文**：《数论颂》(Sāṃkhya Kārikā，72颂)
- **核心学说**：二元论——原初物质(prakṛti)与神我(puruṣa)
- **关键概念**：
  - 三德(guṇas)：萨埵(sattva，明性)、罗阇(rajas，动性)、答摩(tamas，暗性)
  - 二十五谛(tattvas)：从原初物质到神我的二十五层演化

### 瑜伽派(Yoga)

#### 帕坦伽利(Patañjali, 约前2世纪-2世纪)
- **原文**：《瑜伽经》(Yoga Sūtra，196颂)
- **核心学说**：八支瑜伽，心识的止息
- **关键概念**：
  - 八支(aṣṭāṅga)：禁制(yama)、劝制(niyama)、坐法(āsana)、调息(prāṇāyāma)、制感(pratyāhāra)、执持(dhāraṇā)、禅定(dhyāna)、三摩地(samādhi)
  - 瑜伽定义(yogaś citta-vṛtti-nirodhaḥ)：瑜伽是心识波动的止息

### 正理派(Nyāya)

#### 乔答摩/足目(Gautama / Akṣapāda, 约2世纪)
- **原文**：《正理经》(Nyāya Sūtra)
- **核心学说**：十六谛，四量，五支论式
- **关键概念**：
  - 四量(pramāṇas)：现量(pratyakṣa，知觉)、比量(anumāna，推理)、譬喻量(upamāna)、圣言量(śabda)
  - 五支论式(pañcāvayava)：宗、因、喻、合、结——印度逻辑的基本结构

### 胜论派(Vaiśeṣika)

#### 迦那陀(Kaṇāda, 约前2世纪)
- **原文**：《胜论经》(Vaiśeṣika Sūtra)
- **核心学说**：六句义，原子论
- **关键概念**：
  - 六句义(ṣaḍ-padārtha)：实(dravya)、德(guṇa)、业(karma)、同(sāmānya)、异(viśeṣa)、和合(samavāya)
  - 原子(paramāṇu)：不可分割的最小物质单位，由和合力组合

## 核心问题

1. **解脱的路径：知识、祭祀、虔信还是瑜伽？** 吠檀多：知识(jñāna)；弥曼差：祭祀(karma)；罗摩奴阇：虔信(bhakti)；瑜伽：修行(yoga)
2. **梵与世界的关系：同一、属性同一还是根本不同？** 商羯罗：不二；罗摩奴阇：限定不二；摩陀婆：二元——吠檀多三派的根本分歧
3. **六派是否统一？** 正统六派都承认吠陀权威，但哲学立场各异——弥曼差重祭祀，吠檀多重知识，数论重二元，瑜伽重实践，正理重逻辑，胜论重范畴
