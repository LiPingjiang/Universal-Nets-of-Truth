# 05 空与有：中观、唯识与如来藏的张力

## 核心张力

**空(śūnya) vs 有(asti)**

- 空(śūnyatā)：缘起无自性、一切法空
- 有(asti/svabhāva)：实有自性、法体恒存

## 思维流向

从第四章"缘起与空性"的佛教哲学论争，到第五章"空与有的张力"：
- 中观派(Mādhyamika)：龙树、月称——一切法空，缘起即空
- 瑜伽行派(Yogācāra)：无著、世亲——三性说，虚妄分别
- 如来藏思想(Tathāgatagarbha)：《楞伽经》《宝性论》——心性本净，客尘所染
- 三系融合与论争：空有之争在汉传佛教的延续（三论宗、唯识宗、华严宗）

## 人物与文献

### 龙树(Nāgārjuna, 约150-250 CE)
- **原文**：《中论》(Mūlamadhyamakakārikā)
- **核心学说**：中道(śūnyatā)、八不中道
- **关键概念**：
  - 空(śūnyatā)：缘起故空，空即缘起
  - 自性(svabhāva)：独立不依的存在性，空即无自性
  - 八不(aṣṭau na)：不生不灭、不常不断、不一不异、不来不出

### 月称(Candrakīrti, 约600-650 CE)
- **原文**：《入中论》(Madhyamakāvatāra)
- **核心学说**：应成派(Prāsaṅgika)——只破不立，随应破斥
- **关键概念**：应成(prasaṅga)：归谬论证，不立自宗

### 无著(Asaṅga, 约310-390 CE)
- **原文**：《瑜伽师地论》《摄大乘论》
- **核心学说**：瑜伽行派、三性说
- **关键概念**：
  - 三性(tri-svabhāva)：遍计所执性、依他起性、圆成实性
  - 阿赖耶识(ālaya-vijñāna)：根本识，藏识，种子所藏

### 世亲(Vasubandhu, 约320-400 CE)
- **原文**：《唯识三十颂》《俱舍论》
- **核心学说**：唯识无境、百法明门
- **关键概念**：唯识(vijñāpti-mātratā)：一切唯识所现，无外境实有

### 《楞伽经》(Laṅkāvatāra Sūtra)
- **核心思想**：如来藏藏识、五法三自性
- **关键概念**：如来藏(tathāgatagarbha)：佛性内在于一切众生

### 《宝性论》(Ratnagotravibhāga)
- **核心思想**：如来藏为佛性、九喻说明客尘覆障
- **关键概念**：佛性(buddha-dhātu)：一切众生有如来藏，即是有佛性

## 核心问题

1. **空与有的关系：空是虚无？空是缘起？** 龙树：空即缘起，非断灭空；清辩：空是遮遣，非表诠
2. **识与境的关系：唯识无境？境不离识？** 世亲：唯识无境；陈那：带相说
3. **如来藏与阿赖耶识：一心二门？** 《楞伽经》：如来藏藏识，不生不灭与生灭和合
