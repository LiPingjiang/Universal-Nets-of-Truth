# 03 梵与我的关系：不二论、限定不二论与二元论

## 核心张力

**一(不二) vs 多(二元)**

- 一(ekam)：梵是唯一实在，世界如幻，多即一之显现
- 多(anekam)：梵、灵魂、物质各有实体，差异真实不可化约

## 思维流向

从第二章"解脱之道"的六派概览，到第三章"梵与我的关系"的吠檀多深化：
- 商羯罗不二论：梵是唯一实在，世界如幻(māyā)，个体灵魂(jīva)与梵同一
- 罗摩奴阇限定不二论：梵是有人格的，灵魂是梵的属性/部分，既同一又有差异
- 摩陀婆二元论：梵、灵魂、物质三者根本不同，五种差异不可化约
- 其他吠檀多：尼跋伽二不二论、瓦拉巴纯粹不二论、柴坦尼亚不可思议差异同一论
- 数论与瑜伽的二元论：原初物质与神我根本不同——与吠檀多二元论的平行

## 人物与文献

### 商羯罗(Śaṅkara, 约788-820)
- **原文**：《梵经注》(Brahma Sūtra Bhāṣya)、《示教千则》(Upadeśasāhasrī)、《我之觉知》(Ātma Bodha)
- **核心学说**：不二论(Advaita Vedānta)——梵是唯一实在，世界如幻
- **关键概念**：
  - 上梵(nirguṇa Brahman)：无属性之梵，唯一实在，不可言说
  - 下梵(saguṇa Brahman)：有属性之梵，为修行者的方便施设
  - 幻(māyā)：世界的表象性，非真实非虚无，如绳上见蛇
  - 个体灵魂(jīva)：梵受幻力限制而显现的个体化，本质即梵

### 乔荼波陀(Gauḍapāda, 约6-7世纪)
- **原文**：《蛙氏奥义书注》(Māṇḍūkya Kārikā)
 - **核心学说**：幻论(ajāta-vāda)——万物从未生起，如梦中物
- **关键概念**：
  - 不生(ajāti)：万物从未真实生起，唯梵真实
  - 大乘佛教影响：中观空观与吠檀不二论的融合

### 罗摩奴阇(Rāmānuja, 1017-1137)
- **原文**：《梵经注》(Śrī Bhāṣya)、《吠陀义灯》(Vedārthasaṅgraha)
- **核心学说**：限定不二论(Viśiṣṭādvaita)——梵与灵魂既同一又有差异
- **关键概念**：
  - 限定不二(viśiṣṭādvaita)：灵魂是梵的属性(viśeṣaṇa)，如身体之与灵魂
  - 梵即毗湿奴(Brahman as Viṣṇu)：梵是有人格的神，非抽象原理
  - 虔信(bhakti)：对神的爱与奉献是解脱之道，非仅知识

### 摩陀婆(Madhvācārya, 1238-1317)
- **原文**：《梵经注》、《薄伽梵歌注》、《摩诃婆罗多注》
- **核心学说**：二元论(Dvaita)——梵、灵魂、物质三者根本不同
- **关键概念**：
  - 五种差异(pañca-bheda)：神与灵魂、神与物质、灵魂与灵魂、灵魂与物质、物质与物质
  -  gradation of souls：灵魂有等级，部分永远解脱，部分永远受缚
  - 梵即毗湿奴：摩陀婆的虔信比罗摩奴阇更极端

### 尼跋伽(Nimbārka, 约11-12世纪)
- **核心学说**：二不二论(Dvaitādvaita)——梵与灵魂既同一又不同
- **关键概念**：
  - 二不二(dvaitādvaita)：如太阳与光线，既同一又不同
  - 克里希那(Kṛṣṇa)崇拜：梵即克里希那

### 瓦拉巴(Vallabha, 1479-1531)
- **核心学说**：纯粹不二论(Śuddhādvaita)——世界是梵的纯粹显现，非幻
- **关键概念**：
  - 纯粹不二(śuddhādvaita)：世界非幻(māyā)，而是梵的纯粹显现，无杂质
  - 虔信之路(puṣṭimārga)：神的恩典(puṣṭi)是解脱的关键

### 柴坦尼亚(Caitanya, 1486-1534)
- **核心学说**：不可思议差异同一论(Acintya-bhedābheda)——差异与同一同时成立，但不可思议
- **关键概念**：
  - 不可思议(acintya)：梵与灵魂既同一又不同，此关系超越理性
  - 克里希那与拉达的虔信：情感性的虔信(bhakti)至上

## 核心问题

1. **世界是否真实？** 商羯罗：世界如幻，唯梵真实；罗摩奴阇：世界是梵的真实属性；摩陀婆：世界是独立实体
2. **灵魂是否有等级？** 商羯罗：灵魂本质同一，皆即梵；摩陀婆：灵魂有等级，部分永缚——社会等级制度的哲学辩护？
3. **解脱靠知识还是虔信？** 商羯罗：知识(jñāna)即解脱；罗摩奴阇、摩陀婆：虔信(bhakti)是解脱之道——印度教两大传统
