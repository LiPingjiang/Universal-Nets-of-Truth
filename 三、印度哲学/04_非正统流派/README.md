# 04 缘起与空性：佛教哲学的四百年论争

## 核心张力

**缘起(pratītyasamutpāda) vs 空性(śūnyatā)**

- 缘起：一切现象互相依存，无独立自存
- 空性：一切法无自性，缘起即空，空即缘起

## 思维流向

从第三章"梵与我的关系"的印度教正统论争，到第四章"缘起与空性"的佛教革命：
- 原始佛教：四圣谛、八正道、缘起、无我——对吠陀梵我论的颠覆
- 部派佛教：法的精密分析——说一切有部(三世实有) vs 经量部(刹那生灭)
- 大乘中观：龙树八不中道——一切法空，包括空本身
- 大乘瑜伽行：无著三性说——虚妄分别，唯识无境
- 密宗：身口意三密，即身成佛——大乘的修行综合

## 人物与文献

### 佛陀/释迦牟尼(Śākyamuni, 约前5世纪)
- **原文**：巴利三藏(Tipiṭaka)——律藏、经藏、论藏
- **核心学说**：四圣谛、八正道、缘起、无我、涅槃
- **关键概念**：
  - 四圣谛(cattāri ariyasaccāni)：苦、集、灭、道
  - 缘起(paṭiccasamuppāda)：此有故彼有，十二因缘
  - 无我(anattā)：五蕴皆空，无固定不变的自我
  - 涅槃(nibbāna)：贪瞋痴的熄灭，苦的止息

### 说一切有部(Sarvāstivāda, 前2-7世纪)
- **核心学说**：一切法三世实有(sarvaṃ asti)
- **关键概念**：
  - 三世实有：过去、现在、未来一切法皆实有
  - 极微(paramāṇu)：物质的最小单位，实有不可分
  - 四谛十六行相：对四圣谛的精密分析

### 经量部(Sautrāntika, 前2-7世纪)
- **核心学说**：刹那生灭(kṣaṇika)，种子(bīja)说
- **关键概念**：
  - 刹那生灭：一切现象刹那生灭，无暂住
  - 种子(bīja)：潜在业力，如种子生果

### 龙树(Nāgārjuna, 约150-250 CE)
- **原文**：《中论》(Mūlamadhyamakakārikā，27品)
- **核心学说**：八不中道，缘起性空，四重否定
- **关键概念**：
  - 八不(aṣṭau na)：不生不灭、不常不断、不一不异、不来不出
  - 自性(svabhāva)：独立不依的存在性，空即无自性
  - 二谛(dve satye)：世俗谛(有)与胜义谛(空)
  - 涅槃即轮回(nirvāṇaṃ samsāra iti)：二者无本质差异

### 提婆(Āryadeva, 约3世纪)
- **原文**：《百论》(Śataka)
- **核心学说**：破斥外道与佛教内部的实体论，中观派二祖

### 佛护(Buddhapālita, 约470-540)
- **核心学说**：归谬法(prasaṅga)——只破不立，随应破斥

### 清辨(Bhāvaviveka, 约500-570)
- **核心学说**：自立量(svātantrika)——用因明建立正面论证
- **关键概念**：
  - 自立量 vs 归谬法：中观派内部的方法论分歧

### 月称(Candrakīrti, 约600-650)
- **原文**：《入中论》(Madhyamakāvatāra)
- **核心学说**：应成派正统，只破不立
- **关键概念**：
  - 应成(prasaṅga)：归谬论证，不立自宗，随应破斥他宗

### 无著(Asaṅga, 约4世纪)
- **原文**：《瑜伽师地论》(Yogācārabhūmi-śāstra，100卷)、《摄大乘论》
- **核心学说**：瑜伽行派，三性说
- **关键概念**：
  - 三性(tri-svabhāva)：遍计所执性(parikalpita)、依他起性(paratantra)、圆成实性(pariniṣpanna)
  - 阿赖耶识(ālaya-vijñāna)：根本识，藏识，种子所藏

### 世亲(Vasubandhu, 约4-5世纪)
- **原文**：《唯识三十颂》(Triṃśikā)、《唯识二十论》(Viṃśatikā)
- **核心学说**：唯识无境(vijñapti-mātratā)
- **关键概念**：
  - 唯识(vijñapti-mātratā)：一切唯识所现，无外境实有
  - 八识：眼耳鼻舌身意末那阿赖耶

### 陈那(Dignāga, 约480-540)
- **原文**：《因明入正理论》(Nyāyamukha)、《集量论》(Pramāṇasamuccaya)
- **核心学说**：佛教逻辑学(因明)集大成，三支论式，遮诠(apoha)
- **关键概念**：
  - 三支论式(tri-rūpya)：宗、因、喻——简化的逻辑结构
  - 遮诠(apoha)：概念通过排除他者而成立，非正面表诠

### 法称(Dharmakīrti, 约600-660)
- **原文**：《释量论》(Pramāṇavārttika)、《定量论》(Pramāṇaviniścaya)
- **核心学说**：佛教逻辑巅峰，因果推理，瑜伽知觉
- **关键概念**：
  - 因果推理(svabhāva-pratibandha)：必然因果关系是推理的基础
  - 瑜伽知觉(yogi-pratyakṣa)：修行者的直接知觉，超越常人

### 寂天(Śāntideva, 约685-763)
- **原文**：《入菩萨行论》(Bodhicaryāvatāra)
- **核心学说**：大乘修行的系统指南，自他交换
- **关键概念**：
  - 自他交换(parātmāparivartana)：将自己与他人位置互换，培养慈悲

### 莲花戒(Kamalaśīla, 约740-795)
- **原文**：《修次第论》(Bhāvanākrama，三篇)
- **核心学说**：中观与瑜伽行的调和，渐修论
- **关键概念**：
  - 渐修论：与摩诃衍(禅宗)的西藏辩论，主张渐修而非顿悟

## 核心问题

1. **空是虚无还是缘起？** 龙树：空即缘起，非断灭空；批评者：空若一切皆空，空本身是否也空？
2. **唯识无境：外境是否实有？** 世亲：无外境，唯识所现；经量部：外境刹那生灭，非完全无
3. **中观 vs 瑜伽行：空与有的关系？** 中观：一切法空；瑜伽行：三性中依他起性非完全空——大乘两大路线的分歧
