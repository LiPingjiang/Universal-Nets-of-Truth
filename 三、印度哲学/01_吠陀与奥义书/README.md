# 01 梵我合一：吠陀与奥义书的终极追问

## 核心张力

**梵(Brahman) vs 我(Ātman)**

- 梵(Brahman)：宇宙本体、终极实在、遍在之原理
- 我(Ātman)：个体灵魂、内在自我、意识之核心

## 思维流向

印度哲学的开端，从祭祀仪式走向内在冥想：
- 吠陀本集：祭祀万能，通过仪式与神灵沟通——梨俱、娑摩、夜柔、阿闼婆
- 梵书：祭祀仪轨的精密化，婆罗门至上——百道梵书
- 奥义书：从外在祭祀转向内在知识，梵我合一的终极发现——歌者、广林、他氏、蛙氏
- 薄伽梵歌：四种瑜伽的综合，行动、知识、虔信、王瑜伽的统一

## 人物与文献

### 吠陀本集(Ṛgveda, Sāmaveda, Yajurveda, Atharvaveda, 约前1500-前1000)
- **原文**：梨俱吠陀(Ṛgveda，1028首赞歌，10卷)
- **核心思想**：原人(Puruṣa)献祭创世、宇宙秩序(ṛta)、因陀罗(Indra)、伐楼拿(Varuṇa)
- **关键概念**：
  - 原人(Puruṣa)：宇宙人格，千头千眼千足，由献祭产生世界
  - 宇宙秩序(ṛta)：自然与道德的统一法则

### 百道梵书(Śatapatha Brāhmaṇa, 约前8-前6世纪)
- **核心思想**：祭祀万能，宇宙即祭祀，婆罗门至上
- **关键概念**：
  - 祭祀(yajña)：宇宙运行的根本机制
  - 无前(apūrva)：祭祀产生的不可见力量，决定果报

### 歌者奥义书(Chāndogya Upaniṣad, 约前7-前6世纪)
- **核心教导**：「汝即彼」(Tat tvam asi)
- **关键概念**：
  - 存在(sat)：宇宙本源，太初唯存在，有而无二
  - 细谷中的大我(vijñāna)：意识内在于一切存在

### 广林奥义书(Bṛhadāraṇyaka Upaniṣad, 约前7-前6世纪)
- **核心教导**：「我即梵」(Aham brahmāsmi)、「非此非彼」(neti neti)
- **关键概念**：
  - 梵我合一(Brahman-Ātman aikya)：个体灵魂与宇宙本体同一
  - 非此非彼(neti neti)：梵不可正面描述，只能否定性言说

### 蛙氏奥义书(Māṇḍūkya Upaniṣad, 约前3-前1世纪)
- **核心教导**：唵(Oṃ)的四义——A-U-M-无声
- **关键概念**：
  - 四境(catvāri)：醒(jāgrat)、梦(svapna)、深睡(suṣupti)、第四境(turīya)
  - 第四境(turīya)：超越前三，即梵即我，不可言说

### 薄伽梵歌(Bhagavad Gītā, 约前5-前2世纪)
- **核心思想**：四种瑜伽的综合——业瑜伽、智瑜伽、信瑜伽、王瑜伽
- **关键概念**：
  - 达摩(dharma)：职责、正义、自然法则
  - 不执而行(niṣkāma karma)：行动而不执着于结果
  - 虔信(bhakti)：对神的虔诚信仰与爱

## 核心问题

1. **梵与我是同一还是不同？** 奥义书主流：梵我合一；后来吠檀多各派分化：不二论(同一)、限定不二论(属性同一)、二元论(根本不同)
2. **祭祀能否带来解脱？** 梵书：祭祀万能；奥义书：知识(jñāna)即解脱，祭祀非究竟
3. **梵如何被认知？** 正面描述还是否定性言说？「非此非彼」的否定之路 vs 「汝即彼」的肯定之路
