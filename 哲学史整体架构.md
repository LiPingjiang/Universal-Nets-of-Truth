## 一、西方哲学（Western Philosophy）

### 1.1 古希腊哲学（Ancient Greek Philosophy）

**时间跨度**：约公元前6世纪 — 公元前322年（亚里士多德去世）

#### 1.1.1 前苏格拉底哲学（Pre-Socratic Philosophy）

**核心问题**：万物的本原（arche）是什么？

| 流派/人物 | 核心学说 | 本质精髓 |
|-----------|----------|----------|
| **米利都学派** | 泰勒斯：水是万物的本原 | 第一次用自然本身解释自然，摆脱神话思维 |
| | 阿那克西曼德：无定（apeiron） | 本原不可限定，比具体元素更抽象一步 |
| | 阿那克西美尼：气是万物的本原 | 用稀散/浓聚解释变化，引入量变到质变的思路 |
| **毕达哥拉斯学派** | 数是万物的本原 | 第一次把抽象形式（而非物质）当作本质 |
| **赫拉克利特** | 万物流变，火是本原，逻各斯（logos） | 变化本身才是永恒；背后有统一的规律 |
| **巴门尼德** | 存在者存在，非存在者不存在；无变化 | 第一次严格的逻辑论证；思维与存在的同一 |
| **埃利亚学派** | 芝诺悖论 | 用归谬法捍卫"不动"，暴露感官与理性的冲突 |
| **恩培多克勒** | 四根说（水火土气）+ 爱与恨 | 多元本原 + 动力因，为后来原子论铺路 |
| **阿那克萨戈拉** | 种子说 + 努斯（nous，心灵） | 第一次把"心灵"作为独立的本原引入 |
| **德谟克利特/留基伯** | 原子论（atomos，不可分） | 物质由不可分的微粒在虚空中运动构成 |
| **智者学派** | 普罗泰戈拉：人是万物的尺度 | 认识论转向：真理相对于人；修辞与辩论的技术化 |
| | 高尔吉亚：无物存在；即使有也不可认识；即使认识也不可传达 | 怀疑论的极端形式，暴露语言的局限 |

#### 1.1.2 古典希腊哲学（Classical Greek Philosophy）

**核心问题**：什么是善的生活？知识如何可能？

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **苏格拉底**（Socrates，前470-前399） | 认识你自己；德性即知识；苏格拉底式诘问法（elenchus） | 哲学从自然转向人；承认无知是智慧的开端；用对话逼出矛盾 |
| **柏拉图**（Plato，前428-前348） | 理念论（Theory of Forms）；灵魂三分；回忆说；理想国 | 可感世界背后有不可感的真实世界；灵魂不朽；哲学家应当为王 |
| **亚里士多德**（Aristotle，前384-前322） | 实体（ousia）；四因说；中庸之道；逻辑学；形而上学作为"第一哲学" | 对柏拉图理念论的修正：形式在事物之中；经验观察 + 逻辑分析；系统化的学科分类 |

#### 1.1.3 希腊化-罗马哲学（Hellenistic & Roman Philosophy）

**时间跨度**：公元前4世纪末 — 公元5世纪

**核心问题**：如何在不确定的世界中获得内心的宁静（ataraxia）？

| 流派 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **伊壁鸠鲁学派**（Epicureanism） | 快乐是唯一的善，但快乐=无痛苦+无纷扰；原子偏斜（swerve） | 审慎的快乐主义；无神论倾向；原子论的自然科学 |
| **斯多葛学派**（Stoicism） | 芝诺、塞内卡、爱比克泰德、马可·奥勒留；顺应自然（logos）；德性即善；世界公民 | 接受不可控的，改变可控的；理性统治激情；宇宙是一个有生命的整体 |
| **怀疑主义**（Skepticism） | 皮浪：悬置判断；学园派怀疑主义 | 对一切教条保持怀疑，以求得心灵的安宁 |
| **新柏拉图主义**（Neoplatonism） | 普罗提诺：太一（The One）→ 理智 → 灵魂 → 物质；流溢说 | 柏拉图理念的神秘化；万物从太一流出，灵魂渴望回归太一 |

### 1.2 中世纪哲学（Medieval Philosophy）

**时间跨度**：约公元5世纪 — 15世纪文艺复兴

**核心张力**：信仰与理性的关系；雅典与耶路撒冷如何调和？

#### 1.2.1 早期基督教哲学（Patristic Philosophy）

| 人物 | 核心贡献 |
|------|----------|
| **奥古斯丁**（Augustine，354-430） | 时间的主观性；原罪与恩典；上帝之城 vs 地上之城；内在真理的光照说 |
| **波爱修斯**（Boethius，480-524） | 翻译亚里士多德逻辑著作；慰藉哲学；命运与自由意志 |

#### 1.2.2 伊斯兰-阿拉伯哲学（Islamic / Arabic Philosophy）

> 定位：中世纪西方哲学的关键过渡分支，保存古希腊文本，再传回欧洲。

| 人物 | 核心贡献 |
|------|----------|
| **肯迪**（Al-Kindi，801-873） | 阿拉伯哲学之父，翻译希腊哲学 |
| **法拉比**（Al-Farabi，872-950） | 柏拉图与亚里士多德的调和；理想城邦 |
| **阿维森纳/伊本·西那**（Avicenna / Ibn Sina，980-1037） | 存在与本质的区分；灵魂学说；医学典范 |
| **阿威罗伊/伊本·鲁世德**（Averroes / Ibn Rushd，1126-1198） | 亚里士多德评注；理性与信仰的双重真理；对阿奎那的直接影响 |
| **迈蒙尼德**（Maimonides，1135-1204） | 犹太哲学巅峰；《迷途指津》调和犹太教与亚里士多德 |

#### 1.2.3 经院哲学（Scholasticism）

| 人物 | 核心贡献 |
|------|----------|
| **安瑟尔谟**（Anselm，1033-1109） | 本体论证明：上帝是"无法设想比之更大的存在" |
| **阿伯拉尔**（Abelard，1079-1142） | 唯名论 vs 实在论；概念论；伦理学上的意图论 |
| **托马斯·阿奎那**（Thomas Aquinas，1225-1274） | 神学大全；存在与本质的实在区分；五种证明上帝存在；自然法；亚里士多德与基督教的系统综合 |
| **邓斯·司各脱**（Duns Scotus，1266-1308） | 唯意志论：上帝的意志高于理性；个体性原则（haecceitas） |
| **奥卡姆的威廉**（William of Ockham，1287-1347） | 奥卡姆剃刀：如无必要，勿增实体；唯名论；分离神学与哲学 |

### 1.3 近代哲学（Modern Philosophy）

**时间跨度**：约15世纪文艺复兴 — 18世纪末康德

**核心转向**：认识论转向（Epistemological Turn）—— 从"世界是什么"转向"我们如何认识世界"

#### 1.3.1 文艺复兴与人文主义（Renaissance & Humanism）

| 人物 | 核心贡献 |
|------|----------|
| **彼特拉克**（Petrarch，1304-1374） | 人文主义之父；人的尊严与价值 |
| **马基雅维利**（Machiavelli，1469-1527） | 《君主论》；政治与道德的分离；现实主义政治哲学 |
| **伊拉斯谟**（Erasmus，1466-1536） | 基督教人文主义；理性与信仰的温和调和 |

#### 1.3.2 经验主义（Empiricism）

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **培根**（Francis Bacon，1561-1626） | 新工具；归纳法；知识就是力量；四假象说 | 系统化的经验方法；破除偶像 |
| **霍布斯**（Thomas Hobbes，1588-1679） | 利维坦；社会契约；机械论宇宙；一切是运动中的物质 | 政治权威的绝对性；人性本恶，需要强力约束 |
| **洛克**（John Locke，1632-1704） | 白板说（tabula rasa）；观念来自经验；自然权利；政府论 | 经验是知识的唯一来源；生命、自由、财产不可让渡 |
| **贝克莱**（George Berkeley，1685-1753） | 存在即被感知（esse est percipi） | 彻底的经验主义：物质只是观念的集合 |
| **休谟**（David Hume，1711-1776） | 怀疑论：因果只是习惯联想；自我只是一束知觉；是-应问题 | 摧毁了因果必然性的根基；理性不能推出道德 |

#### 1.3.3 理性主义（Rationalism）

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **笛卡尔**（Descartes，1596-1650） | 我思故我在（Cogito, ergo sum）；心物二元；方法论怀疑；天赋观念 | 从怀疑中找到不可动摇的基石；理性是知识的最高法庭 |
| **斯宾诺莎**（Spinoza，1632-1677） | 神即自然（Deus sive Natura）；实体一元论；几何学方法；《伦理学》 | 拒绝笛卡尔二元：思维与广延是同一实体的两种属性；自由是对必然的认识 |
| **莱布尼茨**（Leibniz，1646-1716） | 单子论（monadology）；前定和谐；可能世界；充足理由律 | 宇宙由无数无窗口的精神单子构成；上帝选择了最好的可能世界 |

#### 1.3.4 启蒙哲学（Enlightenment Philosophy）

| 人物 | 核心贡献 |
|------|----------|
| **伏尔泰**（Voltaire，1694-1778） | 理性、自由、宽容；反对教会专制 |
| **孟德斯鸠**（Montesquieu，1689-1755） | 三权分立；法的精神；地理环境决定论 |
| **卢梭**（Rousseau，1712-1778） | 社会契约；人民主权；自然状态；教育哲学；《爱弥儿》 |
| **狄德罗/百科全书派**（Diderot，1713-1784） | 百科全书；唯物主义；无神论倾向 |

### 1.4 现当代哲学（Contemporary Philosophy）

**时间跨度**：18世纪末康德 — 至今

#### 1.4.1 德国古典哲学（German Idealism）

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **康德**（Kant，1724-1804） | 批判哲学：纯粹理性批判、实践理性批判、判断力批判；先天综合判断；物自体（Ding an sich）；绝对命令 | 哥白尼式革命：不是知识符合对象，而是对象符合知识；为理性划界；道德律令的普遍性 |
| **费希特**（Fichte，1762-1814） | 知识学；自我设定自身、设定非我、统一两者 | 绝对自我是唯一的实在 |
| **谢林**（Schelling，1775-1854） | 自然哲学；同一哲学；天启哲学 | 自然与精神的同一；艺术是哲学的最高形态 |
| **黑格尔**（Hegel，1770-1831） | 辩证法：正题-反题-合题；绝对精神；历史是精神的展开；主奴辩证法 | 一切实在都是理性的；矛盾是发展的动力；历史有目的论方向 |

#### 1.4.2 马克思主义哲学（Marxism）

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **马克思**（Marx，1818-1883） | 历史唯物主义；阶级斗争；异化劳动；商品拜物教；意识形态批判 | 不是解释世界，而是改变世界；经济基础决定上层建筑 |
| **恩格斯**（Engels，1820-1895） | 自然辩证法；家庭、私有制和国家的起源 |
| **卢卡奇**（Lukács，1885-1971） | 物化（reification）；阶级意识；总体性 |
| **法兰克福学派** | 霍克海默、阿多诺：启蒙辩证法；文化工业；工具理性批判 | 理性本身变成了支配的工具 |
| | 哈贝马斯（Habermas，1929-）：交往行为理论；公共领域；系统对生活世界的殖民 | 以沟通理性取代工具理性 |

#### 1.4.3 分析哲学（Analytic Philosophy）

> 英美主流：逻辑分析、语言分析、科学哲学

| 人物/流派 | 核心学说 | 本质精髓 |
|-----------|----------|----------|
| **弗雷格**（Frege，1848-1925） | 现代逻辑之父；意义与指称的区分；概念文字 | 数学可以还原为逻辑；语言分析的开端 |
| **罗素**（Russell，1872-1970） | 逻辑原子主义；摹状词理论；类型理论；与怀特海《数学原理》 | 语言表面结构掩盖了真正的逻辑形式 |
| **维特根斯坦早期**（Wittgenstein，1889-1951） | 《逻辑哲学论》：语言的界限就是世界的界限；可说与不可说；图像论 | 对不可言说者必须保持沉默 |
| **维特根斯坦后期** | 《哲学研究》：语言游戏；家族相似；私人语言论证；意义即使用 | 语言没有统一的本质，只有各种"游戏" |
| **逻辑实证主义/维也纳学派** | 卡尔纳普、石里克：可证实性原则；拒斥形而上学；逻辑分析 | 有意义的命题要么是分析命题，要么可被经验证实 |
| **日常语言学派/牛津学派** | 赖尔、奥斯汀、斯特劳森 | 哲学问题源于语言的误用；言语行为理论 |
| **奎因**（Quine，1908-2000） | 经验主义的两个教条；本体论承诺；翻译的不确定性 | 分析与综合的区分站不住脚；整体论的知识观 |
| **戴维森**（Davidson，1917-2003） | 真之语义学；异常一元论 | 意义理论就是真理论 |
| **克里普克**（Kripke，1940-2022） | 命名与必然性；因果指称理论；先验偶然与后验必然 | 专名没有描述意义，只是刚性指称符 |

#### 1.4.4 欧陆哲学（Continental Philosophy）

> 德法主线：现象学、存在主义、解释学、结构主义、后现代

| 人物/流派 | 核心学说 | 本质精髓 |
|-----------|----------|----------|
| **叔本华**（Schopenhauer，1788-1860） | 意志与表象；生命意志的否定；悲观主义 | 世界本质是盲目的意志，理性只是意志的工具 |
| **尼采**（Nietzsche，1844-1900） | 权力意志；超人（Übermensch）；上帝已死；永恒轮回；谱系学；主人道德与奴隶道德 | 重估一切价值；道德是权力的表达；肯定生命本身 |
| **克尔凯郭尔**（Kierkegaard，1813-1855） | 存在主义先驱；主观性即真理；人生三阶段（审美-伦理-宗教）；焦虑与绝望 | 个体面对上帝的绝对关系；信仰之跃 |
| **胡塞尔**（Husserl，1859-1938） | 现象学：回到事物本身；意向性；悬搁（epoché）；先验还原；生活世界 | 研究意识如何构造意义；超越自然态度 |
| **海德格尔**（Heidegger，1889-1976） | 存在与时间；此在（Dasein）；在世存在；向死而生；常人；技术的座架（Gestell） | 追问存在的意义；人是一种特殊的存在者，能理解存在本身 |
| **萨特**（Sartre，1905-1980） | 存在先于本质；自由与责任；虚无；他人即地狱；存在主义是一种人道主义 | 人被判定为自由；人必须为自己的选择负全责 |
| **梅洛-庞蒂**（Merleau-Ponty，1908-1961） | 知觉现象学；身体主体；肉身（chair） | 知觉是知识的基础；身体不是客体，而是主体 |
| **伽达默尔**（Gadamer，1900-2002） | 哲学解释学；视域融合；效果历史；理解即对话 | 理解不是重建作者意图，而是视域的融合 |
| **法兰克福学派**（接上） | （见马克思主义哲学节） | |
| **结构主义** | 索绪尔：语言是差异的系统；列维-斯特劳斯：神话的结构分析；阿尔都塞：意识形态国家机器 | 表层之下有深层结构；主体是被结构建构的 |
| **后结构主义/后现代** | 福柯：权力/知识；规训与惩罚；性史；话语考古学 | 权力生产知识；主体是权力的产物 |
| | 德里达：解构；延异（différance）；逻各斯中心主义批判 | 文本没有固定意义；意义永远在延迟 |
| | 德勒兹：差异与重复；块茎（rhizome）；欲望机器；生成 | 反对同一性思维；肯定差异本身 |
| | 利奥塔：后现代状况；宏大叙事的终结 | 知识变成了性能，合法性危机 |
| | 鲍德里亚：拟像与仿真；消费社会；超真实 | 真实已经消失，只剩下符号的符号 |

#### 1.4.5 实用主义（Pragmatism）

| 人物 | 核心学说 | 本质精髓 |
|------|----------|----------|
| **皮尔士**（Peirce，1839-1914） | 实用主义准则；符号学；可错论 | 概念的意义在于其可观察的实践后果 |
| **詹姆斯**（James，1842-1910） | 真理的效用论；意识流；宗教经验之种种 | 真理是有用的信念；多元宇宙 |
| **杜威**（Dewey，1859-1952） | 工具主义；教育即生活；民主作为生活方式 | 思想是解决问题的工具；实验方法应用于社会 |

#### 1.4.6 当代前沿（20世纪末 — 至今）

| 领域 | 代表人物 | 核心议题 |
|------|----------|----------|
| **心灵哲学** | 塞尔、丹尼特、查尔莫斯、金在权 | 心-脑问题；意识的难问题；功能主义；生物自然主义 |
| **语言哲学（后分析）** | 格赖斯、塞尔、布兰顿 | 言语行为；含义与推论；规范性的语言基础 |
| **科学哲学** | 波普尔、库恩、拉卡托斯、费耶阿本德 | 证伪主义；范式转换；研究纲领；反对方法 |

| 核心人物 | 核心学说 | 本质精髓 |
|----------|----------|----------|
| **迦那陀**（Kanada，约前2世纪-前2世纪） | 《胜论经》：六句义（实、德、业、同、异、和合）；原子论（paramanu）；极微与极微之合 | 印度版的原子论：世界由不可分的原子组合而成；范畴学 |

### 3.3 佛教哲学（Buddhist Philosophy）

**时间跨度**：约公元前5世纪 — 至今

**核心问题**：苦（dukkha）的根源是什么？如何解脱？

#### 3.3.1 原始佛教/上座部（Theravada / Early Buddhism）

| 核心概念 | 内容 | 本质精髓 |
|----------|------|----------|
| **四圣谛** | 苦谛、集谛、灭谛、道谛 | 生命本质是苦；苦的原因是渴爱；苦可以止息；有八条道路 |
| **八正道** | 正见、正思维、正语、正业、正命、正精进、正念、正定 | 从智慧到戒律到禅定，全面修行 |
| **缘起**（pratityasamutpada） | 此有故彼有；十二因缘 | 一切现象互相依存，没有独立自存的东西 |
| **无我**（anatta） | 五蕴皆空；无固定不变的自我 | 人只是身心现象的暂时组合，没有永恒的灵魂 |
| **涅槃**（nibbana / nirvana） | 贪瞋痴的熄灭；有余涅槃与无余涅槃 | 不是天堂，是苦的彻底止息；在此生即可实现 |
| **阿毗达摩**（Abhidhamma） | 法（dhamma）的分析；心所、色法、心不相应行法、涅槃 | 对现象最精密的分类分析 |

#### 3.3.2 部派佛教（Abhidharma Schools）

| 部派 | 核心特点 |
|------|----------|
| **上座部**（Theravada） | 保守，保持早期教义；巴利语三藏；阿毗达摩精密分析 |
| **说一切有部**（Sarvastivada） | 一切法三世实有；极微实有；四谛十六行相 |
| **经量部**（Sautrantika） | 回归经藏，反对阿毗达摩的过度推演；刹那生灭；种子说 |
| **大众部**（Mahasamghika） | 较为开放；佛陀超人间性；心性本净 |

#### 3.3.3 大乘佛教（Mahayana Buddhism）

##### 中观派（Madhyamaka）

| 核心人物 | 核心学说 | 本质精髓 |
|----------|----------|----------|
| **龙树**（Nagarjuna，约150-250 CE） | 《中论》：八不中道；缘起性空；四重否定；世俗谛与胜义谛；涅槃即轮回 | 一切概念（包括"空"本身）都要被否定；空不是虚无，是缘起 |
| **提婆**（Aryadeva） | 《百论》；破斥外道与佛教内部的实体论 | |
| **佛护**（Buddhapalita，约470-540） | 归谬法（prasanga）：只破不立 | |
| **清辨**（Bhaviveka，约500-570） | 自立量（svatantra）：用因明建立论证 | |
| **月称**（Candrakirti，约600-650） | 归谬法正统；入中论 | 中观派的集大成者 |
| **寂天**（Shantideva，约685-763） | 《入菩萨行论》；自他交换；菩萨道 | 大乘修行的系统指南 |

##### 瑜伽行派/唯识派（Yogacara / Vijnanavada）

| 核心人物 | 核心学说 | 本质精髓 |
|----------|----------|----------|
| **弥勒**（Maitreya，传说） | 瑜伽行派根本经典（《瑜伽师地论》等）的托名作者 | |
| **无著**（Asanga，约4世纪） | 《瑜伽师地论》；唯识理论系统化；三性（遍计所执性、依他起性、圆成实性） | |
| **世亲**（Vasubandhu，约4-5世纪） | 《唯识三十颂》；《俱舍论》；从说一切有部转向唯识；阿赖耶识（alaya-vijnana） | 一切现象只是心识的变现；最深层的阿赖耶识储存一切种子 |
| **陈那**（Dignaga，约480-540） | 佛教逻辑学（因明）集大成；现量与比量；三支论式；遮诠理论 | 知识的唯一来源是知觉和推理 |
| **法称**（Dharmakirti，约600-660） | 佛教逻辑的巅峰；《释量论》；因果推理；瑜伽知觉 | 把佛教逻辑系统化到最高水平 |

##### 如来藏思想（Tathagatagarbha）

| 核心文本 | 核心学说 | 本质精髓 |
|----------|----------|----------|
| 《如来藏经》《楞伽经》《大乘起信论》 | 一切众生皆有如来藏（佛性）；真如缘起；心体本净 | 与"无我"的张力：人人本有佛性，但被烦恼遮蔽 |

#### 3.3.4 密宗/金刚乘（Vajrayana / Tantra）

| 核心特点 | 本质精髓 |
|----------|----------|
| 三密相应（身、口、意）；曼荼罗；本尊观想；灌顶；即身成佛 | 利用身体和心理的快速法门；把欲望转化为修行工具 |
