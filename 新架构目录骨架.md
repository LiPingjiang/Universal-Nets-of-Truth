# 窅纂 · 新架构目录骨架（纵向为主干 + 问题节点）· 修正版

> 核心变化：纵向时间线为主干，关键节点设"问题专章"；五个问题专章按存在论→认识论→真理观→自由观的逻辑递进；统一用"理论类型（代表人物）"格式，避免"论"与"人"混排。

---

## 五个问题专章的内在逻辑链条

西方哲学史有一条清晰的递进脉络：先问"世界是什么"，再问"我们如何认识世界"，然后问"认识的结果如何判定为真"，最后问"知道真理后，人还能自由行动吗"。五个问题专章对应这条链条的四个层级。

```
第一层：存在论（Ontology）—— 世界本身是什么？
    └── 专章一：存在与变化
            ↓
第二层：认识论（Epistemology）—— 我们如何认识世界？
    ├── 专章二：共相之争（概念是否对应实在？）
    └── 专章三：心物关系（认识的主体是什么？）
            ↓
第三层：真理观（Theory of Truth）—— 认识的结果如何判定？
    └── 专章四：真理的本质
            ↓
第四层：自由与行动（Freedom & Agency）—— 知道规律后，人还能自由吗？
    └── 专章五：自由与必然
```

这个递进不是人为编排，而是哲学史的真实展开：古希腊人先追问存在，中世纪经院哲学家在神学框架内讨论认识问题，近代认识论转向后心物关系成为核心，19世纪真理问题被明确提出，20世纪自由与必然在存在主义和分析哲学中同时爆发。

### 问题专章一：存在与变化（Being vs Becoming）

**所属层级**：第一层 · 存在论（Ontology）

**前置问题**：世界本身是什么？是永恒不变的存在，还是永远流变的过程？

**为什么这是第一个问题**：古希腊哲学从追问"本原"（arche）开始，但巴门尼德和赫拉克利特把问题推进到了更根本的层面——不是"本原是什么物质"，而是"变化是否真实"。这个问题决定了后来所有哲学问题的地基：如果存在是静止的，现象世界的变化就是幻觉，真理只能在永恒的理念中寻找；如果变化是真实的，那知识就必须面对一个不断变动的对象。

**演进脉络（理论类型 + 代表人物）**：

| 理论类型 | 代表人物 | 核心主张 |
|----------|----------|----------|
| **存在论（Being）** | 巴门尼德 | 存在者存在，非存在者不存在；变化是感官的幻觉 |
| **流变论（Becoming）** | 赫拉克利特 | 万物流变，人不能两次踏入同一条河流；火是本原 |
| **分有论（Participation）** | 柏拉图 | 理念（Form）永恒不变，可感事物分有理念而处于流变之中 |
| **潜能-实现论（Potentiality-Actuality）** | 亚里士多德 | 存在不是静止的，而是从潜能向实现的转化；变化是真实的，但需要不变的实体作为载体 |

**对后续问题的影响**：柏拉图的分有论直接催生了中世纪的共相之争——理念（共相）是否独立于个别事物而存在？

---

### 问题专章二：共相之争（Universals）

**所属层级**：第二层 · 认识论（Epistemology）

**前置问题**：我们用来认识世界的概念（如"人""红""善"），是否对应实在中的某种实体？还是仅仅存在于心灵或语言中？

**为什么接在"存在与变化"之后**：柏拉图说理念永恒、现象流变，但理念（共相）到底在哪里？是独立于心灵的实在，还是心灵的构造？这个问题从柏拉图的理念论直接延伸出来，中世纪经院哲学用基督教神学的框架重新提出，成为连接古代存在论与近代认识论的关键桥梁。

**演进脉络（理论类型 + 代表人物）**：

| 理论类型 | 代表人物 | 核心主张 |
|----------|----------|----------|
| **极端实在论（Extreme Realism）** | 安瑟尔谟 | 共相是独立于心灵和个别事物的客观实在；上帝心中先有"人"的理念，再创造具体的人 |
| **极端唯名论（Extreme Nominalism）** | 罗瑟林 | 共相只是声音（flatus vocis），没有实在对应物；只有个别事物存在 |
| **概念论（Conceptualism）** | 阿伯拉尔 | 共相既非独立于事物的实在，也非纯粹的声音；它是心灵从事物的相似性中抽象出的概念 |
| **温和实在论（Moderate Realism）** | 托马斯·阿奎那 | 共相以三种方式存在：先于事物（在上帝心中）、在事物中（作为事物的本质）、后于事物（在人的心灵中） |
| **唯名论（Nominalism）** | 奥卡姆 | 如无必要，勿增实体；共相只是心灵对个别事物的符号标记；只有个别事物存在 |

**对后续问题的影响**：奥卡姆的唯名论摧毁了经院哲学的实在论基础，为近代经验主义铺路——如果只有个别事物存在，那知识就只能从个别经验出发。同时，心灵与概念的关系问题也引出了"心物关系"：如果共相只是心灵的构造，心灵本身是什么？

---

### 问题专章三：心物关系（Mind-Body Problem）

**所属层级**：第二层 · 认识论（Epistemology）

**前置问题**：认识的主体——心灵（mind）——与被认识的客体——物质世界（body）——是什么关系？心灵是物质的属性，还是独立的实体？

**为什么接在"共相之争"之后**：共相之争已经触及"心灵如何形成概念"的问题，但还没有追问心灵本身的本体论地位。近代认识论转向（笛卡尔）把问题彻底翻转：不是"世界是什么"，而是"我如何认识世界"——而"我"就是一个思维着的心灵。心灵与物质的关系由此成为核心问题。

**演进脉络（理论类型 + 代表人物）**：

| 理论类型 | 代表人物 | 核心主张 |
|----------|----------|----------|
| **实体二元论（Substance Dualism）** | 笛卡尔 | 心灵（res cogitans）与物质（res extensa）是两种根本不同的实体；心灵不占空间，物质不思维 |
| **属性一元论（Attribute Monism）** | 斯宾诺莎 | 心灵与物质是同一实体（神/自然）的两种属性；思维与广延是同一事物的两面 |
| **单子论（Monadology）** | 莱布尼茨 | 宇宙由无数无窗口的精神单子构成；物质是单子的表象；前定和谐使单子之间协调 |
| **现象主义/观念论（Phenomenalism/Idealism）** | 贝克莱 | 存在即被感知；物质只是观念的集合；心灵是唯一的实体 |
| **唯物主义/物理主义（Materialism/Physicalism）** | 霍布斯 → 当代 | 一切存在都是物质；心灵是大脑的属性或功能；没有非物质的心灵 |
| **功能主义（Functionalism）** | 普特南、福多 | 心灵不是某种物质，而是物质组织的功能；心灵状态由因果功能定义，不由实现它的物质定义 |
| **取消唯物主义（Eliminative Materialism）** | 丘奇兰德夫妇 | 民间心理学（信念、欲望等日常概念）将被神经科学取代；心灵概念是前科学时代的错误 |
| **生物自然主义（Biological Naturalism）** | 塞尔 | 意识是大脑的生物过程；但意识有不可还原的主观性（感受质） |
| **异常一元论（Anomalous Monism）** | 戴维森 | 心灵事件就是物理事件（本体论一元），但心灵规律不能还原为物理规律（概念论二元） |
| **泛心论（Panpsychism）** | 查尔莫斯、加尔夫等 | 意识不是大脑的涌现属性，而是宇宙的基本属性；一切物质都有某种程度的意识 |

**对后续问题的影响**：心物关系中的"认识如何可能"直接引出"真理的本质"——如果心灵是物质的，那真理就是大脑状态与外部世界的对应；如果心灵是独立的，那真理可能是一种内在的融贯或实用的效果。

### 问题专章四：真理的本质（The Nature of Truth）

**所属层级**：第三层 · 真理观（Theory of Truth）

**前置问题**：当我们说一个命题是"真"的，我们到底在说什么？真理是命题与事实的对应，还是信念系统内部的融贯，还是信念在实践中的有用性？

**为什么接在"心物关系"之后**：心物关系解决了"认识的主体是什么"，但没有回答"认识的结果如何判定为真"。近代认识论转向后，经验主义和理性主义都预设了某种真理标准，但没有明确反思"真理是什么"这个问题本身。直到19世纪末，真理问题才被作为独立问题明确提出。

**演进脉络（理论类型 + 代表人物）**：

| 理论类型 | 代表人物 | 核心主张 |
|----------|----------|----------|
| **符合论（Correspondence Theory）** | 亚里士多德 → 罗素 → 早期维特根斯坦 | 真理是命题与事实的对应；"雪是白的"为真，当且仅当雪确实是白的 |
| **融贯论（Coherence Theory）** | 黑格尔 → 布兰夏德 | 真理是信念系统内部的逻辑融贯；单个命题的真假取决于它与整个知识系统的一致性 |
| **实用论（Pragmatic Theory）** | 皮尔士 → 詹姆斯 → 杜威 | 真理是有用的信念；一个观念的真理性在于它带来的实践效果 |
| **语义真理论（Semantic Theory）** | 塔尔斯基 → 戴维森 | 真理是语言与世界的语义关系；塔尔斯基的T-约定："P"为真当且仅当P |
| **共识论/商谈论（Consensus/Discourse Theory）** | 哈贝马斯 | 真理是理性主体在理想商谈条件下达成的共识；真理不是个人判断，而是主体间性的产物 |
| **解蔽论（Aletheia / Unconcealment）** | 海德格尔 | 真理不是命题与事实的对应，而是存在从遮蔽中显现（aletheia）；真理是事件，不是属性 |
| **解构论（Deconstruction）** | 德里达 | 没有固定的真理；意义永远在延迟（différance）；文本没有终极意义，只有无限的阐释 |

**对后续问题的影响**：如果真理是符合事实（符合论），那世界就有客观规律，人就必须服从规律，自由就成问题；如果真理是实用的（实用论），那人就可以通过行动创造真理，自由就是核心；如果真理是解蔽（海德格尔），那自由就是让存在显现的能力。

### 问题专章五：自由与必然（Freedom vs Necessity）

**所属层级**：第四层 · 自由与行动（Freedom & Agency）

**前置问题**：如果世界有规律（因果必然性），人还能自由选择和行动吗？自由是幻觉，还是人类独有的能力？

**为什么接在"真理的本质"之后**：真理问题已经揭示了世界是否有客观规律、规律是否可知。如果规律存在且可知（符合论），那人的行为是否也被规律决定？如果真理是实用的（实用论），那人是否可以通过行动创造自己的真理，从而获得自由？自由与必然的问题是所有存在论、认识论、真理观的最终归宿——它问的是：知道了这一切之后，人如何行动？

**演进脉络（理论类型 + 代表人物）**：

| 理论类型 | 代表人物 | 核心主张 |
|----------|----------|----------|
| **先验自由（Transcendental Freedom）** | 康德 | 人作为现象界的一部分服从因果必然性，但作为本体（noumenon）有先验自由；自由不可证明，但必须预设 |
| **自由即对必然的认识（Freedom as Recognized Necessity）** | 黑格尔 | 自由不是任意妄为，而是对必然规律的认识和顺应；真正的自由是理解了规律之后的自觉行动 |
| **历史必然性中的阶级自由（Freedom in Historical Necessity）** | 马克思 | 个人自由受历史条件和阶级关系制约；真正的自由是消灭阶级压迫后的社会自由 |
| **存在先于本质（Existence Precedes Essence）** | 萨特 | 人没有预定的本质，被判定为自由；人必须为自己的选择负全责；自由是存在的重负 |
| **被操纵的自由（Manipulated Freedom）** | 法兰克福学派（马尔库塞等） | 现代社会通过文化工业和工具理性制造虚假需求；人以为自己在自由选择，实际上被系统操纵 |
| **交往自由（Communicative Freedom）** | 哈贝马斯 | 真正的自由不是个人任意，而是在无强制的理想商谈中达成共识；自由是主体间性的 |
| **相容论（Compatibilism）** | 当代分析哲学（法兰克福、刘易斯等） | 决定论与自由意志可以相容；自由不是"免于因果"，而是"按自己的欲望和理性行动" |
| **不相容论（Incompatibilism）** | 范英瓦根等 | 决定论与自由意志不可相容；如果世界是决定论的，人就没有自由 |

**对后续问题的影响**：自由与必然的问题直接通向当代前沿——心灵哲学（自由意志与神经科学）、政治哲学（社会正义与个体自由）、生态哲学（人类自由与自然规律的边界）。

## 问题专章的设计原则

1. **纵向锚定**：每个专章锚定在一个具体问题上，横跨多个时代，展示同一问题的不同回答如何演进
2. **不是替代**：专章不替代纵向章节的完整叙述，而是**补充和聚焦**——纵向章节讲"发生了什么"，专章讲"这个问题如何被不断回答"
3. **可独立阅读**：专章可以独立成篇，也可以作为纵向章节的交叉索引
4. **双向链接**：专章开头列出涉及的所有纵向章节，纵向章节末尾列出相关的专章
5. **统一格式**：每个专章的演进脉络统一用"理论类型（代表人物）"格式，避免"论"与"人"混排
