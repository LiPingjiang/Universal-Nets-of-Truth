# 03 西学东渐：从利玛窦到新儒家

## 核心张力

**西学(科学/理性) vs 中学(道德/人文)**

- 西学：西方科学技术、逻辑分析、实证方法
- 中学：中国道德传统、人文精神、修养工夫

## 思维流向

从明末耶稣会士来华，到近现代新儒家的回应与融合：
- 明末清初（16-17世纪）：利玛窦——"合儒补儒"策略，以基督教义补充儒家
- 清初（17-18世纪）：礼仪之争——中国礼仪是否异端？中西文化冲突的首次爆发
- 晚清（19世纪）：洋务运动——中体西用，技术层面学习西方
- 民国（20世纪）：新文化运动——全盘西化 vs 文化保守主义
- 新儒家（20世纪）：从梁漱溟到牟宗三——会通中西，内圣开出新外王
- 当代：安乐哲、余英时——比较哲学与思想史的新路径

## 人物与文献

### 利玛窦(Matteo Ricci, 1552-1610)
- **原文**：《天主实义》《交友论》《坤舆万国全图》
- **核心策略**：合儒补儒——以基督教义补充儒家，尊重中国礼仪
- **关键概念**：
  - 天主(Tiānzhǔ)：以"天""上帝"等儒家概念翻译Deus
  - 补儒(bǔ rú)：儒家有善无全，基督教补其不足

### 徐光启(1562-1633)
- **原文**：《农政全书》《几何原本》译序
- **核心观点**："欲求超胜，必须会通；会通之前，先须翻译"
- **关键概念**：会通(huì tōng)：中西学问的融合超越

### 礼仪之争(Rites Controversy, 17-18世纪)
- **核心争议**：中国信徒能否祭祖祭孔？中国礼仪是政治还是宗教？
- **教廷立场**：1704年禁止中国礼仪，导致基督教在华传播中断
- **关键概念**：
  - 祭祖(jì zǔ)：儒家伦理还是偶像崇拜？
  - 祭孔(jì kǒng)：尊敬先师还是宗教仪式？

### 严复(1854-1921)
- **译文**：《天演论》(赫胥黎)、《原富》(斯密)、《法意》(孟德斯鸠)
- **核心观点**：信达雅——翻译三原则，引进西方社会科学
- **关键概念**：
  - 天演(tiān yǎn)：evolution，物竞天择，适者生存
  - 信达雅(xìn dá yǎ)：翻译的三重标准——忠实、通顺、典雅

### 胡适(1891-1962)
- **核心观点**：全盘西化、整理国故、大胆假设小心求证
- **关键概念**：
  - 整理国故(zhěng lǐ guó gù)：以科学方法研究中国传统文化
  - 大胆假设，小心求证：杜威实验主义方法

### 牟宗三(1909-1995)
- **原文**：《心体与性体》《智的直觉与中国哲学》《现象与物自身》
- **核心观点**：良知坎陷说——儒家内圣开出民主科学
- **关键概念**：
  - 良知坎陷(liáng zhī kǎn xiàn)：道德主体自我否定，转出知性主体
  - 两层存有论：无执的存有论（本体界）vs 执的存有论（现象界）
  - 智的直觉(zhì de zhí jué)：对应康德intellectuelle Anschauung，中国哲学之所长

### 安乐哲(Roger T. Ames, 1947-)
- **核心观点**：比较哲学——以中西互释的方式重新理解儒家
- **关键概念**：
  - 角色伦理(role ethics)：儒家不是德性伦理，而是关系性、角色性的伦理
  - 焦点-场域(focus-field)模式：个体不是独立实体，而是关系场域中的焦点

## 核心问题

1. **利玛窦的"合儒"是策略还是真诚？** 文化适应(acculturation)的边界——尊重还是妥协？
2. **中体西用是否可行？** 张之洞的框架能否支撑现代转型？技术可学，制度难行？
3. **牟宗三的"坎陷"是否曲折？** 良知自我否定转出知性——是创造性转化还是牵强附会？
4. **比较哲学的方法论：可比性还是不可通约性？** 安乐哲的角色伦理是否过度诠释？
