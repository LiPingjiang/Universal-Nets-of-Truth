# 利玛窦：解读_学术

## 一、思想史定位

### 1.1 在西学东渐进程中的位置

利玛窦（Matteo Ricci, 1552-1610）是中西思想交流史上具有开创性意义的人物。他是第一位深入中国内地、精通汉语、系统接触中国士大夫阶层的西方传教士，其"合儒补儒"策略奠定了明末清初中西对话的基本框架。

| 阶段 | 代表 | 特征 | 与利玛窦的关系 |
|------|------|------|---------------|
| **耶稣会东方传教**（16世纪中后期） | 沙勿略、范礼安 | 从海上抵达东亚，文化适应策略的萌芽 | 范礼安提出"适应中国"方针，利玛窦是其执行者 |
| **利玛窦来华**（1582-1610） | 利玛窦 | 合儒补儒，以学术传教，翻译西学 | **开创者**：奠定中西对话的基本框架 |
| **礼仪之争**（17-18世纪） | 多明我会 vs 耶稣会 | 中国礼仪是否异端？教廷与清廷的直接冲突 | 利玛窦的适应策略受到质疑和否定 |
| **晚清西学**（19世纪后期） | 严复、林则徐 | 系统翻译西方社会科学 | 利玛窦引进自然科学，严复引进社会科学 |
| **新文化运动**（20世纪） | 胡适、陈独秀 | 全盘西化 vs 文化保守 | 利玛窦的"合儒"精神被重新审视 |

### 1.2 "合儒补儒"策略的思想渊源

利玛窦的策略并非个人独创，而是根植于耶稣会的传教传统和经院哲学的理论框架：

- **耶稣会适应主义**（Accommodation）：自罗耀拉创立耶稣会以来，"适应"当地文化是耶稣会传教的核心方针。范礼安（Alessandro Valignano）在日本的传教经验为利玛窦提供了范本。
- **托马斯主义的自然神学**：阿奎那在《神学大全》中区分了"自然理性可知的真理"（自然神学）与"需启示才能知道的真理"（启示神学）。利玛窦据此认为，儒家经典中的"天""上帝"属于自然理性可达知的真理，天主教则补充了启示真理（创世、原罪、救赎、来世）。
- **人文主义教育背景**：利玛窦在罗马学院接受了系统的古典教育（拉丁文、希腊文、数学、天文学、修辞学），这使他能够以学术和文化的形式而非武力形式进行传教。

## 二、核心命题分析

### 2.1 "天主即上帝"：概念的等价化策略

利玛窦最核心的策略是将天主教的Deus与儒家经典中的"上帝""天"等价化。这一策略涉及深层的哲学问题：

**概念等价化的可能性条件**：
1. 儒家经典（《诗》《书》《易》）中的"上帝"确实具有人格化至高神的特征——"降监""命""罚"等，这与天主教的Deus有形式上的可比性。
2. 但宋明理学已将"天""帝"重新解释为"理"——去人格化的形而上原则。利玛窦的策略是绕过理学，回到先秦经典，以"古儒"驳"今儒"。

**概念等价化的困难**：
1. 儒家的"天"兼具自然义（苍天）、道德义（天命）、宗教义（上帝），是多义性的；天主教的Deus是单一的、人格化的、超自然的创造者。
2. 儒家的"天"与"地"相对，在宇宙论框架内；天主教的Deus超越宇宙，不在宇宙之内。
3. 儒家没有"从无创有"（creatio ex nihilo）的概念——中国宇宙论主流是"气化生成"而非"创造"。

### 2.2 对理学"理气论"的批评

利玛窦对宋明理学的批评是中西哲学在本体论层面的首次正面交锋：

**利玛窦的论证**：
1. "理"是万物的法则（规律），不是创造者。
2. 法则不能自足——法则必须依附于某物，不能独立存在。
3. 因此，必须有一个人格化的"作者"（天主）来运用法则创造万物。

**理学的回应（以黄宗羲、王夫之为代表）**：
1. "理"不是外在法则，而是内在于"气"的运行规律——"理在气中"。
2. 不需要一个人格化的"作者"——"气"的自发运动（"生生之谓易"）就是万物的根源。
3. "理"与"气"的关系不是"法则"与"立法者"的关系，而是"体"与"用"的关系。

**哲学意义**：这一争论揭示了中西本体论的根本差异——西方的"创造论"传统（从无到有，需要一个超越的创造者）与中国的"生成论"传统（自然演化，不需要外在创造者）之间的不可通约性。

### 2.3 "补儒"的深层含义

"补儒"不仅是策略性的，更涉及对儒家思想的深层判断：

**利玛窦认为儒家"未尽"之处**：
1. **来世观**：儒家"未知生，焉知死"，缺乏死后灵魂与审判的观念。
2. **创世说**：儒家没有"从无创有"的概念，宇宙是永恒的气化过程。
3. **原罪与救赎**：儒家相信人性本善（或通过修养可善），不需要外在的救赎者。

**"补儒"的哲学意涵**：利玛窦实际上指出了儒家思想在超越性维度上的结构性缺失。这一判断在后世反复出现——从康德对"道德宗教"的讨论到牟宗三对"圆教"的建构，中西思想在"内在超越"与"外在超越"之间的张力始终是核心议题。

## 三、历史影响与学术接受

### 3.1 对中国思想的影响

| 领域 | 影响 |
|------|------|
| **数学与科学** | 与徐光启合译《几何原本》前六卷，引入演绎逻辑和公理化方法，深刻影响清代数学。 |
| **天文学** | 引入西方天文学（球面几何、日心说前夜），崇祯历法改革（崇祯历书）直接受其影响。 |
| **地理学** | 《坤舆万国全图》引入世界地理知识，打破了"天圆地方"的传统宇宙观。 |
| **语言学** | 《西字奇迹》首创以拉丁字母为汉语注音，是汉语拼音化的先驱。 |
| **哲学对话** | "合儒补儒"策略引发持续的中西哲学对话，至礼仪之争达到高潮。 |

### 3.2 礼仪之争：利玛窦遗产的争议

利玛窦去世后不久，耶稣会与其他修会（多明我会、方济各会）围绕中国礼仪（祭祖、祭孔）展开激烈争论：

- **耶稣会立场（继承利玛窦）**：祭祖祭孔是政治/文化仪式，不是宗教崇拜，可以容忍。
- **多明我会立场**：祭祖祭孔是偶像崇拜，违反天主教第一诫。
- **教廷最终裁决**：1704年教皇禁止中国礼仪，1721年康熙帝驱逐传教士。中西文化交流中断近百年。

礼仪之争的深层意义：它不仅是传教策略之争，更是中西文化在最根本的本体论和宗教观层面的冲突——中国的"天"与西方的"God"是否等价？中国的"礼"是宗教还是文化？

### 3.3 当代学术界的重新评价

- **Jacques Gernet**（《中国与基督教》，1982）：从深层文化结构分析中西冲突，认为利玛窦的"合儒"策略掩盖了根本性的不可通约性。
- **Nicolas Standaert**（《礼仪之争中的文化间性》，2016）：以"文化间性"（interculturality）概念重新审视礼仪之争，超越"冲突"或"融合"的二元框架。
- **李天纲**（《中国礼仪之争》，1998）：从中国本土视角分析礼仪之争的历史意义，指出它不仅是宗教之争，更是中西文明秩序观的首次全面碰撞。

## 四、当代研究前沿

### 4.1 全球史视角下的利玛窦

当代研究不再将利玛窦视为单纯的"传教士"或"文化交流者"，而是将其置于16-17世纪全球史（大航海时代、耶稣会全球网络、早期殖民主义）的框架中重新审视：

- **Sanjay Subrahmanyam**（"连接的历史"方法）：利玛窦的传教活动是早期全球化的一部分，应与同时期的印度、日本、美洲传教活动进行比较。
- **R. Po-chia Hsia**（《耶稣会士的世界》，2010）：将利玛窦置于耶稣会全球网络中，分析其策略与同时期其他地区的适应策略的异同。

### 4.2 利玛窦与"跨文化翻译"理论

当代翻译理论家从利玛窦的实践中汲取灵感：

- **劳伦斯·韦努蒂**（Lawrence Venuti）的"归化"与"异化"理论：利玛窦的"合儒"策略是极端"归化"（domestication）的典型——将外来概念完全融入本土话语。
- **"概念旅行"理论**（爱德华·萨义德）：利玛窦将Deus"旅行"到中国语境，概念在旅行过程中发生了不可避免的变形。
