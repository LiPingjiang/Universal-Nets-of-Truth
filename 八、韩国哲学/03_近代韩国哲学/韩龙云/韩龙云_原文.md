# 韩龙云（Han Yong-un）原典摘录

> 韩龙云（1879-1944），号万海，朝鲜近代佛教改革家、独立运动家、诗人。以下选录其代表作《朝鲜佛教维新论》《你的沉默》中的核心段落，附甲译（白话意译）与乙译（学术直译）。

---

## 一、《朝鲜佛教维新论》·佛教改革

### 原文（汉文）

朝鲜佛教之衰，不在于佛教本身之衰，而在于僧侣之堕落、寺院之腐化、教义之僵化。僧侣不务修行，而务经忏；寺院不务教化，而务敛财；教义不务适应，而务守旧。三者皆佛教之蠹虫，非佛教之栋梁也。

### 甲译（白话意译）

**朝鲜佛教的衰落，不是佛教本身衰落，而是三个方面出了问题：僧侣堕落、寺院腐化、教义僵化。僧侣不专心修行，只忙着做经忏法事赚钱；寺院不专心教化民众，只忙着聚敛钱财；教义不与时俱进，只忙着死守旧规矩。这三者是佛教的蛀虫，不是佛教的栋梁。** 韩龙云对朝鲜佛教的批判是切中要害的。在朝鲜后期，佛教虽然失去了官方地位（朝鲜王朝以儒教为国教），但佛教寺院仍然拥有大量土地和财产，僧侣阶层过着舒适的生活，却不再承担精神教化的责任。这种"寺院资本主义"使佛教失去了其宗教本质，变成了社会寄生虫。

### 乙译（学术直译）

朝鲜佛教之衰，不在于佛教本身之衰，而在于僧侣之堕落、寺院之腐化、教义之僵化。僧侣不务修行，而务经忏；寺院不务教化，而务敛财；教义不务适应，而务守旧。三者皆佛教之蠹虫，非佛教之栋梁也。——《朝鲜佛教维新论》

### 原文（汉文）

佛教之真精神，在"慈悲救世"。慈悲者，非仅对一二人之慈悲，乃对一切众生之慈悲；救世者，非仅救个人之灵魂，乃救社会之苦难。故佛教若不与社会结合，不与时代同步，则真精神失，而佛教亡。

### 甲译（白话意译）

**佛教的真正精神在于"慈悲救世"。慈悲不只是对一两个人的好心，而是对一切众生的爱心；救世不只是救个人的灵魂，而是救社会的苦难。所以佛教如果不与社会结合、不与时代同步，就会失去真正的精神，佛教就会灭亡。** 韩龙云将佛教的核心精神从"个人解脱"重新定义为"社会救世"。传统佛教（特别是小乘佛教）强调个人从轮回中解脱，进入涅槃。但韩龙云说，大乘佛教的核心不是个人解脱，而是"菩萨道"——为了救度众生而留在世间，与社会苦难搏斗。这种"入世佛教"或"人间佛教"思想，使佛教从出世的宗教转变为入世的伦理力量。

### 乙译（学术直译）

佛教之真精神，在"慈悲救世"。慈悲者，非仅对一二人之慈悲，乃对一切众生之慈悲；救世者，非仅救个人之灵魂，乃救社会之苦难。故佛教若不与社会结合，不与时代同步，则真精神失，而佛教亡。——《朝鲜佛教维新论》

### 原文（汉文）

改革之道，非废佛而从西教也。改革之道，乃去佛教之糟粕，存佛教之精华，而以时代之要求，充实佛教之内容也。佛教之精华在"无我""慈悲""平等"；时代之要求在"科学""民主""独立"。若能以无我慈悲平等之心，行科学民主独立之业，则佛教之维新可成也。

### 甲译（白话意译）

**改革的方法不是废除佛教、改信西方宗教。改革的方法是去掉佛教的糟粕，保留佛教的精华，然后根据时代的要求，充实佛教的内容。佛教的精华在于"无我""慈悲""平等"；时代的要求在于"科学""民主""独立"。如果能用无我、慈悲、平等的心，来实行科学、民主、独立的事业，那么佛教的维新就可以成功。** 韩龙云提出了一个与朴殷植"儒教求新"相似但路径不同的佛教改革方案。他同样主张"保留精华+引进实用"，但佛教精华与儒家精华不同：佛教的核心是无我（不执着于自我）、慈悲（爱一切众生）、平等（众生平等）。这些精神与科学、民主、独立并不冲突——无我使人谦虚面对科学真理，慈悲使人关心民主社会的公平，平等使人追求民族独立与国际和平。

### 乙译（学术直译）

改革之道，非废佛而从西教也。改革之道，乃去佛教之糟粕，存佛教之精华，而以时代之要求，充实佛教之内容也。佛教之精华在"无我""慈悲""平等"；时代之要求在"科学""民主""独立"。若能以无我慈悲平等之心，行科学民主独立之业，则佛教之维新可成也。——《朝鲜佛教维新论》

## 二、《你的沉默》·诗与觉醒

### 原文（朝鲜文）

당신은 갔습니다 / 말없이 / 눈물 없이 / 당신은 갔습니다

그러나 나는 압니다 / 당신의 침묵이 / 수만 갈래의 말을 담고 있음을

### 甲译（白话意译）

**您走了 / 没有说话 / 没有眼泪 / 您走了**

**但我知道 / 您的沉默里 / 包含着千万种话语**

韩龙云这首诗《你的沉默》创作于1910年代朝鲜亡国之后。诗中的"您"有多重解读：可以指代殉国的独立斗士，可以指代被日本殖民者杀害的同胞，也可以指代失去国家后沉默的朝鲜民族。诗的核心意象是"沉默"——表面上的沉默，实际上是"千万种话语"的压抑。这种沉默不是软弱，而是积蓄力量的方式。诗的深层含义是：殖民统治可以压制人的声音，但压制不了人的意志；表面的沉默之下，是民族独立的渴望和反抗的决心的暗流涌动。这首诗成为韩国独立运动中广为传诵的作品，激发了无数韩国人的民族意识。

### 乙译（学术直译）

당신은 갔습니다 / 말없이 / 눈물 없이 / 당신은 갔습니다

그러나 나는 압니다 / 당신의 침묵이 / 수만 갈래의 말을 담고 있음을

——《你的沉默》（韩龙云诗集）

### 原文（朝鲜文）

나는 이 세상을 사랑합니다 / 고통도 / 불의도 / 전부를 사랑합니다

사랑하지 않으면 / 변화할 수 없기 때문입니다

### 甲译（白话意译）

**我爱这个世界 / 包括痛苦 / 包括不义 / 爱一切**

**因为不爱的话 / 就不能改变它**

韩龙云在这首诗中表达了一种"爱之伦理"：改变世界的前提是爱，不是仇恨。他承认世界充满了痛苦和不义，但他拒绝用仇恨来回应这些痛苦和不义。因为仇恨不能带来真正的改变——只有爱，才能让人真正关心世界，从而真正改变世界。这种"以爱救世"的思想与佛教"慈悲"精神一脉相承，也与甘地的"非暴力抵抗"有精神共鸣。在殖民统治的黑暗年代，这种"爱"的立场不是软弱，而是一种深刻的道德勇气——它拒绝以恶制恶，坚持用善来改变恶。

### 乙译（学术直译）

나는 이 세상을 사랑합니다 / 고통도 / 불의도 / 전부를 사랑합니다

사랑하지 않으면 / 변화할 수 없기 때문입니다

——《韩龙云诗集》
